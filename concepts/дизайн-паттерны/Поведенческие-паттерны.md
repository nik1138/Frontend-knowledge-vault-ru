# Поведенческие паттерны

Поведенческие паттерны определяют способы взаимодействия объектов и распределения обязанностей между ними. Они описывают, как объекты общаются друг с другом.

## Пример: Наблюдатель

```javascript
class EventEmitter {
  constructor() {
    this.events = {};
  }

  on(event, callback) {
    if (!this.events[event]) {
      this.events[event] = [];
    }
    this.events[event].push(callback);
  }

  emit(event, data) {
    if (this.events[event]) {
      this.events[event].forEach(callback => callback(data));
    }
  }
}

const emitter = new EventEmitter();

emitter.on('message', (data) => {
  console.log('Received:', data);
});

emitter.emit('message', 'Hello World'); // Received: Hello World
```

## Заключение

Поведенческие паттерны обеспечивают эффективное взаимодействие между объектами, позволяя системе быть более гибкой и легко поддерживаемой.
