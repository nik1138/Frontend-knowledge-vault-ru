# Семантический HTML

Семантический HTML — это подход к разметке веб-страниц, при котором HTML-теги используются не просто для визуального форматирования, а для передачи смысла и структуры содержимого. Это делает веб-контент более доступным, понятным для поисковых систем и удобным для пользователей с ограниченными возможностями.

## Основные понятия

Семантические теги HTML — это элементы, которые явно описывают свою роль в структуре документа. В отличие от несемантических тегов, таких как `<div>` и `<span>`, семантические теги несут в себе информацию о типе содержимого, которое они оборачивают.

Примеры семантических тегов:
- `<header>` — заголовок секции или документа
- `<nav>` — навигационные ссылки
- `<main>` — основное содержимое страницы
- `<article>` — независимый фрагмент контента
- `<section>` — логическая группа контента
- `<aside>` — боковая информация
- `<footer>` — нижний колонтитул секции или документа

## Преимущества семантического HTML

Использование семантического HTML имеет множество преимуществ:

- **Доступность**: Помогает вспомогательным технологиям, таким как скринридеры, правильно интерпретировать содержимое.
- **SEO**: Поисковые системы используют семантическую структуру для лучшего понимания и индексации контента.
- **Поддерживаемость**: Код становится более понятным для других разработчиков.
- **Адаптивность**: Улучшает взаимодействие с CSS и JavaScript.

## Примеры использования

### Структура типичной веб-страницы

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Семантический HTML</title>
</head>
<body>
    <header>
        <h1>Мой сайт</h1>
        <nav>
            <ul>
                <li><a href="#home">Главная</a></li>
                <li><a href="#about">О нас</a></li>
                <li><a href="#contact">Контакты</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="home">
            <h2>Главная страница</h2>
            <p>Это основное содержимое главной страницы.</p>
        </section>

        <section id="about">
            <h2>О нас</h2>
            <article>
                <h3>Наша миссия</h3>
                <p>Мы стремимся создавать качественные веб-решения.</p>
            </article>
        </section>
    </main>

    <aside>
        <h3>Полезные ссылки</h3>
        <ul>
            <li><a href="#">Ссылка 1</a></li>
            <li><a href="#">Ссылка 2</a></li>
        </ul>
    </aside>

    <footer>
        <p>&copy; 2025 Все права защищены</p>
    </footer>
</body>
</html>
```

### Статья с комментариями

```html
<article>
    <header>
        <h2>Заголовок статьи</h2>
        <p>Опубликовано: <time datetime="2025-01-15">15 января 2025</time></p>
    </header>
    
    <p>Это основное содержимое статьи...</p>
    
    <footer>
        <p>Автор: Иван Иванов</p>
    </footer>
    
    <section>
        <h3>Комментарии</h3>
        <article>
            <header>
                <h4>Комментарий Петра Петрова</h4>
                <time datetime="2025-01-16T10:30">16 января 2025, 10:30</time>
            </header>
            <p>Отличная статья!</p>
        </article>
    </section>
</article>
```

## Семантические теги и их роль

Каждый семантический тег имеет свою специфическую роль:

- `<header>`: Обычно содержит логотип, навигацию или заголовки.
- `<nav>`: Группирует основные навигационные ссылки.
- `<main>`: Основное содержимое, уникальное для конкретной страницы.
- `<article>`: Полностью независимый фрагмент контента, который может быть перепубликован или распространён отдельно.
- `<section>`: Группа связанных по смыслу элементов, часто с заголовком.
- `<aside>`: Содержимое, косвенно связанное с основным содержимым.
- `<footer>`: Информация об авторе, копирайты, ссылки на связанные документы.

## Важные замечания

> [!tip] Совет
> Используйте семантические теги по их прямому назначению, чтобы обеспечить правильную структуру документа и улучшить доступность.

> [!warning] Важно
> Не злоупотребляйте несемантическими тегами `<div>` и `<span>` там, где можно использовать более подходящие семантические элементы.
