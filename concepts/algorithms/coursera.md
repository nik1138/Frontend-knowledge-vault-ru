## Основные курсы алгоритмов

### Принстонский университет - "Algorithms, Part I & II"

Один из самых известных курсов по алгоритмам, преподаваемый профессорами Робертом Седжвиком и Кевином Уэйнлом. Курс охватывает:

- [[concepts/data-structures/arrays|Массивы]], [[concepts/data-structures/linked-lists|связанные списки]], [[concepts/data-structures/stacks|стеки]] и [[concepts/data-structures/queues|очереди]]
- [[concepts/algorithms/sorting/merge-sort|Сортировка слиянием]], [[concepts/algorithms/sorting/quick-sort|быстрая сортировка]], [[concepts/algorithms/sorting/heap-sort|пирамидальная сортировка]]
- [[concepts/data-structures/binary-search-trees|Двоичные деревья поиска]], [[concepts/data-structures/red-black-trees|красно-черные деревья]]
- [[concepts/algorithms/graphs/bfs|Поиск в ширину]] и [[concepts/algorithms/graphs/dfs|поиск в глубину]]
- [[concepts/algorithms/graphs/shortest-path|Кратчайшие пути]] (алгоритмы Дейкстры и Беллмана-Форда)
- [[concepts/algorithms/graphs/mst|Минимальные остовные деревья]]

```java
// Пример реализации DFS из Принстонского курса
public class DepthFirstPaths {
    private boolean[] marked;
    private int[] edgeTo;
    private int s;

    public DepthFirstPaths(Graph G, int s) {
        this.s = s;
        edgeTo = new int[G.V()];
        marked = new boolean[G.V()];
        dfs(G, s);
    }

    private void dfs(Graph G, int v) {
        marked[v] = true;
        for (int w : G.adj(v)) {
            if (!marked[w]) {
                edgeTo[w] = v;
                dfs(G, w);
            }
        }
    }
}
```

## Специализации и треки

### Алгоритмы и структуры данных (Специализация)

- **Продолжительность**: 4-5 курсов (~6 месяцев)
- **Уровень**: От начального до продвинутого
- **Сертификат**: Доступен при оплате
- **Языки**: Java, Python, C++

### Специализация по анализу и разработке алгоритмов

- Курс 1: Основы алгоритмов
- Курс 2: Графовые алгоритмы
- Курс 3: Жадные алгоритмы и динамическое программирование
- Курс 4: NP-полнота и приближенные алгоритмы

## Структура курса

Типичный курс алгоритмов на Coursera включает:

1. **Видео-лекции** (30-60 минут в неделю)
2. **Чтения** и материалы для самостоятельного изучения
3. **Практические задания** с автоматической проверкой
4. **Теоретические задачи** (для понимания концепций)
5. **Финальный экзамен** или проект

## Рекомендации по обучению

1. **Регулярность** - выделяйте время каждый день для изучения
2. **Практика** - реализуйте каждый алгоритм самостоятельно
3. **Анализ** - понимайте, почему алгоритм работает, а не только как
4. **Тестирование** - создавайте тестовые случаи для проверки кода
