# Пользовательские хуки

## Введение

Пользовательские хуки - это функции, которые используют хуки React и позволяют делить логику состояния между компонентами. Они позволяют извлекать компонентную логику в повторно используемые функции, что способствует лучшей организации кода и его переиспользованию.

## Создание пользовательного хука

Пользовательный хук - это просто функция JavaScript, имя которой начинается с "use", и которая может вызывать другие хуки. Это соглашение позволяет React распознавать, что функция содержит хуки, и соблюдать правила их использования.

```javascript
// Пользовательный хук для управления состоянием счетчика
function useCounter(initialValue = 0) {
  const [count, setCount] = useState(initialValue);

  const increment = () => setCount(count + 1);
  const decrement = () => setCount(count - 1);
  const reset = () => setCount(initialValue);

  return { count, increment, decrement, reset };
}

// Использование пользовательного хука
function Counter() {
  const { count, increment, decrement, reset } = useCounter(10);

  return (
    <div>
      <p>Счетчик: {count}</p>
      <button onClick={increment}>+</button>
      <button onClick={decrement}>-</button>
      <button onClick={reset}>Сброс</button>
    </div>
  );
}
```

## Примеры популярных пользовательных хуков

### useLocalStorage

Хук для работы с localStorage с автоматической синхронизацией состояния:

```javascript
import { useState, useEffect } from 'react';

function useLocalStorage(key, initialValue) {
  // Получаем значение из localStorage или используем initialValue
  const [storedValue, setStoredValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;
    } catch (error) {
      console.error(`Ошибка при чтении localStorage ключа "${key}":`, error);
      return initialValue;
    }
  });

  // Обновляем localStorage при изменении значения
  const setValue = (value) => {
    try {
      // Обновляем состояние
      setStoredValue(value);
      // Сохраняем в localStorage
      window.localStorage.setItem(key, JSON.stringify(value));
    } catch (error) {
      console.error(`Ошибка при сохранении в localStorage ключа "${key}":`, error);
    }
  };

  return [storedValue, setValue];
}

// Использование
function Profile() {
  const [name, setName] = useLocalStorage('userName', 'Аноним');

  return (
    <div>
      <p>Имя: {name}</p>
      <input 
        value={name} 
        onChange={(e) => setName(e.target.value)} 
        placeholder="Введите имя" 
      />
    </div>
  );
}
```

### useApi

Хук для управления асинхронными запросами:

```javascript
import { useState, useEffect } from 'react';

function useApi(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        setLoading(true);
        const response = await fetch(url);
        
        if (!response.ok) {
          throw new Error(`Ошибка: ${response.status}`);
        }
        
        const result = await response.json();
        setData(result);
      } catch (err) {
        setError(err.message);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, [url]);

  return { data, loading, error };
}

// Использование
function UserList() {
  const { data: users, loading, error } = useApi('/api/users');

  if (loading) return <div>Загрузка...</div>;
  if (error) return <div>Ошибка: {error}</div>;

  return (
    <ul>
      {users?.map(user => (
        <li key={user.id}>{user.name}</li>
      ))}
    </ul>
  );
}
```

### useDebounce

Хук для дебаунса значений:

```javascript
import { useState, useEffect } from 'react';

function useDebounce(value, delay) {
  const [debouncedValue, setDebouncedValue] = useState(value);

  useEffect(() => {
    const handler = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    // Очищаем таймер при изменении значения
    return () => {
      clearTimeout(handler);
    };
  }, [value, delay]);

  return debouncedValue;
}

// Использование в компоненте поиска
function SearchComponent() {
  const [query, setQuery] = useState('');
  const debouncedQuery = useDebounce(query, 500);

  useEffect(() => {
    if (debouncedQuery) {
      // Выполняем поиск только после паузы в 500мс
      performSearch(debouncedQuery);
    }
  }, [debouncedQuery]);

  return (
    <input
      value={query}
      onChange={(e) => setQuery(e.target.value)}
      placeholder="Поиск..."
    />
  );
}
```

### useToggle

Простой хук для переключения булевых значений:

```javascript
import { useState } from 'react';

function useToggle(initialValue = false) {
  const [value, setValue] = useState(initialValue);

  const toggle = () => setValue(prev => !prev);
  const setTrue = () => setValue(true);
  const setFalse = () => setValue(false);

  return [value, { toggle, setTrue, setFalse }];
}

// Использование
function Modal() {
  const [isOpen, { toggle, setFalse }] = useToggle(false);

  return (
    <div>
      <button onClick={toggle}>Открыть модальное окно</button>
      
      {isOpen && (
        <div className="modal">
          <p>Модальное окно</p>
          <button onClick={setFalse}>Закрыть</button>
        </div>
      )}
    </div>
  );
}
```

### useClickOutside

Хук для обработки кликов вне компонента:

```javascript
import { useEffect, useRef } from 'react';

function useClickOutside(handler) {
  const ref = useRef();

  useEffect(() => {
    const handleClickOutside = (event) => {
      if (ref.current && !ref.current.contains(event.target)) {
        handler();
      }
    };

    document.addEventListener('mousedown', handleClickOutside);
    return () => {
      document.removeEventListener('mousedown', handleClickOutside);
    };
  }, [handler]);

  return ref;
}

// Использование
function Dropdown() {
  const [isOpen, setIsOpen] = useState(false);
  const dropdownRef = useClickOutside(() => setIsOpen(false));

  return (
    <div ref={dropdownRef} className="dropdown">
      <button onClick={() => setIsOpen(!isOpen)}>
        Меню
      </button>
      {isOpen && (
        <div className="dropdown-content">
          <a href="#link1">Ссылка 1</a>
          <a href="#link2">Ссылка 2</a>
        </div>
      )}
    </div>
  );
}
```

## Правила создания пользовательных хуков

### 1. Именование

Все пользовательные хуки должны начинаться с префикса "use". Это позволяет React распознавать их и применять правила хуков.

```javascript
// Правильно
function useCounter() { ... }
function useLocalStorage() { ... }

// Неправильно
function counterHook() { ... }
function localStorageManager() { ... }
```

### 2. Использование других хуков

Пользовательные хуки могут использовать любые встроенные хуки React, а также другие пользовательные хуки.

```javascript
function useDarkMode() {
  const [isDark, setIsDark] = useLocalStorage('darkMode', false);
  
  useEffect(() => {
    if (isDark) {
      document.body.classList.add('dark-theme');
    } else {
      document.body.classList.remove('dark-theme');
    }
  }, [isDark]);

  return [isDark, setIsDark];
}
```

### 3. Состояние изолировано

Каждый вызов пользовательного хука имеет изолированное состояние. Два компонента, использующих один и тот же хук, не разделяют состояние.

```javascript
function ComponentA() {
  const { count } = useCounter(); // Независимое состояние
  return <div>Компонент A: {count}</div>;
}

function ComponentB() {
  const { count } = useCounter(); // Независимое состояние
  return <div>Компонент B: {count}</div>;
}
```

## Продвинутые паттерны

### Композиция хуков

Можно комбинировать несколько пользовательных хуков для создания более сложной логики:

```javascript
function useAuth() {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Логика аутентификации
    checkAuthStatus().then(setUser).finally(() => setLoading(false));
  }, []);

  return { user, loading };
}

function useProtectedRoute() {
  const { user, loading } = useAuth();
  
  useEffect(() => {
    if (!loading && !user) {
      // Перенаправление на страницу входа
      navigate('/login');
    }
  }, [user, loading]);

  return { user, loading, authenticated: !!user && !loading };
}
```

### Хуки с колбэками

Иногда полезно передавать функции в пользовательные хуки:

```javascript
function useAsyncOperation(operation, deps = []) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState(null);

  const execute = useCallback(async (...args) => {
    setLoading(true);
    setError(null);
    
    try {
      const result = await operation(...args);
      setData(result);
      return result;
    } catch (err) {
      setError(err);
      throw err;
    } finally {
      setLoading(false);
    }
  }, deps);

  return { data, loading, error, execute };
}

// Использование
function UserProfile({ userId }) {
  const { data: user, loading, error, execute } = useAsyncOperation(
    (id) => fetch(`/api/users/${id}`).then(r => r.json()),
    []
  );

  useEffect(() => {
    if (userId) {
      execute(userId);
    }
  }, [userId, execute]);

  // Рендер компонента
}
```

## Заключение

Пользовательные хуки являются мощным инструментом для организации и переиспользования логики в React-приложениях. Они позволяют извлекать сложную логику из компонентов и делиться ею между разными частями приложения, улучшая читаемость и поддерживаемость кода.
