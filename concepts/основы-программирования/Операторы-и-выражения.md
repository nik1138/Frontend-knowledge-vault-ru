# Операторы и выражения

## Введение

Операторы и выражения — это основные инструменты для выполнения вычислений, сравнений и логических операций в программировании. Понимание этих концепций критически важно для написания функционального кода.

## Что такое выражения

Выражение — это комбинация значений, переменных, операторов и вызовов функций, которая вычисляется в одно значение. Простейшие выражения — это литералы и переменные.

```javascript
// Простые выражения
let x = 5;
let y = 10;
let sum = x + y; // выражение x + y вычисляется в 15

// Сложные выражения
let result = (x + y) * 2 - 3;
let isPositive = x > 0;
```

## Арифметические операторы

Арифметические операторы используются для выполнения математических операций.

```javascript
let a = 10;
let b = 3;

let addition = a + b;        // 13 - сложение
let subtraction = a - b;     // 7 - вычитание
let multiplication = a * b;  // 30 - умножение
let division = a / b;        // 3.333... - деление
let remainder = a % b;       // 1 - остаток от деления
let exponentiation = a ** b; // 1000 - возведение в степень
```

### Унарные операторы

```javascript
let num = 5;
num++;        // инкремент (увеличение на 1)
num--;        // декремент (уменьшение на 1)
let positive = +num;  // унарный плюс
let negative = -num;  // унарный минус
```

## Операторы сравнения

Операторы сравнения используются для сравнения значений и возвращают булевое значение.

```javascript
let x = 5;
let y = 10;

console.log(x == y);   // false - равенство (с приведением типов)
console.log(x != y);   // true - неравенство
console.log(x === y);  // false - строгое равенство (без приведения типов)
console.log(x !== y);  // true - строгое неравенство
console.log(x < y);    // true - меньше
console.log(x > y);    // false - больше
console.log(x <= y);   // true - меньше или равно
console.log(x >= y);   // false - больше или равно
```

## Логические операторы

Логические операторы используются для объединения или инвертирования булевых значений.

```javascript
let a = true;
let b = false;

let and = a && b;    // false - логическое И
let or = a || b;     // true - логическое ИЛИ
let not = !a;        // false - логическое НЕ
```

### Практическое применение логических операторов

```javascript
// Проверка нескольких условий
let userAge = 25;
let isVerified = true;

if (userAge >= 18 && isVerified) {
  console.log("Пользователь может получить доступ");
}

// Установка значения по умолчанию
let userName = null;
let displayName = userName || "Гость";

// Условное выполнение
let isLoggedIn = true;
isLoggedIn && showUserMenu();
```

## Операторы присваивания

Операторы присваивания используются для присвоения значений переменным.

```javascript
let x = 5;        // обычное присваивание
x += 3;           // x = x + 3, результат: 8
x -= 2;           // x = x - 2, результат: 6
x *= 4;           // x = x * 4, результат: 24
x /= 3;           // x = x / 3, результат: 8
x %= 5;           // x = x % 5, результат: 3
x **= 2;          // x = x ** 2, результат: 9
```

## Операторы условного присваивания

```javascript
// Оператор нулевого слияния (ES2020)
let userRole = null;
let defaultRole = userRole ?? "пользователь"; // "пользователь"

// Оператор логического ИЛИ
let userName = null;
let displayName = userName || "Гость"; // "Гость"

// Оператор логического И
let user = { name: "Иван" };
let userStatus = user && user.name ? "активен" : "неактивен";
```

## Тернарный оператор

Тернарный оператор — это краткая форма условного оператора `if...else`.

```javascript
let age = 18;
let canVote = age >= 18 ? "Можно голосовать" : "Нельзя голосовать";

// Множественные условия
let score = 85;
let grade = score >= 90 ? "A" :
            score >= 80 ? "B" :
            score >= 70 ? "C" : "F";
```

## Операторы для работы с объектами

### Оператор разворота (Spread)
```javascript
let arr1 = [1, 2, 3];
let arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]

let obj1 = { name: "Иван", age: 30 };
let obj2 = { ...obj1, city: "Москва" }; // { name: "Иван", age: 30, city: "Москва" }
```

### Оператор объединения (Rest)
```javascript
function sum(...numbers) {
  return numbers.reduce((total, num) => total + num, 0);
}

console.log(sum(1, 2, 3, 4)); // 10
```

## Практические рекомендации

### Использование строгого равенства

```javascript
// Хорошо - используйте строгое равенство
if (value === 0) { /* ... */ }
if (value === "") { /* ... */ }
if (value === false) { /* ... */ }

// Плохо - избегайте нестрогого равенства
if (value == 0) { /* ... */ } // может вызвать неожиданное поведение
```

### Приоритет операторов

Знание приоритета операторов помогает избежать ошибок:

```javascript
// Оператор && имеет более высокий приоритет, чем ||
let result = a && b || c && d;
// Эквивалентно: (a && b) || (c && d)

// Используйте скобки для ясности
let clearResult = (a && b) || (c && d);
```

## Современные подходы в фронтенд-разработке

### Опциональная цепочка (Optional Chaining)

```javascript
// Безопасный доступ к вложенным свойствам
let user = { profile: { name: "Алексей" } };
let userName = user?.profile?.name; // "Алексей"
let unknown = user?.settings?.theme; // undefined, без ошибки
```

### Логические операторы для проверки данных

```javascript
// Проверка и предоставление значений по умолчанию
const getUserName = (user) => {
  return user?.name || "Анонимный пользователь";
};

// Условное выполнение
const updateUser = (userData) => {
  userData && userData.id && performUpdate(userData);
};
```

## Российские реалии и особенности

При работе с российскими проектами важно учитывать:

- Сравнение строк с кириллицей
- Форматирование чисел и дат
- Локализация сообщений об ошибках

```javascript
// Сравнение строк с кириллицей
const name1 = "Иван";
const name2 = "иван";
const isEqual = name1.toLowerCase() === name2.toLowerCase(); // true

// Форматирование чисел
const price = 12345.67;
const formattedPrice = price.toLocaleString('ru-RU', {
  style: 'currency',
  currency: 'RUB'
}); // "12 345,67 ₽"
```

## Заключение

Операторы и выражения — основа программной логики. Современные фронтенд-фреймворки активно используют эти концепции для создания динамических и интерактивных интерфейсов.

> [!tip] Совет
> Используйте строгое равенство (`===`) вместо нестрогого (`==`) для избежания неожиданного поведения.

> [!warning] Важно
> Обращайте внимание на приоритет операторов при написании сложных выражений.
