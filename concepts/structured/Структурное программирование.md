# Структурное программирование

Структурное программирование — это парадигма программирования, основанная на принципе разбиения программы на логически завершенные блоки и использование структурированного потока управления. Основная идея структурного программирования — сделать программы более понятными, надежными и легко модифицируемыми.

## Основные характеристики

### Использование последовательных блоков
Программа строится из последовательно выполняемых блоков кода.

### Условные операторы
Использование конструкций ветвления (if-else, switch-case) для принятия решений.

### Циклы
Использование циклических конструкций (for, while, do-while) для повторения операций.

### Избегание goto
Отказ от использования оператора goto, который может привести к "спагетти-коду".

## Исторический контекст

Структурное программирование было предложено Эдсгером Дейкстрой в 1968 году в его знаменитом письме "Go To Statement Considered Harmful". Эта парадигма стала ответом на проблемы, возникавшие при использовании непроцедурного стиля программирования с обильным использованием операторов goto.

## Примеры структурного программирования

### Базовая структура
```pascal
// Пример структурного программирования на Pascal
program Example;
var
  numbers: array[1..5] of integer;
  i, sum: integer;
begin
  // Инициализация массива
  numbers[1] := 1;
  numbers[2] := 2;
  numbers[3] := 3;
  numbers[4] := 4;
  numbers[5] := 5;
  
  // Вычисление суммы
  sum := 0;
  for i := 1 to 5 do
  begin
    sum := sum + numbers[i];
  end;
  
  // Вывод результата
  writeln('Сумма: ', sum);
end.
```

### JavaScript
```javascript
// Структурный подход
function findMax(numbers) {
  if (numbers.length === 0) {
    return null;
  }
  
  let max = numbers[0];
  
  for (let i = 1; i < numbers.length; i++) {
    if (numbers[i] > max) {
      max = numbers[i];
    }
  }
  
  return max;
}

const numbers = [3, 7, 2, 9, 1, 5];
const max = findMax(numbers);
console.log(max); // 9
```

### Python
```python
# Структурный подход
def calculate_average(numbers):
    if len(numbers) == 0:
        return 0
    
    total = 0
    for number in numbers:  # Цикл
        total += number
    
    return total / len(numbers)

def main():
    data = [10, 20, 30, 40, 50]
    average = calculate_average(data)
    print(f"Среднее значение: {average}")

if __name__ == "__main__":
    main()
```

## Основные принципы

### Принцип единственной точки входа и выхода
Каждый блок программы должен иметь одну точку входа и одну точку выхода.

### Принцип вложенности
Блоки могут содержать другие блоки, создавая иерархическую структуру.

### Принцип модульности
Программа разбивается на модули (функции, процедуры), каждый из которых выполняет определенную задачу.

## Преимущества структурного программирования

### Читаемость
Структурированный код легче читать и понимать.

### Сопровождаемость
Проще вносить изменения и исправлять ошибки.

### Надежность
Структурированный подход снижает вероятность ошибок.

### Повторное использование
Модули могут быть повторно использованы в других программах.

### Легкость тестирования
Отдельные модули можно тестировать независимо.

## Недостатки структурного программирования

### Ограниченность для сложных систем
Для очень сложных систем структурного программирования может быть недостаточно.

### Сложность моделирования реальных сущностей
Структурный подход не всегда удобен для моделирования сложных реальных объектов.

### Проблемы с управлением данными
При большом объеме глобальных данных может возникнуть путаница.

## Структурное программирование в контексте других парадигм

### Отличие от объектно-ориентированного программирования
В то время как структурное программирование фокусируется на функциях и процедурах, ООП фокусируется на объектах и их взаимодействиях.

```c
// Структурный подход
struct Point {
  int x;
  int y;
};

void movePoint(struct Point* point, int dx, int dy) {
  point->x += dx;
  point->y += dy;
}

// ООП подход
class Point {
  private:
    int x, y;
  public:
    Point(int x, int y) : x(x), y(y) {}
    void move(int dx, int dy) {
      x += dx;
      y += dy;
    }
};
```

### Сочетание с императивным программированием
Структурное программирование является подмножеством императивного программирования.

### Сочетание с процедурным программированием
Структурное программирование часто ассоциируется с процедурным программированием.

## Практическое применение

### Алгоритмы и структуры данных
Структурное программирование отлично подходит для реализации алгоритмов и структур данных.

```c
// Реализация стека в структурном стиле
#define MAX_SIZE 100
int stack[MAX_SIZE];
int top = -1;

void push(int value) {
  if (top < MAX_SIZE - 1) {
    stack[++top] = value;
  }
}

int pop() {
  if (top >= 0) {
    return stack[top--];
  }
  return -1; // Ошибка
}

int isEmpty() {
  return top == -1;
}
```

### Математические вычисления
Структурный подход эффективен для математических вычислений.

```python
# Вычисление факториала структурным подходом
def factorial(n):
    if n < 0:
        return None
    elif n == 0 or n == 1:
        return 1
    else:
        result = 1
        for i in range(2, n + 1):
            result *= i
        return result
```
