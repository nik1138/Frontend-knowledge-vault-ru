# Стили компонентов

Стилизация компонентов — это важный аспект компонентного подхода, который определяет внешний вид и визуальное представление компонентов. Существует несколько подходов к стилизации компонентов, каждый из которых имеет свои преимущества и недостатки.

## Основные подходы к стилизации

### 1. Встроенные стили (Inline Styles)

Встроенные стили применяются непосредственно к элементам через атрибут `style`.

#### React

```jsx
function Button({ children, primary, disabled }) {
  const buttonStyle = {
    padding: '10px 20px',
    border: 'none',
    borderRadius: '4px',
    cursor: disabled ? 'not-allowed' : 'pointer',
    opacity: disabled ? 0.6 : 1,
    backgroundColor: primary ? '#007bff' : '#6c757d',
    color: 'white',
    fontSize: '16px'
  };

  return (
    <button style={buttonStyle} disabled={disabled}>
      {children}
    </button>
  );
}
```

#### Vue.js

```vue
<template>
  <button 
    :style="buttonStyle" 
    :disabled="disabled"
  >
    {{ text }}
  </button>
</template>

<script>
export default {
  name: 'StyledButton',
  props: {
    text: String,
    primary: Boolean,
    disabled: Boolean
  },
  computed: {
    buttonStyle() {
      return {
        padding: '10px 20px',
        border: 'none',
        borderRadius: '4px',
        cursor: this.disabled ? 'not-allowed' : 'pointer',
        opacity: this.disabled ? 0.6 : 1,
        backgroundColor: this.primary ? '#007bff' : '#6c757d',
        color: 'white',
        fontSize: '16px'
      };
    }
  }
}
</script>
```

#### Angular

```typescript
@Component({
  selector: 'app-styled-button',
  template: `
    <button [style]="buttonStyle" [disabled]="disabled">
      {{ text }}
    </button>
  `
})
export class StyledButtonComponent {
  @Input() text: string;
  @Input() primary: boolean = false;
  @Input() disabled: boolean = false;

  get buttonStyle() {
    return {
      'padding': '10px 20px',
      'border': 'none',
      'border-radius': '4px',
      'cursor': this.disabled ? 'not-allowed' : 'pointer',
      'opacity': this.disabled ? 0.6 : 1,
      'background-color': this.primary ? '#007bff' : '#6c757d',
      'color': 'white',
      'font-size': '16px'
    };
  }
}
```

### 3. CSS-модули

CSS-модули позволяют изолировать стили на уровне компонентов.

#### React с CSS-модулями

```jsx
// Button.module.css
.button {
  padding: 8px 16px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  font-size: 14px;
  transition: all 0.2s;
}

.primary {
  background-color: #007bff;
  color: white;
}

.secondary {
  background-color: #6c757d;
  color: white;
}

.large {
  padding: 12px 24px;
  font-size: 16px;
}

.small {
  padding: 4px 8px;
  font-size: 12px;
}

.disabled {
  opacity: 0.6;
  cursor: not-allowed;
}
```

```jsx
// Button.jsx
import styles from './Button.module.css';

function Button({ children, variant = 'primary', size = 'medium', disabled }) {
  const className = [
    styles.button,
    styles[variant],
    styles[size],
    disabled && styles.disabled
  ].filter(Boolean).join(' ');

  return (
    <button className={className} disabled={disabled}>
      {children}
    </button>
  );
}
```

### 4. Styled Components (React)

Styled Components позволяют писать настоящий CSS в JavaScript.

```jsx
import styled from 'styled-components';

const StyledButton = styled.button`
  padding: ${props => props.size === 'large' ? '12px 24px' : props.size === 'small' ? '4px 8px' : '8px 16px'};
  border: none;
  border-radius: 4px;
  cursor: ${props => props.disabled ? 'not-allowed' : 'pointer'};
  font-size: ${props => props.size === 'large' ? '16px' : props.size === 'small' ? '12px' : '14px'};
  transition: all 0.2s;
  opacity: ${props => props.disabled ? 0.6 : 1};
  
  ${props => props.variant === 'primary' && `
    background-color: #007bff;
    color: white;
  `}
  
  ${props => props.variant === 'secondary' && `
    background-color: #6c757d;
    color: white;
  `}
  
  &:hover {
    ${props => !props.disabled && `
      transform: translateY(-1px);
      box-shadow: 0 4px 8px rgba(0,0,0,0.2);
    `}
  }
`;

function Button({ children, variant = 'primary', size = 'medium', disabled, ...props }) {
  return (
    <StyledButton 
      variant={variant} 
      size={size} 
      disabled={disabled} 
      {...props}
    >
      {children}
    </StyledButton>
  );
}
```

### 5. Emotion (React)

Emotion — еще одна библиотека для CSS-in-JS.

```jsx
import { css } from '@emotion/react';

const buttonStyle = (variant, size, disabled) => css`
  padding: ${size === 'large' ? '12px 24px' : size === 'small' ? '4px 8px' : '8px 16px'};
  border: none;
  border-radius: 4px;
  cursor: ${disabled ? 'not-allowed' : 'pointer'};
  font-size: ${size === 'large' ? '16px' : size === 'small' ? '12px' : '14px'};
  transition: all 0.2s;
  opacity: ${disabled ? 0.6 : 1};
  
  ${
    variant === 'primary' ? 
    'background-color: #007bff; color: white;' : 
    variant === 'secondary' ? 
    'background-color: #6c757d; color: white;' : 
    'background-color: #f8f9fa; color: #212529;'
  }
  
  ${!disabled && `
    &:hover {
      transform: translateY(-1px);
      box-shadow: 0 4px 8px rgba(0,0,0,0.2);
    }
  `}
`;

function Button({ children, variant = 'primary', size = 'medium', disabled, ...props }) {
  return (
    <button 
      css={buttonStyle(variant, size, disabled)} 
      disabled={disabled} 
      {...props}
    >
      {children}
    </button>
  );
}
```

### 6. SCSS/SASS с компонентами

Использование препроцессоров CSS с компонентами.

```vue
<template>
  <button :class="buttonClass" :disabled="disabled">
    {{ text }}
  </button>
</template>

<script>
export default {
  name: 'SassButton',
  props: {
    text: String,
    variant: {
      type: String,
      default: 'primary'
    },
    size: {
      type: String,
      default: 'medium'
    },
    disabled: Boolean
  },
  computed: {
    buttonClass() {
      return [
        'btn',
        `btn--${this.variant}`,
        `btn--${this.size}`,
        { 'btn--disabled': this.disabled }
      ];
    }
  }
}
</script>

<style lang="scss">
$primary-color: #007bff;
$secondary-color: #6c757d;
$light-color: #f8f9fa;
$dark-color: #212529;

$button-padding: (
  small: 4px 8px,
  medium: 8px 16px,
  large: 12px 24px
);

$button-font-size: (
  small: 12px,
  medium: 14px,
  large: 16px
);

.btn {
  padding: map-get($button-padding, medium);
  border: none;
  border-radius: 4px;
  cursor: pointer;
  transition: all 0.2s;
  
  @each $variant, $color in (primary: $primary-color, secondary: $secondary-color, light: $light-color) {
    &--#{$variant} {
      background-color: $color;
      color: if($variant == 'light', $dark-color, white);
    }
  }
  
  @each $size, $padding in $button-padding {
    &--#{$size} {
      padding: $padding;
      font-size: map-get($button-font-size, $size);
    }
  }
  
  &--disabled {
    opacity: 0.6;
    cursor: not-allowed;
  }
  
  &:not(.btn--disabled):hover {
    transform: translateY(-1px);
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
  }
}
</style>
```

### 7. Tailwind CSS

Использование утилитарных классов для стилизации.

```jsx
function Button({ children, variant = 'primary', size = 'medium', disabled, className = '' }) {
  const baseClasses = 'border-0 rounded cursor-pointer transition-all duration-200';
  
  const variantClasses = {
    primary: 'bg-blue-500 text-white hover:bg-blue-600',
    secondary: 'bg-gray-500 text-white hover:bg-gray-600',
    success: 'bg-green-500 text-white hover:bg-green-600',
    danger: 'bg-red-500 text-white hover:bg-red-600'
  };
  
  const sizeClasses = {
    small: 'py-1 px-2 text-sm',
    medium: 'py-2 px-4 text-base',
    large: 'py-3 px-6 text-lg'
  };
  
  const disabledClasses = disabled ? 'opacity-60 cursor-not-allowed' : 'hover:shadow-md';
  
  const classes = [
    baseClasses,
    variantClasses[variant],
    sizeClasses[size],
    disabledClasses,
    className
  ].filter(Boolean).join(' ');

  return (
    <button className={classes} disabled={disabled}>
      {children}
    </button>
  );
}
```

## Сравнение подходов

| Подход | Преимущества | Недостатки | Использование |
|--------|-------------|------------|---------------|
| Встроенные стили | Простота, динамические стили | Плохая переиспользуемость, сложность в поддержке | Простые компоненты, динамические стили |
| CSS-классы | Хорошая производительность, знакомый синтаксис | Глобальное пространство имён, конфликты | Традиционные приложения |
| CSS-модули | Изоляция стилей, глобальный синтаксис | Дополнительная настройка, сложность в сложных селекторах | Современные приложения |
| Styled Components | Компонентная изоляция, динамические стили | Больше JavaScript, потенциальная перегрузка | React-приложения |
| Emotion | Гибкость, хорошая производительность | Сложность для новичков | React-приложения |
| Tailwind CSS | Быстрая разработка, консистентность | Большое количество классов, сложность кастомизации | Быстрые прототипы, дизайн-системы |

## Лучшие практики

### 1. Изоляция стилей

```jsx
// Хорошо: стили изолированы
import styles from './Component.module.css';

function MyComponent() {
  return <div className={styles.container}>...</div>;
}

// Плохо: глобальные стили могут конфликтовать
function MyComponent() {
  return <div className="container">...</div>; // Может конфликтовать с другими "container"
}
```

### 2. Темизация

```jsx
// ThemeProvider.jsx
import React, { createContext, useContext } from 'react';

const ThemeContext = createContext();

export function ThemeProvider({ children, theme }) {
  return (
    <ThemeContext.Provider value={theme}>
      {children}
    </ThemeContext.Provider>
  );
}

export function useTheme() {
  return useContext(ThemeContext);
}

// Component.jsx
import { useTheme } from './ThemeProvider';

function Button({ children, variant }) {
  const theme = useTheme();
  
  const style = {
    backgroundColor: theme.colors[variant] || theme.colors.primary,
    color: theme.colors.text,
    padding: theme.spacing.medium,
    border: 'none',
    borderRadius: theme.borderRadius
  };
  
  return <button style={style}>{children}</button>;
}
```

### 4. Состояния компонентов

```jsx
// Button.jsx с различными состояниями
import styled from 'styled-components';

const StyledButton = styled.button`
  padding: 8px 16px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  font-size: 14px;
  transition: all 0.2s;
  
  /* Нормальное состояние */
  background-color: ${props => props.variant === 'primary' ? '#007bff' : '#6c757d'};
  color: white;
  
  /* Hover состояние */
  &:hover:not(:disabled) {
    transform: translateY(-1px);
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
  }
  
  /* Active состояние */
  &:active:not(:disabled) {
    transform: translateY(0);
  }
  
  /* Disabled состояние */
  &:disabled {
    opacity: 0.6;
    cursor: not-allowed;
  }
  
  /* Focus состояние для доступности */
  &:focus {
    outline: 2px solid #0056b3;
    outline-offset: 2px;
  }
`;
```
