# Селекторы

Селекторы — это функции, используемые для извлечения и преобразования данных из хранилища состояния. Они позволяют компонентам получать только необходимую информацию и обеспечивают производительность за счет мемоизации вычислений.

## Основные Концепции

### Извлечение данных
Селекторы извлекают нужные данные из общего состояния, позволяя компонентам не знать о внутренней структуре хранилища.

### Преобразование данных
Селекторы могут преобразовывать данные из хранилища перед передачей в компоненты, например, фильтрацию, сортировку или вычисление производных значений.

### Мемоизация
Многие библиотеки управления состоянием предоставляют мемоизированные селекторы, которые вычисляют результат только при изменении зависимостей, что улучшает производительность.

## Использование

Селекторы особенно полезны в следующих случаях:
- При работе с нормализованными данными
- Для вычисления производных данных
- Для фильтрации или сортировки списков
- Для объединения данных из разных частей состояния

## Плюсы использования селекторов

- Производительность: мемоизация предотвращает ненужные вычисления
- Тестируемость: селекторы легко тестировать как чистые функции
- Поддерживаемость: централизованное место для логики извлечения данных
- Читаемость: отделение логики извлечения данных от представления
