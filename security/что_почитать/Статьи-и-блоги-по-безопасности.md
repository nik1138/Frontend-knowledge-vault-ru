# Статьи и блоги по безопасности

## Российские ресурсы

### Хабрахабр (секция информационная безопасность)
- [https://habr.com/ru/security/](https://habr.com/ru/security/)
- Публикуются как новостные материалы, так и глубокие технические статьи
- Актуальные обзоры уязвимостей и методов защиты в российском контексте

### SecurityLab
- [https://www.securitylab.ru/]
- Один из ведущих российских порталов по информационной безопасности
- Регулярные публикации о новых уязвимостях, атаках и методах защиты

### PT Security (Positive Technologies)
- [https://ptsecurity.ru/blog]
- Технические статьи от экспертов Positive Technologies
- Анализ реальных атак и методов защиты, адаптированных под российские реалии

### Digital Security
- [https://digital-security.ru/]
- Статьи о киберугрозах и методах защиты
- Регулярные обзоры текущей ситуации в области кибербезопасности

## Международные ресурсы

### OWASP Blog
- [https://owasp.org/blog/]
- Официальный блог Open Web Application Security Project
- Регулярные обновления по OWASP Top 10 и новым угрозам

### Krebs on Security
- [https://krebsonsecurity.com/]
- Блог Брайана Кребса, одного из ведущих экспертов в области кибербезопасности
- Глубокий анализ реальных инцидентов безопасности

### Troy Hunt's Blog
- [https://www.troyhunt.com/]
- Блог эксперта по безопасности Troy Hunt
- Много информации о безопасности веб-приложений и API

### Scott Helme
- [https://scotthelme.co.uk/]
- Блог по безопасности веб-приложений и HTTPS
- Практические руководства по настройке безопасности

## Тематические статьи по веб-безопасности

### XSS (Cross-Site Scripting)
- "Understanding XSS: Types and Prevention Methods" - подробный анализ всех типов XSS
- "DOM-based XSS Explained" - особенности клиентских XSS-уязвимостей

### SQL Injection
- "Advanced SQL Injection Techniques" - продвинутые методы атак и защиты
- "SQL Injection Prevention Cheat Sheet" - практические рекомендации

### CSRF (Cross-Site Request Forgery)
- "CSRF Protection Methods Explained" - сравнение различных методов защиты
- "Double-Submit Cookie vs. Samesite Cookie" - современные подходы к защите

### Authentication and Authorization
- "OAuth 2.0 Security Best Practices" - безопасная реализация OAuth
- "Session Management Security" - защита сессий в веб-приложениях
