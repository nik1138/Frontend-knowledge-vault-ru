# Утилиты типов TypeScript

Утилиты типов - это встроенные и пользовательские типы-помощники, которые позволяют преобразовывать и манипулировать типами в TypeScript. Они значительно упрощают работу с типами и помогают создавать более гибкие и переиспользуемые типы.

## Встроенные утилиты TypeScript

TypeScript предоставляет множество встроенных утилит, которые охватывают большинство распространенных сценариев работы с типами:

### Основные категории:

1. **Преобразование свойств объектов**
   - `Partial<T>` - делает все свойства опциональными
   - `Required<T>` - делает все свойства обязательными
   - `Readonly<T>` - делает все свойства только для чтения
   - `Mutable<T>` - обратное к `Readonly<T>`

2. **Работа с подмножествами свойств**
   - `Pick<T, K>` - выбирает определенные свойства
   - `Omit<T, K>` - исключает определенные свойства
   - `Extract<T, U>` - извлекает пересечение типов
   - `Exclude<T, U>` - исключает типы

3. **Работа с функциями**
   - `ReturnType<T>` - получает тип возвращаемого значения функции
   - `Parameters<T>` - получает типы параметров функции
   - `ConstructorParameters<T>` - получает типы параметров конструктора
   - `InstanceType<T>` - получает тип экземпляра класса

4. **Работа со строками**
   - `Uppercase<S>` - преобразует строку в верхний регистр
   - `Lowercase<S>` - преобразует строку в нижний регистр
   - `Capitalize<S>` - делает первую букву заглавной
   - `Uncapitalize<S>` - делает первую букву строчной

5. **Другие утилиты**
   - `Record<K, T>` - создает тип с набором свойств K типа T
   - `NonNullable<T>` - исключает null и undefined
   - `ThisType<T>` - манипулирует типом this в функциях

## Пользовательские утилиты

Кроме встроенных утилит, можно создавать собственные типы-помощники для решения специфических задач проекта.

## Цели изучения:

1. Освоить все встроенные утилиты TypeScript
2. Научиться создавать собственные утилиты типов
3. Понимать, когда и как применять различные утилиты
4. Уметь комбинировать утилиты для сложных преобразований
