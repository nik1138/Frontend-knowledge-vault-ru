## Примеры активности в сообществе

### Как участвовать

```typescript
// Пример кода, который можно обсудить в сообществе
interface Contributor {
  name: string;
  contribution: string;
  date: Date;
}

class CommunityMember {
  private contributions: Contributor[] = [];

  addContribution(contributor: Contributor): void {
    this.contributions.push(contributor);
    console.log(`Добавлен вклад от ${contributor.name}`);
  }

  getContributions(): Contributor[] {
    return this.contributions;
  }
}

// Использование класса
const member = new CommunityMember();
member.addContribution({
  name: 'Алексей',
  contribution: 'Исправление документации',
  date: new Date()
});
```
