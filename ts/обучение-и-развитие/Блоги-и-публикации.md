## Тематические статьи

### Основные концепции

```typescript
// Пример кода из статьи о продвинутой типизации
type StringOrNumber = string | number;

// Conditional Types
type NonNullable<T> = T extends null | undefined ? never : T;

// Mapped Types
type Partial<T> = {
  [P in keyof T]?: T[P];
};

// Utility Types
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P];
};

// Пример использования
interface User {
  id: number;
  name: string;
  email: string;
  age: number;
}

type UserPreview = MyPick<User, 'id' | 'name'>;

const userPreview: UserPreview = {
  id: 1,
  name: 'Алексей'
};
```

## Практические руководства

### Реальные примеры использования

```typescript
// Пример из статьи о паттернах проектирования в TypeScript
interface PaymentProcessor {
  process(amount: number): boolean;
}

class CreditCardProcessor implements PaymentProcessor {
  process(amount: number): boolean {
    console.log(`Обработка платежа на сумму ${amount} через кредитную карту`);
    // Логика обработки платежа
    return true;
  }
}

class PayPalProcessor implements PaymentProcessor {
  process(amount: number): boolean {
    console.log(`Обработка платежа на сумму ${amount} через PayPal`);
    // Логика обработки платежа
    return true;
  }
}

class PaymentService {
  constructor(private processor: PaymentProcessor) {}

  makePayment(amount: number): void {
    if (this.processor.process(amount)) {
      console.log('Платеж успешно обработан');
    } else {
      console.log('Ошибка обработки платежа');
    }
  }
}

// Использование
const creditCardService = new PaymentService(new CreditCardProcessor());
creditCardService.makePayment(100);

const paypalService = new PaymentService(new PayPalProcessor());
paypalService.makePayment(200);
```
