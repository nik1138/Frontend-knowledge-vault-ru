### Руководства и туториалы

```typescript
// Пример кода из официального руководства
interface User {
  name: string;
  age: number;
}

function greetUser(user: User): string {
  return `Привет, ${user.name}! Тебе ${user.age} лет.`;
}

const user: User = { name: 'Алексей', age: 30 };
console.log(greetUser(user));
```

## Практические проекты

### Идеи для практики

> [!tip] Совет
> Лучший способ изучить TypeScript - создавать реальные проекты. Начните с простых приложений и постепенно усложняйте их.

1. **Консольное приложение**: Создайте CLI-инструмент для работы с файлами
2. **Веб-приложение**: Разработайте SPA с использованием TypeScript и фреймворка
3. **Node.js сервер**: Создайте API с типизированными маршрутами
4. **Библиотека компонентов**: Разработайте переиспользуемые компоненты с типами
