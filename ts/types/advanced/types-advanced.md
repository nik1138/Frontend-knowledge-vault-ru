# Продвинутые типы TypeScript

Продвинутые типы включают специальные типы и сложные паттерны типизации, которые предоставляют мощные возможности для моделирования сложных доменных областей и создания безопасных API.

## Содержание

### [Специальные типы](special-types.md)
Типы any, unknown, never, void и null/undefined с особенностями использования и безопасности типов.

### [Шаблонные литеральные типы](template-literal-types.md)
Создание строковых типов с использованием шаблонных литералов, преобразования регистров, сопоставляющие типы с литералами.

### [Псевдонимы типов](type-aliases.md)
Создание пользовательских именованных типов, обобщенные псевдонимы, сравнение с интерфейсами и продвинутые паттерны.

## Обзор продвинутых типов

Продвинутые типы предоставляют мощные инструменты для сложной типизации:

1. **Специальные типы**: уникальные типы с особым поведением
2. **Шаблонные литералы**: генерация строковых типов на основе паттернов
3. **Псевдонимы**: абстракция и переиспользование сложных типов
4. **Комбинации**: интеграция разных типов для создания мощных паттернов

Эти инструменты позволяют создавать очень выразительные и безопасные API.
