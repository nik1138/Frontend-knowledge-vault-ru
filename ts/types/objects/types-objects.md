# Объектные типы TypeScript

Объектные типы включают сложные структуры данных: объекты, массивы, функции, объединения и пересечения. Они позволяют моделировать сложные структуры данных и отношения между ними.

## Содержание

### [Объектные типы](object-types.md)
Типизация объектов, интерфейсы, индексные сигнатуры, геттеры/сеттеры и сложные структуры.

### [Объединения типов](union-types.md)
Комбинации типов с помощью оператора `|`, сужение типов, дискриминированные объединения и проверка исчерпания.

### [Пересечения типов](intersection-types.md)
Комбинации типов с помощью оператора `&`, композиция функциональности, продвинутые паттерны объединения.

## Обзор объектных типов

Объектные типы позволяют создавать сложные структуры данных:

1. **Объектные литералы**: определение форм объектов
2. **Объединения**: выбор одного из нескольких типов
3. **Пересечения**: комбинация нескольких типов
4. **Функциональные типы**: типизация функций как объектов
5. **Массивы и кортежи**: типизация коллекций данных

Эти типы позволяют точно моделировать структуру данных в приложениях.
