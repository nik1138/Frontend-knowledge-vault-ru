# Неизменяемость

Неизменяемость (immutability) - ключевая концепция функционального программирования, которая предполагает, что данные не изменяются после создания. Вместо изменения существующих структур данных создаются новые.

## Понятие неизменяемости

Неизменяемость означает, что после создания объекта его состояние не может быть изменено. Вместо модификации существующих данных создаются новые версии объектов.

```typescript
// Изменяемая структура
let mutableUser = { name: "John", age: 30 };
mutableUser.age = 31; // Изменение существующего объекта

// Неизменяемая структура
const immutableUser = { name: "John", age: 30 };
const updatedUser = { ...immutableUser, age: 31 }; // Создание нового объекта
```
