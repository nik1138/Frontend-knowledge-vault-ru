# Композиция функций

Композиция функций - это фундаментальная концепция функционального программирования, при которой несколько функций объединяются для создания новой функции. Результат одной функции передается как входные данные следующей функции.

## Понятие композиции

Композиция функций - это процесс объединения двух или более функций для создания новой функции. В математике композиция функций f и g обозначается как (f ∘ g)(x) = f(g(x)).

```typescript
// Простая композиция двух функций
const add = (x: number): number => x + 1;
const multiply = (x: number): number => x * 2;

// Композиция: сначала add, затем multiply
const addAndMultiply = (x: number): number => multiply(add(x));
console.log(addAndMultiply(5)); // 12 (5 + 1 = 6, 6 * 2 = 12)

// Композиция: сначала multiply, затем add
const multiplyAndAdd = (x: number): number => add(multiply(x));
console.log(multiplyAndAdd(5)); // 11 (5 * 2 = 10, 10 + 1 = 11)
```

## Преимущества композиции

1. **Повторное использование кода** - Существующие функции могут быть объединены для создания новых функций
2. **Модульность** - Функции могут быть разработаны независимо и затем объединены
3. **Читаемость** - Композиция делает поток данных явным
4. **Тестируемость** - Каждая функция может быть протестирована отдельно
5. **Поддерживаемость** - Изменения в одной функции не влияют на другие
