## Определение чистой функции

Чистая функция - это функция, которая обладает двумя ключевыми свойствами:

1. **Детерминированность** - для одинаковых входных данных всегда возвращает одинаковый результат
2. **Отсутствие побочных эффектов** - не изменяет внешнее состояние и не взаимодействует с внешним миром
