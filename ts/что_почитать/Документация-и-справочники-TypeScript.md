# Документация и справочники TypeScript

## Официальные источники документации

### 1. Официальная документация TypeScript

**URL:** https://www.typescriptlang.org/docs/

**Особенности:**
- Самый авторитетный источник информации
- Регулярно обновляется с новыми версиями
- Содержит руководства, справочники и примеры
- Хорошо структурирована для изучения

**Секции документации:**
- [[Руководство по TypeScript]] - основное руководство для начинающих
- [[Справочник по типам]] - подробное описание систем типов
- [[Конфигурация]] - объяснение опций компилятора
- [[Совместимость]] - информация о совместимости с JavaScript

### 2. Справочник по типам TypeScript

**URL:** https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html

**Особенности:**
- Подробное объяснение всех типов данных
- Примеры использования
- Сравнение с JavaScript
- Практические рекомендации

**Что охватывает:**
- Примитивные типы
- Объектные типы
- Функциональные типы
- Дженерики
- Утилитарные типы

### 3. Руководство по спецификации TypeScript

**URL:** https://github.com/microsoft/TypeScript/blob/main/doc/spec-ARCHIVED.md

**Особенности:**
- Архивированная спецификация языка
- Подробное техническое описание
- Полезно для понимания внутреннего устройства языка
- Обновляется редко

- Обновляются энтузиастами сообщества

**Преимущества:**
- Понятный язык изложения

## Российские реалии и доступность

### Проблемы доступа:
- Официальная документация может быть недоступна из-за ограничений
- Языковой барьер для английской документации
- Медленная загрузка международных ресурсов

### Решения:
- Использовать локальные копии документации
- Следить за переводами в сообществах
- Использовать [[Сообщества-и-ресурсы-TypeScript]] для получения актуальной информации
- Использовать [[Статьи-и-блоги-по-TypeScript]] с объяснением документации

### Кэширование и офлайн-доступ:
- Сохранять важные страницы документации локально
- Использовать инструменты для скачивания документации
- Создавать собственные шпаргалки на основе документации
