# Определение интерфейсов в TypeScript

Интерфейсы в TypeScript - это мощный инструмент для определения контрактов в вашем коде. Они описывают форму объекта, определяя, какие свойства и методы должны быть у объекта, а также их типы.

## Основы интерфейсов

Интерфейс определяется с помощью ключевого слова `interface`, за которым следует имя интерфейса и тело интерфейса в фигурных скобках:

```typescript
interface User {
  name: string;
  age: number;
  email: string;
}
```

Теперь мы можем использовать этот интерфейс для описания объектов:

```typescript
const user: User = {
  name: "Иван",
  age: 30,
  email: "ivan@example.com"
};
```

## Типизация свойств

Интерфейсы позволяют точно определить типы данных для каждого свойства объекта:

```typescript
interface Product {
  id: number;
  title: string;
  price: number;
  inStock: boolean;
}
```

## Методы в интерфейсах

Интерфейсы могут также определять методы, которые должны быть реализованы объектом:

```typescript
interface Calculator {
  add(a: number, b: number): number;
  subtract(a: number, b: number): number;
  multiply(a: number, b: number): number;
}

const calc: Calculator = {
  add(a, b) {
    return a + b;
  },
  subtract(a, b) {
    return a - b;
  },
  multiply(a, b) {
    return a * b;
  }
};
```

## Функции в интерфейсах

Интерфейсы могут описывать сигнатуру функции:

```typescript
interface Greeter {
  (name: string): string;
}

const greet: Greeter = (name) => `Привет, ${name}!`;
```

## Индексные сигнатуры

Интерфейсы могут определять индексные сигнатуры для описания объектов с произвольными ключами:

```typescript
interface StringArray {
  [index: number]: string;
}

interface Dictionary {
  [key: string]: number;
}
```

## Взаимодействие с другими концепциями

Интерфейсы тесно связаны с другими концепциями TypeScript, такими как [[Типы]] и [[Классы]]. Они обеспечивают мощный способ определения формата объектов и контрактов в вашем приложении.

> [!tip] Совет
> Используйте интерфейсы для определения формата объектов, особенно когда вы ожидаете, что эти объекты будут реализованы разными классами или использоваться в разных частях вашего приложения.

> [!warning] Важно
> Интерфейсы существуют только во время компиляции и не влияют на JavaScript, который генерируется TypeScript.
