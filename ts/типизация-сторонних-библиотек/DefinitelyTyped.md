# DefinitelyTyped

**DefinitelyTyped** — это репозиторий с определениями типов для библиотек JavaScript, которые не имеют собственной поддержки TypeScript. Это один из самых важных ресурсов в экосистеме TypeScript, позволяющий использовать сторонние библиотеки с проверкой типов.

## Обзор

DefinitelyTyped — это центральный репозиторий на GitHub, содержащий файлы определений типов (`.d.ts`) для тысяч популярных JavaScript-библиотек. Эти определения позволяют TypeScript понимать API сторонних библиотек, обеспечивая автодополнение, проверку типов и лучшую документацию в редакторе кода.

## Структура репозитория

Репозиторий организован по следующему принципу:
- Каждый пакет определений типов находится в отдельной директории
- Название директории соответствует названию библиотеки (например, `jquery`, `lodash`)
- Внутри директории находится файл `index.d.ts` и тесты
- Пакеты публикуются под префиксом `@types/` (например, `@types/jquery`)

## Установка определений типов

Для установки определений типов из DefinitelyTyped используйте npm:

```bash
npm install --save-dev @types/lodash
npm install --save-dev @types/node
npm install --save-dev @types/react
```

TypeScript автоматически находит и использует эти определения при импорте соответствующих библиотек.

## Качество определений

Определения типов в DefinitelyTyped проходят строгую проверку:
- Автоматические тесты
- Рецензирование кода
- Совместимость с несколькими версиями TypeScript
- Регулярные обновления

## Участие в проекте

DefinitelyTyped — это проект с открытым исходным кодом, и любой разработчик может внести свой вклад:
- Добавление новых определений типов
- Исправление существующих определений
- Обновление определений под новые версии библиотек

## Пример структуры определения

```typescript
// types/lodash/index.d.ts
declare module 'lodash' {
  export function map<T, U>(collection: T[], iteratee: (value: T) => U): U[];
  export function filter<T>(collection: T[], predicate: (value: T) => boolean): T[];
  // ... другие функции
}
```
