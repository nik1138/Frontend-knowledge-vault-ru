# Установка и настройка TypeScript

Для начала работы с TypeScript необходимо установить компилятор TypeScript и настроить проект для его использования. В этом руководстве описаны различные способы установки и настройки TypeScript в вашем проекте.

## Установка TypeScript

TypeScript можно установить несколькими способами:

### Установка через npm

Самый распространенный способ установки TypeScript - через npm (Node.js Package Manager):

```bash
# Установка глобально
npm install -g typescript

# Установка в проект (рекомендуется)
npm install --save-dev typescript
```

После установки вы можете проверить версию TypeScript:

```bash
tsc --version
```

### Установка через yarn

```bash
yarn add -D typescript
```

### Установка через pnpm

```bash
pnpm add -D typescript
```

## Инициализация проекта

Для настройки TypeScript в проекте рекомендуется создать конфигурационный файл `tsconfig.json`:

```bash
tsc --init
```

Эта команда создаст базовый файл конфигурации TypeScript с часто используемыми настройками.

## Базовая настройка

После установки TypeScript можно начать использовать его для компиляции файлов:

```bash
# Компиляция одного файла
tsc myfile.ts

# Компиляция с указанием выходного файла
tsc input.ts --outFile output.js

# Компиляция всего проекта (используя tsconfig.json)
tsc
```

## Настройка редактора кода

Для полноценной поддержки TypeScript рекомендуется использовать один из следующих редакторов:

- **Visual Studio Code** - имеет встроенную поддержку TypeScript
- **WebStorm** - полнофункциональная IDE с отличной поддержкой TypeScript
- **Atom** - с пакетом atom-typescript
- **Sublime Text** - с соответствующими плагинами

## Интеграция с сборщиками

TypeScript может быть интегрирован с различными сборщиками:

### Webpack

```bash
npm install --save-dev ts-loader webpack
```

Конфигурация webpack.config.js:

```js
module.exports = {
  entry: './src/index.ts',
  module: {
    rules: [
      {
        test: /\.tsx?$/,
        use: 'ts-loader',
        exclude: /node_modules/,
      },
    ],
  },
  resolve: {
    extensions: ['.tsx', '.ts', '.js'],
  },
  output: {
    filename: 'bundle.js',
    path: __dirname + '/dist',
  },
};
```

### Parcel

Parcel поддерживает TypeScript из коробки:

```bash
npm install -g parcel-bundler
parcel index.html
```

### Vite

```bash
npm create vite@latest my-app -- --template ts
```

## Создание первого TypeScript файла

Создайте файл с расширением `.ts`:

```ts
// hello.ts
function greet(name: string): string {
    return `Привет, ${name}!`;
}

const user: string = "Иван";
console.log(greet(user));
```

Затем скомпилируйте его:

```bash
tsc hello.ts
```

Это создаст файл `hello.js` с JavaScript-кодом.

## Настройка автоматической компиляции

Для автоматической компиляции при изменении файлов используйте флаг `--watch`:

```bash
tsc --watch
```

## Установка определений типов

Для использования сторонних библиотек, написанных на JavaScript, установите соответствующие определения типов:

```bash
# Установка типов для конкретной библиотеки
npm install --save-dev @types/lodash

# Установка типов для Node.js
npm install --save-dev @types/node
```

## Проверка установки

Проверьте, что все установлено правильно:

```bash
# Проверка версии TypeScript
tsc --version

# Проверка компиляции простого файла
echo 'console.log("Hello TypeScript");' > test.ts
tsc test.ts
node test.js
```
