# ref

## Определение
`ref` - это одна из основных функций реактивности в Vue 3 Composition API, которая создает реактивную ссылку на значение. Используется для создания реактивных примитивных значений (чисел, строк, булей и т.д.).

## Синтаксис
```javascript
import { ref } from 'vue'

const counter = ref(0)
```

## Особенности
- Оборачивает значение в объект с полем `.value`
- Автоматически делает значение реактивным
- Используется для примитивных типов данных (number, string, boolean)
- Также может использоваться для объектов (с глубокой реактивностью)

## Примеры использования

### Примитивные значения
```javascript
import { ref } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const name = ref('Vue')
    
    const increment = () => {
      count.value++
    }
    
    return {
      count,
      name,
      increment
    }
  }
}
```

### В шаблоне
```vue
<template>
  <div>
    <p>Счетчик: {{ count }}</p>
    <p>Имя: {{ name }}</p>
    <button @click="increment">Увеличить</button>
  </div>
</template>
```

### С объектами (глубокая реактивность)
```javascript
const user = ref({ name: 'John', age: 30 })

// Для изменения объекта нужно использовать .value
user.value.name = 'Jane'

// Свойства объекта будут глубоко реактивными при использовании ref
// Vue автоматически оборачивает объект в reactive
```

## Примечания
- В шаблоне `.value` не требуется, Vue автоматически разворачивает ref
- В Composition API `.value` обязательно для доступа к значению
- Для глубокой реактивности объектов лучше использовать [[reactive]]
