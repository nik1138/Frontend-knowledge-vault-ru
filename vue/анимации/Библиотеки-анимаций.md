# Библиотеки анимаций в Vue.js

## Обзор

Библиотеки анимаций в Vue.js предоставляют готовые решения для создания сложных анимационных эффектов. Они значительно упрощают разработку интерактивных интерфейсов и позволяют реализовать профессиональные анимации без глубоких знаний низкоуровневых API.

## Популярные библиотеки анимаций

### Anime.js

Anime.js - мощная и гибкая библиотека анимаций с поддержкой CSS, SVG, DOM и JavaScript объектов.

#### Установка

```bash
npm install animejs
```

#### Пример использования в Vue

```vue
<template>
  <div>
    <button @click="runAnimation">Запустить анимацию</button>
    <div ref="animatedElement" class="box"></div>
  </div>
</template>

<script>
import anime from 'animejs/lib/anime.es.js'

export default {
  methods: {
    runAnimation() {
      anime({
        targets: this.$refs.animatedElement,
        translateX: 250,
        rotate: 180,
        borderRadius: ['0%', '50%'],
        backgroundColor: '#FFF',
        duration: 2000,
        direction: 'alternate',
        loop: true
      })
    }
  }
}
</script>

<style>
.box {
  width: 100px;
  height: 100px;
  background-color: #42b983;
  margin: 20px;
}
</style>
```

### GSAP (GreenSock Animation Platform)

GSAP - профессиональная библиотека анимаций с высокой производительностью и богатыми возможностями.

#### Установка

```bash
npm install gsap
```

#### Пример использования в Vue

```vue
<template>
  <div>
    <button @click="gsapAnimation">GSAP анимация</button>
    <div ref="gsapElement" class="box"></div>
  </div>
</template>

<script>
import { gsap } from 'gsap'

export default {
  methods: {
    gsapAnimation() {
      gsap.to(this.$refs.gsapElement, {
        duration: 2,
        x: 300,
        rotation: 360,
        ease: 'bounce',
        backgroundColor: '#f8f8f8',
        borderRadius: '50%'
      })
    }
  }
}
</script>
```

### Framer Motion (через Vue Motion)

Framer Motion предоставляет интуитивный API для создания анимаций с поддержкой жестов и физики.

#### Установка

```bash
npm install framer-motion vue-motion-next
```

### Vueuse/Motion

Современная библиотека анимаций, интегрированная с VueUse.

#### Установка

```bash
npm install @vueuse/motion
```

#### Пример использования

```vue
<template>
  <div>
    <Motion 
      :initial="{ opacity: 0, y: 100 }"
      :enter="{ opacity: 1, y: 0, transition: { duration: 1000 } }"
      class="box"
    >
      Анимированный элемент
    </Motion>
  </div>
</template>

<script>
import { Motion } from '@vueuse/motion'
import { defineComponent } from 'vue'

export default defineComponent({
  components: { Motion }
})
</script>
```

## Анимации для списков и переходов

### Vue Virtual Scroller + анимации

Для работы с большими списками с анимациями:

```bash
npm install vue-virtual-scroller
```

```vue
<template>
  <RecycleScroller 
    class="scroller"
    :items="items"
    :item-size="54"
    key-field="id"
    v-slot="{ item }"
  >
    <div 
      class="user"
      @click="goToUser(item)"
      v-motion
      :initial="{ opacity: 0, y: 20 }"
      :enter="{ opacity: 1, y: 0, transition: { delay: item.id * 50 } }"
    >
      {{ item.name }}
    </div>
  </RecycleScroller>
</template>
```

## Анимации для российских реалий 2025

В 2025 году в российском Vue-экосистеме особенно важны:

1. **Производительность** - выбор библиотек с минимальным размером и высокой производительностью
2. **Поддержка отечественных браузеров** - проверка совместимости с Яндекс.Браузер, KukuRuzo и другими
3. **Офлайн-работа** - анимации должны корректно работать в условиях нестабильного интернета
4. **Доступность** - поддержка пользовательских настроек, таких как предпочтение уменьшения анимаций

### Оптимизация для российских условий

```javascript
// Проверка системных настроек пользователя
const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)')
const isLowEndDevice = navigator.hardwareConcurrency <= 2

// Условное применение анимаций
const applyAnimations = !prefersReducedMotion.matches && !isLowEndDevice

// Конфигурация анимаций в зависимости от производительности устройства
const animationConfig = {
  duration: isLowEndDevice ? 300 : 600,
  easing: 'ease-out'
}
```

## Лучшие практики

1. Используйте анимации умеренно - избыток анимаций ухудшает UX
2. Обеспечьте fallback для пользователей с отключенными анимациями
3. Тестируйте производительность на слабых устройствах
4. Используйте аппаратное ускорение (`transform`, `opacity`)
5. Оптимизируйте размер бандла при выборе библиотеки
