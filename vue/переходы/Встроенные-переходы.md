# Встроенные-переходы

## Обзор

Vue.js предоставляет встроенный компонент `<transition>`, который позволяет применять анимации перехода к элементам и компонентам при их добавлении, обновлении или удалении из DOM. Это особенно полезно для создания плавных UI-эффектов без необходимости подключения сторонних библиотек.

## Основы использования

Компонент `<transition>` оборачивает элемент или компонент, для которого нужно применить переход:

```vue
<template>
  <div>
    <button @click="show = !show">Переключить</button>
    <transition name="fade">
      <p v-if="show">Привет, это анимированный текст!</p>
    </transition>
  </div>
</template>

<script>
export default {
  data() {
    return {
      show: true
    }
  }
}
</script>

<style>
.fade-enter-active, .fade-leave-active {
  transition: opacity 0.5s;
}
.fade-enter-from, .fade-leave-to {
  opacity: 0;
}
</style>
```

## Типы переходов

### Transition для одиночных элементов

Используется для элементов, которые появляются/исчезают по условию:

```vue
<template>
  <transition name="slide">
    <div v-show="isVisible" class="alert">
      Сообщение с анимацией
    </div>
  </transition>
</template>

<style>
.slide-enter-active {
  transition: all 0.3s ease-out;
}

.slide-leave-active {
  transition: all 0.3s ease-in;
}

.slide-enter-from {
  transform: translateX(20px);
  opacity: 0;
}

.slide-leave-to {
  transform: translateX(-20px);
  opacity: 0;
}
</style>
```

### Transition-group для списков

Для анимации элементов списка используйте `<transition-group>`:

```vue
<template>
  <transition-group name="list" tag="ul">
    <li v-for="item in items" :key="item.id" class="list-item">
      {{ item.text }}
    </li>
  </transition-group>
</template>

<style>
.list-item {
  display: inline-block;
  margin-right: 10px;
}

.list-enter-active, .list-leave-active {
  transition: all 0.3s ease;
}

.list-enter-from {
  opacity: 0;
  transform: translateY(-30px);
}

.list-leave-to {
  opacity: 0;
  transform: translateY(30px);
}

.list-move {
  transition: transform 0.3s ease;
}
</style>
```

## Встроенные CSS классы

Vue автоматически добавляет следующие CSS-классы во время переходов:

- `*-enter-from` - начальное состояние перед вхождением
- `*-enter-active` - активное состояние при вхождении
- `*-enter-to` - конечное состояние при вхождении
- `*-leave-from` - начальное состояние перед выходом
- `*-leave-active` - активное состояние при выходе
- `*-leave-to` - конечное состояние при выходе

## Параметры компонента Transition

### name
Определяет имя CSS-классов перехода и события в hooks.

```vue
<transition name="my-transition">
```

### type
Указывает тип CSS-транзиции или анимации, определяющий момент окончания перехода ('transition' или 'animation').

```vue
<transition type="animation">
```

### duration
Определяет продолжительность перехода в миллисекундах.

```vue
<transition :duration="500">
```

### mode
Управляет временем срабатывания переходов при одновременном входе и выходе элементов.

- `out-in` - сначала исчезает старый, затем появляется новый
- `in-out` - сначала появляется новый, затем исчезает старый

```vue
<transition mode="out-in">
```

## JavaScript hooks

Можно использовать JavaScript-события для более сложных анимаций:

```vue
<template>
  <transition
    v-on:before-enter="beforeEnter"
    v-on:enter="enter"
    v-on:leave="leave"
    css="false"
  >
    <div v-if="show" class="animated-element">JS анимация</div>
  </transition>
</template>

<script>
export default {
  methods: {
    beforeEnter(el) {
      el.style.opacity = 0
    },
    enter(el, done) {
      // Используем Velocity.js или другую библиотеку
      Velocity(el, { opacity: 1, fontSize: '1.4em' }, { duration: 300 })
      Velocity(el, { fontSize: '1em' }, { complete: done })
    },
    leave(el, done) {
      Velocity(el, { translateX: '15px', rotateZ: '50deg' }, { duration: 600 })
      Velocity(el, { rotateZ: '100deg' }, { loop: 2 })
      Velocity(el, { rotateZ: '45deg', translateY: '30px', translateX: '30px', opacity: 0 }, { complete: done })
    }
  }
}
</script>
```

## Практические примеры

### Модальное окно с анимацией

```vue
<template>
  <div class="modal-wrapper">
    <transition name="modal">
      <div v-if="isOpen" class="modal-overlay" @click="close">
        <div class="modal-content" @click.stop>
          <h3>Заголовок модального окна</h3>
          <p>Содержимое модального окна</p>
          <button @click="close">Закрыть</button>
        </div>
      </div>
    </transition>
  </div>
</template>

<style>
.modal-enter-active, .modal-leave-active {
  transition: opacity 0.3s ease;
}

.modal-enter-from, .modal-leave-to {
  opacity: 0;
}

.modal-content {
  transition: transform 0.3s ease;
}

.modal-enter-active .modal-content,
.modal-leave-active .modal-content {
  transition: transform 0.3s ease;
}

.modal-enter-from .modal-content {
  transform: scale(0.8);
}

.modal-leave-to .modal-content {
  transform: scale(0.8);
}
</style>
```

## Особенности реализации в российских условиях (2025)

При реализации переходов в российских условиях в 2025 году следует учитывать:

1. **Производительность на слабых устройствах** - не забывать про оптимизацию анимаций для бюджетных смартфонов, популярных в РФ
2. **Доступность (accessibility)** - обеспечивать опциональное отключение анимаций через `prefers-reduced-motion`
3. **Совместимость с отечественными браузерами** - проверять корректную работу в Яндекс.Браузере и КриптоПро Browser

```css
@media (prefers-reduced-motion: reduce) {
  .fade-enter-active,
  .fade-leave-active {
    transition: none;
  }
}
```
