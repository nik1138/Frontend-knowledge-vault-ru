# Библиотеки-переходов

## Обзор

Библиотеки переходов для Vue.js предоставляют готовые решения для создания сложных анимаций и переходов. Они расширяют возможности встроенных переходов Vue и позволяют реализовать продвинутые анимационные эффекты с минимальными усилиями.

## Популярные библиотеки переходов

### 1. Vue 3 Animation Utils (v3)

Современная библиотека анимаций, разработанная специально для Vue 3:

```bash
npm install @vueuse/motion
```

```vue
<template>
  <div>
    <div 
      v-motion 
      :initial="{ opacity: 0, y: 50 }"
      :enter="{ opacity: 1, y: 0, transition: { duration: 800 } }"
      class="motion-element"
    >
      Анимированный элемент с @vueuse/motion
    </div>
    
    <div 
      v-motion-slide-visible 
      :delay="300"
      class="slide-element"
    >
      Слайд-анимация при появлении
    </div>
  </div>
</template>

<script>
import { defineComponent } from 'vue'
import { MotionPlugin } from '@vueuse/motion'

export default defineComponent({
  name: 'MotionExample',
  directives: {
    Motion: MotionPlugin
  }
})
</script>
```

### 2. Vue Draggable (для анимаций перетаскивания)

Хотя в основном используется для drag-and-drop, также предоставляет отличные анимации:

```bash
npm install vuedraggable@next
```

```vue

    <draggable

      }"
    >
      <template #item="{ element }">
        <li class="draggable-item">{{ element.name }}</li>
      </template>
    </draggable>

<script>
import draggable from 'vuedraggable'

export default {
  components: {

        { id: 2, name: 'Элемент 2' },
        { id: 3, name: 'Элемент 3' }
      ]
    }
  }
}
</script>

<style>
```

### 3. Framer Motion для Vue (через обертки)

Хотя Framer Motion изначально для React, существуют Vue-обертки:

```bash
npm install framer-motion-vue
# или использовать альтернативу
npm install vue3-frappe
```

```vue
<template>
  <div class="framer-style-container">
    <Motion
      :initial="{ opacity: 0, scale: 0.5 }"
      :animate="{ opacity: 1, scale: 1 }"
      :transition="{ duration: 0.5 }"
      class="motion-box"
    >
      Framer Motion стиль
    </Motion>
    
    <Motion
      :initial="{ x: -100 }"
      :animate="{ x: 0 }"
      :while-hover="{ scale: 1.1 }"
      :while-tap="{ scale: 0.9 }"
      class="interactive-motion"
    >
      Интерактивная анимация
    </Motion>
  </div>
</template>

<script>
import { Motion } from 'framer-motion-vue'

export default {
  components: {
    Motion
  }
}
</script>
```

### 4. Vue Animate CSS

Интеграция с популярной библиотекой animate.css:

```bash
npm install vue3-animate
```

```vue
<template>
  <div>
    <div 
      v-if="showElement" 
      class="animate__animated animate__bounce"
    >
      Анимированный элемент
    </div>
    
    <div class="animation-controls">
      <select v-model="selectedAnimation">
        <option value="animate__bounce">Прыжок</option>
        <option value="animate__fadeIn">Появление</option>
        <option value="animate__zoomIn">Увеличение</option>
        <option value="animate__rotateIn">Вращение</option>
      </select>
      <button @click="toggleElement">Переключить</button>
    </div>
    
    <div 
      v-if="showElement" 
      :class="['animated-element', selectedAnimation]"
    >
      Элемент с {{ selectedAnimation }}
    </div>
  </div>
</template>

<script>
import 'animate.css'

export default {
  data() {
    return {
      showElement: true,
      selectedAnimation: 'animate__fadeIn'
    }
  },
  methods: {
    toggleElement() {
      this.showElement = !this.showElement
    }
  }
}
</script>
```

### 5. GreenSock (GSAP) с Vue

GSAP - одна из самых мощных библиотек анимации:

```bash
npm install gsap
```

```vue
<template>
  <div>
    <div ref="animatedElement" class="gsap-element">
      GSAP анимированный элемент
    </div>
    
    <div class="gsap-controls">
      <button @click="animateWithGsap">GSAP Анимация</button>
      <button @click="timelineAnimation">Таймлайн анимация</button>
    </div>
  </div>
</template>

<script>
import { gsap } from 'gsap'
import { ScrollTrigger } from 'gsap/ScrollTrigger'

// Регистрация плагинов
gsap.registerPlugin(ScrollTrigger)

export default {
  mounted() {
    // Анимация при скролле
    gsap.from(this.$refs.animatedElement, {
      scrollTrigger: {
        trigger: this.$refs.animatedElement,
        start: "top 80%",
        end: "bottom 20%",
        toggleActions: "play none none reverse"
      },
      opacity: 0,
      y: 50,
      duration: 1,
      ease: "power2.out"
    })
  },
  methods: {
    animateWithGsap() {
      const el = this.$refs.animatedElement
      
      gsap.to(el, {
        x: 100,
        rotation: 360,
        duration: 1,
        ease: "elastic.out(1, 0.3)",
        yoyo: true,
        repeat: 1
      })
    },
    
    timelineAnimation() {
      const tl = gsap.timeline()
      const el = this.$refs.animatedElement
      
      tl.to(el, { 
        x: 100, 
        duration: 0.5 
      })
      .to(el, { 
        rotation: 180, 
        duration: 0.5 
      })
      .to(el, { 
        y: 50, 
        duration: 0.5 
      })
    }
  }
}
</script>
```

### 2. Vue Visibility

Для анимаций при появлении в области видимости:

```bash
npm install vue-visibility
```

```vue
<template>
  <div>
    <div v-visibility="onVisibilityChange" class="visibility-element">
      Элемент, анимируемый при появлении в области видимости
    </div>
    
    <div v-for="item in items" :key="item.id">
      <div 
        v-visibility.once="() => animateItem(item.id)"
        class="list-item"
      >
        {{ item.name }}
      </div>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      items: [
        { id: 1, name: 'Элемент 1' },
        { id: 2, name: 'Элемент 2' },
        { id: 3, name: 'Элемент 3' }
      ]
    }
  },
  methods: {
    onVisibilityChange(isVisible) {
      if (isVisible) {
        // Запускаем анимацию при появлении
        this.animateElement()
      }
    },
    animateItem(id) {
      // Анимируем конкретный элемент при появлении
      const element = document.querySelector(`.list-item:nth-child(${id})`)
      element.style.animation = 'slideIn 0.5s ease-out'
    },
    animateElement() {
      // Логика анимации
    }
  }
}
</script>

<style>
@keyframes slideIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
</style>
```

## Сравнение библиотек

| Библиотека | Преимущества | Недостатки | Использование |
|------------|--------------|------------|----------------|
| @vueuse/motion | Интеграция с Vue, мощные возможности | Новый, может меняться | Сложные интерактивные анимации |
| GSAP | Очень мощная, кроссплатформенная | Большой размер | Сложные анимации, игры |
| Animate.css | Простота, большая коллекция | Ограниченная кастомизация | Быстрые декоративные анимации |
| Vue Draggable | Отличные анимации при перетаскивании | Только для drag-and-drop | Списки с перетаскиванием |
| Framer Motion (обертки) | Современный API, как в React | Меньше возможностей, чем оригинальная | Современные интерфейсы |

## Интеграция с российскими реалиями (2025)

При использовании библиотек переходов в российских условиях в 2025 году:

1. **Производительность** - выбирать библиотеки с учетом производительности на слабых устройствах
2. **Совместимость** - проверять работу в отечественных браузерах (Яндекс.Браузер, КриптоПро Browser)
3. **Размер бандла** - учитывать влияние на размер итогового приложения

```javascript
// Утилита для выбора библиотеки анимации в зависимости от условий
export function getAnimationLibrary() {
  // Проверяем настройки доступности
  if (window.matchMedia('(prefers-reduced-motion: reduce)').matches) {
    return 'minimal' // Минимальные анимации
  }
  
  // Определяем производительность устройства
  const isLowEndDevice = navigator.hardwareConcurrency < 4 || 
                        (navigator.deviceMemory && navigator.deviceMemory < 4)
  
  if (isLowEndDevice) {
    return 'lightweight' // Легковесные анимации
  }
  
  // Проверяем тип браузера
  const userAgent = navigator.userAgent
  if (userAgent.includes('YaBrowser')) {
    return 'yandex-compatible' // Анимации, протестированные в Я.Браузере
  }
  
  return 'full' // Полнофункциональные анимации
}
```

## Лучшие практики

### 1. Условная загрузка библиотек

```javascript
// lazy-load.js
export async function loadAnimationLibrary(libraryName) {
  switch(libraryName) {
    case 'gsap':
      return await import('gsap')
    case 'motion':
      return await import('@vueuse/motion')
    case 'animate':
      // Для CSS-библиотек подключаем стили
      await import('animate.css')
      return {}
    default:
      return {}
  }
}
```

### 2. Адаптивные анимации

```vue
<template>
  <div>
    <transition
      :name="animationName"
      :duration="animationDuration"
    >
      <div v-if="show" class="adaptive-element">
        Адаптивная анимация
      </div>
    </transition>
  </div>
</template>

<script>
import { computed } from 'vue'

export default {
  setup() {
    const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches
    const isLowEndDevice = navigator.hardwareConcurrency < 4
    
    const animationName = computed(() => {
      if (prefersReducedMotion) return 'no-animation'
      if (isLowEndDevice) return 'simple-fade'
      return 'complex-animation'
    })
    
    const animationDuration = computed(() => {
      if (prefersReducedMotion) return 0
      if (isLowEndDevice) return 200
      return 500
    })
    
    return {
      animationName,
      animationDuration
    }
  }
}
</script>

<style>
.no-animation-enter-active, .no-animation-leave-active {
  transition: none;
}
.no-animation-enter-from, .no-animation-leave-to {
  opacity: 1;
}

.simple-fade-enter-active, .simple-fade-leave-active {
  transition: opacity 0.2s ease;
}
.simple-fade-enter-from, .simple-fade-leave-to {
  opacity: 0;
}

.complex-animation-enter-active, .complex-animation-leave-active {
  transition: all 0.5s ease;
}
.complex-animation-enter-from {
  opacity: 0;
  transform: translateY(20px) scale(0.95);
}
.complex-animation-leave-to {
  opacity: 0;
  transform: translateY(-20px) scale(0.95);
}
</style>
```
