# Скоуп-стили (Scoped Styles) в Vue

## Введение

Скоуп-стили (Scoped Styles) - это функциональность Vue, позволяющая ограничить область действия CSS-стилей только внутри одного компонента. Это достигается с помощью автоматической генерации уникальных атрибутов данных для элементов компонента и соответствующего изменения селекторов CSS.

## Основы использования

Для применения скоуп-стилей в компоненте Vue, нужно добавить атрибут `scoped` к тегу `<style>`:

```vue
<template>
  <div class="container">
    <h1>Заголовок компонента</h1>
    <p>Текст параграфа</p>
  </div>
</template>

<script>
export default {
  name: 'MyComponent'
}
</script>

<style scoped>
.container {
  padding: 20px;
  background-color: #f5f5f5;
}

h1 {
  color: #333;
  font-size: 24px;
}

p {
  color: #666;
  line-height: 1.6;
}
</style>
```

При компиляции компонента Vue добавит уникальный атрибут данных к каждому элементу в шаблоне и изменит CSS-селекторы, чтобы они применялись только к элементам с этим атрибутом:

```vue
<template>
  <div class="container" data-v-f3f3eg9>
    <h1 data-v-f3f3eg9>Заголовок компонента</h1>
    <p data-v-f3f3eg9>Текст параграфа</p>
  </div>
</template>

<style>
.container[data-v-f3f3eg9] {
  padding: 20px;
  background-color: #f5f5f5;
}

h1[data-v-f3f3eg9] {
  color: #333;
  font-size: 24px;
}

p[data-v-f3f3eg9] {
  color: #666;
  line-height: 1.6;
}
</style>
```

## Преимущества скоуп-стилей

1. **Изоляция стилей**: Стили компонента не влияют на другие компоненты
2. **Простота использования**: Не требует дополнительных библиотек или настроек
3. **Читаемость**: Легко понять, какие стили относятся к какому компоненту
4. **Поддержка CSS-переменных**: Работает с современными возможностями CSS

## Продвинутые возможности

### Глубокие селекторы (Deep Selectors)

Иногда требуется применить стили к дочерним компонентам. Для этого используются глубокие селекторы:

```vue
<style scoped>
/* Применяет стили к дочерним компонентам */
.parent ::v-deep .child {
  color: red;
}

/* Альтернативный синтаксис */
.parent :deep(.child) {
  color: blue;
}

/* Для псевдоэлементов */
.parent :slotted(.content) {
  padding: 10px;
}
</style>
```

### CSS-переменные в scoped стилях

Можно использовать CSS-переменные в скоуп-стилях:

```vue
<template>
  <div class="card">
    <h3>{{ title }}</h3>
    <p>{{ content }}</p>
  </div>
</template>

<script>
export default {
  name: 'CardComponent',
  props: {
    title: String,
    content: String
  }
}
</script>

<style scoped>
.card {
  border: 1px solid var(--border-color);
  border-radius: 8px;
  padding: var(--card-padding, 16px);
  background-color: var(--bg-color, white);
}

.card h3 {
  color: var(--title-color, #333);
  margin-bottom: 8px;
}

.card p {
  color: var(--text-color, #666);
  line-height: 1.5;
}
</style>
```

## Практические рекомендации

### 1. Комбинирование scoped и глобальных стилей

В проектах на Vue часто используются как scoped, так и глобальные стили:

```vue
<!-- Компонент MyComponent.vue -->
<style scoped>
/* Стили только для этого компонента */
.component-specific {
  margin: 10px;
}
</style>

<style>
/* Глобальные стили, применяемые ко всему приложению */
.global-button {
  padding: 10px 15px;
  border: none;
  border-radius: 4px;
}
</style>
```

### 2. Использование препроцессоров с scoped стилями

Скоуп-стили работают с препроцессорами CSS, такими как Sass, Less и Stylus:

```vue
<style lang="scss" scoped>
.container {
  $primary-color: #42b983;
  $secondary-color: #f5f5f5;
  
  background-color: $secondary-color;
  
  .title {
    color: $primary-color;
    font-weight: bold;
    
    &:hover {
      color: darken($primary-color, 10%);
    }
  }
}
</style>
```

### 3. Работа с динамическими стилями

Скоуп-стили можно комбинировать с динамическими стилями:

```vue
<template>
  <div 
    class="box" 
    :style="{ backgroundColor: backgroundColor, width: width + 'px' }"
  >
    Динамический блок
  </div>
</template>

<script>
export default {
  name: 'DynamicBox',
  data() {
    return {
      backgroundColor: '#42b983',
      width: 200
    }
  }
}
</script>

<style scoped>
.box {
  height: 100px;
  transition: all 0.3s ease;
  border-radius: 8px;
}
</style>
```

## Лучшие практики

1. **Минимизировать использование глубоких селекторов** - они нарушают принцип изоляции компонентов
2. **Использовать семантические имена классов** - улучшает читаемость кода
3. **Разделять стили по компонентам** - облегчает поддержку кода
4. **Использовать CSS-переменные для темизации** - позволяет легко изменять внешний вид компонентов

## Совместимость с инструментами

Скоуп-стили поддерживаются всеми основными инструментами сборки Vue:

- Vue CLI
- Vite
- Nuxt.js
- Webpack с vue-loader

## Заключение

Скоуп-стили - мощный инструмент для создания изолированных компонентов в Vue. Они позволяют избежать конфликта имен CSS и упрощают поддержку кода. При правильном использовании scoped стили значительно улучшают разработку компонентов в больших приложениях.
