# Курсы по Vue.js

## Обзор онлайн-курсов для изучения Vue.js в 2025 году

### Платформы с курсами по Vue.js

#### Stepik
- Курс "Vue.js для начинающих"
- Бесплатный доступ с возможностью сертификации
- Упражнения и практические задания
- Переводы официальной документации

#### Coursera
- Курс "Vue.js: Frontend Framework с нуля до приложения"
- Преподаватели из российских вузов
- Сертификация при оплате
- Структурированное обучение

#### Udemy
- Курс "Vue - Полный курс по Vue.js для начинающих" (на русском)
- Регулярные скидки и акции
- Практические проекты
- Поддержка сообщества

#### Udemy (зарубежные курсы)
- "Vue - The Complete Guide by Maximilian Schwarzmüller"
- Обновленный курс с Vue 3 и Composition API
- Проекты и практические задания
- Поддержка сообщества

#### Hexlet
- Модуль "Vue.js" в программе "Фронтенд-разработчик"
- Практические задания с проверкой
- Русскоязычные преподаватели
- Интеграция с другими фронтенд-библиотеками

### Российские платформы

#### LoftBlog
- Курс "Vue.js с нуля"
- Подробное изучение фреймворка
- Практические задания
- Видеоуроки на русском языке

#### WebDesign Master
- Курс "Vue.js и Vuex"
- Практические проекты
- Поддержка наставников
- Сертификация по окончании

#### Skillbox
- Профессия "Фронтенд-разработчик"
- Включает модуль по Vue.js
- Практические проекты
- Обратная связь от наставников

#### Netology
- Курс "Vue.js для разработчиков"
- Индивидуальные задания
- Онлайн-встречи с преподавателями
- Проектная работа

- Продвинутые темы платно
- Хорошая структура обучения

- Практические примеры

- Тестирование

- Динамические маршруты
- Защита маршрутов
- Анимации переходов

#### Nuxt.js
- Курс по серверному рендерингу
- Статические сайты

- Практические проекты
