# Directives

Директивы - это специальные атрибуты в шаблонах Vue, которые применяют преобразования к DOM. Встроенные и пользовательские директивы позволяют манипулировать элементами DOM в ответ на изменения данных.

## Жизненный цикл директивы

Пользовательские директивы имеют следующие хуки:

- `created` - вызывается до связывания элемента с узлом
- `beforeMount` - вызывается перед монтированием директивы
- `mounted` - вызывается после монтирования элемента
- `beforeUpdate` - вызывается перед обновлением элемента
- `updated` - вызывается после обновления элемента
- `beforeUnmount` - вызывается перед размонтированием элемента
- `unmounted` - вызывается после размонтирования элемента

## Пример пользовательской директивы

```javascript
// Глобальная регистрация
app.directive('focus', {
  mounted(el) {
    el.focus()
  }
})

// Использование
<template>
  <input v-focus />
</template>
```
