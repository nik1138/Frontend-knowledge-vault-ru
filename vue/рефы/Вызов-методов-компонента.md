# Вызов-методов-компонента

## Обзор

Вызов методов дочернего компонента из родительского осуществляется через component refs и директиву `ref`. Это позволяет родительскому компоненту взаимодействовать с внутренним состоянием и методами дочернего компонента.

## Использование с Composition API

### Родительский компонент

```vue
<template>
  <ChildComponent ref="child" />
  <button @click="callChildMethod">Вызвать метод дочернего</button>
</template>

<script setup>
import { ref } from 'vue'
import ChildComponent from './ChildComponent.vue'

const child = ref(null)

const callChildMethod = () => {
  child.value.childMethod()
}
</script>
```

### Дочерний компонент

```vue
<script setup>
const childMethod = () => {
  console.log('Метод дочернего компонента вызван!')
}

defineExpose({
  childMethod
})
</script>
```

## Использование с Options API

### Родительский компонент

```vue
<template>
  <ChildComponent ref="child" />
  <button @click="callChildMethod">Вызвать метод дочернего</button>
</template>

<script>
import ChildComponent from './ChildComponent.vue'

export default {
  components: {
    ChildComponent
  },
  methods: {
    callChildMethod() {
      this.$refs.child.childMethod()
    }
  }
}
</script>
```

### Дочерний компонент (Options API)

```vue
<script>
export default {
  methods: {
    childMethod() {
      console.log('Метод дочернего компонента вызван!')
    }
  }
}
</script>
```

## Практические рекомендации

- Используйте `defineExpose` в Composition API для явного экспорта методов
- В Options API все публичные методы доступны через ref
- Рассмотрите альтернативы (props, emit) для передачи данных между компонентами
- Избегайте чрезмерного использования refs для взаимодействия между компонентами
