# HTML Элементы

HTML элементы - это основные строительные блоки веб-страниц. Каждый элемент представляет собой определённую часть контента и может содержать атрибуты, которые изменяют его поведение или внешний вид.

## Определение HTML Элементов

HTML элемент состоит из открывающего тега, содержимого и закрывающего тега. Например, `<p>Это абзац</p>` - элемент абзаца. Некоторые элементы, такие как `<img>` или `<br>`, являются одиночными и не требуют закрывающего тега.

HTML элементы определяют структуру документа и семантику содержимого. Они могут быть вложенными друг в друга, образуя иерархическую структуру страницы.

## Типы Элементов

### Блочные Элементы

Блочные элементы занимают всю доступную ширину родительского контейнера и начинаются с новой строки. Примеры:

```html
<div>Блочный элемент</div>
<p>Абзац текста</p>
<h1>Заголовок первого уровня</h1>
<section>Семантическая секция</section>
```

К основным блочным элементам относятся: `div`, `p`, `h1`-`h6`, `section`, `article`, `header`, `footer`, `nav`, `main`, `aside`, `ul`, `ol`, `li`, `table`, `form`, и другие.

### Строчные Элементы

Строчные элементы занимают только необходимое пространство и не начинаются с новой строки. Они находятся в потоке текста:

```html
<span>Текстовый элемент</span>
<a href="#">Ссылка</a>
<strong>Жирный текст</strong>
<em>Курсивный текст</em>
<code>Код</code>
<img src="image.jpg" alt="Изображение">
```

Примеры строчных элементов: `span`, `a`, `strong`, `em`, `code`, `img`, `input`, `label`, `select`, `button`.

### Интерактивные Элементы

Интерактивные элементы позволяют пользователю взаимодействовать со страницей:

```html
<button>Кнопка</button>
<a href="https://example.com">Ссылка</a>
<form>
  <input type="text" placeholder="Текстовое поле">
  <textarea placeholder="Текстовая область"></textarea>
  <select>
    <option>Выбор</option>
  </select>
</form>
```

К ним относятся: `a`, `button`, `input`, `textarea`, `select`, `option`, `form`, `label`, `details`, `summary`, `menu`, `menuitem`.

## Атрибуты Элементов

Атрибуты предоставляют дополнительную информацию об элементе и всегда указываются в открывающем теге:

```html
<!-- Общие атрибуты -->
<div id="main-content" class="container" title="Основной контент">
  <img src="image.jpg" alt="Описание изображения" width="300" height="200">
  <a href="https://example.com" target="_blank" rel="noopener">Ссылка</a>
</div>

<!-- Атрибуты специфичные для элементов -->
<input type="email" required placeholder="email@example.com">
<form method="POST" action="/submit">
  <button type="submit">Отправить</button>
</form>
```

Общие атрибуты включают:
- `id` - уникальный идентификатор элемента
- `class` - имя CSS класса
- `title` - подсказка при наведении
- `style` - встроенные стили CSS
- `data-*` - пользовательские атрибуты данных

## Семантика Элементов

Семантические элементы описывают значение содержимого для браузеров и поисковых систем:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Семантическая разметка</title>
</head>
<body>
  <header>
    <h1>Заголовок сайта</h1>
    <nav>
      <ul>
        <li><a href="/">Главная</a></li>
        <li><a href="/about">О нас</a></li>
      </ul>
    </nav>
  </header>
  
  <main>
    <article>
      <header>
        <h2>Заголовок статьи</h2>
        <time datetime="2023-11-09">9 ноября 2023</time>
      </header>
      <section>
        <h3>Подзаголовок раздела</h3>
        <p>Содержимое статьи...</p>
      </section>
      <footer>
        <p>Автор: Имя Фамилия</p>
      </footer>
    </article>
    
    <aside>
      <h3>Боковая панель</h3>
      <p>Дополнительная информация</p>
    </aside>
  </main>
  
  <footer>
    <p>&copy; 2023 Все права защищены</p>
  </footer>
</body>
</html>
```

Семантические элементы улучшают доступность, SEO и структуру документа.

## Разница между Элементами

Различия между элементами определяются их семантикой, поведением и стилями по умолчанию:

- `div` и `span` - универсальные элементы без семантики
- `section` и `article` - семантические разделы с разными значениями
- `strong` и `b` - оба выделяют текст, но `strong` указывает на важность
- `em` и `i` - оба создают курсив, но `em` указывает на акцент

## Валидация Элементов

HTML документ должен соответствовать стандартам валидации. Валидный HTML:

- Правильно вложен (открывающие и закрывающие теги соответствуют друг другу)
- Содержит корректные атрибуты
- Использует допустимые элементы в правильных контекстах

Используйте [HTML валидатор](https://validator.w3.org/) для проверки документов.
