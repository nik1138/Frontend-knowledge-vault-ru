# Документация по HTML

## Общие сведения

HTML (HyperText Markup Language) — это стандартный язык разметки для создания веб-страниц. Документация по HTML является ключевым ресурсом для разработчиков, позволяющим понимать синтаксис, теги, атрибуты и семантическую структуру веб-страниц.

## Основные ресурсы

### MDN Web Docs (Mozilla Developer Network)
- **Описание:** Самый авторитетный и полный ресурс по HTML, CSS, JavaScript и другим веб-технологиям. Документация регулярно обновляется и содержит примеры кода.
- **Ссылка:** [https://developer.mozilla.org/ru/docs/Web/HTML](https://developer.mozilla.org/ru/docs/Web/HTML)
- **Особенности:** Подробные описания элементов, атрибутов, событий. Руководства по семантике, доступности, безопасности.

### W3C Specifications
- **Описание:** Официальные спецификации HTML, определяющие стандарты. Полезно для понимания "железных" основ, но может быть сложным для новичков.
- **Ссылка:** [https://html.spec.whatwg.org/](https://html.spec.whatwg.org/)
- **Особенности:** Актуальные спецификации HTML Living Standard.

### HTML5.ru
- **Описание:** Российский ресурс, посвященный HTML5 и современным веб-технологиям. Содержит статьи, туториалы и новости.
- **Ссылка:** [https://html5.ru/](https://html5.ru/)
- **Особенности:** Контент на русском языке, адаптированный под российскую аудиторию.

## Практические рекомендации

- **Используйте MDN как основной справочник:** MDN Web Docs — ваш лучший друг при изучении и работе с HTML. [[Справочники]]
- **Обратите внимание на семантику:** Использование правильных HTML-тегов (header, nav, main, article, section и т.д.) улучшает доступность и SEO.
- **Изучайте примеры:** MDN содержит множество примеров кода, которые можно копировать и тестировать в локальной среде или онлайн-редакторах. [[Онлайн-редакторы]]
