# Справочники по HTML

## Общие сведения

Справочники по HTML предоставляют структурированную информацию об HTML-элементах, их атрибутах, значениях атрибутов и примерах использования. Они служат быстрым способом найти нужную информацию без необходимости изучать полную документацию.

## Популярные справочники

### MDN Web Docs - Справочник по элементам
- **Описание:** Содержит полный список всех HTML-элементов с описанием, атрибутами, примерами и совместимостью с браузерами.
- **Ссылка:** [https://developer.mozilla.org/ru/docs/Web/HTML/Element](https://developer.mozilla.org/ru/docs/Web/HTML/Element)
- **Особенности:** Отличная навигация, фильтрация по категориям (структурные, текстовые, встроенные и т.д.).

### HTML Reference от HTML5 Doctor
- **Описание:** Визуально приятный и интуитивно понятный справочник по HTML-элементам. Показывает, как элементы выглядят по умолчанию.
- **Ссылка:** [https://htmlreference.io/](https://htmlreference.io/)
- **Особенности:** Примеры визуального отображения элементов.

### HTML Attribute Reference
- **Описание:** Справочник по атрибутам HTML, как глобальным, так и специфичным для каждого элемента.
- **Ссылка:** [https://developer.mozilla.org/ru/docs/Web/HTML/Attributes](https://developer.mozilla.org/ru/docs/Web/HTML/Attributes)
- **Особенности:** Подробное описание каждого атрибута и его применения.

### HTML.global - справочник на русском языке
- **Описание:** Российский ресурс, содержащий информацию об HTML-тегах и их атрибутах на русском языке.
- **Ссылка:** [https://html.global/](https://html.global/)
- **Особенности:** Локализованный контент, адаптированный под российских разработчиков.

## Практические рекомендации

- **Используйте MDN для глубокого понимания:** MDN не только перечисляет элементы, но и объясняет, *когда* и *почему* их использовать. [[Документация]]
- **Запомните семантические элементы:** Элементы вроде `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>` делают HTML-разметку более осмысленной. [[HTML]]
- **Обращайте внимание на глобальные атрибуты:** Такие атрибуты как `id`, `class`, `data-*`, `style`, `title` доступны для большинства элементов и часто используются.
