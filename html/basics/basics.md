# Основы HTML

HTML (HyperText Markup Language) — это стандартный язык разметки для создания веб-страниц. HTML описывает структуру веб-страницы с помощью тегов, которые интерпретируются браузерами для отображения контента.

## Структура HTML документа

Каждый HTML документ начинается с объявления типа документа `<!DOCTYPE html>` и содержит основные элементы: `html`, `head`, и `body`.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Добро пожаловать на сайт</h1>
    <p>Это пример HTML документа.</p>
</body>
</html>
```

### Объяснение элементов:
- `<!DOCTYPE html>` — объявление типа документа
- `<html>` — корневой элемент всей HTML страницы
- `<head>` — содержит метаданные (например, заголовок, кодировку)
- `<body>` — содержит видимый контент страницы
- `<meta charset="UTF-8">` — определяет кодировку символов
- `<title>` — заголовок страницы, отображаемый во вкладке браузера

## Теги и атрибуты

HTML документ состоит из **элементов**, которые создаются с помощью **тегов**. Теги окружают содержимое и определяют его тип. Атрибуты предоставляют дополнительную информацию об элементе.

```html
<!-- Пример элемента с атрибутами -->
<a href="https://example.com" target="_blank" title="Пример ссылки">Нажмите здесь</a>
```

- `href` — указывает адрес ссылки
- `target` — определяет, где открыть ссылку
- `title` — всплывающая подсказка при наведении

> [!tip] Совет
> Все теги чувствительны к регистру — используйте строчные буквы для согласованности с современными стандартами.

## Основные элементы HTML

| Тег | Описание |
|-----|----------|
| `<h1>` до `<h6>` | Заголовки различного уровня |
| `<p>` | Абзац текста |
| `<a>` | Гиперссылка |
| `<img>` | Вставка изображения |
| `<ul>`, `<ol>`, `<li>` | Списки (ненумерованные, нумерованные, элементы списка) |
| `<div>` | Блочный контейнер |
| `<span>` | Строчный контейнер |
| `<strong>` | Жирный текст (семантически важный) |
| `<em>` | Курсивный текст (акцент) |

## Семантические элементы

Семантические теги описывают значение содержимого. Они улучшают доступность и SEO:

- `<header>` — заголовок страницы или раздела
- `<nav>` — навигационные ссылки
- `<main>` — основное содержимое
- `<section>` — логическая секция
- `<article>` — независимый контент
- `<aside>` — боковая панель
- `<footer>` — нижний колонтитул

```html
<header>
    <h1>Мой сайт</h1>
    <nav>
        <ul>
            <li><a href="#home">Главная</a></li>
            <li><a href="#about">О нас</a></li>
        </ul>
    </nav>
</header>
```

## Заголовки и навигация

Заголовки (`<h1>`-`<h6>`) помогают структурировать контент. Используйте их иерархически: один `<h1>` на страницу, затем `<h2>`, `<h3>` и т.д.

Навигация создается с помощью тега `<nav>` и списков:

```html
<nav>
    <ul>
        <li><a href="/">Главная</a></li>
        <li><a href="/about">О нас</a></li>
        <li><a href="/contact">Контакты</a></li>
    </ul>
</nav>
```

> [!warning] Важно
> Избегайте пропусков в иерархии заголовков (например, перехода от `<h2>` к `<h4>` без `<h3>`).

## Таблицы стилей

HTML сам по себе не отвечает за стилизацию — для этого используется CSS. Связь между HTML и CSS осуществляется через:

1. **Встроенные стили** (атрибут `style`)
2. **Внутренние стили** (тег `<style>` в `<head>`)
3. **Внешние стили** (ссылка на CSS файл через `<link>`)

```html
<head>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <p style="color: blue;">Синий текст</p>
    <p class="highlight">Выделенный текст</p>
</body>
```

См. также: [[CSS Основы]]

## Синтаксис HTML

- Элементы могут быть парными (`<p>...</p>`) или одиночными (`<img>`, `<br>`)
- Атрибуты записываются в виде `name="value"` в открывающем теге
- Значения атрибутов всегда заключаются в кавычки
- HTML игнорирует лишние пробелы и переносы строк

## Лучшие практики

1. **Используйте семантические теги** для улучшения доступности
2. **Проверяйте валидность** HTML кода через [W3C Validator](https://validator.w3.org/)
3. **Добавляйте атрибут `alt`** к изображениям для доступности
4. **Организуйте код** с отступами для читаемости
5. **Используйте `lang` атрибут** в теге `<html>` для указания языка

```html
<img src="image.jpg" alt="Описание изображения">
```

## Примеры кода

Простой HTML документ с основными элементами:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пример страницы</title>
</head>
<body>
    <header>
        <h1>Заголовок страницы</h1>
        <nav>
            <ul>
                <li><a href="/">Главная</a></li>
                <li><a href="/about">О нас</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>Статья</h2>
            <p>Это пример статьи.</p>
        </article>
        
        <aside>
            <h3>Боковая панель</h3>
            <p>Дополнительная информация</p>
        </aside>
    </main>
    
    <footer>
        <p>&copy; 2025 Мой сайт</p>
    </footer>
</body>
</html>
```

## Связи с другими HTML файлами

HTML документы могут ссылаться друг на друга через гиперссылки:

```html
<a href="page2.html">Перейти на вторую страницу</a>
```

Также можно включать внешние ресурсы:

```html
<link rel="stylesheet" href="styles.css">
<script src="script.js"></script>
```

Практикуйтесь в написании HTML кода, следуйте лучшим практикам и используйте современные семантические теги для создания доступного и SEO-оптимизированного контента.
