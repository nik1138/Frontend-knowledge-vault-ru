# Структура HTML-таблиц

## Введение

HTML-таблицы - это важный элемент веб-разработки, который позволяет структурировать и представлять табличные данные в понятном для пользователя формате. Правильная структура таблицы важна не только для визуального отображения, но и для доступности, SEO и адаптивности.

## Основные элементы таблицы

### Основная структура

HTML-таблица создается с помощью элемента `<table>`, внутри которого размещаются другие элементы:

```html
<table>
  <caption>Заголовок таблицы</caption>
  <thead>
    <tr>
      <th>Заголовок столбца 1</th>
      <th>Заголовок столбца 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Данные строки 1, столбец 1</td>
      <td>Данные строки 1, столбец 2</td>
    </tr>
    <tr>
      <td>Данные строки 2, столбец 1</td>
      <td>Данные строки 2, столбец 2</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Итоги</td>
      <td>Итоги</td>
    </tr>
  </tfoot>
</table>
```

### Элементы таблицы

- `<table>` - контейнер для всей таблицы
- `<caption>` - заголовок таблицы
- `<thead>` - группировка заголовков таблицы
- `<tbody>` - группировка основного содержимого таблицы
- `<tfoot>` - группировка итогов таблицы
- `<tr>` - строка таблицы
- `<th>` - ячейка заголовка
- `<td>` - ячейка данных

## Семантическая структура

### Заголовки таблицы

Элемент `<caption>` предоставляет заголовок для всей таблицы и должен быть первым дочерним элементом `<table>`. Он улучшает доступность и помогает пользователям понять назначение таблицы.

```html
<table>
  <caption>Ежемесячные продажи за 2025 год</caption>
  <!-- остальная структура таблицы -->
</table>
```

### Группировка строк

Элементы `<thead>`, `<tbody>` и `<tfoot>` позволяют логически группировать строки таблицы:

- `<thead>` - содержит заголовки столбцов
- `<tbody>` - содержит основные данные
- `<tfoot>` - содержит итоговую информацию

> [!tip] 
> Элемент `<tfoot>` может быть указан перед `<tbody>` в HTML, но отображается внизу таблицы. Это позволяет браузеру отображать итоги сразу, даже если основное содержимое еще загружается.

## Атрибуты таблицы

### Основные атрибуты

- `summary` - описывает структуру таблицы (устаревший, но может использоваться для совместимости)
- `border` - устанавливает границу таблицы (не рекомендуется использовать, стилизация через CSS)

### Атрибуты ячеек

- `colspan` - объединение ячеек по горизонтали
- `rowspan` - объединение ячеек по вертикали
- `headers` - указывает связь ячейки с заголовками

```html
<td colspan="2">Ячейка, объединяющая 2 столбца</td>
<td rowspan="3">Ячейка, объединяющая 3 строки</td>
<th id="header1">Заголовок</th>
<td headers="header1">Связанная ячейка</td>
```

## Лучшие практики структурирования

### Использование семантических элементов

Всегда используйте семантически правильные элементы для структурирования таблиц:

```html
<table>
  <caption>Отчет по продажам</caption>
  <thead>
    <tr>
      <th scope="col">Месяц</th>
      <th scope="col">Продажи</th>
      <th scope="col">Рост</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Январь</th>
      <td>100 000 ₽</td>
      <td>+5%</td>
    </tr>
    <tr>
      <th scope="row">Февраль</th>
      <td>120 000 ₽</td>
      <td>+10%</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th scope="row">Итого</th>
      <td>220 000 ₽</td>
      <td>+7.5%</td>
    </tr>
  </tfoot>
</table>
```

### Атрибут scope для заголовков

Атрибут `scope` помогает программам чтения с экрана определить, к каким данным относится заголовок:

- `scope="col"` - заголовок для столбца
- `scope="row"` - заголовок для строки
- `scope="colgroup"` - заголовок для группы столбцов
- `scope="rowgroup"` - заголовок для группы строк

```html
      <th scope="row">Q2</th>

      <td>650 000 ₽</td>
      <td>450 000 ₽</td>
    </tr>
  </tfoot>
</table>
```
