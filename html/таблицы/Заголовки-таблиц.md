# Заголовки HTML-таблиц

## Введение

Заголовки HTML-таблиц играют ключевую роль в структурировании данных и обеспечении доступности. Правильное использование заголовков позволяет пользователям и вспомогательным технологиям понимать структуру таблицы и связи между данными. В 2025 году требования к доступности таблиц становятся все более строгими, что делает правильное оформление заголовков особенно важным.

## Основные элементы заголовков

### Элемент caption

Элемент `<caption>` предоставляет заголовок для всей таблицы и должен быть первым дочерним элементом `<table>`:

```html
<table>
  <caption>Ежемесячные продажи по регионам в 2025 году</caption>
  <thead>
    <tr>
      <th>Регион</th>
      <th>Январь</th>
      <th>Февраль</th>
      <th>Март</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Москва</td>
      <td>1 200 000 ₽</td>
      <td>1 350 000 ₽</td>
      <td>1 420 000 ₽</td>
    </tr>
  </tbody>
</table>
```

> [!tip]
> Элемент `<caption>` не является обязательным, но его использование улучшает доступность и помогает пользователям быстро понять назначение таблицы.

### Элемент thead

Элемент `<thead>` группирует заголовки таблицы и позволяет браузеру выделять их при печати или прокрутке:

```html
<table>
  <caption>Финансовый отчет за 2025 год</caption>
  <thead>
    <tr>
      <th scope="col">Показатель</th>
      <th scope="col">Q1</th>
      <th scope="col">Q2</th>
      <th scope="col">Q3</th>
      <th scope="col">Q4</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Доход</th>
      <td>5 000 000 ₽</td>
      <td>5 500 000 ₽</td>
      <td>6 000 000 ₽</td>
      <td>6 500 000 ₽</td>
    </tr>
  </tbody>
</table>
```

## Атрибут scope для заголовков

Атрибут `scope` указывает, к каким данным применяется заголовок:

- `scope="col"` - заголовок применяется ко всему столбцу
- `scope="row"` - заголовок применяется ко всей строке
- `scope="colgroup"` - заголовок применяется к группе столбцов
- `scope="rowgroup"` - заголовок применяется к группе строк

```html
<table>
  <thead>
    <tr>
      <th scope="col">Продукт</th>
      <th scope="col">Январь</th>
      <th scope="col">Февраль</th>
      <th scope="col">Март</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Товар A</th>
      <td>100 000 ₽</td>
      <td>120 000 ₽</td>
      <td>110 000 ₽</td>
    </tr>
    <tr>
      <th scope="row">Товар B</th>
      <td>80 000 ₽</td>
      <td>90 000 ₽</td>
      <td>95 000 ₽</td>
    </tr>
  </tbody>
</table>
```

## Сложные заголовки

### Заголовки с объединением ячеек

Для сложных таблиц могут использоваться заголовки, охватывающие несколько столбцов или строк:

```html
<table>
  <caption>Финансовый отчет по подразделениям за 2025 год</caption>
  <thead>
    <tr>
      <th rowspan="2" scope="col">Подразделение</th>
      <th colspan="3" scope="colgroup">Доходы</th>
      <th colspan="3" scope="colgroup">Расходы</th>
    </tr>
    <tr>
      <th scope="col">Q1</th>
      <th scope="col">Q2</th>
      <th scope="col">Q3</th>
      <th scope="col">Q1</th>
      <th scope="col">Q2</th>
      <th scope="col">Q3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">IT-отдел</th>
      <td>500 000 ₽</td>
      <td>550 000 ₽</td>
      <td>600 000 ₽</td>
      <td>300 000 ₽</td>
      <td>320 000 ₽</td>
      <td>340 000 ₽</td>
    </tr>
  </tbody>
</table>
```

### Заголовки с атрибутами headers и id

Для очень сложных таблиц можно использовать атрибуты `headers` и `id`:

```html
<table>
  <thead>
    <tr>
      <th id="region">Регион</th>
      <th id="jan24">Янв 2024</th>
      <th id="jan25">Янв 2025</th>
      <th id="feb24">Фев 2024</th>
      <th id="feb25">Фев 2025</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th id="moscow" headers="region">Москва</th>
      <td headers="moscow jan24">800 000 ₽</td>
      <td headers="moscow jan25">900 000 ₽</td>
      <td headers="moscow feb24">850 000 ₽</td>
      <td headers="moscow feb25">950 000 ₽</td>
    </tr>
  </tbody>
</table>
```

## Семантическая структура заголовков

### Использование элементов th и td

Заголовки всегда должны быть оформлены с помощью элемента `<th>`, а не `<td>`:

```html
<!-- НЕПРАВИЛЬНО -->
<table>
  <tr>
    <td>Регион</td>
    <td>Продажи</td>
  </tr>
</table>

<!-- ПРАВИЛЬНО -->
<table>
  <tr>
    <th scope="col">Регион</th>
    <th scope="col">Продажи</th>
  </tr>
</table>
```

### Группировка заголовков

Используйте элементы `<thead>`, `<tbody>` и `<tfoot>` для логической группировки:

```html
<table>
  <thead>
    <tr>
      <th scope="col">Месяц</th>
      <th scope="col">План</th>
      <th scope="col">Факт</th>
      <th scope="col">Отклонение</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Январь</th>
      <td>1 000 000 ₽</td>
      <td>1 200 000 ₽</td>
      <td>+20%</td>
    </tr>
    <tr>
      <th scope="row">Февраль</th>
      <td>1 000 000 ₽</td>
      <td>950 000 ₽</td>
      <td>-5%</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th scope="row">Итого</th>
      <td>2 000 000 ₽</td>
      <td>2 150 000 ₽</td>
      <td>+7.5%</td>
    </tr>
  </tfoot>
</table>
```

## Доступность заголовков

### ARIA-атрибуты для заголовков

Для дополнительной ясности можно использовать ARIA-атрибуты:

```html
<table role="table" aria-label="Финансовый отчет за 2025 год по подразделениям">
  <thead>
    <tr>
      <th id="dept" scope="col">Подразделение</th>
      <th id="q1" colspan="2" scope="colgroup">Q1</th>
      <th id="q2" colspan="2" scope="colgroup">Q2</th>
    </tr>
    <tr>
      <th></th>
      <th id="q1-rev" headers="q1" scope="col">Доход</th>
      <th id="q1-exp" headers="q1" scope="col">Расход</th>
      <th id="q2-rev" headers="q2" scope="col">Доход</th>
      <th id="q2-exp" headers="q2" scope="col">Расход</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th id="it" headers="dept" scope="row">IT-отдел</th>
      <td headers="it q1-rev">500 000 ₽</td>
      <td headers="it q1-exp">300 000 ₽</td>
      <td headers="it q2-rev">550 000 ₽</td>
      <td headers="it q2-exp">320 000 ₽</td>
    </tr>
  </tbody>
</table>
```

### Стилизация заголовков

CSS для визуального выделения заголовков:

```css
table th {
  background-color: #f8f9fa;
  font-weight: 600;
  text-align: left;
  padding: 12px 15px;
  border-bottom: 2px solid #dee2e6;
}

table th[scope="row"] {
  background-color: #e9ecef;
  font-weight: 500;
}

table thead th {
  position: sticky;
  top: 0;
  z-index: 10;
  background-color: #f8f9fa;
}
```

## Заголовки в адаптивных таблицах

Для адаптивных таблиц особенно важно правильно обозначать заголовки:

```html
<table class="responsive-table">
  <thead>
    <tr>
      <th>Продукт</th>
      <th>Цена</th>
      <th>Количество</th>
      <th>Сумма</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-label="Продукт">Товар 1</td>
      <td data-label="Цена">1 200 ₽</td>
      <td data-label="Количество">5</td>
      <td data-label="Сумма">6 000 ₽</td>
    </tr>
  </tbody>
</table>
```

## Заключение

Правильное использование заголовков HTML-таблиц в 2025 году требует:
- Применения семантически правильных элементов (`<th>`, `<thead>`, `<caption>`)
- Использования атрибута `scope` для указания области применения заголовка
- Обеспечения доступности для пользователей с вспомогательными технологиями
- Тестирования на различных устройствах и с программами чтения с экрана
