# Стилизация HTML-таблиц

## Введение

Стилизация HTML-таблиц с помощью CSS - важный аспект веб-разработки, который влияет на восприятие данных пользователями. Правильно оформленные таблицы улучшают читаемость информации, повышают пользовательский опыт и обеспечивают соответствие современным веб-стандартам в 2025 году.

## Основные свойства CSS для таблиц

### Границы таблицы

#### Свойство border-collapse

```css
table {
  border-collapse: collapse; /* Объединение границ */
}

table {
  border-collapse: separate; /* Раздельные границы */
}
```

`border-collapse: collapse` - это наиболее часто используемый вариант, который устраняет зазоры между границами ячеек и создает более чистый вид.

#### Свойство border-spacing

При использовании `border-collapse: separate` можно контролировать расстояние между границами:

```css
table {
  border-collapse: separate;
  border-spacing: 5px 10px; /* горизонтальный и вертикальный отступ */
}
```

### Свойства границ

```css
table {
  border: 2px solid #333;
}

th, td {
  border: 1px solid #ddd;
  padding: 8px 12px;
}
```

## Современные подходы к стилизации

### Использование CSS-переменных

```css
:root {
  --table-border-color: #e0e0e0;
  --table-header-bg: #f5f5f5;
  --table-row-hover: #f9f9f9;
  --table-text-color: #333;
}

table {
  border: 1px solid var(--table-border-color);
  color: var(--table-text-color);
}

th {
  background-color: var(--table-header-bg);
}
```

### Flexbox и Grid для адаптивных таблиц

В 2025 году все чаще используется альтернативное представление таблиц на мобильных устройствах:

```css
/* Таблица как контейнер flexbox для мобильных */
@media (max-width: 768px) {
  table, thead, tbody, th, td, tr {
    display: block;
  }
  
  thead tr {
    position: absolute;
    top: -9999px;
    left: -9999px;
  }
  
  tr {
    border: 1px solid #ccc;
    margin-bottom: 10px;
    padding: 10px;
  }
  
  tr td {
    border: none;
    position: relative;
    padding-left: 50%;
    text-align: left;
  }
  
  tr td:before {
    content: attr(data-label) ": ";
    position: absolute;
    left: 6px;
    width: 45%;
    text-align: left;
    font-weight: bold;
  }
}
```

## Визуальные улучшения

### Зебра-эффект (чередование строк)

```css
tbody tr:nth-child(even) {
  background-color: #f8f9fa;
}

tbody tr:hover {
  background-color: #e9ecef;
  transition: background-color 0.2s ease;
}
```

### Выделение заголовков

```css
th {
  background: linear-gradient(180deg, #f8f9fa, #e9ecef);
  font-weight: 600;
  text-align: left;
  padding: 12px 15px;
  border-bottom: 2px solid #dee2e6;
}

th:first-child {
  border-top-left-radius: 4px;
}

th:last-child {
  border-top-right-radius: 4px;
}
```

### Стилизация ячеек

```css
td {
  padding: 10px 15px;
  vertical-align: top;
  border-bottom: 1px solid #dee2e6;
}

td:first-child {
  font-weight: 500;
}

td:last-child {
  border-right: none;
}
```

## Адаптивная стилизация

### Фиксированная ширина столбцов

```css
table {
  table-layout: fixed;
  width: 100%;
}

th:nth-child(1), td:nth-child(1) { width: 20%; }
th:nth-child(2), td:nth-child(2) { width: 30%; }
th:nth-child(3), td:nth-child(3) { width: 25%; }
th:nth-child(4), td:nth-child(4) { width: 25%; }
```

### Обрезка текста в ячейках

```css
td {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
```

## Практические примеры

### Пример современной стилизации таблицы

```css
.data-table {
  width: 100%;
  border-collapse: collapse;
  margin: 20px 0;
  font-size: 0.9em;
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
  box-shadow: 0 0 20px rgba(0, 0, 0, 0.15);
  border-radius: 8px;
  overflow: hidden;
}

.data-table thead tr {
  background-color: #009879;
  color: #ffffff;
  text-align: left;
}

.data-table th,
.data-table td {
  padding: 12px 15px;
}

.data-table tbody tr {
  border-bottom: 1px solid #dddddd;
}

.data-table tbody tr:nth-of-type(even) {
  background-color: #f3f3f3;
}

.data-table tbody tr:last-of-type {
  border-bottom: 2px solid #009879;
}

.data-table tbody tr.active-row {
  font-weight: bold;
  color: #009879;
}
```
