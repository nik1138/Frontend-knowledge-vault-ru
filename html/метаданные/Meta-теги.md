# Meta-теги

Meta-теги - это элементы HTML, которые предоставляют информацию о веб-странице, не отображаемую напрямую на странице, но используемую браузерами, поисковыми системами и социальными сетями для понимания содержимого и контекста страницы.

## Основные meta-теги

### Charset
```html
<meta charset="UTF-8">
```
Определяет кодировку документа. Всегда должен быть первым meta-тегом в `<head>`.

### Viewport
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
Контролирует масштабирование и размеры области просмотра на мобильных устройствах.

### Title
```html
<title>Заголовок страницы</title>
```
Хотя не является meta-тегом, влияет на SEO и отображение в результатах поиска.

### Description
```html
<meta name="description" content="Краткое описание содержимого страницы">
```
Описание страницы, отображаемое в поисковой выдаче. Рекомендуемая длина: 150-160 символов.

### Keywords
```html
<meta name="keywords" content="ключевые, слова, страницы">
```
> [!warning] Важно
> Большинство поисковых систем больше не учитывают этот тег для SEO, но может быть полезен для внутреннего использования.

## Практические рекомендации

- Всегда используйте `charset="UTF-8"`
- Оптимизируйте тег `description` для поисковых систем
- Убедитесь, что тег `title` уникален для каждой страницы
- Избегайте дублирования meta-тегов
- Используйте кириллические символы корректно (особенно актуально для российских сайтов)

## Дополнительные теги

### Robots
```html
<meta name="robots" content="index, follow">
```
Инструкции поисковым роботам:
- `index` - индексировать страницу
- `noindex` - не индексировать
- `follow` - переходить по ссылкам
- `nofollow` - не переходить по ссылкам

### Author
```html
<meta name="author" content="Имя автора">
```
Указывает автора страницы.

### Generator
```html
<meta name="generator" content="Название CMS или редактора">
```
Информирует о программном обеспечении, использованном для создания страницы.

## Российские особенности 2025

В 2025 году российские разработчики сталкиваются с необходимостью адаптации к изменениям в отечественных поисковых системах, таких как Яндекс и Mail.ru. Важно учитывать:

- Специфические требования Яндекса к метаданным
- Локализованные теги для российских социальных сетей
- Совместимость с отечественными аналитическими системами
- Соответствие требованиям законодательства РФ о персональных данных

> [!tip] Совет
> Регулярно проверяйте метаданные с помощью инструментов Яндекс Вебмастер и Google Search Console для обеспечения корректного индексирования.
