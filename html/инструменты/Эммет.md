# Эммет (Emmet) для HTML

## Введение в Эммет

**Эммет** — это мощный инструмент для ускорения написания HTML и CSS кода. Он позволяет использовать сокращённые записи, которые автоматически преобразуются в полноценную HTML-разметку. Эммет особенно полезен для российских разработчиков в 2025 году, так как значительно ускоряет процесс верстки и уменьшает количество опечаток.

## Основные возможности Эммета

- **Сокращения HTML-элементов**: Быстрое создание элементов
- **Автоматическое создание структуры**: Генерация каркаса документа
- **Вложенность элементов**: Использование символов для определения иерархии
- **Нумерация и итерации**: Автоматическое добавление индексов
- **CSS-сокращения**: Ускорение написания CSS-свойств

## Синтаксис Эммета

### Основные элементы

```html
div        → <div></div>
p          → <p></p>
span       → <span></span>
a          → <a href=""></a>
img        → <img src="" alt="">
```

### Вложенные элементы

```html
div>ul>li*3     → <div><ul><li></li><li></li><li></li></ul></div>
header+nav+p    → Создаёт header, затем nav, затем p на одном уровне
div>h1+p        → div с h1 и p внутри
```

### Элементы с атрибутами

```html
div#header        → <div id="header"></div>
p.text            → <p class="text"></p>
input[type=text]  → <input type="text">
a{Ссылка}[href=#] → <a href="#">Ссылка</a>
```

### Нумерация

```html
ul>li.item$*3    → <ul><li class="item1"></li><li class="item2"></li><li class="item3"></li></ul>
div>h$*3         → <div><h1></h1><h2></h2><h3></h3></div>
```

## Практические примеры для российских проектов

### Шаблон HTML5 документа

```
!!!+html>head>meta[charset=UTF-8]+meta[name=viewport]+title{Заголовок страницы}+link[rel=stylesheet]+body>header+main+footer
```

Результат:
```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Заголовок страницы</title>
    <link rel="stylesheet" href="">
</head>
<body>
    <header></header>
    <main></main>
    <footer></footer>
</body>
</html>
```

### Структура навигации

```
nav>ul>li*5>a[href=#]{Ссылка $}
```

Результат:
```html
<nav>
    <ul>
        <li><a href="#">Ссылка 1</a></li>
        <li><a href="#">Ссылка 2</a></li>
        <li><a href="#">Ссылка 3</a></li>
        <li><a href="#">Ссылка 4</a></li>
        <li><a href="#">Ссылка 5</a></li>
    </ul>
</nav>
```

### Карточка товара (актуально для e-commerce проектов в РФ)

```
article.card>div.image+div.info>h3.title+p.description+div.price+p.button
```

Результат:
```html
<article class="card">
    <div class="image"></div>
    <div class="info">
        <h3 class="title"></h3>
        <p class="description"></p>
        <div class="price"></div>
        <p class="button"></p>
    </div>
</article>
```

## Расширенные возможности

### Группировка элементов

```
div>(header>ul>li*2>a)+footer>p
```

### Использование текста

```
a{Нажми меня}>img[src=image.jpg]+span
```

### Сокращения для форм

```
form>fieldset>legend+input:text+input:password+textarea+button
```

## Настройка Эммета в редакторах

### VS Code

1. Эммет встроен по умолчанию
2. Для русскоязычных проектов установите язык файлов как `html`
3. Используйте сочетание клавиш `Tab` для разворачивания сокращений
4. Для работы с React JSX используйте `javascriptreact` или `typescriptreact`

### Sublime Text

1. Установите пакет Emmet
2. Используйте `Ctrl+E` для разворачивания сокращений

### Atom

1. Установите пакет emmet
2. Используйте `Tab` для разворачивания

## Практические советы для российских разработчиков

- **Используйте кириллические текстовые узлы**: `{Заголовок}` вместо `{Title}`
- **Учитывайте требования доступности**: Добавляйте `aria-label` и `role` при необходимости
- **Следите за изменениями в стандартах**: В 2025 году могут появиться новые HTML-элементы
- **Интегрируйте с линтерами**: Используйте с ESLint и Stylelint для поддержания качества кода

## Заключение

Эммет — незаменимый инструмент для ускорения разработки HTML-разметки. В условиях высокой конкуренции на российском рынке веб-разработки, использование таких инструментов позволяет значительно повысить производительность и качество кода.

Для получения максимальной пользы от Эммета рекомендуется практиковаться с различными сокращениями и создавать собственные сниппеты под конкретные проекты.
