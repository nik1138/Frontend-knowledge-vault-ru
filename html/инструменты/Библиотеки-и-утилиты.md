# Библиотеки и утилиты для HTML

## Введение

HTML-библиотеки и утилиты - это инструменты, которые помогают упростить и ускорить работу с HTML-документами. В 2025 году среди разработчиков в России особенно популярны инструменты, которые помогают обеспечить семантичность, доступность и совместимость кода с отечественными стандартами.

## Парсеры и обработчики HTML

### 1. Cheerio (для Node.js)

Cheerio - одна из самых популярных библиотек для работы с HTML на сервере. Она предоставляет jQuery-подобный API для манипуляции с HTML-документами.

```javascript
const cheerio = require('cheerio');

// Загрузка HTML
const $ = cheerio.load('<h2 class="title">Hello world</h2>');

// Манипуляции с элементами
$('h2.title').text('Hello there!');
$('h2').addClass('welcome');

console.log($.html()); // Вывод измененного HTML
```

**Преимущества:**
- Знакомый jQuery-подобный синтаксис
- Быстрое выполнение операций
- Большое сообщество поддержки

**Российские аспекты:**
- Может использоваться для парсинга отечественных сайтов
- Подходит для скрапинга данных из российских источников

### 2. PyQuery (для Python)

PyQuery предоставляет возможность работы с HTML, схожую с jQuery, но в Python.

```python
from pyquery import PyQuery as pq

doc = pq('<div class="container"><p>Hello</p></div>')
doc('p').addClass('greeting')
print(doc.outer_html())
```

### 3. BeautifulSoup (для Python)

BeautifulSoup - мощная библиотека для парсинга HTML и XML документов.

```python
from bs4 import BeautifulSoup

html = "<div><p class='text'>Sample</p></div>"
soup = BeautifulSoup(html, 'html.parser')

# Найти элемент
text_element = soup.find('p', class_='text')
text_element.string = "Updated text"

print(soup.prettify())
```

## Инструменты для генерации HTML

### 1. HTMLayout

HTMLayout - это движок рендеринга HTML с возможностью программного управления через API. Особо популярен в отечественной разработке благодаря легковесности.

```javascript
// Пример использования HTMLayout
const htmLayout = require('htm-layout');

const template = `
  <div layout="horizontal"> 
    <div width="200">@menu</div>
    <div weight="1">@content</div>
  </div>
`;

const html = htmLayout.render(template, {
  menu: '<nav>...</nav>',
  content: '<main>...</main>'
});
```

### 2. Template Literals (Vanilla JS)

HTML-шаблонизация на основе шаблонных строк JavaScript:

```javascript
function createCard(title, content) {
  return `
    <article class="card">
      <header><h3>${title}</h3></header>
      <div class="content">${content}</div>
    </article>
  `;
}

const card = createCard("Заголовок", "Содержание");
```

### 3. LitHTML

LitHTML - это библиотека для декларативного рендеринга HTML в веб-компонентах.

```javascript
import {html, render} from 'lit-html';

const template = (name) => html`
  <div>
    <h1>Привет, ${name}!</h1>
    <p>Это компонент с использованием LitHTML</p>
  </div>
`;

render(template('Мир'), document.body);
```

## Утилиты для валидации и проверки

### 1. HTML Validate (Node.js)

Инструмент для проверки соответствия HTML-кода стандартам:

```javascript
const { validate } = require('html-validator');

const options = {
  data: '<!DOCTYPE html><html><head></head><body></body></html>',
  format: 'text'
};

validate(options)
  .then(body => {
    console.log(body);
  })
  .catch(err => {
    console.error(err);
  });
```

### 2. tidy-html5

Tidy-html5 - это инструмент командной строки и библиотека для проверки и очистки HTML.

```bash
tidy -indent -wrap 0 -asxhtml input.html -o output.xhtml
```

## Инструменты для оптимизации

### 1. HTMLMinifier

Инструмент для минификации HTML-кода:

```javascript
const minify = require('html-minifier').minify;

const result = minify('<p title="blah" id="moo">foo</p>', {
  removeAttributeQuotes: true,
  collapseBooleanAttributes: true,
  removeRedundantAttributes: true
});
```

### 2. PostHTML

Плагин для оптимизации HTML после сборки:

```javascript
const posthtml = require('posthtml');
const posthtmlExpressions = require('posthtml-expressions');

posthtml()
  .use(posthtmlExpressions({
    locals: { title: 'Foo' }
  }))
  .process('<title>{{ title }}</title>')
  .then(result => console.log(result.html));
```

## Инструменты для безопасности

### 1. DOMPurify

Библиотека для очистки HTML от потенциально опасного контента:

```javascript
import DOMPurify from 'dompurify';

const clean = DOMPurify.sanitize(dirtyHTML);
```

### 2. Sanitize-HTML

Библиотека для фильтрации HTML-контента:

```javascript
const sanitizeHtml = require('sanitize-html');

const clean = sanitizeHtml(dirtyHtml, {
  allowedTags: [ 'p', 'br', 'strong', 'em', 'u' ],
  allowedAttributes: {
    'a': [ 'href' ]
  }
});
```

## Российские особенности использования

### 1. Соответствие требованиям 152-ФЗ

При использовании библиотек для обработки HTML, особенно с пользовательским контентом, необходимо учитывать требования к защите персональных данных:

```javascript
// Пример очистки персональных данных
function removePersonalInfo(html) {
  let cleaned = html.replace(/\b\d{3}-\d{3}-\d{4}\b/g, '[PHONE HIDDEN]');
  cleaned = cleaned.replace(/\b[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,}\b/gi, '[EMAIL HIDDEN]');
  return cleaned;
}
```

### 2. Интеграция с отечественными решениями

- Использование российских CDN (например, Selectel, CDNvideo)
- Совместимость с отечественными браузерами (Яндекс.Браузер, Касперский и др.)
- Учет специфики российских поисковых систем

### 3. Локализация и международизация

При работе с HTML-шаблонами важно учитывать поддержку кириллицы:

```javascript
// Пример локализации шаблона
const template = (locale) => {
  const translations = {
    ru: 'Привет, мир!',
    en: 'Hello, world!'
  };
  
  return `<div lang="${locale}">${translations[locale]}</div>`;
};
```
