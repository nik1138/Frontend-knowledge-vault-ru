# Производительность и SEO

Производительность веб-сайта напрямую влияет на его позиции в поисковой выдаче. В 2025 году Google и другие поисковые системы активно учитывают метрики производительности как фактор ранжирования, что особенно важно для российских сайтов, конкурирующих в поисковой выдаче Яндекса и Google.

## Введение

Скорость загрузки сайта и общая производительность стали важными факторами SEO. Поисковые системы стремятся предоставлять пользователям лучший опыт, поэтому сайты с высокой производительностью получают преимущество в выдаче.

## История влияния производительности на SEO

### 2010 год - Google

Google впервые объявил, что скорость загрузки страницы является фактором ранжирования для поисковой выдачи.

### 2018 год - Mobile-First Indexing

Google начал использовать мобильную версию сайта для индексации и ранжирования, что усилило значение производительности для мобильных устройств.

### 2021 год - Core Web Vitals

Google внедрил Core Web Vitals как часть Page Experience Update, сделав производительность официальным фактором ранжирования.

### 2023-2025 - Расширение метрик

Поисковые системы начали учитывать дополнительные метрики производительности, включая взаимодействие с сайтом и воспринимаемую скорость.

## Core Web Vitals

### Largest Contentful Paint (LCP)

- Измеряет время загрузки основного контента
- Идеальное значение: < 2.5 секунд
- Влияние на SEO: значительное

```javascript
// Отслеживание LCP
import { getLCP } from 'web-vitals';

getLCP(console.log);
```

### First Input Delay (FID)

- Измеряет время отклика на первый пользовательский ввод
- Идеальное значение: < 100 мс
- Влияние на SEO: умеренное

### Cumulative Layout Shift (CLS)

- Измеряет стабильность макета
- Идеальное значение: < 0.1
- Влияние на SEO: значительное

## Метрики, влияющие на SEO

### Время загрузки страницы

- Первое измерение (First Contentful Paint)
- Полная загрузка (Onload)
- Время до интерактивности (Time to Interactive)

### Мобильная производительность

- Адаптивный дизайн
- Размеры ресурсов для мобильных устройств
- Время отклика на мобильных устройствах

### Воспринимаемая производительность

- Прогрессивная загрузка
- Визуальные индикаторы загрузки
- Кеширование ресурсов

## Технические оптимизации для SEO

### 1. Оптимизация загрузки критических ресурсов

- Встраивание критического CSS
- Отложенная загрузка несущественных ресурсов
- Приоритезация загрузки шрифтов

```html
<!-- Критический CSS -->
<style>
  /* Критические стили для видимой области */
</style>

<!-- Отложенный CSS -->
<link rel="preload" href="styles.css" as="style" onload="this.onload=null;this.rel='stylesheet'">
```

### 2. Оптимизация изображений

- Использование современных форматов (WebP, AVIF)
- Ленивая загрузка изображений
- Установка размеров изображений

```html
<picture>
  <source srcset="image.webp" type="image/webp">
  <img src="image.jpg" width="800" height="600" alt="Описание" loading="lazy">
</picture>
```

### 3. Оптимизация JavaScript

- Уменьшение размера JavaScript-бандлов
- Использование атрибутов `async` и `defer`
- Отложенная загрузка несущественных скриптов

### 4. Серверные оптимизации

- Использование HTTP/2
- Сжатие ресурсов (Gzip, Brotli)
- Кеширование на стороне сервера

## Влияние на конверсии

### Статистика

- Каждые 100 мс улучшения времени загрузки могут увеличить конверсию на 1%
- 53% пользователей мобильных устройств покидают страницу, если она загружается более 3 секунд
- Улучшение производительности может увеличить доход на 15-20%

### Связь с пользовательским опытом

- Быстрые сайты создают положительное впечатление
- Снижение количества отказов
- Увеличение времени на сайте
