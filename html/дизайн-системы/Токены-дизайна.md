# Токены дизайна в HTML дизайн-системе

## Что такое токены дизайна

Токены дизайна — это атомарные значения стилей, используемые в дизайн-системе. Это минимальные единицы, такие как цвета, размеры, шрифты, интервалы, которые можно переиспользовать по всей системе. Токены обеспечивают консистентность и упрощают масштабирование дизайн-системы.

## Типы токенов

### Цветовые токены
- `color-primary` - основной цвет бренда
- `color-secondary` - вспомогательный цвет
- `color-success`, `color-error`, `color-warning` - статусные цвета
- `color-neutral` - нейтральные оттенки

### Типографические токены
- `font-family-primary` - основной шрифт
- `font-size-xs`, `font-size-s`, `font-size-m`, `font-size-l`, `font-size-xl` - размеры шрифтов
- `line-height-tight`, `line-height-normal`, `line-height-loose` - высота строки

### Токены пространства
- `spacing-xs`, `spacing-s`, `spacing-m`, `spacing-l`, `spacing-xl` - отступы
- `border-radius-xs`, `border-radius-s`, `border-radius-m` - скругления

### Токены времени и эффектов
- `transition-fast`, `transition-medium`, `transition-slow` - длительность анимаций
- `shadow-small`, `shadow-medium`, `shadow-large` - тени

## Реализация токенов в HTML/CSS

### CSS Custom Properties
```css
:root {
  /* Цветовые токены */
  --ds-color-primary: #186de1;
  --ds-color-primary-hover: #0d5bc7;
  --ds-color-secondary: #f5f5f7;
  --ds-color-text-primary: #2d2d2f;
  --ds-color-text-secondary: #6e6e73;
  
  /* Типографические токены */
  --ds-font-family-primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
  --ds-font-size-s: 0.875rem;
  --ds-font-size-m: 1rem;
  --ds-font-size-l: 1.125rem;
  --ds-font-size-xl: 1.25rem;
  --ds-font-size-xxl: 1.5rem;
  
  /* Токены пространства */
  --ds-spacing-xs: 0.25rem;
  --ds-spacing-s: 0.5rem;
  --ds-spacing-m: 1rem;
  --ds-spacing-l: 1.5rem;
  --ds-spacing-xl: 2rem;
  
  /* Токены границ */
  --ds-border-radius-s: 0.25rem;
  --ds-border-radius-m: 0.5rem;
  --ds-border-radius-l: 0.75rem;
}
```

### Применение токенов в компонентах
```css
.ds-btn {
  padding: var(--ds-spacing-s) var(--ds-spacing-m);
  font-size: var(--ds-font-size-m);
  background-color: var(--ds-color-primary);
  border-radius: var(--ds-border-radius-m);
}
```

## Архитектура токенов

### Уровни токенов
1. **Семантические токены** - связанные с функцией (например, `btn-primary-bg`)
2. **Атомарные токены** - базовые значения (например, `blue-500`)

### Структура файлов токенов
```
tokens/
├── global/
│   ├── colors.css
│   ├── typography.css
│   ├── spacing.css
│   └── effects.css
├── semantic/
│   ├── button.css
│   ├── form.css
│   └── navigation.css
└── tokens.css (импортирует все токены)
```

## Практические рекомендации

### Именование токенов
- Используйте понятные имена: `color-primary` вместо `blue-500`
- Применяйте консистентную систему: `spacing-{size}` для отступов
- Учитывайте контекст использования: `btn-padding` для отступов кнопки

### Управление темами
```css
:root {
  --ds-color-bg-primary: #ffffff;
  --ds-color-text-primary: #2d2d2f;
}

[data-theme="dark"] {
  --ds-color-bg-primary: #1a1a1c;
  --ds-color-text-primary: #f5f5f7;
}
```

## Российские особенности 2025

### Соответствие стандартам
- Учет требований по контрастности цветов (не менее 4.5:1 для обычного текста)
- Поддержка режима высокой контрастности для пользователей с нарушениями зрения
- Использование отечественных или одобренных российским законодательством решений

### Кириллические шрифты
- Оптимизация токенов для кириллических шрифтов (размеры, интерлиньяж)
- Учет особенностей восприятия кириллического текста

## Заключение

Токены дизайна обеспечивают гибкость и консистентность в дизайн-системе. Их правильное использование позволяет легко обновлять стили, поддерживать темизацию и обеспечивать единообразие по всему интерфейсу.
