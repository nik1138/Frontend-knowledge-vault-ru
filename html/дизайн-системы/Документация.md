# Документация дизайн-системы

## Важность документации

### Структура документации компонента
1. Описание назначения компонента

7. Доступность (a11y)

### Пример документации компонента
```
# Кнопка (Button)

## Описание
Кнопка используется для инициации действий пользователя. Является интерактивным элементом интерфейса.

## Пример
<div class="preview">
  <button class="ds-btn ds-btn--primary">Основное действие</button>
</div>

## Код
```html
<button class="ds-btn ds-btn--primary">Основное действие</button>
```

## Модификаторы
- `ds-btn--primary` - основной стиль
- `ds-btn--secondary` - второстепенный стиль
- `ds-btn--danger` - стиль для опасных действий
- `ds-btn--disabled` - неактивное состояние

## Доступность
Компонент соответствует стандартам WCAG 2.1:
- Поддерживает клавиатурную навигацию
- Имеет корректные ARIA-атрибуты
- Обеспечивает достаточный контраст цветов
```
