# HTML Мультимедиа

## Обзор

HTML предоставляет широкие возможности для включения мультимедийного контента на веб-страницы. Это включает изображения, аудио, видео, векторную графику и другие медиафайлы. В этом руководстве рассмотрим основные элементы, атрибуты, форматы файлов, доступность и оптимизацию производительности.

## Включение изображений (img)

Элемент `<img>` используется для вставки изображений на веб-страницу. Он является самозакрывающимся тегом.

```html
<img src="path/to/image.jpg" alt="Описание изображения" width="300" height="200">
```

### Основные атрибуты

- `src` — путь к изображению
- `alt` — альтернативный текст (важно для доступности)
- `width`, `height` — размеры изображения
- `loading` — стратегия загрузки (например, `lazy`)
- `srcset` — несколько источников изображения для адаптивности

### Форматы изображений

- JPEG — для фотографий
- PNG — для изображений с прозрачностью
- GIF — для анимированных изображений
- WebP — современный формат с высоким качеством и низким размером
- SVG — векторная графика

## Включение аудио (audio)

Элемент `<audio>` позволяет вставлять аудиофайлы на страницу.

```html
<audio controls>
  <source src="path/to/audio.mp3" type="audio/mpeg">
  <source src="path/to/audio.ogg" type="audio/ogg">
  Ваш браузер не поддерживает элемент audio.
</audio>
```

### Атрибуты

- `controls` — отображение элементов управления
- `autoplay` — автоматическое воспроизведение
- `loop` — зацикленное воспроизведение
- `muted` — отключение звука
- `preload` — стратегия предварительной загрузки

### Форматы аудио

- MP3 — наиболее распространенный
- OGG — альтернатива с открытым кодом
- WAV — высокое качество, большой размер

## Включение видео (video)

Элемент `<video>` используется для вставки видеофайлов.

```html
<video width="640" height="360" controls poster="path/to/poster.jpg">
  <source src="path/to/video.mp4" type="video/mp4">
  <source src="path/to/video.webm" type="video/webm">
  Ваш браузер не поддерживает элемент video.
</video>
```

### Атрибуты

- `controls` — элементы управления
- `autoplay` — автовоспроизведение
- `loop` — зацикливание
- `muted` — без звука
- `poster` — изображение-заглушка
- `preload` — стратегия предварительной загрузки

### Форматы видео

- MP4 — наиболее совместимый
- WebM — открытый формат
- OGV — альтернатива с открытым исходным кодом

## Векторная графика (svg)

SVG (Scalable Vector Graphics) позволяет вставлять векторные изображения, которые масштабируются без потерь качества.

```html
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red" />
</svg>
```

SVG можно вставлять как элемент, так и через тег `<img>` или iframe.

## Доступность медиа контента

Для обеспечения доступности:

- Используйте атрибут `alt` для изображений
- Добавляйте субтитры и описания для видео
- Обеспечивайте контрастность и читаемость
- Используйте семантические теги
- Проверяйте совместимость с вспомогательными технологиями

## Оптимизация для производительности

- Сжимайте изображения и видео
- Используйте форматы нового поколения (WebP, AVIF)
- Применяйте lazy loading
- Используйте CDN для раздачи медиафайлов
- Оптимизируйте размеры и кодеки
