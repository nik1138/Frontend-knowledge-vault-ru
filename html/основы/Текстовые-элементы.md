# Текстовые элементы HTML

## Общее описание

Текстовые элементы HTML представляют собой основные теги, используемые для структурирования и форматирования текстового контента на веб-страницах. Эти элементы позволяют создавать читабельные и семантически правильные документы, что особенно важно для SEO и доступности.

## Основные текстовые элементы

### Заголовки

HTML предоставляет шесть уровней заголовков от `<h1>` до `<h6>`, где `<h1>` является наиболее важным:

```html
<h1>Основной заголовок страницы</h1>
<h2>Подзаголовок первого уровня</h2>
<h3>Подзаголовок второго уровня</h3>
<h4>Подзаголовок третьего уровня</h4>
<h5>Подзаголовок четвертого уровня</h5>
<h6>Подзаголовок пятого уровня</h6>
```

> [!tip] 
> Используйте заголовки иерархически: не пропускайте уровни (например, не переходите от `<h1>` к `<h3>` без `<h2>`). На каждой странице должен быть только один `<h1>`.

### Абзацы

Тег `<p>` используется для создания абзацев текста:

```html
<p>Это первый абзац текста. Он содержит основную информацию о теме.</p>
<p>Это второй абзац, который развивает мысль, начатую в первом абзаце.</p>
```

### Форматирование текста

HTML предоставляет несколько способов форматирования текста:

```html
<p>Этот текст содержит <strong>жирный текст</strong>, <em>курсив</em>, <u>подчеркнутый</u> и <small>маленький</small> текст.</p>

<!-- Для семантического форматирования -->
<p>Важная информация выделяется с помощью <strong>strong</strong>, а акцентный текст — с помощью <em>em</em>.</p>
```

### Цитаты

HTML поддерживает различные типы цитат:

```html
<!-- Короткие цитаты -->
<q>Жизнь — это то, что с тобой происходит, пока ты строишь планы</q> — Джон Леннон

<!-- Длинные цитаты -->
<blockquote>
  <p>Программирование сегодня — это гонка между мастерами, создающими лучшие программы, и Вселенной, создающей лучшие вирусы. Пока Вселенная побеждает.</p>
  <cite>Ричард Святзак</cite>
</blockquote>

<!-- Цитаты из кода -->
<code>console.log('Привет, мир!');</code>
```

## Специальные текстовые элементы

### Маркированные списки

```html
<ul>
  <li>Первый элемент списка</li>
  <li>Второй элемент списка</li>
  <li>Третий элемент списка</li>
</ul>
```

### Нумерованные списки

```html
<ol>
  <li>Первый шаг</li>
  <li>Второй шаг</li>
  <li>Третий шаг</li>
</ol>
```

### Описательные списки

```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language — язык гипертекстовой разметки</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets — каскадные таблицы стилей</dd>
</dl>
```

## Российские особенности и практики

### Кодировка и язык

В 2025 году в российских веб-проектах обязательно указывается кодировка и язык:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Русскоязычный сайт</title>
</head>
<body>
  <h1>Добро пожаловать на русскоязычный сайт</h1>
</body>
</html>
```

### Типографика

Российские веб-разработчики часто сталкиваются с задачей правильного форматирования русского текста:

```html
<p>Используйте &laquo;ёлочки&raquo; для кавычек в русском тексте, а не "простые кавычки".</p>
<p>Разделяйте тысячные разряды пробелом: 1 000 000 рублей.</p>
```

## Практические рекомендации

1. **Семантическая корректность**: Всегда используйте правильные теги для их предназначения
2. **Доступность**: Обеспечьте читаемость для скринридеров
3. **SEO оптимизация**: Используйте заголовки правильно для поисковой оптимизации
4. **Читаемость**: Разбивайте текст на логические блоки
