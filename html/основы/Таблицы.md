# Таблицы в HTML

HTML-таблицы используются для представления табличных данных в структурированном виде. Правильное использование таблиц важно для доступности, SEO и корректного отображения данных.

## Основы HTML-таблиц

### Базовая структура таблицы

```html
<table>
    <thead>
        <tr>
            <th>Заголовок 1</th>
            <th>Заголовок 2</th>
            <th>Заголовок 3</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Данные 1</td>
            <td>Данные 2</td>
            <td>Данные 3</td>
        </tr>
        <tr>
            <td>Данные 4</td>
            <td>Данные 5</td>
            <td>Данные 6</td>
        </tr>
    </tbody>
</table>
```

### Основные элементы таблицы

- `<table>` — контейнер для всей таблицы
- `<thead>` — группа заголовков таблицы
- `<tbody>` — группа основного содержимого таблицы
- `<tfoot>` — группа футера таблицы
- `<tr>` — строка таблицы
- `<th>` — ячейка заголовка
- `<td>` — ячейка данных

## Структура таблицы

### Заголовки таблицы

```html
<table>
    <caption>Продажи по кварталам 2025 года</caption>
    <thead>
        <tr>
            <th scope="col">Квартал</th>
            <th scope="col">Продажи (тыс. руб.)</th>
            <th scope="col">Рост (%)</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th scope="row">Q1</th>
            <td>1500</td>
            <td>+12%</td>
        </tr>
        <tr>
            <th scope="row">Q2</th>
            <td>1800</td>
            <td>+15%</td>
        </tr>
    </tbody>
</table>
```

### Атрибут `scope` для доступности

Атрибут `scope` указывает, к каким ячейкам применяется заголовок:

- `scope="col"` — заголовок для столбца
- `scope="row"` — заголовок для строки
- `scope="colgroup"` — заголовок для группы столбцов
- `scope="rowgroup"` — заголовок для группы строк

## Атрибуты таблицы

### Основные атрибуты

```html
<table 
    border="1"           <!-- Границы ячеек (устаревший, используйте CSS) -->
    cellpadding="5"      <!-- Отступы внутри ячеек (устаревший, используйте CSS) -->
    cellspacing="0"      <!-- Расстояние между ячейками (устаревший, используйте CSS) -->
    summary="Описание таблицы для доступности (устаревший)" 
>
    <!-- Содержимое таблицы -->
</table>
```

> [!warning] Важно
> Атрибуты `border`, `cellpadding`, `cellspacing` устарели. Используйте CSS для стилизации таблиц.

### Современный подход с CSS

```html
<table class="data-table">
    <thead>
        <tr>
            <th>Продукт</th>
            <th>Цена</th>
            <th>Наличие</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Ноутбук</td>
            <td>50000 ₽</td>
            <td>В наличии</td>
        </tr>
    </tbody>
</table>
```

```css
.data-table {
    border-collapse: collapse;
    width: 100%;
}

.data-table th,
.data-table td {
    border: 1px solid #ddd;
    padding: 8px;
    text-align: left;
}

.data-table th {
    background-color: #f2f2f2;
    font-weight: bold;
}
```

## Российские особенности работы с таблицами

В 2025 году в России при работе с HTML-таблицами важно учитывать:

- **Требования 449-ФЗ** — обеспечение доступности табличных данных для людей с ограниченными возможностями
- **Соответствие требованиям РКН** — корректное представление табличной информации на государственных сайтах
- **Форматирование чисел и дат** — в соответствии с российскими стандартами
- **Валютные обозначения** — использование рубля (₽) и правильное форматирование

## Практические рекомендации

1. **Используйте таблицы только для табличных данных**, а не для макета страницы
2. **Добавляйте `<caption>`** для описания таблицы
3. **Используйте `<thead>`, `<tbody>`, `<tfoot>`** для структурирования
4. **Применяйте атрибут `scope`** к заголовкам для доступности
5. **Используйте CSS** для стилизации, а не атрибуты таблицы
6. **Объединяйте ячейки** только когда это улучшает восприятие данных
7. **Проверяйте доступность** с помощью скринридеров

## Адаптивные таблицы

Для мобильных устройств таблицы могут быть проблемой. Вот несколько подходов:

### Вариант 1: Горизонтальный скролл

```html
<div class="table-container">
    <table>
        <thead>
            <tr>
                <th>Продукт</th>
                <th>Цена</th>
                <th>Описание</th>
                <th>Наличие</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Ноутбук</td>
                <td>50000 ₽</td>
                <td>15.6" дюймов, 16GB RAM</td>
                <td>В наличии</td>
            </tr>
        </tbody>
    </table>
</div>
```

```css
.table-container {
    overflow-x: auto;
    width: 100%;
}
```

### Вариант 2: Преобразование в список на мобильных

С помощью CSS можно преобразовать таблицу в список на мобильных устройствах:

```css
/* На мобильных устройствах */
@media (max-width: 768px) {
    table, thead, tbody, th, td, tr {
        display: block;
    }
    
    thead tr {
        position: absolute;
        top: -9999px;
        left: -9999px;
    }
    
    tr {
        border: 1px solid #ccc;
        margin-bottom: 10px;
        padding: 10px;
    }
    
    td {
        border: none;
        position: relative;
        padding-left: 50%;
    }
    
    td:before {
        content: attr(data-label) ": ";
        position: absolute;
        left: 6px;
        width: 45%;
        padding-right: 10px;
        white-space: nowrap;
        font-weight: bold;
    }
}
```

```html
<table>
    <thead>
        <tr>
            <th>Продукт</th>
            <th>Цена</th>
            <th>Наличие</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td data-label="Продукт">Ноутбук</td>
            <td data-label="Цена">50000 ₽</td>
            <td data-label="Наличие">В наличии</td>
        </tr>
    </tbody>
</table>
```
