# Структура документа HTML

HTML-документ имеет строго определенную структуру, которая обеспечивает правильное отображение веб-страницы в браузере. Понимание этой структуры критически важно для создания валидных и доступных веб-страниц.

## Основная структура HTML-документа

Каждый HTML-документ начинается с объявления типа документа и имеет определенные элементы:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Заголовок страницы</title>
</head>
<body>
    <!-- Содержимое страницы -->
</body>
</html>
```

### Объявление типа документа

`<!DOCTYPE html>` — это объявление типа документа, которое указывает браузеру, что документ следует стандарту HTML5. Это важно для правильной обработки содержимого.

### Корневой элемент `<html>`

Элемент `<html>` — это корневой элемент любого HTML-документа. Он содержит все остальные элементы. Атрибут `lang` указывает язык документа, что важно для доступности и SEO.

### Элемент `<head>`

Элемент `<head>` содержит метаданные о документе. Он не отображается на странице, но предоставляет важную информацию браузеру и поисковым системам:

- `<title>` — заголовок страницы, отображаемый во вкладке браузера
- `<meta>` — метаданные, такие как кодировка, описание, ключевые слова
- `<link>` — ссылки на внешние ресурсы (CSS-файлы, иконки)
- `<style>` — встроенные стили
- `<script>` — встроенные скрипты

### Элемент `<body>`

Элемент `<body>` содержит видимое содержимое страницы: текст, изображения, ссылки, таблицы и другие элементы.

## Семантическая структура HTML5

HTML5 вводит семантические элементы, которые делают структуру документа более понятной:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пример семантической структуры</title>
</head>
<body>
    <header>
        <h1>Заголовок сайта</h1>
        <nav>
            <ul>
                <li><a href="#home">Главная</a></li>
                <li><a href="#about">О нас</a></li>
                <li><a href="#contact">Контакты</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <header>
                <h2>Заголовок статьи</h2>
                <time datetime="2025-01-01">1 января 2025</time>
            </header>
            <section>
                <h3>Раздел статьи</h3>
                <p>Содержимое раздела...</p>
            </section>
        </article>
        
        <aside>
            <h3>Боковая панель</h3>
            <p>Дополнительная информация</p>
        </aside>
    </main>
    
    <footer>
        <p>&copy; 2025 Все права защищены</p>
    </footer>
</body>
</html>
```

## Важные мета-теги

### Кодировка символов

```html
<meta charset="UTF-8">
```

Обязательный тег, указывающий кодировку документа. UTF-8 поддерживает все языки мира, включая русский.

### Адаптивный дизайн

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Необходим для адаптивного дизайна, особенно важен для мобильных устройств.

### SEO-мета-данные

```html
<title>Заголовок страницы - важен для SEO</title>
<meta name="description" content="Краткое описание содержимого страницы">
<meta name="keywords" content="ключевые, слова, страницы">
```

## Российские особенности структуры

В 2025 году в России при создании HTML-документов важно учитывать:

- **Требования 449-ФЗ** о доступности веб-сайтов для людей с ограниченными возможностями
- **Соответствие требованиям РКН** к структуре и содержанию веб-сайтов
- **Поддержка кириллических доменов** и русскоязычного контента
- **Локализация дат и времени** в соответствии с российскими стандартами

## Практические рекомендации

1. **Всегда используйте объявление DOCTYPE** в начале документа
2. **Указывайте язык документа** с помощью атрибута `lang`
3. **Включайте обязательные мета-теги** (кодировка, viewport)
4. **Используйте семантические элементы** для лучшей структуры
5. **Проверяйте структуру документа** с помощью валидаторов

## Пример полного документа

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Пример правильно структурированного HTML-документа">
    <title>Пример HTML-документа</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header role="banner">
        <h1>Мой веб-сайт</h1>
        <nav role="navigation">
            <ul>
                <li><a href="/">Главная</a></li>
                <li><a href="/about">О нас</a></li>
                <li><a href="/contact">Контакты</a></li>
            </ul>
        </nav>
    </header>
    
    <main role="main">
        <article>
            <header>
                <h2>Интересная статья</h2>
                <p>Опубликовано: <time datetime="2025-01-01">1 января 2025</time></p>
            </header>
            <section>
                <h3>Первый раздел</h3>
                <p>Содержимое первого раздела статьи...</p>
            </section>
            <section>
                <h3>Второй раздел</h3>
                <p>Содержимое второго раздела статьи...</p>
            </section>
        </article>
    </main>
    
    <aside role="complementary">
        <h3>Дополнительная информация</h3>
        <p>Боковая панель с дополнительным контентом</p>
    </aside>
    
    <footer role="contentinfo">
        <p>&copy; 2025 Все права защищены</p>
    </footer>
</body>
</html>
```
