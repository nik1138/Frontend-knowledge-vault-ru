# Теги и элементы HTML

HTML-теги и элементы — это основные строительные блоки веб-страниц. Понимание различий между тегами и элементами, а также знание основных тегов, критически важно для создания веб-сайтов.

## Что такое теги и элементы

**Теги** — это ключевые слова, заключенные в угловые скобки (`< >`), которые определяют структуру и семантику содержимого. 

**Элементы** — это теги вместе с их содержимым. Например, `<p>Привет, мир!</p>` — это элемент параграфа, а `<p>` и `</p>` — это теги.

## Классификация HTML-элементов

### Блочные элементы

Блочные элементы занимают всю доступную ширину и начинаются с новой строки:

- `<div>` — универсальный контейнер
- `<p>` — параграф
- `<h1>` до `<h6>` — заголовки
- `<header>` — заголовок секции
- `<nav>` — навигация
- `<main>` — основное содержимое
- `<section>` — раздел
- `<article>` — самостоятельный контент
- `<aside>` — боковая панель
- `<footer>` — нижний колонтитул
- `<ul>`, `<ol>`, `<dl>` — списки
- `<form>` — форма
- `<table>` — таблица

```html
<div>
    <h2>Заголовок раздела</h2>
    <p>Это параграф в блочном элементе.</p>
</div>
```

### Строчные элементы

Строчные элементы занимают только необходимое пространство и не начинаются с новой строки:

- `<span>` — универсальный строчный контейнер
- `<a>` — ссылка
- `<strong>` — жирный текст
- `<em>` — курсивный текст
- `<img>` — изображение
- `<input>` — поле ввода
- `<label>` — метка
- `<button>` — кнопка
- `<small>` — мелкий текст
- `<abbr>` — аббревиатура

```html
<p>Это <strong>жирный</strong> и <em>курсивный</em> текст внутри строчного элемента.</p>
```

### Элементы с двойными тегами и одиночные теги

**Элементы с двойными тегами** имеют открывающий и закрывающий теги:

```html
<p>Это элемент с открывающим и закрывающим тегами</p>
<div>Еще один элемент</div>
```

**Одиночные теги (самозакрывающиеся)** не требуют закрывающего тега:

```html
<img src="image.jpg" alt="Описание изображения">
<br> <!-- Перевод строки -->
<hr> <!-- Горизонтальная линия -->
<input type="text" name="username">
<meta charset="UTF-8">
<link rel="stylesheet" href="style.css">
```

## Основные структурные элементы

### Заголовки

```html
<h1>Заголовок первого уровня (должен быть только один на странице)</h1>
<h2>Заголовок второго уровня</h2>
<h3>Заголовок третьего уровня</h3>
<h4>Заголовок четвертого уровня</h4>
<h5>Заголовок пятого уровня</h5>
<h6>Заголовок шестого уровня</h6>
```

### Параграфы и структура текста

```html
<p>Это параграф текста.</p>
<br> <!-- Перевод строки -->
<hr> <!-- Горизонтальная линия -->
<pre>Предварительно отформатированный текст, сохраняющий пробелы и переносы</pre>
<blockquote>Цитата от другого источника</blockquote>
<code>Код в тексте</code>
<samp>Вывод программы</samp>
<kbd>Клавиши на клавиатуре</kbd>
<var>Переменная в тексте</var>
```

### Семантические элементы HTML5

```html
<header>Шапка сайта или раздела</header>
<nav>Навигационные ссылки</nav>
<main>Основное содержимое страницы</main>
<article>Самостоятельный контент (статья, пост)</article>
<section>Раздел содержимого</section>
<aside>Дополнительная информация</aside>
<footer>Нижний колонтитул</footer>
<figure>Контейнер для медиа-контента с подписью</figure>
<figcaption>Подпись к медиа-контенту</figcaption>
<time datetime="2025-01-01">1 января 2025</time>
<details>Детали, которые можно раскрыть</details>
<summary>Заголовок для details</summary>
```

## Текстовые элементы

### Форматирование текста

```html
<strong>Жирный текст (семантически важный)</strong>
<b>Жирный текст (только визуальное выделение)</b>
<em>Курсивный текст (семантическое выделение)</em>
<i>Курсивный текст (только визуальное выделение)</i>
<u>Подчеркнутый текст</u>
<mark>Выделенный маркером текст</mark>
<small>Мелкий текст (например, копирайт)</small>
<del>Перечеркнутый текст</del>
<ins>Вставленный текст</ins>
```

### Ссылки

```html
<a href="https://example.com">Ссылка на внешний сайт</a>
<a href="/page.html">Ссылка на внутреннюю страницу</a>
<a href="#section">Ссылка на якорь на этой странице</a>
<a href="mailto:email@example.com">Ссылка на email</a>
<a href="tel:+71234567890">Ссылка на телефон</a>
```

## Российские особенности использования тегов

В 2025 году в России при использовании HTML-тегов важно учитывать:

- **Требования 449-ФЗ** — использовать семантические теги для улучшения доступности
- **Соответствие требованиям РКН** — правильная структура для государственных сайтов
- **Поддержка кириллического контента** — корректная кодировка и языковые атрибуты
- **SEO-оптимизация** — правильное использование заголовков и структурных элементов

## Практические рекомендации

1. **Используйте семантические теги** вместо `<div>` и `<span>` когда это возможно
2. **Следите за иерархией заголовков** — не пропускайте уровни (h1 → h2 → h3)
3. **Используйте теги по их предназначению**, а не только для визуального оформления
4. **Добавляйте атрибут `alt`** к изображениям для доступности
5. **Проверяйте валидность** с помощью HTML-валидатора

## Пример использования тегов в реальной структуре

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пример структуры с тегами</title>
</head>
<body>
    <header>
        <h1>Мой веб-сайт</h1>
        <nav>
            <ul>
                <li><a href="/">Главная</a></li>
                <li><a href="/about">О нас</a></li>
                <li><a href="/services">Услуги</a></li>
                <li><a href="/contact">Контакты</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <header>
                <h2>Интересная статья</h2>
                <p>Опубликовано: <time datetime="2025-01-01">1 января 2025</time></p>
            </header>
            
            <section>
                <h3>Первый раздел статьи</h3>
                <p>Это <strong>важная информация</strong>, представленная в виде <em>первого раздела</em> статьи.</p>
                
                <figure>
                    <img src="image.jpg" alt="Описание изображения">
                    <figcaption>Подпись к изображению</figcaption>
                </figure>
            </section>
            
            <section>
                <h3>Второй раздел статьи</h3>
                <p>Во втором разделе мы рассмотрим <mark>ключевые моменты</mark> темы.</p>
                <blockquote>
                    <p>Это важная цитата, которую стоит запомнить.</p>
                    <cite>— Известный человек</cite>
                </blockquote>
            </section>
        </article>
    </main>
    
    <aside>
        <h3>Полезная информация</h3>
        <p>Дополнительные сведения, связанные с основным контентом.</p>
    </aside>
    
    <footer>
        <p>&copy; 2025 Все права защищены</p>
        <address>
            Контактная информация: <a href="mailto:info@example.com">info@example.com</a>
        </address>
    </footer>
</body>
</html>
```
