# HTML3 и HTML4: Развитие возможностей языка разметки

HTML3 и HTML4 стали важными этапами в развитии языка разметки, добавив значительные возможности для стилизации, макета и мультимедиа. Эти версии заложили основы для современного веб-дизайна и стали краеугольным камнем веб-технологий, которые использовались в России и по сей день.

## HTML3.2: Значительные улучшения (1997)

HTML 3.2 был опубликован W3C в январе 1997 года и стал важным шагом в развитии веб-стандартов. Он включал возможности, которые ранее использовались только в браузерах Netscape и Microsoft Internet Explorer.

### Основные нововведения HTML3.2

- **Таблицы**: Поддержка сложных таблиц с тегами `<TABLE>`, `<TR>`, `<TD>`, `<TH>`
- **Форматирование**: Улучшенные возможности форматирования текста
- **Изображения**: Расширенная поддержка изображений с атрибутами
- **Скрипты**: Поддержка клиентских скриптов (JavaScript)
- **Фреймы**: Возможность создания многооконных интерфейсов

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 3.2 Final//EN">
<HTML>
<HEAD>
<TITLE>Пример HTML3.2 документа</TITLE>
</HEAD>
<BODY>
<H1>Заголовок документа</H1>
<TABLE BORDER="1">
<TR>
<TH>Заголовок колонки 1</TH>
<TH>Заголовок колонки 2</TH>
</TR>
<TR>
<TD>Данные 1</TD>
<TD>Данные 2</TD>
</TR>
</TABLE>
<P>Это абзац с <B>жирным</B> и <I>курсивным</I> текстом.</P>
<IMG SRC="image.jpg" ALT="Описание изображения">
</BODY>
</HTML>
```

### Поддержка браузерами

HTML3.2 был принят большинством браузеров к 1997 году:

- Netscape Navigator 3.0+
- Internet Explorer 3.0+
- Opera 3.0+
- Mosaic (последние версии)

## HTML4.0: Четыре уровня спецификации (1997-1999)

HTML4.0, опубликованный W3C в декабря 1997 года, представил более строгую спецификацию с тремя уровнями (Strict, Transitional, Frameset). В 1999 году была опубликована HTML4.01 с небольшими исправлениями.

### Три уровня HTML4.0

1. **Strict (Строгий)**: Только структурные элементы, без презентационных тегов
2. **Transitional (Переходный)**: Разрешены презентационные теги и атрибуты
3. **Frameset**: Для документов, использующих фреймы

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" 
    "http://www.w3.org/TR/html4/strict.dtd">
<HTML>
<HEAD>
<TITLE>HTML4.0 Строгий пример</TITLE>
</HEAD>
<BODY>
<H1>Заголовок</H1>
<P>Этот документ соответствует строгому стандарту HTML4.0</P>
</BODY>
</HTML>
```

### Основные нововведения HTML4

- **CSS-интеграция**: Поддержка каскадных таблиц стилей
- **Скриптовая поддержка**: Улучшенная поддержка JavaScript
- **Улучшенная доступность**: Новые теги и атрибуты для доступности
- **Семантические элементы**: Улучшенная семантика документа
- **Международизация**: Поддержка различных языков и кодировок

## Влияние на российскую веб-индустрию

### 1997-2000: Развитие российского веба

В 1997-2000 годах в России происходило активное развитие веб-индустрии:

- Появление первых крупных российских веб-студий
- Развитие электронной коммерции
- Создание первых российских поисковых систем (например, WebALTA, Rambler)
- Формирование сообщества веб-разработчиков

### Особенности российского контекста

- **Кириллическая поддержка**: HTML4 улучшил поддержку кириллических символов
- **Медленные соединения**: Необходимость оптимизации для модемных соединений
- **Разнообразие браузеров**: Необходимость тестирования в различных браузерах

### Практические рекомендации для российских разработчиков

Даже в 2025 году знание HTML4 важно по нескольким причинам:

1. **Поддержка устаревших систем**: Многие государственные и корпоративные системы до сих пор используют технологии на основе HTML4

2. **Обратная совместимость**: Понимание HTML4 помогает при поддержке унаследованного кода

3. **Образовательные цели**: Изучение эволюции HTML помогает понять современные стандарты

```html
<!-- Пример кода, совместимого с HTML4 -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" 
    "http://www.w3.org/TR/html4/loose.dtd">
<HTML>
<HEAD>
<TITLE>Совместимый с HTML4 документ</TITLE>
<META HTTP-EQUIV="Content-Type" CONTENT="text/html; charset=windows-1251">
</HEAD>
<BODY>
<H1>Пример HTML4 документа</H1>
<TABLE>
<TR>
<TD>Таблица в HTML4</TD>
</TR>
</TABLE>
<P>Абзац с <FONT COLOR="red">цветным текстом</FONT></P>
</BODY>
</HTML>
```

## Семантические улучшения

HTML4 внес значительные улучшения в семантическую разметку:

- `<STRONG>` и `<EM>` для логического выделения
- `<CITE>` для цитирования источников
- `<CODE>` для программного кода
- Улучшенная поддержка таблиц с заголовками и описаниями

## Безопасность и доступность

HTML4 включал новые возможности для повышения безопасности и доступности:

- Атрибут `title` для дополнительной информации
- Улучшенная поддержка alt-атрибутов для изображений
- Возможности для создания доступных форм

## Значение для современности

Хотя HTML4 официально устарел, его влияние все еще ощутимо:

1. **Множество унаследованных систем**: В России до сих пор эксплуатируется множество систем, созданных в эпоху HTML4

2. **Образовательные программы**: В российских вузах и техникумах изучение истории веб-стандартов включает HTML4

3. **Понимание эволюции**: Знание HTML4 помогает понять, почему современные стандарты принимают определенные решения

## Заключение

HTML3 и HTML4 стали важными этапами в развитии веб-технологий, добавив возможности, которые легли в основу современного веб-дизайна. В российском контексте 2025 года знание этих стандартов важно для понимания эволюции веб-технологий и работы с унаследованными системами, особенно в государственном и корпоративном секторах.
