# Карта сайта в HTML

## Обзор

Карта сайта (sitemap) - это веб-страница, которая содержит список ссылок на все важные страницы сайта, организованных в иерархическом порядке. Карта сайта помогает пользователям и поисковым роботам находить и индексировать контент на сайте.

## Типы карты сайта

### 1. Пользовательская карта сайта

Предназначена для пользователей, помогает им ориентироваться на сайте:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Карта сайта - Мой сайт</title>
</head>
<body>
  <header>
    <h1>Карта сайта</h1>
    <p>Все разделы нашего сайта в одном месте</p>
  </header>

  <main>
    <nav aria-label="Карта сайта">
      <section>
        <h2>Главная</h2>
        <ul>
          <li><a href="/">Главная страница</a></li>
          <li><a href="/about">О нас</a></li>
          <li><a href="/contact">Контакты</a></li>
        </ul>
      </section>

      <section>
        <h2>Продукты</h2>
        <ul>
          <li><a href="/products">Все продукты</a></li>
          <li><a href="/products/electronics">Электроника</a>
            <ul>
              <li><a href="/products/electronics/phones">Телефоны</a></li>
              <li><a href="/products/electronics/laptops">Ноутбуки</a></li>
            </ul>
          </li>
          <li><a href="/products/clothing">Одежда</a></li>
        </ul>
      </section>

      <section>
        <h2>Услуги</h2>
        <ul>
          <li><a href="/services">Все услуги</a></li>
          <li><a href="/services/consulting">Консалтинг</a></li>
          <li><a href="/services/support">Поддержка</a></li>
        </ul>
      </section>

      <section>
        <h2>Информация</h2>
        <ul>
          <li><a href="/blog">Блог</a></li>
          <li><a href="/news">Новости</a></li>
          <li><a href="/faq">Часто задаваемые вопросы</a></li>
        </ul>
      </section>

      <section>
        <h2>Поддержка</h2>
        <ul>
          <li><a href="/help">Помощь</a></li>
          <li><a href="/support">Техническая поддержка</a></li>
          <li><a href="/terms">Условия использования</a></li>
          <li><a href="/privacy">Политика конфиденциальности</a></li>
        </ul>
      </section>
    </nav>
  </main>

  <footer>
    <p>&copy; 2025 Мой сайт. Все права защищены.</p>
  </footer>
</body>
</html>
```

### 2. XML-карта сайта

Предназначена для поисковых роботов:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://www.mysite.com/</loc>
    <lastmod>2025-01-15</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://www.mysite.com/about</loc>
    <lastmod>2025-01-10</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://www.mysite.com/products</loc>
    <lastmod>2025-01-12</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.9</priority>
  </url>
  <url>
    <loc>https://www.mysite.com/products/electronics</loc>
    <lastmod>2025-01-05</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.7</priority>
  </url>
</urlset>
```

## Структура пользовательской карты сайта

### Организация по категориям

```html
<main>
  <nav aria-label="Карта сайта">
    <div class="sitemap-categories">
      <section class="category">
        <h2>Компания</h2>
        <ul>
          <li><a href="/about">О компании</a></li>
          <li><a href="/team">Наша команда</a></li>

        <ul>
          <li><a href="/products">Все продукты</a></li>
          <li><a href="/products/new">Новинки</a></li>
          <li><a href="/products/popular">Популярные</a></li>
```

## Доступность карты сайта

### ARIA-атрибуты

```html
<nav aria-label="Карта сайта" role="navigation">
  <h1>Карта сайта</h1>
  <p>Навигация по всем разделам нашего сайта</p>
  
  <section aria-labelledby="company-heading">
    <h2 id="company-heading">Компания</h2>
    <ul>
      <li><a href="/about">О компании</a></li>
      <li><a href="/team">Наша команда</a></li>
    </ul>
  </section>
  
  <section aria-labelledby="products-heading">
    <h2 id="products-heading">Продукты</h2>
    <ul>
      <li><a href="/products">Все продукты</a></li>
      <li><a href="/catalog">Каталог</a></li>
    </ul>
  </section>
</nav>
```

### Клавиатурная навигация

Карта сайта должна быть легко доступна с клавиатуры:

```css
.sitemap a {
  padding: 0.5rem;
  display: block;
  text-decoration: none;
  border-radius: 4px;
  transition: background-color 0.3s;
}

.sitemap a:focus {
  background-color: #e9ecef;
  outline: 2px solid #007cba;
  outline-offset: 2px;
}
```

## SEO-оптимизация карты сайта

### Метаданные

```html
<head>
  <title>Карта сайта - Название сайта</title>
  <meta name="description" content="Полная карта сайта с ссылками на все важные страницы">
  <link rel="sitemap" type="application/xml" title="XML Sitemap" href="/sitemap.xml">
</head>
```

### Schema.org разметка

```html
<nav itemscope itemtype="http://schema.org/SiteNavigationElement">
  <ul>
    <li itemprop="name"><a href="/" itemprop="url">Главная</a></li>
    <li itemprop="name"><a href="/about" itemprop="url">О нас</a></li>
    <li itemprop="name"><a href="/products" itemprop="url">Продукты</a></li>
  </ul>
</nav>
```

## Стилизация карты сайта

### Базовая стилизация

```css
.sitemap {
  max-width: 1200px;
  margin: 0 auto;
  padding: 2rem;
  font-family: Arial, sans-serif;
}

.sitemap h2 {
  color: #333;
  border-bottom: 2px solid #007cba;
  padding-bottom: 0.5rem;
  margin-top: 2rem;
  margin-bottom: 1rem;
}

.sitemap ul {
  list-style: none;
  padding: 0;
  margin: 0.5rem 0;
}

.sitemap li {
  margin: 0.25rem 0;
  padding: 0.25rem 0;
}

.sitemap a {
  text-decoration: none;
  color: #007cba;
  padding: 0.25rem 0;
  display: inline-block;
  transition: color 0.3s;
}

.sitemap a:hover {
  color: #005a87;
  text-decoration: underline;
}

.sitemap-categories {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 2rem;
}

.category {
  background-color: #f8f9fa;
  padding: 1.5rem;
  border-radius: 8px;
  border-left: 4px solid #007cba;
}
```

### Адаптивная карта сайта

```css
@media (max-width: 768px) {
  .sitemap {
    padding: 1rem;
  }
  
  .sitemap-categories {
    grid-template-columns: 1fr;
    gap: 1rem;
  }
  
  .sitemap h2 {
    font-size: 1.25rem;
  }
}
```

## Лучшие практики

### 1. Обновление карты сайта

Регулярно обновляйте карту сайта при добавлении новых страниц:

```html
<!-- Укажите дату последнего обновления -->
<footer class="sitemap-footer">
  <p>Карта сайта последний раз обновлена: <time datetime="2025-01-15">15 января 2025</time></p>
</footer>
```

### 2. Ограничение количества ссылок

Не размещайте слишком много ссылок на одной странице:

```html
<!-- Разделите на несколько страниц, если у вас более 100 ссылок -->
<nav aria-label="Карта сайта">
  <section>
    <h2>Страницы 1-50</h2>
    <!-- Ссылки 1-50 -->
  </section>
  <section>
    <h2>Страницы 51-100</h2>
    <!-- Ссылки 51-100 -->
  </section>
</nav>
```

### 3. Использование понятных названий

```html
<!-- Хорошо -->
<ul>
  <li><a href="/customer-support">Поддержка клиентов</a></li>
  <li><a href="/product-catalog">Каталог продуктов</a></li>
</ul>

<!-- Не рекомендуется -->
<ul>
  <li><a href="/page123">Страница 123</a></li>
  <li><a href="/item456">Элемент 456</a></li>
</ul>
```

## Интеграция с навигацией

### Связь с основной навигацией

```html
<!-- Добавьте ссылку на карту сайта в основную навигацию -->
<footer>
  <nav aria-label="Нижняя навигация">
    <ul>
      <li><a href="/sitemap">Карта сайта</a></li>
      <li><a href="/privacy">Политика конфиденциальности</a></li>
      <li><a href="/terms">Условия использования</a></li>
    </ul>
  </nav>
</footer>

<!-- Или в шапке сайта -->
<header>
  <nav aria-label="Верхняя навигация">
    <ul>
      <li><a href="/">Главная</a></li>
      <li><a href="/sitemap">Карта сайта</a></li>
    </ul>
  </nav>
</header>
```

## Заключение

Карта сайта - важный элемент навигации, который улучшает пользовательский опыт и помогает поисковым роботам индексировать ваш сайт. Правильно реализованная карта сайта с использованием семантических элементов HTML, ARIA-атрибутов и соблюдением принципов доступности делает сайт более удобным для всех пользователей.
