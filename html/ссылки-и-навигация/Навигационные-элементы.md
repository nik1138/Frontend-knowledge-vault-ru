# Навигационные элементы в HTML

## Обзор

Навигационные элементы позволяют пользователям перемещаться по веб-сайту и находить нужную информацию. В HTML существуют специальные семантические элементы, которые обозначают различные типы навигации, что улучшает доступность и SEO.

## Семантические навигационные элементы

### 1. `<nav>`

Элемент `<nav>` обозначает раздел, содержащий навигационные ссылки:

```html
<nav>
  <ul>
    <li><a href="/">Главная</a></li>
    <li><a href="/about">О нас</a></li>
    <li><a href="/services">Услуги</a></li>
    <li><a href="/contact">Контакты</a></li>
  </ul>
</nav>
```

### 2. `<header>`

Элемент `<header>` может содержать навигационные элементы в верхней части страницы:

```html
<header>
  <h1>Мой сайт</h1>
  <nav>
    <ul>
      <li><a href="/">Главная</a></li>
      <li><a href="/blog">Блог</a></li>
    </ul>
  </nav>
</header>
```

### 3. `<footer>`

Элемент `<footer>` может содержать навигационные ссылки в нижней части страницы:

```html
<footer>
  <nav>
    <ul>
      <li><a href="/privacy">Политика конфиденциальности</a></li>
      <li><a href="/terms">Условия использования</a></li>
    </ul>
  </nav>
  <p>&copy; 2025 Мой сайт</p>
</footer>
```

## Типы навигации

### 1. Основная навигация

Основная навигация сайта, обычно размещается в шапке:

```html
<nav aria-label="Основная навигация">
  <ul>
    <li><a href="/">Главная</a></li>
    <li><a href="/products">Продукты</a></li>
    <li><a href="/about">О компании</a></li>
    <li><a href="/contacts">Контакты</a></li>
  </ul>
</nav>
```

### 2. Боковая навигация

Навигация в боковой панели (сайдбаре):

```html
<aside>
  <nav aria-label="Навигация по разделам">
    <h3>Разделы</h3>
    <ul>
      <li><a href="/category/tech">Технологии</a></li>
      <li><a href="/category/design">Дизайн</a></li>
      <li><a href="/category/business">Бизнес</a></li>
    </ul>
  </nav>
</aside>
```

### 3. Пагинация

Навигация по страницам контента:

```html
<nav aria-label="Навигация по страницам">
  <ul class="pagination">
    <li><a href="?page=1" aria-label="Первая страница">««</a></li>
    <li><a href="?page=1" aria-label="Предыдущая страница">«</a></li>
    <li><a href="?page=1">1</a></li>
    <li><a href="?page=2" aria-current="page">2</a></li>
    <li><a href="?page=3">3</a></li>
    <li><a href="?page=3" aria-label="Следующая страница">»</a></li>
    <li><a href="?page=3" aria-label="Последняя страница">»»</a></li>
  </ul>
</nav>
```

### 4. Пошаговая навигация

Для многошаговых процессов (например, оформление заказа):

```html
<nav aria-label="Пошаговая навигация">
  <ol>
    <li><a href="/cart">Корзина</a></li>
    <li><a href="/delivery">Доставка</a></li>
    <li><a href="/payment" aria-current="page">Оплата</a></li>
    <li>Подтверждение</li>
  </ol>
</nav>
```

## Структура навигационных списков

### Вертикальное меню

```html
<nav>
  <ul class="vertical-menu">
    <li><a href="/home">Главная</a></li>
    <li><a href="/products">Продукты</a>
      <ul>
        <li><a href="/products/electronics">Электроника</a></li>
        <li><a href="/products/clothing">Одежда</a></li>
      </ul>
    </li>
    <li><a href="/services">Услуги</a></li>
    <li><a href="/contact">Контакты</a></li>
  </ul>
</nav>
```

### Горизонтальное меню

```html
<nav>
  <ul class="horizontal-menu">
    <li><a href="/home">Главная</a></li>
    <li><a href="/about">О нас</a></li>
    <li><a href="/services">Услуги</a></li>
    <li><a href="/portfolio">Портфолио</a></li>
    <li><a href="/blog">Блог</a></li>
    <li><a href="/contact">Контакты</a></li>
  </ul>
</nav>
```

## Атрибуты доступности

### ARIA-атрибуты для навигации

```html
<nav aria-label="Основная навигация" role="navigation">
  <ul>
    <li><a href="/" aria-current="page">Главная</a></li>
    <li><a href="/products">Продукты</a></li>
    <li><a href="/services">Услуги</a></li>
  </ul>
</nav>
```

### Атрибут `aria-current`

Указывает текущую активную страницу или элемент:

```html
<!-- Для страницы -->
<a href="/current-page" aria-current="page">Текущая страница</a>

<!-- Для шага в процессе -->
<a href="/step-2" aria-current="step">Шаг 2</a>

<!-- Для выбранного элемента -->
<a href="/item-3" aria-current="true">Элемент 3</a>
```

## Стилизация навигационных элементов

### Базовая стилизация меню

```css
nav ul {
  list-style: none;
  padding: 0;
  margin: 0;
  display: flex;
  flex-wrap: wrap;
}

nav li {
  margin-right: 1rem;
}

nav a {
  text-decoration: none;
  padding: 0.5rem 1rem;
  display: block;
  transition: background-color 0.3s;
}

nav a:hover,
nav a:focus {
  background-color: #f0f0f0;
  outline: 2px solid #007cba;
  outline-offset: -2px;
}

nav a[aria-current="page"] {
  background-color: #007cba;
  color: white;
}
```

## Лучшие практики

### 1. Использование семантических элементов

```html
<!-- Хорошо -->
<nav aria-label="Основная навигация">
  <ul>
    <li><a href="/">Главная</a></li>
  </ul>
</nav>

<!-- Не рекомендуется -->
<div>
  <span><a href="/">Главная</a></span>
</div>
```

### 2. Описание навигационных элементов

```html
<nav aria-label="Основная навигация">
  <ul>
    <li><a href="/">Главная страница сайта</a></li>
    <li><a href="/products">Каталог продуктов</a></li>
  </ul>
</nav>
```

### 3. Консистентность навигации

Навигация должна быть последовательной на всех страницах сайта:

- Одинаковое расположение
- Одинаковые названия пунктов
- Одинаковый визуальный стиль

### 4. Мобильная адаптация

```html
<nav aria-label="Основная навигация">
  <button class="menu-toggle" aria-expanded="false" aria-controls="main-menu">
    Меню
  </button>
  <ul id="main-menu">
    <li><a href="/">Главная</a></li>
    <li><a href="/products">Продукты</a></li>
    <li><a href="/services">Услуги</a></li>
  </ul>
</nav>
```

## Современные подходы

### Навигация с выпадающими меню

```html
<nav>
  <ul>
    <li><a href="/products">Продукты</a>
      <ul class="dropdown">
        <li><a href="/products/electronics">Электроника</a></li>
        <li><a href="/products/clothing">Одежда</a></li>
        <li><a href="/products/books">Книги</a></li>
      </ul>
    </li>
    <li><a href="/services">Услуги</a></li>
  </ul>
</nav>
```

### Таб-навигация

```html
<nav role="tablist">
  <button role="tab" aria-selected="true" aria-controls="tab1">Вкладка 1</button>
  <button role="tab" aria-selected="false" aria-controls="tab2">Вкладка 2</button>
</nav>

<section id="tab1" role="tabpanel">
  <p>Содержимое первой вкладки</p>
</section>

<section id="tab2" role="tabpanel" hidden>
  <p>Содержимое второй вкладки</p>
</section>
```

## Доступность

### Клавиатурная навигация

- Обеспечьте логичный порядок фокуса
- Визуально выделяйте элементы при фокусе
- Используйте `tabindex` при необходимости

### Скринридеры

- Используйте ARIA-атрибуты для улучшения восприятия
- Обеспечьте контекст для навигационных элементов
- Правильно используйте `aria-current`

## Заключение

Навигационные элементы играют ключевую роль в создании удобного и доступного веб-сайта. Использование семантически правильных HTML-элементов, ARIA-атрибутов и соблюдение принципов доступности позволяет создать эффективную систему навигации, которая будет работать для всех пользователей.
