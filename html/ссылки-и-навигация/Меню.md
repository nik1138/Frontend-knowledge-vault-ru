# Меню в HTML

## Обзор

Меню в HTML - это структурированная коллекция ссылок, позволяющая пользователям перемещаться по веб-сайту. Правильная реализация меню важна для удобства использования, доступности и SEO.

## Типы меню

### 1. Основное меню (Главное меню)

Наиболее важное меню сайта, обычно размещается в шапке:

```html
<nav aria-label="Основное меню">
  <ul>
    <li><a href="/">Главная</a></li>
    <li><a href="/about">О нас</a></li>
    <li><a href="/services">Услуги</a></li>
    <li><a href="/portfolio">Портфолио</a></li>
    <li><a href="/blog">Блог</a></li>
    <li><a href="/contact">Контакты</a></li>
  </ul>
</nav>
```

### 2. Боковое меню (Сайдбар)

Дополнительная навигация, обычно размещается в боковой панели:

```html
<aside>
  <nav aria-label="Боковое меню">
    <h3>Категории</h3>
    <ul>
      <li><a href="/category/tech">Технологии</a></li>
      <li><a href="/category/design">Дизайн</a></li>
      <li><a href="/category/business">Бизнес</a></li>
      <li><a href="/category/lifestyle">Образ жизни</a></li>
    </ul>
  </nav>
</aside>
```

### 3. Выпадающее меню

Меню с вложенными подпунктами:

```html
<nav aria-label="Выпадающее меню">
  <ul class="dropdown-menu">
    <li><a href="/products">Продукты</a>
      <ul class="submenu">
        <li><a href="/products/electronics">Электроника</a></li>
        <li><a href="/products/clothing">Одежда</a></li>
        <li><a href="/products/books">Книги</a></li>
      </ul>
    </li>
    <li><a href="/services">Услуги</a>
      <ul class="submenu">
        <li><a href="/services/consulting">Консалтинг</a></li>
        <li><a href="/services/support">Поддержка</a></li>
      </ul>
    </li>
    <li><a href="/about">О нас</a></li>
  </ul>
</nav>
```

### 4. Мобильное меню

Адаптированное меню для мобильных устройств:

```html
<nav aria-label="Мобильное меню">
  <button class="mobile-menu-toggle" aria-expanded="false" aria-controls="mobile-menu">
    <span>Меню</span>
    <span class="hamburger-icon"></span>
  </button>
  <ul id="mobile-menu" hidden>
    <li><a href="/">Главная</a></li>
    <li><a href="/products">Продукты</a></li>
    <li><a href="/services">Услуги</a></li>
    <li><a href="/about">О нас</a></li>
    <li><a href="/contact">Контакты</a></li>
  </ul>
</nav>
```

## Структура меню

### Базовая структура

```html
<nav aria-label="Название меню">
  <ul>
    <li><a href="/link1">Ссылка 1</a></li>
    <li><a href="/link2">Ссылка 2</a></li>
    <li><a href="/link3">Ссылка 3</a></li>
  </ul>
</nav>
```

### Многоуровневое меню

```html
<nav aria-label="Многоуровневое меню">
  <ul>
    <li><a href="/category1">Категория 1</a>
      <ul>
        <li><a href="/subcategory1-1">Подкатегория 1.1</a></li>
        <li><a href="/subcategory1-2">Подкатегория 1.2</a></li>
      </ul>
    </li>
    <li><a href="/category2">Категория 2</a>
      <ul>
        <li><a href="/subcategory2-1">Подкатегория 2.1</a></li>
        <li><a href="/subcategory2-2">Подкатегория 2.2</a></li>
      </ul>
    </li>
  </ul>
</nav>
```

## Стилизация меню

### Горизонтальное меню

```css
.horizontal-menu {
  display: flex;
  list-style: none;
  padding: 0;
  margin: 0;
  background-color: #f8f9fa;
  border-bottom: 1px solid #dee2e6;
}

.horizontal-menu li {
  position: relative;
}

.horizontal-menu a {
  display: block;
  padding: 1rem 1.5rem;
  text-decoration: none;
  color: #212529;
  transition: background-color 0.3s;
}

.horizontal-menu a:hover,
.horizontal-menu a:focus {
  background-color: #e9ecef;
  outline: 2px solid #007cba;
  outline-offset: -2px;
}

.horizontal-menu a[aria-current="page"] {
  background-color: #007cba;
  color: white;
}
```

### Вертикальное меню

```css
.vertical-menu {

.vertical-menu a:hover,
.vertical-menu a:focus {
  background-color: #e9ecef;
  outline: 2px solid #007cba;
  outline-offset: -2px;

```

### Выпадающее меню

```css
.dropdown-menu {
  list-style: none;
  padding: 0;
  margin: 0;
  display: flex;
}

.dropdown-menu > li {
  position: relative;
}

.dropdown-menu a {
  display: block;
  padding: 1rem 1.5rem;
  text-decoration: none;
  color: #212529;
}

.dropdown-menu .submenu {
  position: absolute;
  top: 100%;
  left: 0;
  background-color: white;
  list-style: none;
  padding: 0;
  margin: 0;
```

## Доступность меню

### ARIA-атрибуты

```html
<nav aria-label="Основное меню" role="navigation">
  <ul role="menubar">
    <li role="none">
      <a href="/" role="menuitem" aria-current="page">Главная</a>
    </li>
    <li role="none">
      <a href="/products" role="menuitem">Продукты</a>
      <ul role="menu" aria-label="Подменю продуктов">
        <li role="none">
          <a href="/products/electronics" role="menuitem">Электроника</a>
        </li>
        <li role="none">
          <a href="/products/clothing" role="menuitem">Одежда</a>
        </li>
      </ul>
    </li>
  </ul>
</nav>
```

### Клавиатурная навигация

Для полноценной клавиатурной навигации по меню может потребоваться JavaScript:

```javascript
document.querySelectorAll('.dropdown-menu a').forEach(item => {
  item.addEventListener('keydown', function(e) {
    if (e.key === 'Enter' || e.key === ' ') {
      // Обработка нажатия Enter или Space
      window.location.href = this.href;
    }
  });
});
```

## Адаптивное меню

### Мобильное меню с JavaScript

```html
<nav aria-label="Навигация">
  <div class="nav-container">
    <a href="/" class="logo">Мой сайт</a>
    <button class="menu-toggle" aria-expanded="false" aria-controls="main-menu">
      <span class="sr-only">Открыть меню</span>
      <span class="hamburger"></span>
      <span class="hamburger"></span>
      <span class="hamburger"></span>
    </button>
    <ul id="main-menu" class="main-menu">
      <li><a href="/">Главная</a></li>
      <li><a href="/products">Продукты</a></li>
      <li><a href="/services">Услуги</a></li>
      <li><a href="/about">О нас</a></li>
      <li><a href="/contact">Контакты</a></li>
    </ul>
  </div>
</nav>
```

```css
.menu-toggle {
  display: none;
  flex-direction: column;
  background: none;
  border: none;
  cursor: pointer;
  padding: 0.5rem;
}

.hamburger {
  width: 25px;
  height: 3px;
  background-color: #333;
  margin: 3px 0;
  transition: 0.3s;
}

@media (max-width: 768px) {
  .menu-toggle {
    display: flex;
  }
  
  .main-menu {
    position: fixed;
    left: -100%;
    top: 70px;
    flex-direction: column;
    background-color: white;
    width: 100%;
    transition: 0.3s;
    box-shadow: 0 10px 27px rgba(0, 0, 0, 0.05);
  }
  
  .main-menu.active {
    left: 0;
  }
}
```

```javascript
const menuToggle = document.querySelector('.menu-toggle');
const mainMenu = document.querySelector('.main-menu');

menuToggle.addEventListener('click', function() {
  const isExpanded = this.getAttribute('aria-expanded') === 'true';
  this.setAttribute('aria-expanded', !isExpanded);
  mainMenu.classList.toggle('active');
});
```

## Лучшие практики

### 1. Использование семантических элементов

```html
<!-- Хорошо -->
<nav aria-label="Основное меню">
  <ul>
    <li><a href="/">Главная</a></li>
  </ul>
</nav>

<!-- Не рекомендуется -->
<div class="menu">
  <span><a href="/">Главная</a></span>
</div>
```

### 2. Осмысленные тексты ссылок

```html
<!-- Хорошо -->
<ul>
  <li><a href="/products">Каталог продуктов</a></li>
  <li><a href="/services">Наши услуги</a></li>
</ul>

<!-- Не рекомендуется -->
<ul>
  <li><a href="/products">Здесь</a></li>
  <li><a href="/services">Нажмите</a></li>
</ul>
```

### 3. Индикация текущей страницы

```html
<nav aria-label="Основное меню">
  <ul>
    <li><a href="/" aria-current="page">Главная</a></li>
    <li><a href="/about">О нас</a></li>
  </ul>
</nav>
```

### 4. Контрастность и размер

- Обеспечьте достаточный контраст текста меню
- Сделайте элементы меню достаточно большими для удобного нажатия (минимум 44x44 пикселя)

### 5. Последовательность навигации

Сохраняйте логичный порядок навигации по всему сайту.

## Современные подходы

### Мега-меню

Для сайтов с большим количеством разделов:

```html
<nav aria-label="Мега-меню">
  <ul class="mega-menu">
    <li class="dropdown">
      <a href="/products">Продукты</a>
      <div class="mega-menu-content">
        <div class="mega-menu-column">
          <h4>Электроника</h4>
          <ul>
            <li><a href="/electronics/phones">Телефоны</a></li>
            <li><a href="/electronics/laptops">Ноутбуки</a></li>
          </ul>
        </div>
        <div class="mega-menu-column">
          <h4>Одежда</h4>
          <ul>
            <li><a href="/clothing/men">Мужская одежда</a></li>
            <li><a href="/clothing/women">Женская одежда</a></li>
          </ul>
        </div>
      </div>
    </li>
  </ul>
</nav>
```

### Меню с иконками

```html
<nav aria-label="Меню с иконками">
  <ul>
    <li><a href="/home">
      <svg class="icon"><use href="#home-icon"></use></svg>
      <span>Главная</span>
    </a></li>
    <li><a href="/search">
      <svg class="icon"><use href="#search-icon"></use></svg>
      <span>Поиск</span>
    </a></li>
  </ul>
</nav>
```

## Заключение

Меню являются ключевым элементом навигации по веб-сайту. Их правильная реализация с использованием семантических элементов HTML, ARIA-атрибутов и соблюдением принципов доступности позволяет создать удобный и интуитивно понятный интерфейс для всех пользователей.
