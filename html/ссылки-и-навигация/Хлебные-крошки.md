# Хлебные крошки в HTML

## Обзор

Хлебные крошки (breadcrumb navigation) - это навигационный элемент, который показывает пользователю текущее местоположение на сайте в иерархии страниц. Они особенно полезны для многоуровневых сайтов с глубокой структурой.

## Структура хлебных крошек

### Базовая структура

```html
<nav aria-label="Навигационная цепочка" role="navigation">
  <ol>
    <li><a href="/">Главная</a></li>
    <li><a href="/category">Категория</a></li>
    <li><a href="/subcategory">Подкатегория</a></li>
    <li aria-current="page">Текущая страница</li>
  </ol>
</nav>
```

### Структура с Schema.org разметкой

```html
<nav aria-label="Навигационная цепочка" vocab="https://schema.org/" typeof="BreadcrumbList">
  <ol>
    <li property="itemListElement" typeof="ListItem">
      <a property="item" typeof="WebPage" href="/">
        <span property="name">Главная</span>
      </a>
      <meta property="position" content="1">
    </li>
    <li property="itemListElement" typeof="ListItem">
      <a property="item" typeof="WebPage" href="/category">
        <span property="name">Категория</span>
      </a>
      <meta property="position" content="2">
    </li>
    <li property="itemListElement" typeof="ListItem">
      <a property="item" typeof="WebPage" href="/category/subcategory">
        <span property="name">Подкатегория</span>
      </a>
      <meta property="position" content="3">
    </li>
    <li property="itemListElement" typeof="ListItem">
      <span property="name" aria-current="page">Текущая страница</span>
      <meta property="position" content="4">
    </li>
  </ol>
</nav>
```

## Типы хлебных крошек

### 1. Показывающие иерархию

Показывают структуру сайта:

```html
<nav aria-label="Иерархия сайта">
  <ol>
    <li><a href="/">Мой сайт</a></li>
    <li><a href="/products">Продукты</a></li>
    <li><a href="/products/electronics">Электроника</a></li>
    <li><a href="/products/electronics/smartphones">Смартфоны</a></li>
    <li aria-current="page">Смартфон XYZ</li>
  </ol>
</nav>
```

### 2. Показывающие путь к текущей странице

Отображают фактический путь, по которому пользователь попал на страницу:

```html
<nav aria-label="Путь к странице">
  <ol>
    <li><a href="/">Главная</a></li>
    <li><a href="/search">Поиск</a></li>
    <li><a href="/search?q=смартфон">Результаты поиска</a></li>
    <li aria-current="page">Смартфон ABC</li>
  </ol>
</nav>
```

### 3. Атрибутивные хлебные крошки

Показывают атрибуты текущего элемента:

```html
<nav aria-label="Атрибуты товара">
  <ol>
    <li><a href="/">Главная</a></li>
    <li><a href="/products">Продукты</a></li>
    <li><a href="/products/electronics">Электроника</a></li>
    <li>Цвет: Черный</li>
    <li>Объем: 256 ГБ</li>
    <li aria-current="page">Смартфон XYZ</li>
  </ol>
</nav>
```

## Семантическая разметка

### Использование `<nav>` и `<ol>`

```html
<nav aria-label="Навигационная цепочка">
  <ol>
    <li><a href="/">Главная</a></li>
    <li><a href="/blog">Блог</a></li>
    <li><a href="/blog/tech">Технологии</a></li>
    <li aria-current="page">Статья о веб-разработке</li>
  </ol>
</nav>
```

### Атрибут `aria-current`

Используется для обозначения текущей страницы:

```html
<!-- Для текущей страницы -->
<li aria-current="page">Текущая страница</li>

<!-- Для текущего шага в процессе -->
<li aria-current="step">Текущий шаг</li>

<!-- Для текущего элемента в коллекции -->
<li aria-current="true">Текущий элемент</li>
```

## Стилизация хлебных крошек

### Базовая стилизация

```css
.breadcrumb {
  padding: 0.75rem 1rem;
  margin-bottom: 1rem;
  list-style: none;
  background-color: #e9ecef;
  border-radius: 0.25rem;
  font-size: 0.875rem;
}

.breadcrumb ol {
  display: flex;
  flex-wrap: wrap;
  padding: 0;
  margin-bottom: 0;
  list-style: none;
}

.breadcrumb li:not(:last-child)::after {
  content: " / ";
  padding: 0 0.5rem;
  color: #6c757d;
}

.breadcrumb a {
  text-decoration: none;
  color: #007bff;
}

.breadcrumb a:hover {
  text-decoration: underline;
}

.breadcrumb [aria-current] {
  color: #6c757d;
}
```

### Альтернативные разделители

```css
/* Стрелки */
.breadcrumb li:not(:last-child)::after {
  content: " → ";
}

/* Угловые скобки */
.breadcrumb li:not(:last-child)::after {
  content: " › ";
}

/* Путь */
.breadcrumb li:not(:last-child)::after {
  content: " > ";
}
```

### Адаптивная стилизация

```css
.breadcrumb {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

@media (max-width: 768px) {
  .breadcrumb {
    font-size: 0.75rem;
  }
  
  .breadcrumb li:not(:last-child) {
    display: none;
  }
  
  .breadcrumb li:first-child,
  .breadcrumb li:last-child {
    display: inline;
  }
  
  .breadcrumb li:not(:last-child)::after {
    content: " / ";
  }
  
  .breadcrumb li:nth-last-child(2)::after {
    content: " / ";
  }
}
```

## Доступность

### ARIA-атрибуты

```html
<nav aria-label="Навигационная цепочка" role="navigation">
  <ol>
    <li><a href="/">Главная</a></li>
    <li><a href="/category">Категория</a></li>
    <li aria-current="page">Текущая страница</li>
  </ol>
</nav>
```

### Альтернативные тексты

Для изображений в хлебных крошках:

```html
<nav aria-label="Навигационная цепочка">
  <ol>
    <li><a href="/">
      <img src="/home-icon.svg" alt="Главная">
    </a></li>
    <li><a href="/category">Категория</a></li>
    <li aria-current="page">Текущая страница</li>
  </ol>
</nav>
```

## SEO-оптимизация

### Schema.org разметка

```html
<nav itemscope itemtype="http://schema.org/BreadcrumbList">
  <ol>
    <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem">
      <a itemprop="item" href="/">
        <span itemprop="name">Главная</span>
      </a>
      <meta itemprop="position" content="1" />
    </li>
    <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem">
      <a itemprop="item" href="/category">
        <span itemprop="name">Категория</span>
      </a>
      <meta itemprop="position" content="2" />
```

## Лучшие практики

### 1. Использование подходящих элементов

```html
<!-- Хорошо -->
<nav aria-label="Навигационная цепочка">
  <ol>
    <li><a href="/">Главная</a></li>
  </ol>
</nav>

<!-- Не рекомендуется -->
<div class="breadcrumb">
  <span>Главная</span> / <span>Категория</span> / <span>Текущая страница</span>
</div>
```

### 2. Показывать только значимые уровни

Не включайте каждый промежуточный URL-адрес:

```html
<!-- Хорошо -->
<ol>
  <li><a href="/">Главная</a></li>
  <li><a href="/products">Продукты</a></li>
  <li aria-current="page">Смартфон</li>
</ol>

<!-- Не рекомендуется (слишком много уровней) -->
<ol>
  <li><a href="/">Главная</a></li>
  <li><a href="/category">Категория</a></li>
  <li><a href="/subcategory">Подкатегория</a></li>
  <li><a href="/subsubcategory">Под-подкатегория</a></li>
  <li aria-current="page">Смартфон</li>
</ol>
```

### 3. Использовать понятные названия

```html
<!-- Хорошо -->
<ol>
  <li><a href="/">Главная</a></li>
  <li><a href="/products">Продукты</a></li>
  <li aria-current="page">Смартфон XYZ</li>
</ol>

<!-- Не рекомендуется -->
<ol>
  <li><a href="/">Главная</a></li>
  <li><a href="/p">Продукты</a></li>
  <li aria-current="page">pr_12345</li>
</ol>
```

### 4. Обновлять при изменении контента

Хлебные крошки должны отражать актуальное местоположение пользователя.

### 5. Не использовать на главной странице

Хлебные крошки не обязательны на главной странице сайта.

## Современные подходы

### Интерактивные хлебные крошки

С выпадающими меню для каждого уровня:

```html
<nav aria-label="Интерактивная навигационная цепочка">
  <ol>
    <li class="dropdown">
      <a href="/">Главная</a>
      <ul class="dropdown-menu">
        <li><a href="/category1">Категория 1</a></li>
        <li><a href="/category2">Категория 2</a></li>
      </ul>
    </li>
    <li class="dropdown">
      <a href="/products">Продукты</a>
      <ul class="dropdown-menu">
        <li><a href="/products/electronics">Электроника</a></li>
        <li><a href="/products/clothing">Одежда</a></li>
      </ul>
    </li>
    <li aria-current="page">Смартфон</li>
  </ol>
</nav>
```

### Хлебные крошки с иконками

```html
<nav aria-label="Навигационная цепочка">
  <ol class="breadcrumb-icons">
    <li><a href="/"><span class="icon-home"></span>Главная</a></li>
    <li><a href="/products"><span class="icon-products"></span>Продукты</a></li>
    <li aria-current="page"><span class="icon-smartphone"></span>Смартфон</li>
  </ol>
</nav>
```

## Заключение

Хлебные крошки - важный элемент навигации, который помогает пользователям понимать структуру сайта и легко перемещаться по нему. Их правильная реализация с использованием семантических элементов HTML, ARIA-атрибутов и Schema.org разметки улучшает как пользовательский опыт, так и SEO-показатели сайта.
