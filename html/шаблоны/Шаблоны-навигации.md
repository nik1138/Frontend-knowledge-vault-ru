# Шаблоны навигации

## Общие шаблоны навигации

Навигация - один из ключевых элементов веб-сайта, обеспечивающий пользователю возможность перемещения по страницам. В 2025 году в российской веб-разработке особое внимание уделяется доступности, адаптивности и юзабилити навигационных элементов.

        const mobileToggle = document.querySelector('.mobile-toggle');
        const navMenu = document.querySelector('.nav-menu');

            const isExpanded = this.getAttribute('aria-expanded') === 'true';
            this.setAttribute('aria-expanded', !isExpanded);
            navMenu.classList.toggle('active');
            
            // Изменение текста aria-label в зависимости от состояния
            this.setAttribute('aria-label', isExpanded ? 'Открыть меню' : 'Закрыть меню');
        });
        
        // Закрытие меню при клике на ссылку (на мобильных устройствах)
        document.querySelectorAll('.nav-menu a').forEach(link => {
            link.addEventListener('click', () => {
                if (window.innerWidth <= 768) {
                    mobileToggle.setAttribute('aria-expanded', 'false');
                    navMenu.classList.remove('active');
                }
            });
        });
        
        // Закрытие меню при изменении размера экрана
        window.addEventListener('resize', () => {
            if (window.innerWidth > 768) {
                navMenu.classList.remove('active');
                mobileToggle.setAttribute('aria-expanded', 'false');
            }
        });
    </script>
</body>
</html>

            <li><a href="/dashboard" aria-current="page">Панель управления</a></li>
            <li><a href="/profile">Профиль</a></li>
            <li><a href="/settings">Настройки</a></li>
            <li><a href="/messages">Сообщения</a></li>
            <li><a href="/documents">Документы</a></li>
            <li><a href="/reports">Отчеты</a></li>
            <li><a href="/calendar">Календарь</a></li>

    <script>
        const sidebar = document.querySelector('.sidebar');
        const toggleBtn = document.querySelector('.mobile-sidebar-toggle');
        
        toggleBtn.addEventListener('click', () => {
            sidebar.classList.toggle('active');
            
            const isActive = sidebar.classList.contains('active');
            toggleBtn.setAttribute('aria-label', isActive ? 'Закрыть меню' : 'Открыть меню');
        });
        
        // Закрытие меню при клике вне его области (на мобильных устройствах)
        document.addEventListener('click', (e) => {
            if (window.innerWidth <= 768 && 
                !sidebar.contains(e.target) && 
                e.target !== toggleBtn &&
                sidebar.classList.contains('active')) {
                sidebar.classList.remove('active');
                toggleBtn.setAttribute('aria-label', 'Открыть меню');
            }
        });
    </script>

## Навигация по вкладкам

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Навигация по вкладкам</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
        
        body {
            font-family: Arial, sans-serif;
            padding: 20px;
        }
        
        .tabs {
            display: flex;
            border-bottom: 1px solid #dee2e6;
            margin-bottom: 20px;
        }
        
        .tab {
            padding: 12px 20px;
            cursor: pointer;
            background-color: #f8f9fa;
            border: 1px solid transparent;
            border-bottom: none;
            margin-right: 5px;
            border-top-left-radius: 5px;
            border-top-right-radius: 5px;
        }
        
        .tab:hover {
            background-color: #e9ecef;
        }
        
        .tab.active {
            background-color: white;
            border-color: #dee2e6;
            border-bottom: 1px solid white;
            margin-bottom: -1px;
            font-weight: bold;
        }
        
        .tab-content {
            display: none;
            padding: 20px;
            border: 1px solid #dee2e6;
            border-top: none;
            border-radius: 0 0 5px 5px;
        }
        
        .tab-content.active {
            display: block;
        }
        
        /* Стили для доступности */
        .tab:focus {
            outline: 2px solid #007bff;
            outline-offset: -2px;
        }
    </style>
</head>
<body>
    <h1>Навигация по вкладкам</h1>
    
    <div class="tabs-container">
        <div class="tabs" role="tablist">
            <button 
                class="tab active" 
                role="tab" 
                aria-selected="true" 
                aria-controls="panel-1" 
                id="tab-1"
            >
                Общая информация
            </button>
            <button 
                class="tab" 
                role="tab" 
                aria-selected="false" 
                aria-controls="panel-2" 
                id="tab-2"
                tabindex="-1"
            >
                Детали
            </button>
            <button 
                class="tab" 
                role="tab" 
                aria-selected="false" 
                aria-controls="panel-3" 
                id="tab-3"
                tabindex="-1"
            >
                Документы
            </button>
            <button 
                class="tab" 
                role="tab" 
                aria-selected="false" 
                aria-controls="panel-4" 
                id="tab-4"
                tabindex="-1"
            >
                Комментарии
            </button>
        </div>
        
        <div class="tab-content active" role="tabpanel" id="panel-1" aria-labelledby="tab-1">
            <h2>Общая информация</h2>
            <p>Содержимое первой вкладки. Здесь отображается общая информация.</p>
        </div>
        
        <div class="tab-content" role="tabpanel" id="panel-2" aria-labelledby="tab-2" hidden>
            <h2>Детали</h2>
            <p>Содержимое второй вкладки. Здесь отображаются подробные детали.</p>
        </div>
        
        <div class="tab-content" role="tabpanel" id="panel-3" aria-labelledby="tab-3" hidden>
            <h2>Документы</h2>
            <p>Содержимое третьей вкладки. Здесь отображаются связанные документы.</p>
        </div>
        
        <div class="tab-content" role="tabpanel" id="panel-4" aria-labelledby="tab-4" hidden>
            <h2>Комментарии</h2>
            <p>Содержимое четвертой вкладки. Здесь отображаются комментарии пользователей.</p>
        </div>
    </div>

    <script>
        // Обработчики для вкладок
        const tabs = document.querySelectorAll('.tab');
        const tabPanels = document.querySelectorAll('.tab-content');
        
        tabs.forEach(tab => {
            tab.addEventListener('click', () => {
                // Сброс активной вкладки
                tabs.forEach(t => {
                    t.classList.remove('active');
                    t.setAttribute('aria-selected', 'false');
                    t.setAttribute('tabindex', '-1');
                });
                
                // Сброс активной панели
                tabPanels.forEach(panel => {
                    panel.classList.remove('active');
                    panel.hidden = true;
                });
                
                // Установка новой активной вкладки
                tab.classList.add('active');
                tab.setAttribute('aria-selected', 'true');
                tab.setAttribute('tabindex', '0');
                
                // Установка новой активной панели
                const panelId = tab.getAttribute('aria-controls');
                const activePanel = document.getElementById(panelId);
                activePanel.classList.add('active');
                activePanel.hidden = false;
                
                // Фокус на активной вкладке
                tab.focus();
            });
            
            // Обработка клавиатурной навигации
            tab.addEventListener('keydown', (e) => {
                let targetIndex;
                
                switch(e.key) {
                    case 'ArrowLeft':
                        targetIndex = Array.from(tabs).indexOf(tab) - 1;
                        if (targetIndex < 0) targetIndex = tabs.length - 1;
                        tabs[targetIndex].focus();
                        tabs[targetIndex].click();
                        break;
                    case 'ArrowRight':
                        targetIndex = Array.from(tabs).indexOf(tab) + 1;
                        if (targetIndex >= tabs.length) targetIndex = 0;
                        tabs[targetIndex].focus();
                        tabs[targetIndex].click();
                        break;
                    case 'Home':
                        tabs[0].focus();
                        tabs[0].click();
                        break;
                    case 'End':
                        tabs[tabs.length - 1].focus();
                        tabs[tabs.length - 1].click();
                        break;
                }
            });
        });
    </script>
</body>
</html>
```

## Практические рекомендации

1. **Доступность**: Используйте семантические элементы и ARIA-атрибуты для обеспечения доступности навигации.

2. **Адаптивность**: Обеспечьте корректное отображение навигации на всех устройствах, особенно на мобильных.

3. **Интуитивность**: Структура навигации должна быть логичной и понятной пользователям.

4. **Контрастность**: Обеспечьте достаточный контраст между текстом и фоном для удобства чтения.

5. **Клавиатурная навигация**: Проверьте, что все элементы навигации доступны через клавиатуру.

6. **Скорость загрузки**: Оптимизируйте навигационные элементы для быстрой загрузки страницы.
