# Базовые шаблоны HTML

## Общее понимание HTML-шаблонов

HTML-шаблоны - это заранее подготовленные структуры HTML-документов, которые служат основой для создания новых веб-страниц. Они обеспечивают согласованность, улучшают производительность разработки и помогают соблюдать лучшие практики веб-стандартов.

В 2025 году, с учетом российских реалий, важно учитывать требования к локализации, доступности и производительности, особенно при работе с ограниченной пропускной способностью интернета в отдаленных регионах.

## Стандартный HTML5 шаблон

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Заголовок страницы</title>
    <meta name="description" content="Описание страницы">
    <!-- Подключение CSS -->
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Заголовок страницы</h1>
    </header>
    
    <main>
        <p>Основное содержимое страницы</p>
    </main>
    
    <footer>
        <p>&copy; 2025 Название компании</p>
    </footer>
    
    <!-- Подключение JavaScript -->
    <script src="script.js"></script>
</body>
</html>
```

## Шаблон с поддержкой темизации

```html
<!DOCTYPE html>
<html lang="ru" data-theme="light">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Темизированный шаблон</title>
    <meta name="description" content="Шаблон с поддержкой светлой и темной темы">
    <link rel="stylesheet" href="styles.css">
    <style>
        /* Базовые стили для темизации */
        :root {
            --bg-color: #ffffff;
            --text-color: #000000;
            --accent-color: #007bff;
        }
        
        [data-theme="dark"] {
            --bg-color: #1a1a1a;
            --text-color: #ffffff;
            --accent-color: #0d6efd;
        }
        
        body {
            background-color: var(--bg-color);
            color: var(--text-color);
            transition: background-color 0.3s, color 0.3s;
        }
    </style>
</head>
<body>
    <header>
        <nav>
            <button id="theme-toggle">Сменить тему</button>
        </nav>
    </header>
    
    <main>
        <h1>Темизированный контент</h1>
        <p>Этот контент адаптируется под выбранную тему</p>
    </main>
    
    <script>
        // Простая реализация переключения темы
        document.getElementById('theme-toggle').addEventListener('click', function() {
            const currentTheme = document.documentElement.getAttribute('data-theme');
            const newTheme = currentTheme === 'light' ? 'dark' : 'light';
            document.documentElement.setAttribute('data-theme', newTheme);
            localStorage.setItem('theme', newTheme);
        });
        
        // Восстановление темы из localStorage
        const savedTheme = localStorage.getItem('theme') || 'light';
        document.documentElement.setAttribute('data-theme', savedTheme);
    </script>
</body>
</html>
```

## Шаблон с поддержкой локализации

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title data-i18n="title">Заголовок страницы</title>
    <meta name="description" content="data-i18n=description">
    <link rel="stylesheet" href="styles.css">
    <script>
        // Простая система локализации
        const translations = {
            ru: {
                title: "Заголовок страницы",
                description: "Описание страницы на русском",
                welcome: "Добро пожаловать!"
            },
            en: {
                title: "Page Title",
                description: "Page description in English",
                welcome: "Welcome!"
            }
        };
        
        function localizePage() {
            const lang = localStorage.getItem('lang') || 'ru';
            const elements = document.querySelectorAll('[data-i18n]');
            
            elements.forEach(element => {
                const key = element.getAttribute('data-i18n');
                const translation = translations[lang][key];
                
                if (translation) {
                    if (element.tagName === 'TITLE') {
                        element.textContent = translation;
                    } else {
                        element.textContent = translation;
                    }
                }
            });
        }
        
        // Вызов функции локализации после загрузки DOM
        document.addEventListener('DOMContentLoaded', localizePage);
    </script>
</head>
<body>
    <header>
        <h1 data-i18n="welcome">Добро пожаловать!</h1>
    </header>
    
    <main>
        <p>Контент, который можно локализовать</p>
    </main>
</body>
</html>
```

## Шаблон с минимальной производительностью

Для российских реалий 2025 года, где в некоторых регионах скорость интернета может быть ограничена, важно использовать оптимизированные шаблоны:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Минималистичный шаблон</title>
    <meta name="description" content="Минималистичный шаблон для быстрой загрузки">
    <!-- Инлайн-стили для критического CSS -->
    <style>
        body { font-family: Arial, sans-serif; margin: 0; padding: 20px; }
        header { margin-bottom: 20px; }
        main { max-width: 800px; margin: 0 auto; }
    </style>
    <!-- Отложенная загрузка остальных стилей -->
    <link rel="preload" href="non-critical.css" as="style" onload="this.onload=null;this.rel='stylesheet'">
    <noscript><link rel="stylesheet" href="non-critical.css"></noscript>
</head>
<body>
    <header>
        <h1>Заголовок</h1>
    </header>
    
    <main>
        <p>Контент страницы</p>
    </main>
    
    <!-- Отложенная загрузка JavaScript -->
    <script>
        function loadScript(src, callback) {
            var script = document.createElement('script');
            script.src = src;
            script.onload = callback;
            document.head.appendChild(script);
        }
        
        // Загрузка основного скрипта после загрузки DOM
        if ('loading' in document.createElement('img')) {
            // Поддержка lazy loading
            loadScript('main.js', function() {
                console.log('Скрипт загружен');
            });
        } else {
            // Резервный вариант для старых браузеров
            var script = document.createElement('script');
            script.src = 'main.js';
            document.head.appendChild(script);
        }
    </script>
</body>
</html>
```

## Практические рекомендации

1. **Используйте семантические элементы HTML5**: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>` - они улучшают доступность и SEO.

2. **Подключайте критические стили инлайн**: Вставляйте критические CSS-правила непосредственно в `<head>`, чтобы ускорить первоначальную отрисовку.

3. **Оптимизируйте заголовки**: Используйте теги `<h1>`-`<h6>` в правильной иерархии для лучшего SEO и доступности.

4. **Добавляйте атрибуты доступности**: Используйте `aria-label`, `aria-describedby`, `role` для улучшения доступности.

5. **Учитывайте российские требования**: Включайте необходимые метатеги для соответствия требованиям российского законодательства (например, обработка персональных данных).

## Заключение

Базовые HTML-шаблоны служат фундаментом для всех веб-проектов. Правильное использование шаблонов позволяет ускорить разработку, обеспечить согласованность интерфейса и улучшить доступность веб-сайтов. В условиях российского сегмента интернета особенно важно учитывать производительность и локализацию при создании шаблонов.
