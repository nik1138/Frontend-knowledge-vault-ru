# Документация и справочники HTML

## Официальные спецификации и документация по HTML в 2025 году

### Официальные источники

#### 1. [WHATWG HTML Specification](https://html.spec.whatwg.org/)
- **Описание**: Современная, постоянно обновляемая спецификация HTML
- **Преимущества**:
  - Актуальная информация
  - Постоянные обновления
  - Официальный стандарт
- **Язык**: Английский
- **Частота обновления**: Постоянно
- **Формат**: Веб-документация

#### 2. [W3C HTML Living Standard](https://www.w3.org/TR/html/)
- **Описание**: Официальная спецификация от W3C
- **Преимущества**:
  - Официальный стандарт
  - Подробные описания
  - Рекомендации
- **Язык**: Английский
- **Формат**: Веб-документация

#### 3. [MDN Web Docs - HTML](https://developer.mozilla.org/ru/docs/Web/HTML)
- **Описание**: Самый популярный справочник для веб-разработчиков
- **Разделы**:
  - [[https://developer.mozilla.org/ru/docs/Web/HTML/Element][HTML элементы]]
  - [[https://developer.mozilla.org/ru/docs/Web/HTML/Attributes][HTML атрибуты]]
  - [[https://developer.mozilla.org/ru/docs/Web/HTML/Global_attributes][Глобальные атрибуты]]
  - [[https://developer.mozilla.org/ru/docs/Web/Guide/HTML/HTML5][HTML5]]
- **Преимущества**:
  - Русскоязычная версия
  - Примеры кода
  - Совместимость с браузерами
  - Интерактивные примеры
- **Язык**: Русский и английский
- **Формат**: Веб-документация

### Справочники и шпаргалки

#### 1. [HTML Шпаргалка от HTML Academy](https://htmlacademy.ru/blog/boost/tools/html-cheatsheet)
- **Описание**: Удобная шпаргалка по основным HTML-тегам
- **Преимущества**:
  - Удобный формат
  - Основные теги
  - Русскоязычный контент
- **Формат**: Веб-страница

#### 2. [HTML Attribute Reference](https://developer.mozilla.org/ru/docs/Web/HTML/Attributes)
- **Описание**: Полный список HTML атрибутов
- **Преимущества**:
  - Полный охват
  - Примеры использования
  - Совместимость с браузерами
- **Формат**: Веб-документация

#### 3. [HTML Global Attributes](https://developer.mozilla.org/ru/docs/Web/HTML/Global_attributes)
- **Описание**: Глобальные атрибуты, доступные всем HTML элементам
- **Преимущества**:
  - Универсальность
  - Примеры использования
  - Ссылки на спецификации
- **Формат**: Веб-документация

  - Кроссплатформенность
- **Формат**: Настольное приложение

### Практическое применение документации

#### 1. Проверка совместимости
- Используйте [[https://caniuse.com/][Can I Use]] для проверки поддержки HTML-возможностей в браузерах
- Сравнивайте спецификации с реальной поддержкой

#### 2. Поиск примеров
- MDN содержит интерактивные примеры
- Используйте примеры в своей практике

#### 3. Изучение новых возможностей
- Регулярно проверяйте WHATWG и W3C спецификации
- Следите за изменениями в HTML
