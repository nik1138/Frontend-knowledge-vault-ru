# Метатеги адаптивности

Метатеги адаптивности - это специальные HTML-теги, которые помогают браузерам и поисковым системам правильно отображать веб-страницы на различных устройствах. В условиях российского рынка 2025 года, где мобильные устройства доминируют в интернет-трафике, правильное использование метатегов становится критически важным для успеха веб-проектов.

## Основные метатеги

### Viewport метатег

Viewport метатег уже подробно рассмотрен в статье [[Viewport]], но его важность требует упоминания:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### Open Graph метатеги

Open Graph метатеги определяют, как контент будет отображаться при публикации в социальных сетях:

```html
<meta property="og:title" content="Заголовок страницы">
<meta property="og:description" content="Описание страницы">
<meta property="og:image" content="https://example.com/image.jpg">
<meta property="og:url" content="https://example.com/page">
<meta property="og:type" content="website">
```

### Twitter Card метатеги

Для корректного отображения контента в Twitter:

```html
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Заголовок страницы">
<meta name="twitter:description" content="Описание страницы">
<meta name="twitter:image" content="https://example.com/twitter-image.jpg">
```

## Метатеги для мобильных устройств

### Apple мобильные метатеги

Для корректного отображения на iOS устройствах:

```html
<!-- Значок на домашнем экране -->
<link rel="apple-touch-icon" href="/apple-touch-icon.png">

<!-- Скрытие адресной строки -->
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black">
<meta name="apple-mobile-web-app-title" content="Название приложения">

<!-- Запуск в полноэкранном режиме -->
<meta name="apple-mobile-web-app-capable" content="yes">
```

### Android метатеги

Для корректного отображения на Android устройствах:

```html
<!-- Цвет темы -->
<meta name="theme-color" content="#4285f4">

<!-- Значок для PWA -->
<link rel="manifest" href="/manifest.json">
```

## Практические рекомендации

### 1. Комплексный подход

Вот пример полного набора метатегов для адаптивной страницы:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- SEO метатеги -->
    <title>Заголовок страницы</title>
    <meta name="description" content="Описание страницы для поисковых систем">
    
    <!-- Open Graph метатеги -->
    <meta property="og:title" content="Заголовок страницы">
    <meta property="og:description" content="Описание страницы">
    <meta property="og:image" content="https://example.com/og-image.jpg">
    <meta property="og:url" content="https://example.com/page">
    <meta property="og:type" content="website">
    <meta property="og:locale" content="ru_RU">
    
    <!-- Twitter Card метатеги -->
    <meta name="twitter:card" content="summary_large_image">
    <meta name="twitter:title" content="Заголовок страницы">
    <meta name="twitter:description" content="Описание страницы">
    <meta name="twitter:image" content="https://example.com/twitter-image.jpg">
    
    <!-- Мобильные метатеги -->
    <link rel="apple-touch-icon" href="/apple-touch-icon.png">
    <meta name="apple-mobile-web-app-capable" content="yes">
    <meta name="apple-mobile-web-app-status-bar-style" content="black">
    <meta name="theme-color" content="#4285f4">
    
    <!-- Значки -->
    <link rel="icon" type="image/x-icon" href="/favicon.ico">
    <link rel="manifest" href="/manifest.json">
</head>
<body>
    <!-- Содержимое страницы -->
</body>
</html>
```

### 2. Динамическая генерация метатегов

Для динамических страниц рекомендуется генерировать метатеги программно:

```javascript
// Пример динамической генерации метатегов
function updateMetaTags(title, description, image) {
    // Обновление заголовка
    document.title = title;
    
    // Обновление метатега description
    let metaDescription = document.querySelector('meta[name="description"]');
    if (!metaDescription) {
        metaDescription = document.createElement('meta');
        metaDescription.name = 'description';
        document.head.appendChild(metaDescription);
    }
    metaDescription.content = description;
    
    // Обновление Open Graph метатегов
    let ogTitle = document.querySelector('meta[property="og:title"]');
    if (!ogTitle) {
        ogTitle = document.createElement('meta');
        ogTitle.setAttribute('property', 'og:title');
        document.head.appendChild(ogTitle);
    }
    ogTitle.content = title;
    
    // Аналогично для других метатегов...
}
```

## Современные тренды 2025 года

### 1. Поддержка PWA

Метатеги для прогрессивных веб-приложений:

```html
<link rel="manifest" href="/manifest.json">
<meta name="mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-title" content="Название PWA">
```

### 2. Адаптация под складные устройства

С учетом появления складных устройств в 2025 году, важно учитывать:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover">
```

### 3. Темизация

Поддержка светлой и темной тем:

```html
<meta name="color-scheme" content="light dark">
```

## Типичные ошибки

- **Отсутствие viewport метатега** - приводит к неправильному отображению на мобильных устройствах
- **Неправильные размеры изображений для Open Graph** - снижает эффективность шаринга
- **Устаревшие метатеги** - использование устаревших или неподдерживаемых метатегов
- **Несовпадение содержимого** - расхождение между метатегами и реальным содержимым страницы
