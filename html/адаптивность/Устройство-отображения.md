# Устройство отображения

Устройство отображения - это физическое устройство, на котором отображается веб-страница, включая его характеристики экрана, разрешение, плотность пикселей и другие параметры, влияющие на отображение контента. Понимание характеристик устройств отображения критически важно для создания действительно адаптивных веб-сайтов в условиях российского рынка 2025 года.

## Основные характеристики устройств

### Физические размеры экрана

Современные устройства представлены в широком диапазоне размеров:
- **Смартфоны**: 4-7 дюймов
- **Планшеты**: 7-13 дюймов
- **Ноутбуки**: 11-17 дюймов
- **Десктопные мониторы**: 19-32+ дюймов

### Разрешение экрана

Разрешение экрана измеряется в пикселях и определяет детализацию изображения:
- **HD (1280x720)** - устаревший стандарт
- **Full HD (1920x1080)** - стандарт для большинства устройств
- **Quad HD (2560x1440)** - распространено в флагманских смартфонах
- **4K (3840x2160)** - стандарт для высококачественных дисплеев

### Плотность пикселей (DPI/PPI)

Плотность пикселей измеряется в точках на дюйм (DPI) или пикселях на дюйм (PPI):
- **Обычные экраны**: 72-120 PPI
- **Retina/HD+ экраны**: 264-460 PPI
- **Ультра HD экраны**: 400+ PPI

## CSS Media Queries для устройств

### Мобильные устройства
```css
/* Мобильные устройства (максимальная ширина 767px) */
@media (max-width: 767px) {
    .container {
        padding: 10px;
    }
}

/* Мобильные устройства (ширина от 320px до 767px) */
@media (min-width: 320px) and (max-width: 767px) {
    .mobile-specific {
        font-size: 14px;
    }
}
```

### Планшеты
```css
/* Планшеты (ширина от 768px до 1023px) */
@media (min-width: 768px) and (max-width: 1023px) {
    .tablet-specific {
        padding: 20px;
    }
}
```

### Десктопы
```css
/* Десктопы (минимальная ширина 1024px) */
@media (min-width: 1024px) {
    .desktop-specific {
        max-width: 1200px;
        margin: 0 auto;
    }
}
```

### Высокоразрешенные экраны
```css
/* Для устройств с высокой плотностью пикселей */
@media (-webkit-min-device-pixel-ratio: 2),
       (min-resolution: 192dpi) {
    .high-dpi-image {
        background-image: url('image@2x.jpg');
    }
}
```

## Практические рекомендации

### 1. Многоточечный подход (Multi-Point Approach)

Вместо традиционного mobile-first подхода используйте многоточечный:

```css
/* Базовые стили */
.content {
    padding: 15px;
    font-size: 16px;
}

/* Мобильные устройства */
@media (max-width: 767px) {
    .content {
        padding: 10px;
        font-size: 14px;
    }
}

/* Планшеты */
@media (min-width: 768px) and (max-width: 1023px) {
    .content {
        padding: 20px;
        font-size: 15px;
    }
}

/* Десктопы */
@media (min-width: 1024px) and (max-width: 1439px) {
    .content {
        padding: 30px;
        font-size: 16px;
        max-width: 1000px;
        margin: 0 auto;
    }
}

/* Большие экраны */
@media (min-width: 1440px) {
    .content {
        padding: 40px;
        font-size: 17px;
        max-width: 1200px;
        margin: 0 auto;
    }
}
```

### 2. Адаптация под складные устройства

С учетом появления складных устройств:

```css
/* Для складных устройств */
@media (display-mode: standalone) and (max-width: 1023px) {
    .foldable-specific {
        /* Стили для складных устройств */
    }
}

/* Использование environment variables для складных устройств */
.foldable-container {
    padding: env(fold-area);
}
```

### 3. Оптимизация под плотность пикселей

```css
/* Использование изображений с разной плотностью */
.avatar {
    width: 100px;
    height: 100px;
    background-image: url('avatar.png');
}

@media (-webkit-min-device-pixel-ratio: 2),
       (min-resolution: 192dpi) {
    .avatar {
        background-image: url('avatar@2x.png');
        background-size: 100px 100px;
    }
}

@media (-webkit-min-device-pixel-ratio: 3),
       (min-resolution: 288dpi) {
    .avatar {
        background-image: url('avatar@3x.png');
        background-size: 100px 100px;
    }
}
```

## Практический пример

Пример адаптивного макета для различных устройств:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Адаптивный макет для различных устройств</title>
    <style>
        /* Базовые стили */
        body {
            margin: 0;
            padding: 0;
            font-family: Arial, sans-serif;
        }
        
        .container {
            padding: 15px;
            box-sizing: border-box;
        }
        
        .header {
            background-color: #333;
            color: white;
            padding: 20px;
            text-align: center;
        }
        
        .main-content {
            display: flex;
            flex-direction: column;
            gap: 15px;
        }
        
        .sidebar, .content {
            background-color: #f5f5f5;
            padding: 15px;
            border-radius: 8px;
        }
        
        /* Мобильные устройства */
        @media (max-width: 767px) {
            .main-content {
                flex-direction: column;
            }
            
            .container {
                padding: 10px;
            }
        }
        
        /* Планшеты */
        @media (min-width: 768px) and (max-width: 1023px) {
            .main-content {
                flex-direction: row;
            }
            
            .sidebar {
                flex: 1;
            }
            
            .content {
                flex: 3;
            }
        }
        
        /* Десктопы */
        @media (min-width: 1024px) {
            .container {
                max-width: 1200px;
                margin: 0 auto;
            }
            
            .main-content {
                flex-direction: row;
            }
            
            .sidebar {
                flex: 1;
                min-width: 250px;
            }
            
            .content {
                flex: 3;
                min-width: 700px;
            }
        }
        
        /* Высокоразрешенные экраны */
        @media (-webkit-min-device-pixel-ratio: 2),
               (min-resolution: 192dpi) {
            .logo {
                background-image: url('logo@2x.png');
                background-size: 200px 50px;
            }
        }
    </style>
</head>
<body>
    <div class="container">
        <header class="header">
            <h1>Адаптивный заголовок</h1>
        </header>
        
        <main class="main-content">
            <aside class="sidebar">
                <h3>Боковая панель</h3>
                <p>Содержимое боковой панели</p>
            </aside>
            
            <section class="content">
                <h2>Основной контент</h2>
                <p>Этот контент адаптируется под различные устройства.</p>
            </section>
        </main>
    </div>
</body>
</html>
```
