# Адаптивные таблицы

Адаптивные таблицы - это подход к отображению табличных данных, который обеспечивает корректное отображение информации на устройствах с различными размерами экрана. В условиях российского рынка 2025 года, где мобильные устройства составляют значительную часть трафика, правильная реализация адаптивных таблиц становится критически важной для доступности и удобства использования данных.

## Основные проблемы адаптивных таблиц

- **Ширина экрана**: Таблицы с множеством колонок не помещаются на узких экранах
- **Читаемость**: Трудности с восприятием данных на мобильных устройствах
- **Взаимодействие**: Сложности с навигацией по табличным данным на сенсорных устройствах
- **Производительность**: Большое количество данных может замедлить работу на мобильных устройствах

## Подходы к созданию адаптивных таблиц

### 1. Горизонтальная прокрутка

Простейший подход - добавление горизонтальной прокрутки:

```html
<div class="table-container">
    <table class="responsive-table">
        <thead>
            <tr>
                <th>Имя</th>
                <th>Email</th>
                <th>Телефон</th>
                <th>Должность</th>
                <th>Отдел</th>
                <th>Зарплата</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Иванов Иван</td>
                <td>ivanov@example.com</td>
                <td>+7 (999) 123-45-67</td>
                <td>Менеджер</td>
                <td>Продажи</td>
                <td>80 000 ₽</td>
            </tr>
            <tr>
                <td>Петров Петр</td>
                <td>petrov@example.com</td>
                <td>+7 (999) 234-56-78</td>
                <td>Разработчик</td>
                <td>IT</td>
                <td>120 000 ₽</td>
            </tr>
        </tbody>
    </table>
</div>

<style>
.table-container {
    overflow-x: auto;
    -webkit-overflow-scrolling: touch; /* Для плавной прокрутки на iOS */
}

.responsive-table {
    width: 100%;
    border-collapse: collapse;
    min-width: 600px; /* Минимальная ширина для отображения всех колонок */
}

.responsive-table th,
.responsive-table td {
    padding: 12px 15px;
    text-align: left;
    border-bottom: 1px solid #ddd;
}

.responsive-table th {
    background-color: #f8f9fa;
    font-weight: bold;
}
</style>
```

### 3. Виртуализация таблиц (для больших данных)

Для таблиц с большим количеством строк:

```javascript
// Пример виртуализированной таблицы
class VirtualTable {
    constructor(container, data, options = {}) {
        this.container = container;
        this.data = data;
        this.options = {
            rowHeight: options.rowHeight || 50,
            visibleRows: options.visibleRows || 10,
            ...options
        };
        
        this.render();
    }
    
    render() {
        const totalHeight = this.data.length * this.options.rowHeight;
        const visibleHeight = this.options.visibleRows * this.options.rowHeight;
        
        this.container.innerHTML = `
            <div class="virtual-table-container" style="height: ${visibleHeight}px; overflow-y: auto;">
                <div class="virtual-table-content" style="height: ${totalHeight}px; position: relative;">
                    <div class="virtual-table-rows" style="position: absolute; top: 0; width: 100%;"></div>
                </div>
            </div>
        `;
        
        this.bindScroll();
    }
    
    bindScroll() {
        const container = this.container.querySelector('.virtual-table-container');
        let timeout;
        
        container.addEventListener('scroll', () => {
            clearTimeout(timeout);
            timeout = setTimeout(() => {
                this.renderVisibleRows();
            }, 16); // ~60fps
        });
    }
    
    renderVisibleRows() {
        const container = this.container.querySelector('.virtual-table-container');
        const startRow = Math.floor(container.scrollTop / this.options.rowHeight);
        const endRow = Math.min(
            startRow + this.options.visibleRows + 5, // 5 дополнительных строк для плавности
            this.data.length
        );
        
        const rowsContainer = this.container.querySelector('.virtual-table-rows');
        rowsContainer.style.top = `${startRow * this.options.rowHeight}px`;
        
        // Отрисовка видимых строк
        let rowsHTML = '';
        for (let i = startRow; i < endRow; i++) {
            rowsHTML += this.renderRow(this.data[i], i);
        }
        
        rowsContainer.innerHTML = rowsHTML;
    }
    
    renderRow(rowData, index) {
        return `
            <div class="virtual-table-row" style="height: ${this.options.rowHeight}px; display: flex;">
                <div class="cell">${rowData.id}</div>
                <div class="cell">${rowData.name}</div>
                <div class="cell">${rowData.email}</div>
            </div>
        `;
    }
}
```

## Заключение

Адаптивные таблицы - это сложная, но важная часть современного веб-дизайна. В условиях российского рынка 2025 года, где мобильные устройства доминируют, правильная реализация адаптивных таблиц обеспечивает доступность и удобство использования табличных данных для всех пользователей, независимо от устройства, которое они используют.

> [!tip]
> Всегда тестируйте адаптивные таблицы на реальных мобильных устройствах, чтобы убедиться в удобстве использования и корректности отображения данных.
