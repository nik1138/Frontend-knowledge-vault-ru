# Микроразметка для SEO-оптимизации фронтенд-приложений

## Введение

Микроразметка (структурированные данные) - это специальный формат представления информации на веб-страницах, который помогает поисковым системам лучше понимать содержимое. В 2025 году, особенно в российском сегменте интернета, микроразметка играет важную роль в улучшении отображения сайта в поисковой выдаче и увеличении CTR. Правильная реализация микроразметки может значительно повысить видимость сайта в результатах поиска Яндекса и Google.

## Основные форматы микроразметки

### JSON-LD

Наиболее рекомендуемый формат от Google и Яндекса. Размещается в теге `<head>` или в теле страницы:

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "iPhone 15 Pro",
  "image": "https://example.com/iphone15.jpg",
  "description": "Новейший iPhone с передовыми технологиями",
  "sku": "IP15P-256GB-NAT",
  "offers": {
    "@type": "Offer",
    "price": "99990",
    "priceCurrency": "RUB",
    "availability": "https://schema.org/InStock",
    "seller": {
      "@type": "Organization",
      "name": "Магазин электроники"
    }
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "reviewCount": "1247"
  }
}
</script>
```

### Microdata

Встраивается непосредственно в HTML-разметку:

```html
<div itemscope itemtype="https://schema.org/Product">
  <h1 itemprop="name">iPhone 15 Pro</h1>
  <img src="iphone15.jpg" itemprop="image" alt="iPhone 15 Pro">
  <div itemprop="description">Новейший iPhone с передовыми технологиями</div>
  <div itemprop="offers" itemscope itemtype="https://schema.org/Offer">
    <span itemprop="price">99 990 ₽</span>
    <link itemprop="availability" href="https://schema.org/InStock">В наличии
  </div>
</div>
```

### RDFa

Альтернативный формат, похожий на Microdata:

```html
<div vocab="https://schema.org/" typeof="Product">
  <h1 property="name">iPhone 15 Pro</h1>
  <img src="iphone15.jpg" property="image" alt="iPhone 15 Pro">
  <div property="description">Новейший iPhone с передовыми технологиями</div>
  <div property="offers" typeof="Offer">
    <span property="price">99 990 ₽</span>
  </div>
</div>
```

## Российские особенности 2025 года

### Яндекс и его требования

Яндекс в 2025 году активно развивает поддержку структурированных данных:

- Поддержка Rich-контента в поисковой выдаче
- Интеграция с Яндекс.Маркетом через микроразметку
- Учет локализованных схем для российских сервисов

### Региональные особенности

Для российских сайтов важно учитывать:

- Использование кодов стран и валют в формате ISO
- Указание часовых поясов
- Учет региональных особенностей

### Специфика отраслей

Различные отрасли требуют специфичных схем:

- **E-commerce**: Product, Offer, AggregateOffer
- **Услуги**: Service, ServiceChannel
- **Образование**: Course, EducationalOrganization
- **Здравоохранение**: MedicalBusiness, MedicalProcedure

## Реализация в фронтенд-приложениях

### Динамическая микроразметка в SPA

В одностраничных приложениях микроразметка должна обновляться при навигации:

```javascript
// Пример динамического добавления микроразметки в React
import { useEffect } from 'react';

function ProductPage({ product }) {
  useEffect(() => {
    const script = document.createElement('script');
    script.type = 'application/ld+json';
    script.text = JSON.stringify({
      '@context': 'https://schema.org',
      '@type': 'Product',
      'name': product.name,
      'image': product.image,
      'description': product.description,
      'offers': {
        '@type': 'Offer',
        'price': product.price,
        'priceCurrency': 'RUB',
        'availability': 'https://schema.org/InStock'
      }
    });
    
    document.head.appendChild(script);
    
    return () => {
      document.head.removeChild(script);
    };
  }, [product]);

  return (
    <div>
      {/* Содержимое страницы */}
    </div>
  );
}
```

### Генерация на стороне сервера

Для лучшей SEO-оптимизации рекомендуется генерировать микроразметку на сервере:

```javascript
// Пример серверного рендеринга с микроразметкой
app.get('/product/:id', async (req, res) => {
  const product = await getProductById(req.params.id);
  
  const structuredData = {
    '@context': 'https://schema.org',
    '@type': 'Product',
    'name': product.name,
    'image': product.image,
    'description': product.description,
    'offers': {
      '@type': 'Offer',
      'price': product.price,
      'priceCurrency': 'RUB',
      'availability': product.inStock ? 'https://schema.org/InStock' : 'https://schema.org/OutOfStock'
    }
  };

  const html = `
    <!DOCTYPE html>
    <html>
      <head>
        <title>${product.name}</title>
        <script type="application/ld+json">${JSON.stringify(structuredData)}</script>
      </head>
      <body>
        <div id="root">${renderApp(product)}</div>
      </body>
    </html>
  `;

  res.send(html);
});
```

## Практические рекомендадции

### Общие правила

1. **Актуальность** - данные должны соответствовать реальному содержимому страницы
2. **Полнота** - использование всех релевантных свойств схемы
3. **Консистентность** - единообразное форматирование данных
4. **Валидация** - регулярная проверка корректности микроразметки

### Российские особенности

1. **Валюты** - использование RUB, USD, EUR в соответствии с ISO 4217
2. **Языки** - указание русского языка как "Russian" или "ru"
3. **Адреса** - корректное форматирование российских адресов
4. **Телефоны** - международный формат для телефонных номеров

## Инструменты для работы с микроразметкой

## Заключение

Микроразметка остается важной частью SEO-архитектуры фронтенд-приложений в 2025 году, особенно в российских реалиях, где поисковые системы активно используют структурированные данные для улучшения поисковой выдачи. Правильная реализация микроразметки способствует лучшему пониманию контента поисковыми системами и может привести к улучшению позиций в результатах поиска.
