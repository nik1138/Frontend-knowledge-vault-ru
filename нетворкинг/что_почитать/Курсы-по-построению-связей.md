# Курсы по построению связей

## Онлайн-курсы по нетворкингу для разработчиков

### 1. "Профессиональные связи в IT" - Coursera
Курс от ведущих университетов, посвященный построению профессиональных связей в IT-среде. Включает практические задания и кейсы из реальной практики.

**Темы курса:**
- Основы нетворкинга
- Построение личного бренда
- Участие в профессиональных сообществах
- Создание и поддержание связей
- Влияние и убеждение

[[Coursera]] [[Профессиональные связи]] [[Личный бренд]]

### 2. "Эффективное общение для технических специалистов" - Udemy
Курс, разработанный специально для технических специалистов, включая разработчиков. Уделяет особое внимание особенностям общения в IT-среде.

**Преимущества курса:**
- Практические навыки общения
- Техники установления контакта
- Управление конфликтами
- Презентационные навыки
- Нетворкинг в цифровом формате

[[Udemy]] [[Эффективное общение]] [[Презентационные навыки]]

### 3. "Карьерный рост разработчика" - Stepik
Российская платформа Stepik предлагает курс, посвященный карьерному росту разработчиков, включая модуль по построению профессиональных связей.

**Содержание курса:**
- Построение карьеры в IT
- Профессиональные связи и их значение
- Участие в open source
- Создание портфолио
- Подготовка к собеседованиям

[[Stepik]] [[Карьерный рост]] [[Open Source]]

### 4. "Мягкие навыки для разработчиков" - Hexlet
Курс, разработанный российской образовательной платформой Hexlet. Включает модуль по нетворкингу и построению профессиональных связей.

**Модули:**
- Коммуникация
