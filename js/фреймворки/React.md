# React

## Обзор

React - это библиотека JavaScript для создания пользовательских интерфейсов, разработанная Facebook. Это один из самых популярных инструментов для построения интерактивных веб-приложений в 2025 году.

## Основные особенности

- **Компонентный подход** - Разделение интерфейса на независимые части
- **Виртуальный DOM** - Эффективное обновление реального DOM
- **Односторонний поток данных** - Упрощение логики приложения
- **Хуки** - Возможность использования состояния и других возможностей React без написания классов

## Основные концепции

### JSX

JSX - это синтаксическое расширение JavaScript, которое позволяет писать код, похожий на HTML, внутри JavaScript.

```jsx
const element = <h1>Привет, мир!</h1>;
```

### Компоненты

Компоненты - это переиспользуемые части интерфейса, которые можно использовать в различных частях приложения.

```jsx
function Welcome(props) {
  return <h1>Привет, {props.name}!</h1>;
}
```

### Props и State

Props (свойства) - это входные данные, передаваемые компоненту. State - это внутреннее состояние компонента, которое может изменяться.

```jsx
function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Счетчик: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Увеличить
      </button>
    </div>
  );
}
```

## Хуки

### useState

Для управления состоянием компонента.

```jsx
const [state, setState] = useState(initialValue);
```

### useEffect

Для выполнения побочных эффектов.

```jsx
useEffect(() => {
  // код эффекта
  return () => {
    // код очистки
  };
}, [dependencies]);
```

### useContext

Для работы с контекстом.

```jsx
const value = useContext(MyContext);
```

## Практические примеры

### Простое приложение "Список задач"

```jsx
import React, { useState } from 'react';

function TodoApp() {
  const [todos, setTodos] = useState([]);
  const [inputValue, setInputValue] = useState('');

  const addTodo = () => {
    if (inputValue.trim() !== '') {
      setTodos([...todos, { id: Date.now(), text: inputValue, completed: false }]);
      setInputValue('');
    }
  };

  const toggleTodo = (id) => {
    setTodos(todos.map(todo =>
      todo.id === id ? { ...todo, completed: !todo.completed } : todo
    ));
  };

  return (
    <div>
      <h1>Список задач</h1>
      <input
        type="text"
        value={inputValue}
        onChange={(e) => setInputValue(e.target.value)}
        placeholder="Добавить задачу"
      />
      <button onClick={addTodo}>Добавить</button>
      <ul>
        {todos.map(todo => (
          <li key={todo.id}>
            <span
              style={{ textDecoration: todo.completed ? 'line-through' : 'none' }}
              onClick={() => toggleTodo(todo.id)}
            >
              {todo.text}
            </span>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default TodoApp;
```

## Заключение

React остается одним из самых востребованных инструментов для разработки пользовательских интерфейсов в 2025 году. Его гибкость, производительность и широкая экосистема делают его отличным выбором для разработки как небольших, так и крупномасштабных приложений, особенно с учетом специфики российского рынка.

> [!tip] Совет
> Начинайте с простых компонентов и постепенно переходите к более сложным паттернам. Используйте [[TypeScript]] с самого начала для повышения надежности кода.

> [!warning] Важно
> В условиях санкционных ограничений в России, важно следить за альтернативными решениями и отечественными разработками, которые могут заменить заблокированные сервисы и инструменты.
