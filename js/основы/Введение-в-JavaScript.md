# Введение в JavaScript

JavaScript (JS) - это динамический, интерпретируемый язык программирования, который используется для добавления интерактивности к веб-страницам. В 2025 году JavaScript остается одним из самых востребованных языков программирования в мире, особенно в сфере веб-разработки.

## История JavaScript

JavaScript был создан Бренданом Айком в 1995 году за 10 дней в компании Netscape. Изначально язык назывался Mocha, затем LiveScript, и только потом - JavaScript. Несмотря на схожесть в названии, JavaScript не имеет отношения к языку Java.

## Основные особенности

- **Динамическая типизация** - типы данных определяются во время выполнения
- **Прототипное наследование** - вместо классического наследования используется прототипное
- **Функции первого класса** - функции могут передаваться как переменные
- **Асинхронность** - поддержка асинхронного программирования через Promise и async/await
- **Кроссплатформенность** - работает в браузерах и на серверах (Node.js)

## Применение JavaScript

В 2025 году JavaScript используется не только в браузерах, но и:

- **Серверная разработка** (Node.js, Deno)
- **Мобильная разработка** (React Native, Ionic)
- **Десктопные приложения** (Electron)
- **Игры** (Phaser, Three.js)
- **Машинное обучение** (TensorFlow.js)
- **Блокчейн** (Web3, DApps)

## Среды выполнения

### Браузер
JavaScript изначально был создан для работы в браузерах. Современные браузеры предоставляют богатый API для взаимодействия с DOM, сетью, локальным хранилищем и другими возможностями.

### Node.js
Платформа Node.js позволяет запускать JavaScript на сервере. Это открыло возможности для создания серверных приложений, инструментов сборки и утилит.

### Deno
Альтернатива Node.js, созданная тем же автором (Райаном Даллом), с акцентом на безопасность и современные возможности языка.

## Заключение

JavaScript - мощный и гибкий язык, который продолжает развиваться. Его универсальность делает его незаменимым инструментом в арсенале современного разработчика. Независимо от того, работаете ли вы на фрилансе в Москве или в IT-компании в Новосибирске, знание JavaScript откроет перед вами множество возможностей.

> Начинайте изучение JavaScript с понимания [[Типы-данных]] и [[Область-видимости]], так как это основы, на которых строится всё остальное.

> [!warning] Важно
> Не забывайте про безопасность при работе с JavaScript, особенно при обработке пользовательского ввода и работе с API.
