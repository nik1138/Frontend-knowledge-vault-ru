# Переменные в JavaScript: Основы

## Что такое переменные

Переменные в JavaScript - это именованные контейнеры для хранения данных. Они позволяют сохранять значения и обращаться к ним по имени. Переменные являются фундаментальной концепцией любого языка программирования.

## Способы объявления переменных

В JavaScript существует три способа объявления переменных:

### var

`var` - это старый способ объявления переменных, который использовался до появления ES6:

```javascript
var name = "Иван";
var age = 25;
```

### let

`let` - современный способ объявления переменных, значения которых могут изменяться:

```javascript
let name = "Иван";
let age = 25;
age = 26; // Можно изменить значение
```

### const

`const` - используется для объявления констант, значения которых не должны изменяться:

```javascript
const PI = 3.14159;
const userName = "Иван";
// PI = 3.14; // Ошибка! Нельзя изменить значение константы
```

## Правила именования переменных

1. Имя переменной должно начинаться с буквы, символа подчеркивания (_) или знака доллара ($)
2. Имя может содержать буквы, цифры, подчеркивания и знаки доллара
3. Имена переменных чувствительны к регистру (myVar и myvar - разные переменные)
4. Нельзя использовать зарезервированные слова JavaScript в качестве имен переменных

```javascript
// Правильные имена переменных
let userName = "Иван";
let _private = "скрытое значение";
let $element = document.getElementById("myId");
let camelCase = "рекомендуемый стиль";

// Неправильные имена переменных
// let 123name = "неправильно"; // Начинается с цифры
// let my-name = "неправильно"; // Дефис не допускается
// let let = "неправильно"; // Зарезервированное слово
```

## Типы данных в переменных

JavaScript - это язык с динамической типизацией, что означает, что переменная может хранить значения разных типов:

```javascript
let dynamicVar; // undefined
dynamicVar = 5; // number
dynamicVar = "строка"; // string
dynamicVar = true; // boolean
dynamicVar = { name: "Иван" }; // object
```

## Область видимости переменных

### Глобальная область видимости

Переменные, объявленные вне любых функций, имеют глобальную область видимости:

```javascript
let globalVar = "Я глобальная переменная";

function example() {
  console.log(globalVar); // Доступна внутри функции
}

example(); // "Я глобальная переменная"
```

### Локальная область видимости

Переменные, объявленные внутри функции или блока, имеют локальную область видимости:

```javascript
function example() {
  let localVar = "Я локальная переменная";
  console.log(localVar); // Доступна внутри функции
}

example(); // "Я локальная переменная"
// console.log(localVar); // Ошибка! Переменная не доступна вне функции
```

## Различия между var, let и const

### Поднятие (Hoisting)

```javascript
console.log(a); // undefined (var поднимается)
var a = 5;

// console.log(b); // ReferenceError (let не инициализируется до объявления)
let b = 10;
```

### Область видимости

```javascript
function example() {
  if (true) {
    var functionScoped = "функциональная область";
    let blockScoped = "блочная область";
  }
  
  console.log(functionScoped); // "функциональная область"
  // console.log(blockScoped); // ReferenceError
}
```

## Рекомендации по использованию

1. По умолчанию используйте `const` для переменных, которые не будут изменяться
2. Используйте `let` для переменных, значения которых будут изменяться
3. Избегайте использования `var` в современном коде

```javascript
// Хорошо
const userName = "Иван";
const users = [];
let counter = 0;

// Плохо
var oldStyle = "устаревший способ";
```

## Практические примеры

### Работа с переменными

```javascript
// Объявление нескольких переменных
const firstName = "Иван";
const lastName = "Иванов";
let age = 25;
let isStudent = true;

// Использование переменных
const fullName = firstName + " " + lastName;
console.log(`Меня зовут ${fullName}, мне ${age} лет`);

// Изменение значения переменной
age = age + 1;
console.log(`Через год мне будет ${age} лет`);

// Работа с массивами и объектами
const person = {
  name: firstName,
  age: age,
  student: isStudent
};

console.log(person); // { name: "Иван", age: 26, student: true }
```

## Заключение

Понимание переменных - это первый шаг к освоению JavaScript. Правильное использование `let` и `const` вместо `var`, соблюдение правил именования и понимание области видимости помогут писать более чистый и предсказуемый код.
