# Оптимизация для мобильных

Оптимизация для мобильных устройств — это комплекс мер, направленных на улучшение производительности, скорости загрузки и пользовательского опыта на мобильных устройствах. В 2025 году это критически важно для успеха веб-приложений.

## Основные аспекты оптимизации

### Производительность JavaScript

- Минимизация и сжатие JavaScript-кода
- Отложенная загрузка скриптов
- Использование веб-воркеров для тяжелых вычислений
- Избегание лишних перерисовок и вычислений стилей

### Оптимизация изображений

- Использование современных форматов (WebP, AVIF)
- Адаптивные изображения с использованием `srcset` и `sizes`
- Ленивая загрузка изображений

### Сеть и загрузка

- Минимизация HTTP-запросов
- Использование HTTP/2 и HTTP/3
- Кеширование ресурсов
- Использование CDN

## Практические рекомендации

### Ленивая загрузка контента

```javascript
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      loadContent(entry.target);
      observer.unobserve(entry.target);
    }
  });
});

document.querySelectorAll('.lazy-load').forEach(el => {
  observer.observe(el);
});
```

### Оптимизация DOM

```javascript
// Используйте documentFragment для множественных изменений DOM
const fragment = document.createDocumentFragment();
items.forEach(item => {
  const element = createItemElement(item);
  fragment.appendChild(element);
});
container.appendChild(fragment);
```

## Особенности российского рынка

В России часто используются устройства с ограниченными ресурсами и медленные соединения. Также популярны бюджетные смартфоны, которые могут иметь проблемы с производительностью. Учитывайте это при оптимизации приложений.
