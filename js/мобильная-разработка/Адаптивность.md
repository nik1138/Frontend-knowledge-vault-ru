# Адаптивность

Адаптивность в мобильной разработке — это способность веб-приложения адаптироваться к различным размерам экрана и устройствам. В 2025 году, с ростом разнообразия мобильных устройств, адаптивность стала критически важной для обеспечения доступности и удобства использования.

## Основные принципы адаптивности

- **Гибкие сетки**: Использование относительных единиц измерения (%, em, rem, vw, vh) вместо фиксированных пикселей.
- **Медиа-запросы**: CSS-правила, которые применяются при определенных условиях (например, ширина экрана).
- **Адаптивные изображения**: Изображения, которые изменяют размер в зависимости от экрана.

## Практические примеры

### Гибкая сетка

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}
```

### Медиа-запросы

```css
@media (max-width: 768px) {
  .sidebar {
    display: none;
  }
}
```

## JavaScript и адаптивность

JavaScript может использоваться для динамического изменения поведения интерфейса в зависимости от размера экрана.

```javascript
function handleResize() {
  if (window.innerWidth < 768) {
    // Логика для мобильных устройств
  } else {
    // Логика для настольных устройств
  }
}

window.addEventListener('resize', handleResize);
```

## Особенности российского рынка

В России популярны недорогие смартфоны с небольшими экранами. Также важна поддержка различных ориентаций экрана (портретная/ландшафтная). Учитывайте это при проектировании интерфейса.
