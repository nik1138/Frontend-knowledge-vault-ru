# React

## Общее описание

React - это библиотека JavaScript для создания пользовательских интерфейсов, разработанная Facebook. Он используется для построения интерактивных веб-приложений и веб-сайтов, а также мобильных приложений с помощью React Native.

## Архитектура и основные концепции

### Компоненты
React построен на компонентной архитектуре. Компоненты могут быть функциональными или классовыми. В современном React предпочтение отдается функциональным компонентам с хуками.

```jsx
function Welcome(props) {
  return <h1>Привет, {props.name}!</h1>;
}
```

### JSX
JSX - это синтаксическое расширение JavaScript, которое позволяет писать HTML-подобный код в JavaScript. Это необязательно, но делает код более читаемым.

### Виртуальный DOM
React использует виртуальный DOM для оптимизации производительности. При изменении данных React сравнивает виртуальное представление DOM с предыдущим и обновляет только те части реального DOM, которые действительно изменились.

## Основные возможности

### Хуки (Hooks)
Хуки позволяют использовать состояние и другие возможности React без написания классов.

#### useState
```jsx
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Счетчик: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Увеличить
      </button>
    </div>
  );
}
```

#### useEffect
```jsx
import React, { useState, useEffect } from 'react';

function UserStatus({ userId }) {
  const [status, setStatus] = useState('offline');

  useEffect(() => {
    const subscription = subscribeToUserStatus(userId, (status) => {
      setStatus(status);
    });
    
    return () => {
      subscription.unsubscribe();
    };
  }, [userId]);

  return <div>Статус: {status}</div>;
}
```

## Экосистема React

### Create React App
Официальный инструмент для создания проектов React с нуля. Предоставляет готовую конфигурацию для разработки.

### React Router
Библиотека для реализации навигации между компонентами в одностраничных приложениях.

### Redux
Библиотека управления состоянием, часто используется с React для управления глобальным состоянием приложения.

### Next.js
Фреймворк на основе React, который предоставляет расширенные возможности для серверного рендеринга, статической генерации и других функций.

- Используйте TypeScript для лучшей типизации
- Оптимизируйте производительность с помощью React.memo и useMemo
