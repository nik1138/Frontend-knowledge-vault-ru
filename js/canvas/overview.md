## Обзор

Canvas API предоставляет мощные возможности для рендеринга 2D и 3D графики в браузере. Canvas 2D API идеально подходит для рисования графиков, создания пользовательских компонентов и простых игр, а WebGL позволяет создавать сложные 3D сцены и визуализации.

## Основные технологии

- **Canvas 2D API** - 2D графика с помощью элемента canvas
- **WebGL** - 3D графика с использованием OpenGL ES
- **WebGL2** - улучшенная версия WebGL
- **WebGPU** - новое поколение API для графики (в разработке)
