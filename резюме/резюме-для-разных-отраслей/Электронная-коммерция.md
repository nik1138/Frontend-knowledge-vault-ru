# Электронная коммерция: Резюме для российского рынка 2025

## Общая информация

Российский рынок электронной коммерции в 2025 году продолжает адаптироваться к новым реалиям после ухода западных компаний и санкционного давления. Рынок характеризуется ростом отечественных платформ, развитием локальных логистических решений и усилением роли государственных инициатив в поддержке цифровой торговли.

## Ключевые особенности резюме для e-commerce

### 1. Понимание российской специфики

В 2025 году российский e-commerce имеет свои особенности:
- Ориентация на отечественные платформы (Вайлдберриз, Озон, Яндекс.Маркет)
- Развитие собственных систем доставки и логистики
- Адаптация под требования 152-ФЗ о персональных данных
- Учет особенностей оплаты через отечественные системы (Мир, СБП)

### 2. Технические навыки для разных позиций

#### Для разработчиков:
- Языки программирования: Python, JavaScript, PHP, Java
- Фреймворки: Django, React, Vue.js, Laravel
- Платформы: WooCommerce, 1С-Битрикс, OpenCart
- Базы данных: PostgreSQL, MySQL
- Опыт работы с API маркетплейсов (Wildberries, Ozon)
- Знание систем управления запасами (WMS)

#### Для маркетологов:
- Знание таргетинга в ВКонтакте, Одноклассниках, Яндекс.Директ
- Опыт работы с SEO для российских поисковиков
- Понимание воронки продаж в e-commerce
- Аналитика через Яндекс.Метрику, Google Analytics (российские аналоги)

#### Для логистов:
- Знание систем доставки (CDEK, Boxberry, Почта России)
- Понимание международных перевозок (в рамках доступных направлений)
- Опыт работы с 1С:Управление торговлей
- Знание таможенных процедур (при наличии международных поставок)

### 3. Опыт работы с маркетплейсами

Важно указать опыт работы с российскими маркетплейсами:
- Вайлдберриз: настройка кабинета, управление ассортиментом, реклама
- Озон: FBS/FBO, аналитика продаж, маркетинг
- Яндекс.Маркет: модели продаж, реклама, отзывы
