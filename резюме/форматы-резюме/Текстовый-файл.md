# Текстовый файл

**Текстовый файл** (TXT) — простейший формат резюме, представляющий собой документ с минимальным форматированием. В 2025 году этот формат используется в особых случаях, когда важна максимальная совместимость и простота обработки.

## Преимущества

- **Максимальная совместимость** — открывается на любом устройстве и операционной системе
- **Минимальный размер файла** — занимает минимальное место
- **Безопасность** — не содержит вирусов или вредоносных скриптов
- **Простота обработки** — легко читается ATS-системами и текстовыми редакторами
- **Надежность** — не зависит от специфических программ или версий
- **Быстрая загрузка** — мгновенное открытие без ожидания

## Недостатки

- **Отсутствие форматирования** — невозможно использовать жирный шрифт, цвета, таблицы или изображения
- **Ограниченная визуальная привлекательность** — выглядит сухо и неинтересно
- **Сложность восприятия** — без визуальных акцентов информация может быть трудночитаемой
- **Непрофессиональный вид** — может восприниматься как небрежно подготовленный документ
- **Ограниченные возможности** — нельзя использовать сложные структуры данных

## Особенности подготовки

При создании текстового резюме в 2025 году важно учитывать особенности этого формата:

### Структура документа
- Используйте простую иерархию с разделением на блоки с помощью пустых строк
- Применяйте символы (например, *, -, #) для обозначения заголовков и подзаголовков
- Организуйте информацию в логическом порядке: контактная информация, краткое описание, опыт, образование, навыки
- Используйте табуляцию и пробелы для создания примитивных таблиц и списков

### Форматирование через символы
- Используйте символы подчеркивания или решетки для выделения заголовков
- Применяйте звездочки или тире для маркированных списков
- Разделяйте разделы пустыми строками или линиями из символов (например, ---)
- Используйте двоеточия и тире для структурирования информации

### Совместимость с ATS
- Текстовый формат идеально подходит для ATS-систем
- Избегайте сложных символов, которые могут неправильно интерпретироваться
- Убедитесь, что все ключевые слова и важная информация представлены в простом текстовом виде

## Рекомендации по оформлению

### Структурное оформление
- Начинайте с имени и фамилии, выделенных заглавными буквами
- Используйте заголовки для каждого раздела (например, "ОПЫТ РАБОТЫ", "ОБРАЗОВАНИЕ")
- Применяйте отступы для визуального отделения подразделов
- Используйте последовательное форматирование для всех элементов одного типа

### Технические требования
- Сохраняйте файл в кодировке UTF-8 для корректного отображения русских символов
- Используйте формат .txt
- Ограничьте длину строк примерно 80 символами для лучшей читаемости
- Название файла должно быть профессиональным (например, "Иванов_Иван_Программист.txt")

### Контент
- Пишите кратко и по существу
- Используйте ключевые слова из описания вакансии
- Включайте конкретные достижения с цифрами
- Избегайте сложных конструкций и двусмысленностей

## Использование в российских реалиях 2025 года

В российском контексте 2025 года текстовый формат резюме используется в следующих ситуациях:

- **При работе с ATS-системами**, которые требуют максимальной простоты обработки
- **Для отправки через формы**, которые не поддерживают другие форматы
- **В специфических отраслях**, где важна максимальная совместимость
- **При необходимости быстрой отправки** без подготовки сложного документа
- **Для резервных копий**, когда другие форматы недоступны

Однако большинство работодателей предпочитают более визуально привлекательные форматы, такие как [[PDF]] или [[Онлайн-профиль]].

## Практические рекомендации

1. **Создание**:
   - Используйте простой текстовый редактор (Notepad, TextEdit, VS Code)
   - Следите за структурой и читаемостью текста
   - Проверяйте форматирование в разных редакторах

2. **Проверка**:
   - Откройте файл в нескольких текстовых редакторах
   - Убедитесь, что структура сохраняется при открытии
   - Проверьте корректность отображения русских символов

3. **Отправка**:
   - В теме письма укажите "Текстовое резюме"
   - Объясните в сопроводительном письме, почему выбран этот формат
   - При необходимости предоставьте также версию в формате [[PDF]]
