# AR/VR - подготовка резюме для позиций в области дополненной и виртуальной реальности

## Общая информация
В 2025 году рынок AR/VR в России развивается в условиях импортозамещения и локальных решений. Сфера применения расширяется от развлечений до промышленного обучения, медицины и образования. Работодатели ищут специалистов с опытом создания решений для отечественного оборудования и платформ.

## Ключевые навыки для резюме
- Знание игровых движков: Unity 3D, Unreal Engine
- Опыт программирования на C#, C++, Python
- Понимание принципов 3D-моделирования и анимации
- Опыт работы с AR/VR SDK (OpenXR, Oculus SDK, ARCore, ARKit)
- Знание принципов UX/UI для 3D-интерфейсов
- Понимание физики движений и пространственного звучания
- Опыт оптимизации производительности для VR-устройств

## Технические проекты для портфолио
- Разработка AR-приложений для обучения и тренинга
- Создание VR-симуляторов для промышленного использования
- Разработка интерактивных 3D-презентаций
- Создание AR-фильтров для отечественных соцсетей
- Разработка VR-терапевтических решений
- Интеграция AR с IoT-устройствами

## Примеры формулировок для резюме
- "Разработал AR-приложение для обучения персонала, сократив время адаптации на 40%"
- "Создал VR-симулятор для промышленного оборудования с поддержкой отечественных VR-шлемов"
- "Оптимизировал производительность VR-приложения до стабильных 90 FPS на доступном оборудовании"
