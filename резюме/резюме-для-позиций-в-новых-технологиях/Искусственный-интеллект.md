# Искусственный интеллект - подготовка резюме для позиций в области ИИ

## Общая информация
В 2025 году рынок труда в области искусственного интеллекта в России продолжает расти несмотря на санкционные ограничения. Компании активно ищут специалистов по машинному обучению, нейросетям и автоматизации процессов.

## Ключевые навыки для резюме
- Знание фреймворков: TensorFlow, PyTorch, Keras
- Опыт работы с моделями трансформеров (GPT, BERT, T5)
- Понимание архитектур нейросетей
- Знание Python и библиотек: scikit-learn, pandas, numpy
- Опыт работы с GPU-ускорением (CUDA, OpenCL)
- Знание русского языка (критично для локальных компаний)

## Технические проекты для портфолио
- Разработка чат-ботов на основе LLM
- Создание систем рекомендаций
- Компьютерное зрение (распознавание изображений, OCR)
- Обработка естественного языка (NLP)
- Автоматизация бизнес-процессов с помощью ИИ

## Примеры формулировок для резюме
- "Разработал модель машинного обучения, улучшившую эффективность бизнес-процесса на 25%"
- "Создал нейросеть для анализа текстовых данных с точностью 92%"
- "Оптимизировал архитектуру модели под ограниченные вычислительные ресурсы"

## Российские особенности
- Упоминание опыта работы с русскоязычными данными
- Знание локальных ИИ-платформ (например, Салют, Алиса)
- Опыт работы с отечественными решениями (Нейросеть Mail.Ru, Сбер AI)
- Понимание требований 152-ФЗ и 242-ФЗ
