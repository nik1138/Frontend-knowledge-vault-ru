# IoT - подготовка резюме для позиций в области интернета вещей

## Общая информация
В 2025 году российский рынок IoT активно развивается в рамках программ цифровой трансформации и импортозамещения. Акцент делается на создание отечественных решений для умного города, промышленного интернета вещей (IIoT) и систем мониторинга. Работодатели ищут специалистов с комплексным пониманием как аппаратной, так и программной сторон IoT-систем.

## Ключевые навыки для резюме
- Знание микроконтроллеров и одноплатных компьютеров (Arduino, ESP32, Raspberry Pi)
- Опыт программирования на C/C++, Python, JavaScript
- Понимание протоколов передачи данных (MQTT, CoAP, HTTP, LoRaWAN)
- Опыт работы с облачными платформами IoT (AWS IoT, Azure IoT, отечественные решения)
- Знание принципов работы датчиков и исполнительных механизмов
- Понимание принципов безопасности IoT-устройств
- Опыт работы с системами реального времени (RTOS)

## Технические проекты для портфолио
- Разработка систем умного дома на отечественном оборудовании
- Создание промышленных сенсорных систем мониторинга
- Разработка систем автоматизации сельского хозяйства
- Создание систем мониторинга окружающей среды
- Разработка решений для умного города
- Интеграция IoT с ИИ и машинным обучением

## Примеры формулировок для резюме
- "Разработал систему мониторинга параметров производства, сократив аварийные остановки на 35%"
- "Создал сеть из 100+ сенсоров для мониторинга микроклимата в тепличном комплексе"
- "Интегрировал IoT-устройства с отечественной системой управления зданием"

## Российские особенности
- Опыт работы с отечественными IoT-платформами и решениями
- Понимание требований ФСТЭК и ФСБ к защите IoT-устройств
- Знание специфики государственных IoT-проектов
- Опыт работы с отечественными компонентами и чипами
- Понимание требований к локализации данных и их хранению
