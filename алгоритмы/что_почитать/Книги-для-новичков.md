# Книги для новичков в алгоритмах

## Введение

Выбор правильной книги для начала изучения алгоритмов играет ключевую роль в формировании прочного фундамента знаний. Ниже представлены лучшие книги, подходящие для начинающих программистов, которые хотят познакомиться с основами алгоритмов и структур данных.

## Рекомендуемые книги

### 1. "Грокаем алгоритмы" - Адитья Бхаргава

Одна из самых доступных книг для начинающих. Автор использует визуальный подход с иллюстрациями и простыми объяснениями, что делает сложные концепции понятными даже для тех, кто только начинает свой путь в программировании.

- **Язык:** перевод
- **Сложность:** ★☆☆☆☆
- **Темы:**
  - [[Сортировка]] и [[Поиск]]
  - [[Графы]] и [[Алгоритмы на графах]]
  - [[Динамическое программирование]]
  - [[Хеш-таблицы]]

Пример алгоритма из книги - бинарный поиск:

**Псевдокод:**
```
ФУНКЦИЯ бинарный_поиск(массив, цель)
    НАЧАЛО
        лево = 0
        право = длина(массив) - 1
        ПОКА лево <= право
            середина = (лево + право) / 2
            ЕСЛИ массив[середина] = цель
                ВОЗВРАТ середина
            ИНАЧЕ ЕСЛИ массив[середина] < цель
                лево = середина + 1
            ИНАЧЕ
                право = середина - 1
            КОНЕЦ ЕСЛИ
        КОНЕЦ ПОКА
        ВОЗВРАТ -1
    КОНЕЦ
```

**JavaScript реализация:**
```javascript
function binarySearch(arr, target) {
    let left = 0;
    let right = arr.length - 1;

    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        if (arr[mid] === target) {
            return mid;
        } else if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    return -1;
}
```

> [!tip] Совет
> Эта книга особенно рекомендуется для самообучения и подготовки к собеседованиям в IT-компании.
