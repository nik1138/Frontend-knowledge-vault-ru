# Документация и справочники по алгоритмам

## Введение

Документация и справочники по алгоритмам являются важным ресурсом для разработчиков, позволяя быстро находить информацию о реализации, сложности и применении различных алгоритмов. Доступны как традиционные печатные справочники, так и интерактивные онлайн-ресурсы.

```
СТРУКТУРА ХешТаблица
    ПОЛЕ массив_бакетов[размер]
    ПОЛЕ размер
    ПОЛЕ количество_элементов

    ФУНКЦИЯ инициализировать(размер)
        НАЧАЛО
            ЭТО.массив_бакетов = новый_массив(размер)
            ЭТО.размер = размер
            ЭТО.количество_элементов = 0
            ДЛЯ i = 0 ДО размер - 1
                ЭТО.массив_бакетов[i] = пустой_список
            КОНЕЦ ДЛЯ
        КОНЕЦ

    ФУНКЦИЯ хеш(ключ)
        НАЧАЛО
            хеш_значение = 0
            ДЛЯ каждый символ в ключ
                хеш_значение = (хеш_значение * 31 + символ) % ЭТО.размер
            КОНЕЦ ДЛЯ
            ВОЗВРАТ хеш_значение
        КОНЕЦ

    ФУНКЦИЯ вставить(ключ, значение)
        НАЧАЛО
            индекс = ЭТО.хеш(ключ)
            бакет = ЭТО.массив_бакетов[индекс]

            ДЛЯ каждая пара в бакете
                ЕСЛИ пара.ключ = ключ
                    пара.значение = значение
                    ВОЗВРАТ
```

**JavaScript реализация:**
```javascript
class HashTable {
    constructor(size = 100) {
        this.buckets = new Array(size);
        this.size = size;
        this.count = 0;

        for (let i = 0; i < size; i++) {
            this.buckets[i] = [];
        }
    }

    hash(key) {
        let hashValue = 0;
        for (let i = 0; i < key.length; i++) {
            hashValue = (hashValue * 31 + key.charCodeAt(i)) % this.size;
        }
        return hashValue;
    }

    set(key, value) {
        const index = this.hash(key);
        const bucket = this.buckets[index];

        for (let i = 0; i < bucket.length; i++) {
            if (bucket[i][0] === key) {
                bucket[i][1] = value;
                return;
            }
        }

        bucket.push([key, value]);
        this.count++;
    }

    get(key) {
        const index = this.hash(key);
        const bucket = this.buckets[index];

        for (let i = 0; i < bucket.length; i++) {
            if (bucket[i][0] === key) {
                return bucket[i][1];
            }
        }

        return null;
    }
}
```
