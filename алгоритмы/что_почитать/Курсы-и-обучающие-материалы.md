Пример алгоритма, изучаемого на курсе - обход дерева в глубину (DFS):

**Псевдокод:**
```
ФУНКЦИЯ DFS(дерево, узел)
    НАЧАЛО
        стек = []
        посещенные = []
        результат = []

        добавить(стек, узел)

        ПОКА стек не пусто
            текущий = извлечь_с_конца(стек)
            ЕСЛИ текущий не в посещенные
                добавить(результат, текущий)
                добавить(посещенные, текущий)

                ДЛЯ каждого потомка в текущий.потомки
                    добавить(стек, потомок)
                КОНЕЦ ДЛЯ
            КОНЕЦ ЕСЛИ
        КОНЕЦ ПОКА

        ВОЗВРАТ результат
    КОНЕЦ
```

**JavaScript реализация:**
```javascript
function dfs(tree, node) {
    const stack = [node];
    const visited = new Set();
    const result = [];

    while (stack.length > 0) {
        const current = stack.pop();

        if (!visited.has(current)) {
            result.push(current.value);
            visited.add(current);

            // Добавляем потомков в обратном порядке для правильного обхода
            if (current.children) {
                for (let i = current.children.length - 1; i >= 0; i--) {
                    stack.push(current.children[i]);
                }
            }
        }
    }

    return result;
}
```
