# Ресурсы в интернете по алгоритмам

## Введение

Интернет остаётся богатым источником информации по алгоритмам. Онлайн-ресурсы позволяют получать актуальные знания, практические примеры и интерактивное обучение. Ниже представлены лучшие ресурсы для изучения алгоритмов.

## Специализированные сайты

### 1. VisuAlgo

Интерактивная визуализация алгоритмов и структур данных.

- **Особенности:**
  - Анимированные объяснения
  - [[Алгоритмы на графах]]
  - [[Сортировка]]
  - [[Деревья]]

Пример алгоритма, который можно визуализировать - сортировка пузырьком:

**Псевдокод:**
```
ФУНКЦИЯ сортировка_пузырьком(массив)
    НАЧАЛО
        n = длина(массив)
        ПОВТОРИТЬ
            swapped = ЛОЖЬ
            ДЛЯ i = 0 ДО n - 2
                ЕСЛИ массив[i] > массив[i + 1]
                    ПОМЕНЯТЬ местами(массив[i], массив[i + 1])
                    swapped = ИСТИНА
                КОНЕЦ ЕСЛИ
            КОНЕЦ ДЛЯ
            n = n - 1
        ПОКА swapped = ИСТИНА
        ВОЗВРАТ массив
    КОНЕЦ
```

**JavaScript реализация:**
```javascript
function bubbleSort(arr) {
    const n = arr.length;
    let swapped;

    do {
        swapped = false;
        for (let i = 0; i < n - 1; i++) {
            if (arr[i] > arr[i + 1]) {
                // Меняем местами элементы
                [arr[i], arr[i + 1]] = [arr[i + 1], arr[i]];
                swapped = true;
            }
        }
    } while (swapped);

    return arr;
}
```
