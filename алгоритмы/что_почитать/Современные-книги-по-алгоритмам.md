Пример алгоритма из книги - поиск в ширину (BFS) для обработки графов:

**Псевдокод:**
```
ФУНКЦИЯ BFS(граф, стартовая_вершина)
    НАЧАЛО
        ОЧЕРЕДЬ = []
        ПОСЕЩЕННЫЕ = []
        РЕЗУЛЬТАТ = []

        добавить(ОЧЕРЕДЬ, стартовая_вершина)
        добавить(ПОСЕЩЕННЫЕ, стартовая_вершина)

        ПОКА ОЧЕРЕДЬ не пуста
            вершина = извлечь_с_начала(ОЧЕРЕДЬ)
            добавить(РЕЗУЛЬТАТ, вершина)

            ДЛЯ каждой смежной_вершины в граф[вершина]
                ЕСЛИ смежная_вершина не в ПОСЕЩЕННЫЕ
                    добавить(ПОСЕЩЕННЫЕ, смежная_вершина)
                    добавить(ОЧЕРЕДЬ, смежная_вершина)
                КОНЕЦ ЕСЛИ
            КОНЕЦ ДЛЯ
        КОНЕЦ ПОКА

        ВОЗВРАТ РЕЗУЛЬТАТ
    КОНЕЦ
```

**JavaScript реализация:**
```javascript
function bfs(graph, startVertex) {
    const queue = [];
    const visited = new Set();
    const result = [];

    queue.push(startVertex);
    visited.add(startVertex);

    while (queue.length > 0) {
        const vertex = queue.shift();
        result.push(vertex);

        for (const adjacentVertex of graph[vertex] || []) {
            if (!visited.has(adjacentVertex)) {
                visited.add(adjacentVertex);
                queue.push(adjacentVertex);
            }
        }
    }

    return result;
}
```
