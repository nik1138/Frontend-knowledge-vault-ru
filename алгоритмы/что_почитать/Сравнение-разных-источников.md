Пример алгоритма, часто рассматриваемого в современных книгах - жадный алгоритм для задачи о покрытии множеств:

**Псевдокод:**
```
ФУНКЦИЯ жадное_покрытие_множеств(множества, универсум)
    НАЧАЛО
        покрытие = []
        оставшиеся_элементы = универсум

        ПОКА оставшиеся_элементы не пусто
            лучшее_множество = НИЛЬ
            макс_покрытие = 0

            ДЛЯ каждое_множество в множества
                покрыто = пересечение(каждое_множество, оставшиеся_элементы)
                ЕСЛИ длина(покрыто) > макс_покрытие
                    макс_покрытие = длина(покрыто)
                    лучшее_множество = каждое_множество
                КОНЕЦ ЕСЛИ
            КОНЕЦ ДЛЯ

            ЕСЛИ лучшее_множество = НИЛЬ
                ВОЗВРАТ НИЛЬ  // Невозможно покрыть
            КОНЕЦ ЕСЛИ

            добавить(покрытие, лучшее_множество)
            оставшиеся_элементы = разность(оставшиеся_элементы, лучшее_множество)
        КОНЕЦ ПОКА

        ВОЗВРАТ покрытие
    КОНЕЦ
```

**JavaScript реализация:**
```javascript
function greedySetCover(sets, universe) {
    const cover = [];
    let remainingElements = new Set(universe);

    while (remainingElements.size > 0) {
        let bestSet = null;
        let maxCoverage = 0;

        for (const set of sets) {
            const covered = set.filter(element => remainingElements.has(element));
            if (covered.length > maxCoverage) {
                maxCoverage = covered.length;
                bestSet = set;
            }
        }

        if (bestSet === null) {
            return null; // Невозможно покрыть
        }

        cover.push(bestSet);
        bestSet.forEach(element => remainingElements.delete(element));
    }

    return cover;
}
```

## Сравнение форматов

### Печатные книги
**Преимущества:**
- Структурированное изложение
- Глубокое понимание
- Отсутствие отвлекающих факторов

**Недостатки:**
- Устаревание информации
- Высокая цена
- Отсутствие интерактивности

### Электронные книги
**Преимущества:**
- Доступность
- Поиск по тексту
- Обновления

**Недостатки:**
- Утомляемость глаз
- Отвлечение внимания

### Видеокурсы
**Преимущества:**
- Наглядность
- Объяснение голосом
- Визуализация

**Недостатки:**
- Медленное обучение
- Отсутствие возможности перечитать

### Интерактивные ресурсы
**Преимущества:**
- Практика в реальном времени
- [[Визуализация алгоритмов]]
- Немедленная обратная связь

**Недостатки:**
- Ограниченная теория
- Зависимость от интернета
