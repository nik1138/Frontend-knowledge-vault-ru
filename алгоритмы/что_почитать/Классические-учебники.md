# Классические учебники по алгоритмам

## Введение

Классические учебники по алгоритмам заслужили своё признание за десятилетия благодаря глубине изложения, фундаментальности и научной строгости. Эти книги остаются актуальными, несмотря на развитие технологий.

## Основные произведения

Пример алгоритма из книги - сортировка слиянием:

**Псевдокод:**
```
ФУНКЦИЯ сортировка_слиянием(массив)
    НАЧАЛО
        ЕСЛИ длина(массив) <= 1
            ВОЗВРАТ массив
        КОНЕЦ ЕСЛИ

        середина = длина(массив) / 2
        левая_часть = сортировка_слиянием(массив[0:середина])
        правая_часть = сортировка_слиянием(массив[середина:конец])

        ВОЗВРАТ слияние(левая_часть, правая_часть)
    КОНЕЦ

ФУНКЦИЯ слияние(левая, правая)
    НАЧАЛО
        результат = []
        левый_индекс = 0
        правый_индекс = 0

        ПОКА левый_индекс < длина(левая) И правый_индекс < длина(правая)
            ЕСЛИ левая[левый_индекс] <= правая[правый_индекс]
                добавить(результат, левая[левый_индекс])
                левый_индекс = левый_индекс + 1
            ИНАЧЕ
                добавить(результат, правая[правый_индекс])
                правый_индекс = правый_индекс + 1
            КОНЕЦ ЕСЛИ
        КОНЕЦ ПОКА

        ПОКА левый_индекс < длина(левая)
            добавить(результат, левая[левый_индекс])
            левый_индекс = левый_индекс + 1
        КОНЕЦ ПОКА

        ПОКА правый_индекс < длина(правая)
            добавить(результат, правая[правый_индекс])
            правый_индекс = правый_индекс + 1
        КОНЕЦ ПОКА

        ВОЗВРАТ результат
    КОНЕЦ
```

**JavaScript реализация:**
```javascript
function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }

    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));

    return merge(left, right);
}

function merge(left, right) {
    let result = [];
    let leftIndex = 0;
    let rightIndex = 0;

    while (leftIndex < left.length && rightIndex < right.length) {
        if (left[leftIndex] <= right[rightIndex]) {
            result.push(left[leftIndex]);
            leftIndex++;
        } else {
            result.push(right[rightIndex]);
            rightIndex++;
        }
    }

    return result.concat(left.slice(leftIndex)).concat(right.slice(rightIndex));
}
```

> Книга содержит строгие математические доказательства и глубокое теоретическое обоснование каждого алгоритма.
