# Bootstrap

Этот файл описывает CSS-фреймворк Bootstrap.

Bootstrap — один из самых популярных CSS-фреймворков для быстрой разработки адаптивных и кроссплатформенных веб-сайтов. Он предоставляет готовые компоненты и сеточную систему, что ускоряет процесс разработки.

## Основные особенности:

- Адаптивная сеточная система
- Готовые компоненты (кнопки, формы, навигация и т.д.)
- Система утилит для быстрого стилизования
- Темизация и настройка
