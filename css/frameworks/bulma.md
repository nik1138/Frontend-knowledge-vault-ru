# Bulma

Этот файл описывает CSS-фреймворк Bulma.

Bulma — современный CSS-фреймворк, основанный на Flexbox. Он предоставляет чистый, современный и адаптивный дизайн без необходимости в JavaScript, что делает его легким в использовании и настройке.

## Основные особенности:

- Основан на Flexbox
- Чистый и минималистичный дизайн
- Адаптивная сетка
- Множество готовых компонентов
- Легкая настройка через Sass-переменные
