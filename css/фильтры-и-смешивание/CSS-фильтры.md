# CSS-фильтры

CSS-фильтры - это мощный инструмент для применения визуальных эффектов к элементам HTML без использования внешних редакторов изображений. Они позволяют создавать эффекты, аналогичные тем, что применяются в графических редакторах, прямо в браузере.

## Основные понятия

CSS-фильтры применяются с помощью свойства `filter`, которое может принимать различные функции для изменения визуального представления элемента. Эти фильтры работают как "слои эффектов", которые изменяют отображение элемента без изменения его содержимого.

## Доступные фильтры

### 1. blur() - размытие

```css
.blur-element {
  filter: blur(5px);
}
```

Функция `blur()` создает эффект размытия по Гауссу. Значение указывает радиус размытия - чем больше значение, тем сильнее размытие. Используется для создания эффектов "background blur" и визуального акцента на других элементах.

### 2. brightness() - яркость

```css
.brighter {
  filter: brightness(1.5);
}

.darker {
  filter: brightness(0.5);
}
```

Функция `brightness()` регулирует яркость элемента. Значение 1.0 оставляет изображение без изменений, значения больше 1.0 делают его ярче, меньше 1.0 - темнее.

### 3. contrast() - контрастность

```css
.high-contrast {
  filter: contrast(200%);
}

.low-contrast {
  filter: contrast(50%);
}
```

Функция `contrast()` изменяет контрастность элемента. Значение 100% оставляет изображение без изменений, значения больше 100% увеличивают контрастность, меньше - уменьшают.

### 4. grayscale() - оттенки серого

```css
.gray-scale {
  filter: grayscale(100%);
}
```

Функция `grayscale()` преобразует изображение в оттенки серого. Значение 100% полностью преобразует изображение в черно-белое, 0% оставляет без изменений.

### 5. hue-rotate() - поворот оттенка

```css
.rotated-hue {
  filter: hue-rotate(90deg);
}
```

Функция `hue-rotate()` изменяет оттенок всех цветов в изображении. Значение задается в градусах (0-360deg), где 0deg оставляет изображение без изменений.

### 6. invert() - инверсия

```css
.inverted {
  filter: invert(100%);
}
```

Функция `invert()` инвертирует все цвета в изображении. Значение 100% полностью инвертирует цвета, 0% оставляет без изменений.

### 7. opacity() - непрозрачность

```css
.semi-transparent {
  filter: opacity(50%);
}
```

Функция `opacity()` изменяет непрозрачность элемента. Значение 100% оставляет элемент полностью непрозрачным, 0% делает его полностью прозрачным.

### 8. saturate() - насыщенность

```css
.high-saturation {
  filter: saturate(200%);
}

.desaturated {
  filter: saturate(50%);
}
```

Функция `saturate()` изменяет насыщенность цветов. Значение 100% оставляет изображение без изменений, значения больше 100% увеличивают насыщенность, меньше - уменьшают.

### 9. sepia() - сепия

```css
.sepia-effect {
  filter: sepia(100%);
}
```

Функция `sepia()` применяет эффект сепии к изображению. Значение 100% полностью преобразует изображение в сепию, 0% оставляет без изменений.

## Комбинирование фильтров

CSS-фильтры можно комбинировать, указывая несколько функций в одном свойстве `filter`:

```css
.combined-filters {
  filter: blur(2px) brightness(110%) contrast(120%);
}
```

При комбинировании фильтров порядок применения может влиять на конечный результат, особенно при использовании фильтров, которые изменяют цвета.

## Производительность и совместимость

CSS-фильтры могут быть ресурсоемкими, особенно при анимации или применении к большим элементам. В 2025 году поддержка фильтров в основных браузерах составляет:
- Chrome: полная поддержка с версии 18+
- Firefox: полная поддержка с версии 35+
- Safari: полная поддержка с версии 6+
- Edge: полная поддержка с версии 13+

Для оптимизации производительности рекомендуется:
- Использовать `will-change: filter` для элементов, которые будут анимироваться
- Ограничивать область применения фильтра с помощью `transform: translateZ(0)` для включения аппаратного ускорения
- Избегать применения фильтров к большому количеству элементов одновременно

## Практические примеры использования

### Эффект ховера для изображений

```css
.image-container img {
  transition: filter 0.3s ease;
}

.image-container:hover img {
  filter: brightness(110%) contrast(110%);
}
```

### Создание эффекта "заглушения" для фона при открытии модального окна

```css
.background-blur {
  filter: blur(3px);
}
```

### Адаптивный режим "ночь/день" с помощью фильтров

```css
.night-mode {
  filter: invert(100%) hue-rotate(180deg) contrast(90%) brightness(110%);
}
```

## Совместимость с российскими веб-стандартами

В 2025 году российские веб-разработчики активно используют CSS-фильтры в соответствии с национальными стандартами веб-доступности. При использовании фильтров важно учитывать:
- Сохранение читаемости текста при применении фильтров
- Возможность отключения анимаций для пользователей с ограниченными возможностями
- Совместимость с российскими браузерами, такими как Яндекс.Браузер и Mail.ru Browser

## Заключение

CSS-фильтры предоставляют мощный инструмент для создания визуальных эффектов без необходимости использования внешних графических редакторов. Их правильное использование позволяет создавать более привлекательный и интерактивный пользовательский интерфейс, при этом соблюдая требования к производительности и доступности.

Для дальнейшего изучения рекомендуется ознакомиться с [[Режимы-смешивания]], [[Backdrop-filter]] и [[Маски]].

> [!tip] 
> При работе с фильтрами в российских проектах рекомендуется тестировать производительность на устройствах с разными характеристиками, учитывая широкий спектр используемых устройств в регионе.

> [!warning]
> Избегайте чрезмерного использования фильтров, особенно в сочетании с анимациями, так как это может негативно сказаться на производительности и доступности интерфейса.
