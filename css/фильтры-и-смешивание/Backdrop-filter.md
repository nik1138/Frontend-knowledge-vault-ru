# Backdrop-filter

Свойство `backdrop-filter` в CSS позволяет применять графические эффекты к области, находящейся под элементом, но перед его содержимым. Это особенно полезно для создания эффектов размытия фона, как в современных интерфейсах операционных систем, а также для создания полупрозрачных панелей с эффектом "стекла".

## Основные понятия

Свойство `backdrop-filter` работает аналогично `filter`, но вместо применения эффекта к содержимому элемента, оно применяет эффект к области под элементом. Это позволяет создавать эффекты "стеклянной панели" (glassmorphism), когда элемент частично прозрачен, но при этом имеет визуальные эффекты, примененные к фону под ним.

## Поддерживаемые фильтры

Все CSS-фильтры, которые можно использовать с `filter`, также можно использовать с `backdrop-filter`:

### 1. blur() - размытие фона

```css
.backdrop-blur {
  backdrop-filter: blur(10px);
  background-color: rgba(255, 255, 255, 0.1);
}
```

Эффект размытия фона под элементом, создавая эффект "стеклянной панели". Особенно популярен в современном дизайне.

### 2. brightness() - яркость фона

```css
.backdrop-brightness {
  backdrop-filter: brightness(1.2);
  background-color: rgba(255, 255, 255, 0.2);
}
```

Изменение яркости области под элементом.

### 3. contrast() - контрастность фона

```css
.backdrop-contrast {
  backdrop-filter: contrast(1.5);
  background-color: rgba(0, 0, 0, 0.1);
}
```

Изменение контрастности области под элементом.

### 4. grayscale() - оттенки серого фона

```css
.backdrop-grayscale {
  backdrop-filter: grayscale(50%);
  background-color: rgba(255, 255, 255, 0.15);
}
```

Преобразование фона под элементом в оттенки серого.

### 5. hue-rotate() - поворот оттенка фона

```css
.backdrop-hue-rotate {
  backdrop-filter: hue-rotate(90deg);
  background-color: rgba(255, 255, 255, 0.1);
}
```

Поворот оттенка фона под элементом.

### 6. invert() - инверсия фона

```css
.backdrop-invert {
  backdrop-filter: invert(20%);
  background-color: rgba(255, 255, 255, 0.1);
}
```

Инверсия цветов фона под элементом.

### 7. opacity() - непрозрачность фона

```css
.backdrop-opacity {
  backdrop-filter: opacity(50%);
  background-color: rgba(255, 255, 255, 0.1);
}
```

Изменение непрозрачности фона под элементом.

### 8. saturate() - насыщенность фона

```css
.backdrop-saturate {
  backdrop-filter: saturate(1.8);
  background-color: rgba(255, 255, 255, 0.1);
}
```

Изменение насыщенности цветов фона под элементом.

### 9. sepia() - сепия фона

```css
.backdrop-sepia {
  backdrop-filter: sepia(60%);
  background-color: rgba(255, 255, 255, 0.1);
}
```

Применение эффекта сепии к фону под элементом.

## Комбинирование фильтров

Как и с обычным `filter`, можно комбинировать несколько эффектов:

```css
.combined-backdrop {
  backdrop-filter: blur(5px) brightness(110%) contrast(120%);
  background-color: rgba(255, 255, 255, 0.15);
}
```

## Практические примеры использования

### Эффект "стеклянной панели" (Glassmorphism)

```css
.glass-panel {
  backdrop-filter: blur(10px);
  background: rgba(255, 255, 255, 0.1);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 16px;
  padding: 20px;
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.37);
}
```

### Полупрозрачное модальное окно с размытым фоном

```css
.modal-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  backdrop-filter: blur(5px);
  background-color: rgba(0, 0, 0, 0.3);
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### Навигационная панель с эффектом размытия

```css
.nav-bar {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  backdrop-filter: blur(12px);
  background: rgba(255, 255, 255, 0.1);
  border-bottom: 1px solid rgba(255, 255, 255, 0.2);
  padding: 1rem;
  z-index: 100;
}
```

### Карточка с эффектом стекла

```css
.glass-card {
  backdrop-filter: blur(16px) saturate(180%);
  background-color: rgba(17, 25, 40, 0.25);
  border-radius: 12px;
  border: 1px solid rgba(255, 255, 255, 0.18);
  padding: 24px;
  margin: 20px;
}
```

## Совместимость и производительность

Поддержка `backdrop-filter` в браузерах в 2025 году:
- Chrome: полная поддержка с версии 76+
- Firefox: полная поддержка с версии 74+
- Safari: полная поддержка с версии 9+
- Edge: полная поддержка с версии 79+

Важные моменты:
- `backdrop-filter` может быть ресурсоемким, особенно на мобильных устройствах
- Рекомендуется использовать умеренные значения размытия (обычно 5-20px)
- В российских проектах особенно важно тестировать производительность на бюджетных устройствах

## Оптимизация производительности

### Использование transform для аппаратного ускорения

```css
.optimized-backdrop {
  backdrop-filter: blur(10px);
  transform: translateZ(0); /* Включает аппаратное ускорение */
}
```

### Ограничение области применения

```css
.limited-backdrop {
  backdrop-filter: blur(8px);
  contain: layout style paint; /* Ограничивает область рендеринга */
}
```

### Альтернативы для слабых устройств

```css
@media (prefers-reduced-motion: reduce) {
  .backdrop-element {
    backdrop-filter: none;
    background-color: rgba(255, 255, 255, 0.7);
  }
}

/* Или определение по производительности */
@supports not (backdrop-filter: blur(1px)) {
  .fallback-backdrop {
    background-color: rgba(255, 255, 255, 0.8);
  }
}
```

## Сравнение с другими методами

### Backdrop-filter vs CSS Filters

- `backdrop-filter` применяет эффект к области под элементом
- `filter` применяет эффект к содержимому элемента
- `backdrop-filter` требует частичной прозрачности для видимости эффекта

### Backdrop-filter vs SVG Filters

- `backdrop-filter` проще в использовании
- `backdrop-filter` интегрируется непосредственно в CSS
- SVG фильтры могут быть более гибкими, но сложнее в реализации

## Современные тенденции использования

В 2025 году популярны следующие подходы к использованию `backdrop-filter`:
- Эффект "стеклянной панели" в навигационных элементах
- Размытие фона для модальных окон и всплывающих подсказок
- Создание визуального контраста между элементами и фоном
- Интеграция с темными/светлыми темами интерфейса

## Заключение

Свойство `backdrop-filter` предоставляет мощный инструмент для создания современных визуальных эффектов в веб-интерфейсах. Его правильное использование позволяет создавать привлекательные и современные дизайны, особенно в стиле glassmorphism, при этом сохраняя производительность и доступность.

Для более глубокого понимания рекомендуется изучить [[CSS-фильтры]], [[Режимы-смешивания]] и [[Маски]].

> При работе с `backdrop-filter` в российских проектах рекомендуется использовать умеренные значения размытия и тестировать производительность на различных устройствах, особенно учитывая разнообразие используемых устройств в регионе.

> [!warning]
> Избегайте чрезмерного использования `backdrop-filter`, особенно с сильным размытием, так как это может негативно сказаться на производительности и читаемости контента.
