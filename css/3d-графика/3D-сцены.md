# 3D-сцены

3D-сцены в CSS представляют собой комплексное использование свойств перспективы, трансформаций и стилизации для создания объемных визуальных композиций. В 2025 году в российской веб-разработке 3D-сцены становятся важным инструментом для создания интерактивных интерфейсов и привлекательных пользовательских впечатлений.

## Основы построения 3D-сцен

Для создания 3D-сцены необходимо правильно организовать HTML-структуру и применить соответствующие CSS-свойства:

```html
<div class="scene">
  <div class="cube">
    <div class="face front">Передняя</div>
    <div class="face back">Задняя</div>
    <div class="face right">Правая</div>
    <div class="face left">Левая</div>
    <div class="face top">Верхняя</div>
    <div class="face bottom">Нижняя</div>
  </div>
</div>
```

```css
.scene {
  width: 200px;
  height: 200px;
  perspective: 1000px;
}

.cube {
  position: relative;
  width: 100%;
  height: 100%;
  transform-style: preserve-3d;
  transform: rotateX(-25deg) rotateY(-25deg);
}

.face {
  position: absolute;
  width: 200px;
  height: 200px;
  border: 2px solid #333;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 18px;
  opacity: 0.9;
}

.front  { transform: rotateY(0deg) translateZ(100px); background: rgba(255, 0, 0, 0.7); }
.back   { transform: rotateY(180deg) translateZ(100px); background: rgba(0, 255, 0, 0.7); }
.right  { transform: rotateY(90deg) translateZ(100px); background: rgba(0, 0, 255, 0.7); }
.left   { transform: rotateY(-90deg) translateZ(100px); background: rgba(255, 255, 0, 0.7); }
.top    { transform: rotateX(90deg) translateZ(100px); background: rgba(255, 0, 255, 0.7); }
.bottom { transform: rotateX(-90deg) translateZ(100px); background: rgba(0, 255, 255, 0.7); }
```

## Организация 3D-пространства

### Контейнер сцены

Контейнер сцены определяет перспективу и служит основой для всех 3D-объектов:

```css
.scene-container {
  perspective: 1200px;
  perspective-origin: 50% 50%;
  width: 400px;
  height: 400px;
  margin: 50px auto;
}
```

### Сохранение 3D-стиля

Свойство `transform-style: preserve-3d` необходимо для сохранения 3D-положения дочерних элементов:

```css
.scene-3d {
  transform-style: preserve-3d;
  transform: translateZ(0); /* Для активации GPU */
}
```

## Создание сложных 3D-объектов

### 3D-куб с анимацией

```css
.animated-cube {
  position: relative;
  width: 200px;
  height: 200px;
  transform-style: preserve-3d;
  animation: rotateScene 15s infinite linear;
}

@keyframes rotateScene {
  0% { transform: rotateX(0) rotateY(0) rotateZ(0); }
  100% { transform: rotateX(360deg) rotateY(360deg) rotateZ(360deg); }
}

.cube-face {
  position: absolute;
  width: 200px;
  height: 200px;
  background: rgba(255, 255, 255, 0.85);
  border: 2px solid #333;
  display: flex;
  align-items: center;
  justify-content: center;
  backface-visibility: hidden;
}
```

### 3D-сетка

```css
.grid-3d {
  transform-style: preserve-3d;
  transform: rotateX(-30deg) rotateY(-30deg);
}

.grid-cell {
  position: absolute;
  width: 50px;
  height: 50px;
  transform-style: preserve-3d;
}

/* Распределение ячеек в 3D-пространстве */
.cell-1 { transform: translate3d(0, 0, 0); }
.cell-2 { transform: translate3d(60px, 0, 0); }
.cell-3 { transform: translate3d(0, 60px, 0); }
.cell-4 { transform: translate3d(0, 0, 60px); }
```

## Свет и тени в 3D-сценах

Хотя CSS не предоставляет полноценного управления освещением, можно имитировать световые эффекты с помощью градиентов и теней:

```css
.lighted-cube {
  transform-style: preserve-3d;
}

.face {
  background: 
    linear-gradient(
      135deg, 
      rgba(255, 255, 255, 0.2) 0%, 
      rgba(0, 0, 0, 0.1) 100%
    );
  box-shadow: inset 5px 5px 15px rgba(0, 0, 0, 0.2);
}
```

## Оптимизация производительности

### Использование GPU

```css
.optimized-scene {
  transform: translateZ(0); /* Принудительное использование GPU */
  will-change: transform; /* Подсказка браузеру о предстоящей анимации */
}
```

### Управление сложностью

В 2025 году в России важно учитывать разнообразие устройств, на которых будет отображаться контент:

- Минимизируйте количество одновременно анимируемых 3D-элементов
- Используйте `backface-visibility: hidden` для скрытия задних сторон элементов
- Применяйте `contain: paint` для изоляции сложных 3D-объектов

```css
.complex-object {
  contain: paint;
  backface-visibility: hidden;
}
```

## Интерактивные 3D-сцены

Создание сцен, реагирующих на действия пользователя:

```css
.interactive-scene {
  perspective: 1000px;
  width: 300px;
  height: 300px;
  margin: 50px auto;
}

.interactive-object {
  transform-style: preserve-3d;
  transition: transform 0.5s ease;
}

.interactive-scene:hover .interactive-object {
  transform: rotateX(20deg) rotateY(30deg) translateZ(50px);
}
```

## Практические примеры использования

### 3D-галерея

```css
.gallery-3d {
  perspective: 1500px;
  width: 400px;
  height: 300px;
  margin: 100px auto;
}

.gallery-container {
  transform-style: preserve-3d;
  transform: translateZ(-1000px);
}

.gallery-item {
  position: absolute;
  width: 300px;
  height: 200px;
  transform-origin: center center -500px;
  transition: transform 1s ease, opacity 1s ease;
}
```

### 3D-меню

```css
.menu-3d {
  perspective: 1200px;
  display: flex;
  justify-content: center;
  align-items: center;
}

.menu-item {
  transform-style: preserve-3d;
  margin: 0 10px;
  transition: transform 0.3s ease;
}

.menu-item:hover {
  transform: translateZ(30px) rotateX(10deg);
}
```

## Совместимость и реализация

Для обеспечения совместимости с различными браузерами в России в 2025 году:

```css
.scene {
  -webkit-perspective: 1000px;
  -moz-perspective: 1000px;
  perspective: 1000px;
  
  -webkit-transform-style: preserve-3d;
  -moz-transform-style: preserve-3d;
  transform-style: preserve-3d;
}
```
