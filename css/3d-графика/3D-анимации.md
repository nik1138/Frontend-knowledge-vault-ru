# 3D-анимации

3D-анимации в CSS позволяют создавать динамичные и привлекательные визуальные эффекты, используя возможности 3D-трансформаций. Они особенно популярны в 2025 году благодаря улучшенной производительности браузеров и росту интереса к интерактивному веб-дизайну в России.

## Основы 3D-анимаций

3D-анимации создаются с помощью CSS-свойств `animation` и `@keyframes`, которые работают с 3D-трансформациями. Ключевые свойства:

- `animation`: определяет параметры анимации
- `@keyframes`: задает промежуточные состояния анимации
- `transform`: применяет 3D-преобразования в ключевых кадрах

### Пример простой 3D-анимации

```css
.cube {
  width: 100px;
  height: 100px;
  animation: rotateCube 4s infinite linear;
}

@keyframes rotateCube {
  0% {
    transform: rotateX(0deg) rotateY(0deg);
  }
  100% {
    transform: rotateX(360deg) rotateY(360deg);
  }
}
```

## Типы 3D-анимаций

### Вращение
Наиболее распространенный тип 3D-анимации - вращение вокруг одной или нескольких осей:

```css
.rotation {
  animation: spin 2s infinite ease-in-out;
}

@keyframes spin {
  0% { transform: rotateX(0deg) rotateY(0deg) rotateZ(0deg); }
  100% { transform: rotateX(360deg) rotateY(360deg) rotateZ(360deg); }
}
```

### Параллакс-эффект
Создание иллюзии глубины с помощью различных скоростей движения элементов:

```css
.parallax-layer {
  animation: float 8s infinite ease-in-out;
}

@keyframes float {
  0%, 100% { transform: translateZ(-50px) rotateY(10deg); }
  50% { transform: translateZ(50px) rotateY(-10deg); }
}
```

## Продвинутые техники

### Анимация с использованием transform3d

```css
.floating-card {
  animation: floatCard 3s ease-in-out infinite;
}

@keyframes floatCard {
  0% {
    transform: translateY(0) rotateX(10deg) translateZ(0);
  }
  50% {
    transform: translateY(-20px) rotateX(20deg) translateZ(30px);
  }
  100% {
    transform: translateY(0) rotateX(10deg) translateZ(0);
  }
}
```

### Комбинирование анимаций

```css
.complex-animation {
  animation: 
    rotate 4s linear infinite,
    pulse 2s ease-in-out infinite alternate,
    float 3s ease-in-out infinite alternate;
}

@keyframes rotate {
  from { transform: rotateY(0deg); }
  to { transform: rotateY(360deg); }
}

@keyframes pulse {
  from { transform: scale3d(1, 1, 1); }
  to { transform: scale3d(1.05, 1.05, 1.05); }
}

@keyframes float {
  from { transform: translateY(0); }
  to { transform: translateY(-15px); }
}
```

## Производительность и оптимизация

В 2025 году при создании 3D-анимаций в России важно учитывать:

- Использование свойства `will-change` для предварительной подготовки анимируемых элементов
- Оптимизация через `transform` и `opacity` для использования GPU
- Минимизация количества анимируемых элементов одновременно
- Тестирование на различных устройствах, особенно в регионах с ограниченными ресурсами

```css
.optimized-animation {
  will-change: transform;
  transform: translateZ(0); /* Принудительное использование GPU */
  animation: myAnimation 2s infinite;
}
```

## Практические примеры

### Анимированный 3D-куб

```css
.cube-container {
  perspective: 1000px;
}

.cube {
  position: relative;
  width: 200px;
  height: 200px;
  transform-style: preserve-3d;
  animation: rotateCube 10s infinite linear;
}

.face {
  position: absolute;
  width: 200px;
  height: 200px;
  background: rgba(255, 255, 255, 0.9);
  border: 2px solid #333;
  display: flex;
  align-items: center;
  justify-content: center;
}

.front { transform: translateZ(100px); }
.back { transform: rotateY(180deg) translateZ(100px); }
.right { transform: rotateY(90deg) translateZ(100px); }
.left { transform: rotateY(-90deg) translateZ(100px); }
.top { transform: rotateX(90deg) translateZ(100px); }
.bottom { transform: rotateX(-90deg) translateZ(100px); }

@keyframes rotateCube {
  0% { transform: rotateX(0) rotateY(0) rotateZ(0); }
  100% { transform: rotateX(360deg) rotateY(360deg) rotateZ(360deg); }
}
```
