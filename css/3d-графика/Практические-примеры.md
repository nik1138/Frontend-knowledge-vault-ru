# Практические примеры

В этой статье представлены реальные примеры использования 3D-графики в CSS с учетом современных тенденций российской веб-разработки в 2025 году. Все примеры готовы к использованию и содержат пошаговые инструкции по реализации.

## Пример 1: 3D-карусель товаров

Популярный элемент интерфейса для интернет-магазинов, особенно актуален для российского рынка электронной коммерции.

### HTML-структура

```html
<div class="product-carousel-3d">
  <div class="carousel-track">
    <div class="carousel-item">
      <img src="product1.jpg" alt="Товар 1">
      <div class="product-info">Товар 1</div>
    </div>
    <div class="carousel-item">
      <img src="product2.jpg" alt="Товар 2">
      <div class="product-info">Товар 2</div>
    </div>
    <div class="carousel-item">
      <img src="product3.jpg" alt="Товар 3">
      <div class="product-info">Товар 3</div>
    </div>
  </div>
  <button class="carousel-nav prev">‹</button>
  <button class="carousel-nav next">›</button>
</div>
```

### CSS-реализация

```css
.product-carousel-3d {
  width: 300px;
  height: 200px;
  perspective: 1500px;
  margin: 100px auto;
  position: relative;
}

.carousel-track {
  width: 100%;
  height: 100%;
  position: absolute;
  transform-style: preserve-3d;
  transition: transform 0.8s cubic-bezier(0.17, 0.67, 0.83, 0.67);
}

.carousel-item {
  position: absolute;
  width: 250px;
  height: 150px;
  background: white;
  border-radius: 10px;
  box-shadow: 0 10px 30px rgba(0, 0, 0, 0.3);
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  backface-visibility: hidden;
}

/* Расположение элементов в 3D-пространстве */
.carousel-item:nth-child(1) { transform: rotateY(0deg) translateZ(300px); }
.carousel-item:nth-child(2) { transform: rotateY(60deg) translateZ(300px); }
.carousel-item:nth-child(3) { transform: rotateY(120deg) translateZ(300px); }

.product-info {
  text-align: center;
  font-weight: bold;
  margin-top: 10px;
}
```

## Пример 2: Интерактивная 3D-кнопка

Современный элемент интерфейса, добавляющий глубину и интерактивность.

### HTML

```html
<button class="button-3d">
  <span class="button-text">Нажми меня</span>
</button>
```

### CSS

```css
.button-3d {
  position: relative;
  padding: 15px 30px;
  background: linear-gradient(45deg, #6a11cb 0%, #2575fc 100%);
  color: white;
  border: none;
  border-radius: 50px;
  font-size: 16px;
  font-weight: bold;
  cursor: pointer;
  transform: translateZ(0);
  transition: transform 0.2s, box-shadow 0.2s;
  overflow: hidden;
}

.button-3d:before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: linear-gradient(45deg, #2575fc 0%, #6a11cb 100%);
  z-index: -1;
  transform: translateZ(-20px);
  border-radius: 50px;
}

.button-3d:after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.1);
  z-index: -2;
  transform: translateZ(-40px);
  border-radius: 50px;
}

.button-3d:hover {
  transform: translateZ(10px);
  box-shadow: 0 15px 35px rgba(37, 117, 252, 0.4);
}

.button-3d:active {
  transform: translateZ(5px);
  transition: transform 0.1s;
}
```

## Пример 3: 3D-меню навигации

Интерактивное меню с объемными эффектами, подходящее для современных веб-сайтов.

### HTML

```html
<nav class="menu-3d">
  <a href="#" class="menu-item">Главная</a>
  <a href="#" class="menu-item">О нас</a>
  <a href="#" class="menu-item">Услуги</a>
  <a href="#" class="menu-item">Контакты</a>
</nav>
```

### CSS

```css
.menu-3d {
  display: flex;
  justify-content: center;
  gap: 20px;
  perspective: 1000px;
  padding: 20px;
}

.menu-item {
  padding: 12px 24px;
  background: #f0f0f0;
  color: #333;
  text-decoration: none;
  border-radius: 8px;
  transform-style: preserve-3d;
  transition: transform 0.3s ease, background 0.3s ease;
  position: relative;
  box-shadow: 0 5px 15px rgba(0, 0, 0, 0.1);
}

.menu-item:before {
  content: '';
  position: absolute;
  top: 5px;
  left: 5px;
  right: 5px;
  bottom: 5px;
  background: inherit;
  border-radius: 8px;
  transform: translateZ(-5px);
  z-index: -1;
}

.menu-item:hover {
  transform: translateZ(15px) rotateX(5deg);
  background: #e0e0e0;
}

.menu-item.active {
  background: #2575fc;
  color: white;
  transform: translateZ(10px);
}
```

  gap: 12px;

.btn {

  border: none;
  border-radius: 10px;
  font-size: 18px;
  font-weight: bold;

  transform-style: preserve-3d;

  bottom: 3px;
  background: inherit;
  border-radius: 8px;
  transform: translateZ(-5px);
  z-index: -1;

## Рекомендации по оптимизации

### Производительность

> [!tip] Совет
> Используйте `transform: translateZ(0)` или `will-change: transform` для активации аппаратного ускорения GPU, особенно важно для российских пользователей с разнообразными устройствами.

### Адаптивность

В 2025 году в России особенно важно учитывать разнообразие устройств:

```css
@media (max-width: 768px) {
  .portfolio-3d {
    flex-direction: column;
    align-items: center;
  }
  
  .project-card {
    transform-style: preserve-3d;
  }
}
```

### Совместимость

Для обеспечения работы на всех устройствах:

```css
.compatible-3d-element {
  -webkit-transform-style: preserve-3d;
  -moz-transform-style: preserve-3d;
  transform-style: preserve-3d;
  
  -webkit-backface-visibility: hidden;
  -moz-backface-visibility: hidden;
  backface-visibility: hidden;
}
```
