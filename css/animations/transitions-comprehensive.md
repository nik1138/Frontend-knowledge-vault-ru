# CSS Transitions

Этот файл описывает CSS-переходы.

CSS Transitions позволяют плавно изменять значения свойств элемента за определённый период времени. Это позволяет создавать плавные анимационные эффекты без использования JavaScript или Flash.

## Основные свойства:

- transition-property: Определяет, какие свойства будут анимироваться
- transition-duration: Определяет, как долго длится переход
- transition-timing-function: Определяет кривую скорости перехода
- transition-delay: Определяет задержку перед началом перехода

## Сокращённый синтаксис:

```css
transition: property duration timing-function delay;
```
