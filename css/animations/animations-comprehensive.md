# CSS Animations

Этот файл описывает CSS-анимации в целом.

CSS-анимации позволяют создавать сложные анимационные эффекты без использования JavaScript. Они обеспечивают более широкие возможности по сравнению с переходами (transitions), позволяя определять промежуточные состояния и более сложное поведение.

## Основные компоненты:

1. **@keyframes**: Определяет анимационный ключевой кадр и его свойства
2. **Свойства анимации**: Управляют воспроизведением и поведением анимации
3. **Тайминги**: Определяют, как быстро или медленно будет воспроизводиться анимация

## Применения:

- Анимации элементов при наведении курсора
- Загрузочные анимации
- Анимации при появлении элементов
- Интерактивные анимации
