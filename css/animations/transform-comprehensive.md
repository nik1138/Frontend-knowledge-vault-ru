# CSS Transform

Этот файл описывает CSS-трансформации.

CSS transform позволяет изменять форму, размер и положение элемента. Он не влияет на нормальный поток документа и не влияет на положение других элементов. Transform может применяться к элементам в 2D или 3D пространстве.

## Основные функции трансформации:

- translate(): Перемещает элемент по осям X и Y
- rotate(): Поворачивает элемент на заданный угол
- scale(): Изменяет размер элемента
- skew(): Наклоняет элемент
- matrix(): Комбинирует несколько трансформаций

## 3D трансформации:

- translate3d(), rotate3d(), scale3d(): 3D версии 2D функций
- perspective(): Определяет глубину для 3D-позиционирования
