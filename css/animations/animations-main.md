# css/animations

Этот файл описывает анимации в CSS.

CSS Animations охватывает различные методы создания анимационных эффектов с использованием CSS. Анимации улучшают пользовательский опыт, делая интерфейсы более живыми и интуитивно понятными.

## Основные типы анимаций:

1. **Переходы (Transitions)**: Плавные изменения значений свойств
2. **Ключевые кадры (Keyframes)**: Более сложные анимации с промежуточными состояниями
3. **Трансформации (Transforms)**: Изменение формы, размера и положения элементов
4. **Анимации на основе свойств**: Анимации, оптимизированные для производительности

## Свойства анимаций:

- transition: Плавные изменения значений свойств
- animation: Более сложные анимации с ключевыми кадрами
- transform: Изменение формы, размера и положения элементов
- will-change: Подсказка браузеру о предстоящих изменениях
