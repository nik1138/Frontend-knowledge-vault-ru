# Статьи и блоги по CSS

## Российские ресурсы и блоги

### Хабр
Хабр остается одним из самых популярных ресурсов для изучения CSS в российском сегменте интернета. Регулярно публикуются статьи по современным возможностям CSS, методологиям и практикам.

Популярные рубрики:
- [[CSS]] - основные статьи по CSS
- [[Веб-разработка]] - более широкая тематика
- [[Frontend]] - комплексные материалы

### CSS-лента на Дев.to (русская локализация)
Платформа, где российские разработчики делятся опытом и знаниями. Особенно полезны статьи по современным возможностям CSS, которые официально не вошли в спецификации, но уже доступны в браузерах.

### CSS-академия
Сайт с подробными статьями на русском языке. Покрывает как основы, так и продвинутые темы CSS. Особенно полезны статьи по [[CSS-анимациям]] и [[адаптивной верстке]].

## Зарубежные ресурсы с переводами

### Smashing Magazine
Один из ведущих ресурсов по веб-дизайну и разработке. Многие статьи переводятся на русский язык российскими сообществами.

Популярные статьи:
- "The Complete Guide to Flexbox"
- "The Complete Guide to Grid"
- "Modern CSS Solutions for Old CSS Problems"

### CSS-Tricks
Классический ресурс с огромной коллекцией статей по CSS. Многие статьи доступны в переводе на русскоязычных ресурсах.

### Web.dev (от Google)
Официальные статьи от Google по современным возможностям CSS. Особенно полезны разделы по производительности и новым возможностям.

## Тематические блоги российских экспертов

### Блог Ильи Кантора
Регулярно публикуются статьи по современным возможностям CSS, особенно по анимациям и интерактивным элементам.

### Блог Артема Кузнецова
Специализируется на архитектуре CSS и методологиях. Полезные статьи по [[БЭМ]] и [[ITCSS]].

### Блог Яндекса
Официальные статьи от команды Яндекса. Особое внимание уделяется производительности и поддержке устаревших браузеров в российских реалиях.

## Полезные статьи по темам

### Начинающим
- "CSS для чайников" - базовые понятия
- "Как работает специфичность в CSS" - важная концепция
- "Понимание box model" - основы визуального форматирования

### Средний уровень
- "Flexbox и Grid: когда что использовать" - практические рекомендации
- "CSS Custom Properties: практическое руководство" - переменные в CSS
- "Адаптивная верстка без медиа-запросов" - современные подходы

### Продвинутый уровень
- "Container Queries: новая эра адаптивности" - передовые технологии
- "CSS Subgrid: полное руководство" - сложные грид-структуры
- "CSS Houdini: API для CSS" - программное управление стилями

## Блоги компаний

### Блог Авито
Статьи по производительности и масштабируемости CSS в крупных проектах. Особенно актуально для российского рынка.

### Блог Тинькофф
Регулярные публикации по компонентному подходу и архитектуре CSS в банковских приложениях.

### Блог Сбер
Статьи по доступности (accessibility) и верстке для государственных проектов.

## RSS-каналы и рассылки

### CSS Weekly
Еженедельная рассылка с лучшими статьями по CSS. Многие материалы доступны с переводом.

### Frontend Weekly
Русскоязычная рассылка, включающая статьи по CSS. Часто публикуются обзоры новых возможностей.
