# Документация и справочники CSS

## Официальные спецификации CSS

### W3C Specifications
Официальные спецификации от World Wide Web Consortium - основной источник информации о текущем состоянии CSS. Включает черновики и рекомендованные версии.

**CSS 2.2 Specification**
- Последняя стабильная версия, полностью поддерживаемая браузерами
- Базовые свойства и правила

**CSS 3 Modules**
- Модульная структура спецификаций
- Постепенное внедрение новых возможностей

**CSS 2025 Specification**
- Современные возможности CSS
- Поддержка новых браузерных технологий

### CSS Working Group Drafts
Промежуточные версии спецификаций, доступные на GitHub. Полезны для отслеживания будущих возможностей CSS.

## Онлайн-справочники

### MDN Web Docs
Самый популярный справочник по веб-технологиям. Обновляется регулярно и содержит информацию на русском языке.

**Преимущества:**
- Подробное описание свойств
- Примеры кода
- Совместимость с браузерами
- Переводы на русский язык

**Полезные разделы:**
- [[CSS Reference]] - полный справочник по свойствам
- [[CSS Guide]] - руководства по использованию
- [[CSS Examples]] - практические примеры

### CSS-Tricks Almanac
Интерактивный справочник с живыми примерами. Особенно полезен для изучения новых свойств.

### Can I Use
Справочник совместимости CSS-свойств с различными браузерами. Критически важен при разработке под российские реалии, где до сих пор используются устаревшие версии браузеров.

## Инструменты для работы с документацией

### DevDocs.io
Офлайн-документация для множества технологий, включая CSS. Особенно полезна при ограниченном доступе к интернету.

### Dash (для macOS)
Популярный инструмент для просмотра документации. Включает CSS-справочник с возможностью поиска.

### Zeal (для Windows/Linux)
Аналог Dash для других операционных систем. Бесплатная альтернатива с поддержкой CSS-документации.

## Справочники по префиксу браузеров

### Autoprefixer Documentation
Документация по автоматическому добавлению префиксов. Особенно важно для поддержки старых браузеров в России.

### Can I Use (детализированная версия)
Подробная информация о необходимости префиксов для каждого свойства в зависимости от версии браузера.

## Современные возможности CSS

### CSS Custom Properties (Переменные)
- Спецификация CSS Custom Properties
- Практические примеры использования
- Темизация интерфейсов

### CSS Environment Variables
- Системные переменные (например, безопасные области на мобильных устройствах)
- Практическое применение

### CSS @supports (Feature Queries)
- Условное применение стилей
- Постепенное улучшение (progressive enhancement)

### CSS Nesting
- Вложенные правила (новое в CSS 2025)
- Совместимость и реализация

## Ресурсы для проверки синтаксиса

### CSS Validator (W3C)
Официальный валидатор CSS. Проверяет синтаксис и соответствие спецификациям.

### CSS Lint
Инструмент для анализа качества CSS-кода. Проверяет на наличие потенциальных проблем.

## Справочники по доступности (a11y)

### MDN Accessibility
Раздел документации, посвященный доступности в CSS. Особенно важен для государственных и коммерческих сайтов в России.

### WebAIM CSS и доступность
Руководство по использованию CSS с учетом требований доступности.
