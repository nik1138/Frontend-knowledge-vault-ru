# Книги по CSS

## Основные рекомендованные книги

### 1. "CSS. Карманный справочник" - Дж. ДUCKETТ

Классическая книга для начинающих и опытных разработчиков. Подробно описаны основы CSS, селекторы, позиционирование, флексбокс и гриды. Отличный справочник для повседневной работы.

**Преимущества:**
- Простое объяснение сложных концепций
- Много практических примеров
- Подходит для начинающих

### 2. "CSS. Секреты верстки" - Л. ЗАЙЦЕВА

Российская литература, ориентированная на российские реалии. Книга охватывает современные методы верстки, адаптивный дизайн и работу с фреймворками.

### 3. "Выразительный CSS" - К. КОНЕЛЬ

Одна из лучших книг по архитектуре CSS. Автор рассматривает принципы написания поддерживаемого кода, методологии (включая [[БЭМ]] и [[ITCSS]]).

### 4. "CSS для веб-дизайнеров" - Д. М. ШЕРРИС

Хорошая книга для дизайнеров, переходящих в веб-разработку. Помогает понять, как правильно использовать CSS для реализации дизайнерских решений.

## Современные издания 2024-2025

### "CSS в подлиннике" - А. ПОНОМАРЕВ

Новейшая книга российского автора, посвященная современным возможностям CSS. Рассматриваются последние спецификации, контейнерные запросы, логические свойства и пространство имен.

### "Архитектура CSS в условиях российского сегмента интернета"

Специализированное издание, посвященное особенностям верстки под российские реалии: поддержка устаревших браузеров, особенности шрифтов, локализация и т.д.

## Электронные книги и PDF-ресурсы

- [[Современный CSS. Руководство 2025]] - доступна в формате PDF на российских ресурсах
- "CSS Grid и Flexbox. Практическое руководство" - интерактивное издание с примерами кода

## Книги на английском языке (с переводом)

### "CSS Master" - Б. КАССЕЛЬ

Отличное издание для продвинутых разработчиков. Рассматриваются сложные концепции, оптимизация производительности и современные методы работы с CSS.

### "CSS Secrets" - Л. ХАВЕРБЕКЕ

Книга, раскрывающая малоизвестные возможности CSS. Отлично подходит для расширения знаний и изучения нестандартных решений.

## Как выбрать книгу по CSS

1. **Для новичков**: начните с "CSS. Карманный справочник" или "CSS для веб-дизайнеров"
2. **Для среднего уровня**: "CSS. Секреты верстки" или "Выразительный CSS"
3. **Для продвинутых**: "CSS Master" или "CSS в подлиннике"
