# CSS Variables

Этот файл описывает CSS-переменные (кастомные свойства).

CSS-переменные, также известные как кастомные свойства, позволяют определять переиспользуемые значения в CSS. Они помогают создавать более поддерживаемый и гибкий CSS-код, позволяя легко изменять значения в одном месте, которые затем применяются ко всему проекту.

## Определение переменных:

```css
:root {
  --main-color: #007bff;
  --secondary-color: #6c757d;
  --font-size: 16px;
}
```

## Использование переменных:

```css
.element {
  color: var(--main-color);
  font-size: var(--font-size);
}
```

## Преимущества:

- Упрощение поддержки: изменение значения в одном месте
- Повышение согласованности: единые значения для цветов, размеров и т.д.
- Динамическое изменение: переменные могут изменяться через JavaScript
