# typography

Этот файл описывает типографику в CSS.

CSS Typography — это практика оформления текстового контента с использованием CSS. Хорошая типографика улучшает читаемость, доступность и визуальную привлекательность веб-сайтов.

## Основные аспекты типографики:

- Размер шрифта: Установка подходящих размеров для различных элементов
- Межстрочный интервал: Настройка интервала для улучшения читаемости
- Высота строки: Баланс между читаемостью и плотностью текста
- Пространство между буквами: Регулировка кернинга и трекинга
- Выравнивание: Выбор подходящего выравнивания для различных типов контента

## Свойства типографики:

- font-family: Выбор гарнитуры шрифта
- font-size: Размер шрифта
- font-weight: Насыщенность шрифта
- line-height: Высота строки
- text-align: Выравнивание текста
- letter-spacing: Расстояние между буквами
- text-transform: Преобразование регистра
