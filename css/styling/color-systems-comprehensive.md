# CSS Color Systems and Palettes

Этот файл описывает системы цветов и палитры в CSS.

CSS предоставляет несколько способов определения цветов для элементов. Понимание различных систем цветов и подходов к созданию палитр критически важно для создания визуально привлекательных и доступных интерфейсов.

## Системы цветов:

1. **Named Colors**: Ключевые слова (red, blue, black и т.д.)
2. **Hexadecimal**: Шестнадцатеричные значения (#ff0000, #336699 и т.д.)
3. **RGB/RGBA**: Значения красного, зеленого, синего (rgb(255, 0, 0), rgba(255, 0, 0, 0.5))
4. **HSL/HSLA**: Оттенок, насыщенность, светлота (hsl(0, 100%, 50%), hsla(0, 100%, 50%, 0.5))

## Создание палитр:

- Использование CSS-переменных для определения цветовой схемы
- Поддержание контрастности для доступности
- Учет особенностей цветовосприятия
