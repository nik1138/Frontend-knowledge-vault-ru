# Продвинутая CSS-типографика

Продвинутая CSS-типографика - это искусство и наука создания читаемого, доступного и визуально привлекательного текста на веб-страницах. Правильная типографика играет ключевую роль в пользовательском опыте и доступности веб-сайтов.

## Основы веб-типографики

Веб-типографика отличается от печатной типографики рядом особенностей:

- **Адаптивность** - текст должен хорошо отображаться на различных устройствах и размерах экрана
- **Доступность** - типографика должна быть доступна для пользователей с нарушениями зрения
- **Читаемость** - выбор шрифта, размера и интервалов должен способствовать легкому восприятию текста
- **Производительность** - загрузка шрифтов не должна замедлять сайт

## Единицы измерения в типографике

В CSS доступно несколько единиц измерения, подходящих для типографики:

### `em` - относительная единица
Единица `em` относительно размера шрифта родительского элемента:

```css
.container {
  font-size: 16px;
}

.text {
  margin: 1em; /* 16px относительно родительского font-size */
  line-height: 1.5em; /* 24px относительно родительского font-size */
}
```

### `rem` - корневая относительная единица
Единица `rem` относительно размера шрифта корневого элемента (html):

```css
html {
  font-size: 16px;
}

.text {
  font-size: 1.25rem; /* 20px независимо от родительского font-size */
  margin: 2rem; /* 32px независимо от родительского font-size */
}
```

### `ex` и `ch` - единицы на основе шрифта
- `ex` - высота строчной буквы 'x' в текущем шрифте
- `ch` - ширина символа '0' в текущем шрифте

```css
.text {
  margin-top: 1ex; /* Высота строчной 'x' */
  padding-left: 2ch; /* Ширина двух символов '0' */
}
```

### `lh` - единица на основе высоты строки
Единица `lh` основана на высоте строки текущего элемента:

```css
.text {
  line-height: 1.5;
  margin-bottom: 1lh; /* Высота строки текущего элемента */
}
```

## Свойства для управления текстом

### `line-height` - высота строки
Определяет высоту блока строки текста:

```css
.text {
  line-height: 1.6; /* Рекомендуемое значение для основного текста */
}
```

### `letter-spacing` - интервал между буквами
Управляет пространством между буквами:

```css
.headline {
  letter-spacing: 0.1em; /* Расширение между буквами */
}

.caps {
  letter-spacing: 0.2em; /* Для заглавных букв */
}
```

### `word-spacing` - интервал между словами
Управляет пространством между словами:

```css
.quote {
  word-spacing: 0.1em; /* Небольшое расширение между словами */
}
```

## Вертикальный ритм и сетка

Вертикальный ритм - это согласование всех элементов на странице с базовой сеткой, что создает визуальную гармонию:

```css
/* Установка базовой сетки */
:root {
  --base-font-size: 1rem;
  --base-line-height: 1.5;
  --baseline-grid: calc(var(--base-font-size) * var(--base-line-height));
}

html {
  font-size: var(--base-font-size);
  line-height: var(--base-line-height);
}

/* Выравнивание элементов по сетке */
.text, .heading, .image {
  margin-bottom: var(--baseline-grid);
}
```

## Адаптивная типографика (fluid typography)

Адаптивная типографика позволяет тексту плавно изменяться в зависимости от размера экрана:

```css
/* Использование clamp() для адаптивного размера шрифта */
.responsive-text {
  font-size: clamp(1rem, 2.5vw, 2rem);
  /* минимальный размер, желаемый размер, максимальный размер */
}

/* Более сложный пример с расчетами */
.adaptive-heading {
  font-size: clamp(1.5rem, 4vw, 3rem);
  line-height: 1.2;
}
```

## Переменные шрифтов и их использование

Переменные шрифты позволяют использовать диапазоны толщины, ширины и наклона в одном файле шрифта:

```css
.variable-font {
  font-family: 'VariableFont', sans-serif;
  font-variation-settings: 'wght' 400, 'wdth' 100;
}

.font-controls {
  font-weight: 600;
  font-stretch: 125%;
}
```

## Современные свойства

### `font-synthesis`
Управляет созданием жирного и курсивного начертания:

```css
.text {
  font-synthesis: none; /* Запрет генерации жирного/курсива браузером */
}
```

### `font-variant`
Управляет альтернативными глифами шрифта:

```css
.numbers {
  font-variant-numeric: tabular-nums; /* Фиксированная ширина чисел */
}

.caps {
  font-variant-caps: small-caps; /* Малые заглавные */
}
```

## Практические примеры типографских решений

### Основной текстовый блок
```css
.article-text {
  font-size: 1.125rem; /* 18px при 16px базе */
  line-height: 1.6;
  letter-spacing: -0.003em;
  max-width: 65ch; /* Оптимальная длина строки */
  margin: 0 auto;
  padding: 0 1rem;
}
```

### Адаптивный заголовок
```css
.main-heading {
  font-size: clamp(1.75rem, 5vw, 3rem);
  font-weight: 700;
  line-height: 1.2;
  letter-spacing: -0.02em;
  margin-bottom: 1rem;
}
```

### Типографская шкала
```css
:root {
  --font-size-xs: 0.75rem;
  --font-size-sm: 0.875rem;
  --font-size-base: 1rem;
  --font-size-lg: 1.125rem;
  --font-size-xl: 1.25rem;
  --font-size-2xl: 1.5rem;
  --font-size-3xl: 1.875rem;
  --font-size-4xl: 2.25rem;
  --font-size-5xl: 3rem;
  --font-size-6xl: 3.75rem;
}

/* Использование шкалы */
.body-text { font-size: var(--font-size-base); }
.subheading { font-size: var(--font-size-xl); }
.main-title { font-size: var(--font-size-4xl); }
```

## Доступность текста

### Контрастность
Текст должен иметь достаточный контраст с фоном:

```css
.high-contrast-text {
  color: #000;
  background-color: #fff;
  /* Соотношение контрастности не менее 4.5:1 для обычного текста */
}
```

### Адаптация под пользовательские настройки
```css
.accessible-text {
  font-size: 1rem; /* Позволяет масштабирование текста */
  line-height: 1.5;
}
```

## Лучшие практики CSS-типографики

1. **Используйте относительные единицы** - `rem` и `em` для масштабируемости
2. **Ограничьте длину строки** - используйте `max-width: 65ch` для основного текста
3. **Поддерживайте вертикальный ритм** - используйте кратные значения для отступов
4. **Проверяйте доступность** - обеспечивайте достаточный контраст и читаемость
5. **Используйте адаптивную типографику** - для лучшего пользовательского опыта на всех устройствах
6. **Оптимизируйте загрузку шрифтов** - используйте `font-display: swap` для улучшения производительности
