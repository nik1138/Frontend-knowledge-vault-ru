# Выравнивание в Flexbox

## Общее описание

Выравнивание в Flexbox - это комплекс свойств, позволяющих точно контролировать расположение flex-элементов внутри контейнера. Flexbox предоставляет мощные инструменты для выравнивания как по главной оси (в направлении `flex-direction`), так и по поперечной оси (перпендикулярно главной оси).

## Оси в Flexbox

Прежде чем рассматривать выравнивание, важно понимать две оси в flex-контейнере:

1. **Главная ось (main axis)** - направление, определяемое `flex-direction`
2. **Поперечная ось (cross axis)** - перпендикулярна главной оси

При `flex-direction: row`:
- Главная ось - горизонтально (слева направо)
- Поперечная ось - вертикально (сверху вниз)

При `flex-direction: column`:
- Главная ось - вертикально (сверху вниз)
- Поперечная ось - горизонтально (слева направо)

## Свойства выравнивания

### justify-content

Выравнивание элементов по **главной оси**. Работает с flex-контейнером.

```css
.container {
  display: flex;
  justify-content: center; /* выравнивание по центру главной оси */
}
```

#### Значения justify-content:

- `flex-start` - элементы прижаты к началу главной оси
- `flex-end` - элементы прижаты к концу главной оси
- `center` - элементы выравниваются по центру главной оси
- `space-between` - элементы равномерно распределены по главной оси; первый прижат к началу, последний к концу
- `space-around` - элементы равномерно распределены с равными отступами вокруг
- `space-evenly` - элементы равномерно распределены с равными отступами между ними

### align-items

Выравнивание всех flex-элементов по **поперечной оси**. Работает с flex-контейнером.

```css
.container {
  display: flex;
  align-items: center; /* выравнивание по центру поперечной оси */
}
```

#### Значения align-items:

- `stretch` - элементы растягиваются по поперечной оси (по умолчанию)
- `flex-start` - элементы прижаты к началу поперечной оси
- `flex-end` - элементы прижаты к концу поперечной оси
- `center` - элементы выравниваются по центру поперечной оси
- `baseline` - элементы выравниваются по базовой линии текста

### align-content

Выравнивание строк при **множественных строках** flex-элементов. Работает только при наличии нескольких строк (когда `flex-wrap: wrap`).

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: center; /* выравнивание строк по поперечной оси */
}
```

#### Значения align-content:

- `flex-start` - строки прижаты к началу поперечной оси
- `flex-end` - строки прижаты к концу поперечной оси
- `center` - строки выравниваются по центру поперечной оси
- `space-between` - строки равномерно распределены по поперечной оси
- `space-around` - строки равномерно распределены с отступами
- `stretch` - строки растягиваются равномерно (по умолчанию)

### align-self

Выравнивание **конкретного элемента** по поперечной оси, переопределяя значение `align-items` контейнера.

```css
.item {
  align-self: flex-end; /* переопределяет выравнивание для этого элемента */
}
```

#### Значения align-self:

- `auto` - элемент наследует значение от родительского контейнера
- `flex-start` - элемент прижат к началу поперечной оси
- `flex-end` - элемент прижат к концу поперечной оси
- `center` - элемент выравнивается по центру
- `baseline` - элемент выравнивается по базовой линии
- `stretch` - элемент растягивается по поперечной оси

## Практические рекомендации

1. Используйте `justify-content: center` и `align-items: center` для простого центрирования элементов в контейнере
2. В российских реалиях 2025 года, при создании интерфейсов для государственных сервисов, часто требуется строгое выравнивание элементов - `space-between` идеально подходит для навигационных панелей
3. Свойство `align-self` особенно полезно для выделения одного элемента из группы, например, кнопки действия
4. При работе с многострочными flex-контейнерами используйте `align-content` для контроля расположения строк
5. В мобильных интерфейсах, популярных в России, часто используется `flex-direction: column` с `align-items: center` для вертикальных списков
6. Для создания адаптивных карточек часто комбинируют `justify-content: space-around` с `align-items: stretch`

## Примеры использования

### Центрирование элемента
```css
.center-container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
```

### Навигационное меню с равномерным распределением
```css
.nav-menu {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
}
```

### Карточки с разным выравниванием
```css
.card-container {
  display: flex;
  align-items: stretch;
}

.special-card {
  align-self: center; /* эта карточка будет по центру, остальные растянуты */
}
```

### Многострочный контейнер с выравниванием строк
```css
.multiline-container {
  display: flex;
  flex-wrap: wrap;
  align-content: space-around;
}
```

### Адаптивный список с выравниванием по базовой линии
```css
.text-list {
  display: flex;
  align-items: baseline;
}
```
