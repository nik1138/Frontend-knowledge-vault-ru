# Элементы Flex

## Общее описание

Flex-элементы - это непосредственные дочерние элементы flex-контейнера. После того как родительский элемент становится flex-контейнером (с помощью `display: flex`), все его прямые потомки автоматически становятся flex-элементами и подчиняются правилам flex-макета. Каждый flex-элемент может иметь свои собственные свойства, которые определяют его поведение внутри контейнера.

## Основные свойства flex-элементов

### flex-grow

Определяет, насколько элемент может увеличиваться относительно других flex-элементов в контейнере. Значение по умолчанию - 0 (не увеличивается).

```css
.item {
  flex-grow: 1; /* элемент займет доступное пространство */
}
```

- `0` - элемент не увеличивается (значение по умолчанию)
- Положительное число - определяет пропорциональное увеличение относительно других элементов

### flex-shrink

Определяет, насколько элемент может уменьшаться относительно других flex-элементов при отсутствии достаточного места. Значение по умолчанию - 1.

```css
.item {
  flex-shrink: 1; /* элемент может уменьшаться */
}
```

- `0` - элемент не уменьшается
- `1` - элемент уменьшается пропорционально (значение по умолчанию)
- Положительное число - определяет степень уменьшения

### flex-basis

Определяет начальный размер элемента до распределения свободного пространства. Может быть задан в единицах длины (px, %, em и т.д.) или ключевыми словами.

```css
.item {
  flex-basis: 100px; /* начальный размер 100px */
}
```

- `auto` - размер определяется содержимым элемента (значение по умолчанию)
- `0` - элемент не учитывает содержимое при определении размера
- Значение длины - фиксированный начальный размер

### flex

Сокращенное свойство для `flex-grow`, `flex-shrink` и `flex-basis`. Значение по умолчанию - `0 1 auto`.

```css
.item {
  flex: 1; /* эквивалентно flex: 1 1 0% */
}
```

- `flex: 1` - элемент будет равномерно распределять доступное пространство
- `flex: 0 0 auto` - элемент сохраняет свой естественный размер и не растягивается
- `flex: 2 1 200px` - элемент имеет двойной приоритет роста, обычный приоритет сжатия и начальный размер 200px

### order

Определяет порядок, в котором элемент будет расположен внутри flex-контейнера, независимо от его положения в HTML. Значение по умолчанию - 0.

```css
.item {
  order: 1; /* элемент будет расположен после элементов с order 0 */
}
```

- Отрицательные значения - элемент будет расположен раньше
- Положительные значения - элемент будет расположен позже
- Элементы с одинаковым значением order сохраняют исходный порядок

### align-self

Позволяет переопределить выравнивание элемента, заданное свойством `align-items` контейнера. Значение по умолчанию - `auto`.

```css
.item {
  align-self: center; /* переопределяет выравнивание для этого элемента */
}
```

- `auto` - элемент наследует значение от родительского контейнера (значение по умолчанию)
- `flex-start` - элемент прижат к началу поперечной оси
- `flex-end` - элемент прижат к концу поперечной оси
- `center` - элемент выравнивается по центру
- `baseline` - элемент выравнивается по базовой линии
- `stretch` - элемент растягивается по поперечной оси

## Практические рекомендации

1. Используйте `flex: 1` для создания элементов, которые равномерно распределяют доступное пространство
2. В российских реалиях 2025 года, при создании интерфейсов для банковских приложений или государственных сервисов, учитывайте доступность - свойство `order` может помочь изменить визуальный порядок без изменения HTML
3. Свойство `align-self` особенно полезно для выравнивания отдельных элементов в группе, например, выделения важного элемента в ряду
4. Используйте `flex-basis: 0` вместе с `flex-grow: 1` для равномерного распределения ширины между элементами
5. При работе с текстовыми элементами в flex-контейнере учитывайте переполнение и свойства `overflow`
6. В российском сегменте важно учитывать особенности отображения на мобильных устройствах - используйте `flex-shrink: 0` для элементов, которые не должны уменьшаться (например, иконки)

## Примеры использования

### Равномерное распределение пространства
```css
.flex-container {
  display: flex;
}

.flex-item {
  flex: 1; /* все элементы займут равное пространство */
}
```

### Один элемент занимает больше места
```css
.main-content {
  flex: 2; /* занимает в 2 раза больше места */
}

.sidebar {
  flex: 1; /* занимает меньше места */
}
```

### Элемент с фиксированным размером
```css
.fixed-width {
  flex: 0 0 200px; /* не растет, не сжимается, фиксированная ширина 200px */
}
```

### Изменение порядка элементов
```css
.first {
  order: 2; /* будет отображен последним */
}

.second {
  order: 1; /* будет отображен вторым */
}

.third {
  order: 0; /* будет отображен первым (значение по умолчанию) */
}
```

### Индивидуальное выравнивание
```css
.container {
  display: flex;
  align-items: flex-start;
}

.special-item {
  align-self: center; /* этот элемент будет по центру, остальные прижаты к началу */
}
```
