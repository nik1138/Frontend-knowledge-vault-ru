# Основы Flexbox

## Общее описание

Flexbox (Flexible Box Layout) - это модуль CSS, предназначенный для создания гибких и адаптивных макетов. Он позволяет легко управлять выравниванием, направлением и порядком элементов внутри контейнера, даже когда размеры этих элементов неизвестны или могут изменяться.

## Основные концепции

Flexbox работает с двумя типами элементов:

1. **Flex-контейнер** - родительский элемент, к которому применяется `display: flex`
2. **Flex-элементы** - дочерние элементы, которые подчиняются правилам flex-контейнера

## Основные свойства

### Свойства flex-контейнера

- `display: flex` - превращает элемент в flex-контейнер
- `flex-direction` - определяет направление главной оси (row, column, row-reverse, column-reverse)
- `justify-content` - выравнивание элементов по главной оси
- `align-items` - выравнивание элементов по поперечной оси
- `flex-wrap` - управление переносом строк
- `align-content` - выравнивание строк при нескольких строках

### Свойства flex-элементов

- `flex-grow` - определяет, насколько элемент может увеличиваться
- `flex-shrink` - определяет, насколько элемент может уменьшаться
- `flex-basis` - начальный размер элемента
- `order` - порядок элемента
- `align-self` - выравнивание конкретного элемента

## Значения свойств

### flex-direction

- `row` - элементы располагаются по горизонтали (по умолчанию)
- `column` - элементы располагаются по вертикали
- `row-reverse` - элементы располагаются по горизонтали в обратном порядке
- `column-reverse` - элементы располагаются по вертикали в обратном порядке

### justify-content

- `flex-start` - элементы выравниваются по началу главной оси
- `flex-end` - элементы выравниваются по концу главной оси
- `center` - элементы выравниваются по центру главной оси
- `space-between` - элементы распределяются равномерно, первый и последний прижаты к краям
- `space-around` - элементы распределяются с равными отступами вокруг
- `space-evenly` - элементы распределяются с равными отступами между ними

### align-items

- `stretch` - элементы растягиваются по поперечной оси (по умолчанию)
- `flex-start` - элементы выравниваются по началу поперечной оси
- `flex-end` - элементы выравниваются по концу поперечной оси
- `center` - элементы выравниваются по центру поперечной оси
- `baseline` - элементы выравниваются по базовой линии текста

### flex-wrap

- `nowrap` - элементы не переносятся (по умолчанию)
- `wrap` - элементы переносятся при необходимости
- `wrap-reverse` - элементы переносятся в обратном порядке

## Практические рекомендации

1. Используйте Flexbox для создания адаптивных макетов, особенно для компонентов интерфейса
2. Не злоупотребляйте Flexbox для всего макета страницы - он лучше подходит для локальных компоновок
3. Помните, что Flexbox работает в одном направлении (одна ось), для более сложных макетов используйте CSS Grid
4. В российских реалиях 2025 года поддержка Flexbox отличная во всех современных браузерах, включая Яндекс.Браузер и Opera
5. При работе с текстовыми элементами учитывайте особенности выравнивания и переноса строк
6. Используйте Flexbox для создания адаптивных навигационных меню, карточек товаров и других интерфейсных элементов

## Примеры использования

```css
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}
```
