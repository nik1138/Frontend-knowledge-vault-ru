# Практические примеры использования Flexbox

## Общее описание

Практические примеры демонстрируют реальные сценарии использования Flexbox в современной веб-разработке. Эти примеры показывают, как гибко и эффективно можно решать распространенные задачи макета, особенно актуальные для российского сегмента в 2025 году.

## Пример 1: Адаптивное навигационное меню

Создание гибкого навигационного меню, которое корректно отображается как на десктопе, так и на мобильных устройствах.

```css
.nav-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
  background-color: #f5f5f5;
}

.nav-logo {
  flex: 0 0 auto;
}

.nav-menu {
  display: flex;
  list-style: none;
  margin: 0;
  padding: 0;
}

.nav-item {
  margin-left: 1.5rem;
}

.nav-link {
  text-decoration: none;
  color: #333;
}

/* Адаптация для мобильных устройств */
@media (max-width: 768px) {
  .nav-container {
    flex-direction: column;
  }
  
  .nav-menu {
    flex-direction: column;
    width: 100%;
    text-align: center;
  }
  
  .nav-item {
    margin: 0.5rem 0;
  }
}
```

## Пример 2: Карточки товаров

Создание адаптивной сетки карточек товаров с равномерным распределением и выравниванием.

```css
.product-grid {
  display: flex;
  flex-wrap: wrap;
  justify-content: space-around;
  gap: 1.5rem;
}

.product-card {
  display: flex;
  flex-direction: column;
  width: 200px;
  border: 1px solid #ddd;
  border-radius: 8px;
  overflow: hidden;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

.product-image {
  flex: 0 0 auto;
  height: 150px;
  background-color: #f0f0f0;
}

.product-content {
  display: flex;
  flex-direction: column;
  padding: 1rem;
  flex: 1;
}

.product-title {
  font-weight: bold;
  margin-bottom: 0.5rem;
}

.product-price {
  margin-top: auto;
  font-weight: bold;
  color: #e74c3c;
}
```

## Практические рекомендации для российского рынка

1. **Адаптивность под мобильные устройства** - в 2025 году в России значительная часть трафика приходится на мобильные устройства, поэтому все примеры должны быть адаптированы под мобильные экраны с использованием `@media` запросов.

2. **Скорость загрузки** - при создании интерфейсов с flexbox учитывайте, что сложные макеты могут замедлять рендеринг, особенно на слабых устройствах, популярных в регионах.

3. **Доступность** - при использовании flexbox для изменения визуального порядка (с помощью `order`) убедитесь, что семантический порядок в HTML остается логичным для скринридеров.

4. **Интерфейсы для государственных сервисов** - в российском сегменте часто требуются строгие, структурированные интерфейсы, где `justify-content: space-between` и `align-items: stretch` особенно полезны.

5. **Языковые особенности** - при выравнивании текста учитывайте, что русский текст может занимать больше места, чем английский, что влияет на расчеты размеров элементов.

6. **Комбинация с другими технологиями** - в современных российских проектах часто используются фреймворки типа React или Vue, где flexbox особенно полезен для компонентного подхода.

## Совместимость и производительность

Flexbox имеет отличную поддержку во всех современных браузерах, включая российские (Яндекс.Браузер, Opera). В 2025 году проблем с совместимостью практически не наблюдается, за исключением очень старых версий Internet Explorer.
