# PostCSS

Этот файл описывает инструмент PostCSS.

PostCSS — это инструмент для преобразования CSS с помощью JavaScript-плагинов. В отличие от традиционных препроцессоров, PostCSS работает с обычным CSS и может имитировать функции препроцессоров с помощью плагинов. Это позволяет создавать мощные инструменты для обработки CSS.

## Основные возможности:

- Плагины: Обширная экосистема плагинов для различных задач
- Автопрефиксы: Добавление вендорных префиксов
- Транспиляция: Преобразование новых возможностей CSS в совместимые версии
- Инлайн SVG: Встраивание SVG-изображений в CSS
- Оптимизация: Минификация и оптимизация CSS

## Популярные плагины:

- Autoprefixer: Автоматическое добавление вендорных префиксов
- CSS Modules: Локальное применение стилей к компонентам
- PostCSS Preset Env: Использование будущих возможностей CSS сегодня

## Пример:

```css
/* Входной CSS */
.example {
  display: grid;
  color: hwb(75, 60%, 40%);
}
```

```css
/* Выходной CSS после обработки плагинами */
.example {
  display: -ms-grid;
  display: grid;
  color: #99CD4E;
}
```
