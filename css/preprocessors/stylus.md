# Stylus

Этот файл описывает CSS-препроцессор Stylus.

Stylus — это гибкий CSS-препроцессор с возможностью писать код с минимальным синтаксисом. Он предоставляет богатый набор возможностей для написания CSS с более чистым и кратким синтаксисом по сравнению с другими препроцессорами.

## Основные возможности:

- Гибкий синтаксис: Возможность писать код без скобок, точек с запятой и даже без двоеточий
- Переменные: Для хранения значений
- Вложенность: Возможность вкладывать CSS-правила
- Миксины: Переиспользуемые блоки кода
- Операции: Математические и логические операции
- Функции: Пользовательские и встроенные функции
- Условия и циклы: Для динамической генерации CSS

## Пример:

```stylus
primary-color = #007bff

.button
  background-color primary-color
  padding 10px 20px
  
  &.large
    padding 15px 30px
  
  &:hover
    background-color darken(primary-color, 10%)
```
