# Адаптивные-переменные

Адаптивные переменные — это CSS-переменные, значения которых изменяются в зависимости от условий окружающей среды, таких как размер экрана, ориентация устройства, разрешение или другие факторы. В 2025 году адаптивные переменные стали важным инструментом для создания гибких и адаптивных веб-интерфейсов, которые корректно отображаются на различных устройствах и при разных условиях использования.

## Введение в адаптивные переменные

Адаптивные переменные позволяют создавать более гибкие и масштабируемые стили, которые автоматически адаптируются к различным условиям отображения. Это особенно важно в эпоху множества устройств с разными характеристиками экрана, где статичные значения могут не обеспечивать оптимальный пользовательский опыт.

## Подходы к созданию адаптивных переменных

### 1. Использование медиа-запросов

Наиболее распространенный способ создания адаптивных переменных — использование медиа-запросов для изменения значений переменных в зависимости от размера экрана или других характеристик устройства.

```css
:root {
  /* Значения по умолчанию (мобильные устройства) */
  --font-size-base: 14px;
  --spacing-unit: 8px;
  --container-width: 100%;
  --border-radius: 4px;
  --header-height: 50px;
}

/* Планшеты */
@media (min-width: 768px) {
  :root {
    --font-size-base: 16px;
    --spacing-unit: 12px;
    --container-width: 750px;
    --border-radius: 6px;
    --header-height: 60px;
  }
}

/* Настольные компьютеры */
@media (min-width: 1024px) {
  :root {
    --font-size-base: 18px;
    --spacing-unit: 16px;
    --container-width: 1000px;
    --border-radius: 8px;
    --header-height: 70px;
  }
}

/* Большие экраны */
@media (min-width: 1400px) {
  :root {
    --font-size-base: 20px;
    --spacing-unit: 20px;
    --container-width: 1200px;
    --border-radius: 10px;
    --header-height: 80px;
  }
}
```

### 2. Использование функции `clamp()`

Функция `clamp()` позволяет создавать переменные с плавным изменением значений в зависимости от размера экрана, обеспечивая более плавный переход между размерами.

```css
:root {
  --font-size-responsive: clamp(1rem, 2.5vw, 1.5rem);
  --spacing-responsive: clamp(0.5rem, 4vw, 2rem);
  --container-width: clamp(300px, 90%, 1200px);
  --border-radius-responsive: clamp(4px, 1vw, 12px);
}
```

### 3. Использование `env()` для системных переменных

Системные переменные, доступные через функцию `env()`, позволяют адаптироваться к системным настройкам устройства, таким как безопасные области (safe areas) на мобильных устройствах.

```css
.component {
  padding-top: env(safe-area-inset-top);
  padding-right: env(safe-area-inset-right);
  padding-bottom: env(safe-area-inset-bottom);
  padding-left: env(safe-area-inset-left);
}
```

## Практические примеры адаптивных переменных

### Пример 1: Адаптивные размеры шрифтов

```css
:root {
  /* Базовые размеры шрифтов */
  --font-size-xs: clamp(0.75rem, 2vw, 0.8rem);
  --font-size-sm: clamp(0.875rem, 2.5vw, 1rem);
  --font-size-base: clamp(1rem, 3vw, 1.125rem);
  --font-size-lg: clamp(1.125rem, 3.5vw, 1.25rem);
  --font-size-xl: clamp(1.25rem, 4vw, 1.5rem);
  --font-size-2xl: clamp(1.5rem, 5vw, 2rem);
  --font-size-3xl: clamp(1.875rem, 6vw, 3rem);
  --font-size-4xl: clamp(2.25rem, 7vw, 4rem);
  --font-size-5xl: clamp(3rem, 8vw, 5rem);
}

.text-xs { font-size: var(--font-size-xs); }
.text-sm { font-size: var(--font-size-sm); }
.text-base { font-size: var(--font-size-base); }
.text-lg { font-size: var(--font-size-lg); }
.text-xl { font-size: var(--font-size-xl); }
.text-2xl { font-size: var(--font-size-2xl); }
.text-3xl { font-size: var(--font-size-3xl); }
.text-4xl { font-size: var(--font-size-4xl); }
.text-5xl { font-size: var(--font-size-5xl); }
```

### Пример 2: Адаптивные отступы и размеры

```css
:root {
  /* Адаптивные отступы */
  --spacing-xs: clamp(0.25rem, 1vw, 0.5rem);
  --spacing-sm: clamp(0.5rem, 2vw, 0.75rem);
  --spacing-md: clamp(1rem, 3vw, 1.5rem);
  --spacing-lg: clamp(1.5rem, 4vw, 2rem);
  --spacing-xl: clamp(2rem, 5vw, 3rem);
  --spacing-2xl: clamp(3rem, 6vw, 4rem);
  --spacing-3xl: clamp(4rem, 8vw, 6rem);
  
  /* Адаптивные размеры компонентов */
  --button-height: clamp(36px, 8vw, 48px);
  --input-height: clamp(40px, 8vw, 48px);
  --card-padding: clamp(1rem, 4vw, 2rem);
}
```

### Пример 3: Адаптивные размеры контейнеров

```css
:root {
  --container-padding: clamp(1rem, 5vw, 2rem);
  --max-content-width: clamp(300px, 90%, 1200px);
  --sidebar-width: clamp(200px, 25vw, 300px);
  --navigation-height: clamp(50px, 8vh, 70px);
}

.container {
  max-width: var(--max-content-width);
  padding: var(--container-padding);
  margin: 0 auto;
}

.sidebar {
  width: var(--sidebar-width);
}

.navigation {
  height: var(--navigation-height);
}
```

## Расширенные возможности адаптивных переменных

### 1. Адаптация к ориентации устройства

```css
:root {
  --header-height: 50px;
  --sidebar-width: 250px;
}

/* Портретная ориентация */
@media (orientation: portrait) {
  :root {
    --header-height: 60px;
    --sidebar-width: 100%;
  }
}

/* Альбомная ориентация */
@media (orientation: landscape) {
  :root {
    --header-height: 50px;
    --sidebar-width: 250px;
  }
}
```

### 2. Адаптация к разрешению экрана

```css
:root {
  --image-quality: 1;
  --animation-speed: 1;
}

/* Высокое разрешение */
@media (min-resolution: 2dppx) {
  :root {
    --image-quality: 2;
  }
}

/* Ограничение анимаций (для пользователей с чувствительностью к движениям) */
@media (prefers-reduced-motion: reduce) {
  :root {
    --animation-speed: 0;
  }
}
```

### 3. Адаптация к предпочтениям пользователя

```css
:root {
  --color-scheme: light;
}

/* Если пользователь предпочитает темную тему */
@media (prefers-color-scheme: dark) {
  :root {
    --color-scheme: dark;
  }
}

/* Если пользователь предпочитает высокую контрастность */
@media (prefers-contrast: high) {
  :root {
    --color-scheme: high-contrast;
  }
}
```

## Реализация адаптивных переменных в компонентах

### Адаптивная сетка

```css
:root {
  --grid-gap: clamp(1rem, 3vw, 2rem);
  --grid-columns: 1;
}

@media (min-width: 768px) {
  :root {
    --grid-columns: 2;
  }
}

@media (min-width: 1024px) {
  :root {
    --grid-columns: 3;
  }
}

@media (min-width: 1400px) {
  :root {
    --grid-columns: 4;
  }
}

.grid {
  display: grid;
  grid-template-columns: repeat(var(--grid-columns), 1fr);
  gap: var(--grid-gap);
}
```

### Адаптивные карточки

```css
:root {
  --card-padding: clamp(1rem, 4vw, 2rem);
  --card-radius: clamp(8px, 2vw, 16px);
  --card-shadow: 0 clamp(2px, 1vw, 8px) clamp(4px, 2vw, 16px) rgba(0, 0, 0, 0.1);
}

.card {
  padding: var(--card-padding);
  border-radius: var(--card-radius);
  box-shadow: var(--card-shadow);
}
```

## Совместимость с браузерами

Функция `clamp()` поддерживается всеми современными браузерами (Chrome, Firefox, Safari, Edge) с 2020 года. CSS-переменные также поддерживаются всеми современными браузерами. Для поддержки старых браузеров (IE) можно использовать полифиллы или альтернативные решения.

## Практические советы

- Используйте `clamp()` для создания плавных переходов между размерами
- Группируйте адаптивные переменные по функциональности (шрифты, отступы, размеры и т.д.)
- Документируйте адаптивные переменные, особенно если они используются в разных частях проекта
- Тестируйте адаптивные переменные на различных устройствах и размерах экрана
- Используйте медиа-запросы для значительных изменений и `clamp()` для плавных переходов
- Учитывайте системные предпочтения пользователя при создании адаптивных переменных

## Примеры использования в реальных проектах

### Адаптивный типографский модуль

```css
:root {
  /* Типографский модуль - базовый размер, от которого вычисляются все шрифты */
  --typographic-ratio: 1.2;
  --font-size-base: clamp(1rem, 2.5vw, 1.125rem);
  --font-size-h1: calc(var(--font-size-base) * var(--typographic-ratio) * var(--typographic-ratio) * var(--typographic-ratio));
  --font-size-h2: calc(var(--font-size-base) * var(--typographic-ratio) * var(--typographic-ratio));
  --font-size-h3: calc(var(--font-size-base) * var(--typographic-ratio));
}
```

### Адаптивная навигация

```css
:root {
  --nav-height: clamp(50px, 8vh, 70px);
  --nav-item-padding: clamp(0.5rem, 2vw, 1rem);
  --nav-font-size: clamp(0.875rem, 2.5vw, 1rem);
}

.navigation {
  height: var(--nav-height);
}

.nav-item {
  padding: var(--nav-item-padding);
  font-size: var(--nav-font-size);
}
```

В 2025 году они особенно важны для обеспечения оптимального пользовательского опыта на множестве устройств с разными характеристиками экрана.
