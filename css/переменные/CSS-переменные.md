# CSS-переменные

CSS-переменные (также известные как CSS Custom Properties) — это мощный инструмент для хранения и повторного использования значений CSS в стилях. Они позволяют определять значения в одном месте и использовать их в других местах, что упрощает обслуживание и темизацию CSS.

## Основы CSS-переменных

CSS-переменные определяются с помощью префикса `--` и используются с функцией `var()`. Они могут быть определены на любом элементе и наследуются потомками.

### Определение переменной

```css
:root {
  --primary-color: #007bff;
  --secondary-color: #6c757d;
  --font-size-base: 16px;
}
```

### Использование переменной

```css
.button {
  background-color: var(--primary-color);
  font-size: var(--font-size-base);
}
```

## Преимущества использования CSS-переменных

- **Повторное использование**: Определите цвета, размеры и другие значения один раз и используйте их везде.
- **Легкая темизация**: Легко переключайтесь между темами, изменяя значения переменных.
- **Поддержка**: Упрощает внесение изменений в стили проекта.

## Практические примеры

### Пример 1: Цветовая палитра

```css
:root {
  --color-primary: #007bff;
  --color-secondary: #6c757d;
  --color-success: #28a745;
  --color-danger: #dc3545;
  --color-warning: #ffc107;
  --color-info: #17a2b8;
  --color-light: #f8f9fa;
  --color-dark: #343a40;
}
```

### Пример 2: Размеры шрифтов

```css
:root {
  --font-size-xs: 0.75rem;
  --font-size-sm: 0.875rem;
  --font-size-base: 1rem;
  --font-size-lg: 1.125rem;
  --font-size-xl: 1.25rem;
  --font-size-2xl: 1.5rem;
  --font-size-3xl: 1.875rem;
  --font-size-4xl: 2.25rem;
  --font-size-5xl: 3rem;
}
```

### Пример 3: Отступы и размеры

```css
:root {
  --spacing-xs: 0.25rem;
  --spacing-sm: 0.5rem;
  --spacing-md: 1rem;
  --spacing-lg: 1.5rem;
  --spacing-xl: 2rem;
  --spacing-2xl: 3rem;
  --spacing-3xl: 4rem;
}
```

## Область видимости переменных

CSS-переменные наследуются от родительского элемента к дочернему. Это означает, что переменную, определенную на элементе, можно использовать в стилях его потомков.

```css
.component {
  --component-color: red;
}

.component .sub-element {
  color: var(--component-color); /* Будет красным */
}
```

## Значения по умолчанию

Функция `var()` позволяет задавать значения по умолчанию, которые используются, если переменная не определена:

```css
.element {
  color: var(--main-color, blue); /* Если --main-color не определена, будет синим */
}
```

## Работа с цветами

CSS-переменные особенно полезны для работы с цветами. Можно комбинировать их с функциями `hsl()`, `rgb()` и другими для создания гибких цветовых схем:

```css
:root {
  --hue-primary: 210;
  --saturation-primary: 50%;
  --lightness-primary: 50%;
  --primary-color: hsl(var(--hue-primary), var(--saturation-primary), var(--lightness-primary));
}
```

## Совместимость с браузерами

CSS-переменные поддерживаются всеми современными браузерами (Chrome, Firefox, Safari, Edge). IE не поддерживает CSS-переменные, но для поддержки IE можно использовать полифиллы.

## Практические советы

- Используйте семантические имена переменных (например, `--color-primary` вместо `--blue`)
- Группируйте связанные переменные (цвета, размеры, отступы и т.д.)
- Документируйте переменные, особенно если они используются в разных частях проекта
- Используйте `:root` для глобальных переменных
- Рассмотрите возможность использования препроцессоров для дополнительной логики с переменными

## Заключение

CSS-переменные — это важный инструмент для современной разработки. Они позволяют создавать более гибкие и поддерживаемые стили, упрощают темизацию и улучшают читаемость кода. Использование CSS-переменных особенно актуально в 2025 году, когда требования к адаптивности и темизации веб-сайтов и приложений продолжают расти.
