# Responsive Design

Этот файл описывает адаптивный дизайн.

Responsive Design — это подход к веб-дизайну, при котором веб-страницы адаптируются под различные размеры экранов и устройства. Это обеспечивает оптимальное отображение контента на любом устройстве: от мобильных телефонов до настольных компьютеров.

## Основные принципы:

- Гибкие сетки
- Гибкие изображения
- Медиазапросы

## Техники:

- Mobile-first подход
- Desktop-first подход
- Использование относительных единиц измерения
- Адаптивные изображения
