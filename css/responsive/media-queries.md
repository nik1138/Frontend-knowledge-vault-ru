# Медиазапросы

Этот файл описывает медиазапросы в CSS.

Медиазапросы (media queries) — это CSS-технология, позволяющая применять разные стили в зависимости от характеристик устройства (ширина экрана, ориентация, разрешение и т.д.). Это ключевой инструмент для создания адаптивного дизайна.

## Структура медиазапроса:

Медиазапрос состоит из медиатипа и одного или нескольких медиавыражений, которые описывают условия отображения стилей.

## Примеры:

```css
/* Для экранов шириной до 768px */
@media (max-width: 768px) {
  .container {
    width: 100%;
  }
}

/* Для экранов от 768px до 1024px */
@media (min-width: 768px) and (max-width: 1024px) {
  .container {
    width: 80%;
  }
}

/* Для печати */
@media print {
  .no-print {
    display: none;
  }
}
```

## Распространенные точки останова:

- Мобильные устройства: до 768px
- Планшеты: 768px - 1024px
- Настольные компьютеры: свыше 1024px
