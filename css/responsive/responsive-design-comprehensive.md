# CSS Responsive Design

Этот файл описывает адаптивный дизайн с использованием CSS.

Адаптивный дизайн — это подход к веб-дизайну, при котором веб-страницы адаптируются под различные размеры экранов и устройства. CSS предоставляет мощные инструменты для создания адаптивных интерфейсов.

## Основные компоненты:

1. **Медиазапросы**: Позволяют применять разные стили в зависимости от характеристик устройства
2. **Гибкие сетки**: Использование относительных единиц измерения для создания гибких макетов
3. **Гибкие изображения**: Изображения, которые адаптируются к размеру контейнера

## Техники:

- Flexbox для одномерных макетов
- Grid для двумерных макетов
- Относительные единицы измерения (%, em, rem, vw, vh)
- Mobile-first или desktop-first подходы
