# CSS Media Queries

Этот файл описывает медиазапросы в CSS.

CSS Media Queries позволяют применять разные стили в зависимости от характеристик устройства (ширина экрана, ориентация, разрешение и т.д.). Это ключевой инструмент для создания адаптивного дизайна.

## Основные применения:

1. **Адаптивный дизайн**: Создание макетов, которые адаптируются к различным размерам экрана
2. **Условия отображения**: Применение стилей в зависимости от типа устройства (печатное издание, экран, речь)
3. **Характеристики устройства**: Адаптация к различным характеристикам устройства (разрешение, ориентация, цветность)

## Примеры:

```css
/* Для экранов шириной до 768px */
@media (max-width: 768px) {
  .container {
    width: 100%;
  }
}

/* Для печати */
@media print {
  .no-print {
    display: none;
  }
}
```
