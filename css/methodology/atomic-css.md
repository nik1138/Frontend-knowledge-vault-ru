# Atomic CSS

Этот файл описывает методологию Atomic CSS.

Atomic CSS — это методология CSS, при которой создаются небольшие, однозначные классы, каждый из которых отвечает за одну конкретную задачу. Вместо создания семантических классов, Atomic CSS использует утилитарные классы для стилизации элементов.

## Принципы Atomic CSS:

- Каждый класс отвечает за одно CSS-свойство
- Классы имеют предсказуемые имена
- Повторное использование классов по всему проекту
- Минимальная специфичность

## Пример:

```html
<div class="text-center padding-20 margin-bottom-10 bg-blue-500">
  Содержимое
</div>
```

## Преимущества:

- Быстрая разработка
- Маленький размер CSS-файлов
- Консистентность визуальных компонентов
- Легкость масштабирования

## Недостатки:

- Длинные HTML-классы
- Сложность переиспользования в разных проектах
- Не семантическая разметка
