# CSS Methodology

Этот файл описывает методологии CSS.

CSS Methodology — это систематизированный подход к написанию CSS-кода, который помогает создавать более структурированный, понятный и поддерживаемый код. Методологии предоставляют правила и рекомендации по организации CSS-кода, именованию классов и структурированию проекта.

## Популярные методологии:

- BEM (Block Element Modifier)
- OOCSS (Object-Oriented CSS)
- SMACSS (Scalable and Modular Architecture for CSS)
- Atomic CSS
- ITCSS (Inverted Triangle CSS)

## Преимущества использования методологий:

- Повышение читаемости кода
- Упрощение сопровождения
- Повышение согласованности
- Улучшение командной работы
- Упрощение масштабирования проекта
