# SMACSS

Этот файл описывает методологию SMACSS (Scalable and Modular Architecture for CSS).

SMACSS (Scalable and Modular Architecture for CSS) — это методология CSS, разработанная Джонатаном Снупером. Она предлагает пятиуровневую систему категоризации CSS-правил для создания более масштабируемой и поддерживаемой архитектуры стилей.

## Пять категорий SMACSS:

1. **Base**: Базовые стили для элементов (body, h1, a и т.д.)
2. **Layout**: Стили, отвечающие за структуру страницы (шапка, боковая панель, подвал)
3. **Module**: Повторяющиеся элементы интерфейса (кнопки, формы, навигация)
4. **State**: Правила, описывающие различные состояния элементов (скрытый, активный)
5. **Theme**: Правила, отвечающие за визуальное оформление (цвета, шрифты, изображения)

## Преимущества:

- Лучшая организация кода
- Повышенная масштабируемость
- Повышенная переиспользуемость
- Легкость сопровождения
