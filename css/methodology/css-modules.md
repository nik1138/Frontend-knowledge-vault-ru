# CSS Modules

Этот файл описывает CSS Modules.

CSS Modules — это подход к локальному применению стилей к компонентам. Он автоматически создает уникальные имена классов для CSS-правил, что предотвращает конфликты имен и обеспечивает инкапсуляцию стилей на уровне компонентов.

## Основные особенности:

- Локальная область видимости: Классы автоматически становятся локальными
- Глобальные классы: Возможность создания глобальных стилей при необходимости
- Совместимость: Работает с существующими CSS-файлами
- Интеграция: Легко интегрируется с webpack и другими сборщиками

## Пример:

```css
/* styles.module.css */
.button {
  padding: 10px 20px;
  background-color: #007bff;
}
```

```javascript
// Component.js
import styles from './styles.module.css';

function Button() {
  return <button className={styles.button}>Кнопка</button>;
}
```

## Преимущества:

- Изоляция стилей: Отсутствие конфликтов имен
- Предсказуемость: Легче отслеживать, какие стили применяются к каким компонентам
- Совместимость: Работает с существующими CSS-файлами
