# CSS Accessibility

Этот файл описывает доступность в CSS.

CSS Accessibility — это практика написания CSS-кода, который обеспечивает доступность веб-сайтов для людей с ограниченными возможностями. Доступный CSS помогает создавать интерфейсы, которые могут использоваться людьми с различными типами ограничений.

## Основные аспекты доступности:

1. **Контрастность**: Обеспечение достаточного контраста между текстом и фоном
2. **Фокусировка**: Обеспечение видимости элементов в фокусе
3. **Размер шрифта**: Поддержка масштабирования текста
4. **Визуальные альтернативы**: Обеспечение альтернатив для визуальных элементов

## Техники:

- Использование семантических CSS-селекторов
- Обеспечение видимости фокуса для клавиатурной навигации
- Избегание стилей, которые могут вызвать проблемы у людей с мигренями
- Поддержка пользовательских настроек доступности
