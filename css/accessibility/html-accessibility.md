# HTML Accessibility

Этот файл описывает доступность в HTML.

HTML Accessibility — это практика создания HTML-разметки, которая обеспечивает доступность веб-сайтов для людей с ограниченными возможностями. Доступная HTML-разметка позволяет вспомогательным технологиям (например, скринридерам) правильно интерпретировать и представлять содержимое пользователям.

## Основные принципы:

1. **Семантика**: Использование правильных HTML-элементов для их предполагаемого назначения
2. **Заголовки**: Правильная иерархия заголовков
3. **Альтернативный текст**: Предоставление описаний для изображений
4. **Формы**: Правильная разметка форм с метками и инструкциями

## Техники:

- Использование семантических элементов (header, nav, main, article, section и т.д.)
- Добавление атрибутов доступности (ARIA)
- Обеспечение клавиатурной навигации
- Обеспечение контрастности цветов
