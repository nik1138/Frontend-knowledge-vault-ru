# Темные темы

Темные темы (Dark Mode) - это специальные режимы оформления веб-страниц и приложений, в которых используются преимущественно темные цвета для фона и светлые для текста. В 2025 году темные темы стали неотъемлемой частью современных интерфейсов, обеспечивая комфортное взаимодействие в условиях низкой освещенности и снижая усталость глаз.

## Преимущества темных тем

Темные темы предлагают несколько важных преимуществ:

- **Снижение усталости глаз** - особенно в условиях низкой освещенности
- **Энергосбережение** - на OLED-экранах черный цвет потребляет меньше энергии
- **Улучшенная доступность** - некоторые пользователи предпочитают темные темы
- **Современный вид** - воспринимаются как более стильные и современные
- [[Переключение тем]] - позволяют пользователям выбирать наиболее удобный режим

## Реализация темных тем

### Использование media query prefers-color-scheme

Самый простой способ - автоматическое применение темной темы на основе системных настроек:

```css
/* Светлая тема по умолчанию */
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
  --surface-color: #f8f9fa;
  --border-color: #dee2e6;
  --primary-color: #007bff;
}

/* Темная тема */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-color: #1a1a1a;
    --text-color: #f0f0f0;
    --surface-color: #2d2d2d;
    --border-color: #444444;
    --primary-color: #4dabf7;
  }
}
```

### Альтернативный подход с data-атрибутом

Более гибкий подход с возможностью ручного переключения:

```css
/* Светлая тема по умолчанию */
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
  --surface-color: #f8f9fa;
  --border-color: #dee2e6;
  --primary-color: #007bff;
}

/* Темная тема при ручном переключении */
[data-theme="dark"] {
  --bg-color: #1a1a1a;
  --text-color: #f0f0f0;
  --surface-color: #2d2d2d;
  --border-color: #444444;
  --primary-color: #4dabf7;
}

/* Автоматическое применение темной темы при системной настройке */
@media (prefers-color-scheme: dark) {
  :root:not([data-theme="light"]) {
    --bg-color: #1a1a1a;
    --text-color: #f0f0f0;
    --surface-color: #2d2d2d;
    --border-color: #444444;
    --primary-color: #4dabf7;
  }
}
```

## Принципы создания эффективных темных тем

### 1. Поддержание контраста

Важно обеспечить достаточный контраст между текстом и фоном:

```css
[data-theme="dark"] {
  /* Правильный контраст для основного текста */
  --text-primary: #e6e6e6; /* Не белый, чтобы не был слишком ярким */
  --text-secondary: #a3a3a3; /* Менее важный текст */
  --text-hint: #737373; /* Подсказки и вспомогательный текст */
  
  /* Фоновые цвета */
  --bg-primary: #121212;
  --bg-secondary: #1e1e1e;
  --bg-tertiary: #2d2d2d;
}
```

### 2. Избегание чрезмерной яркости

Избегайте использования чистого белого цвета в темной теме:

```css
/* ПЛОХО - чрезмерная яркость */
--text-color: #ffffff;

/* ХОРОШО - более мягкий светлый цвет */
--text-color: #e6e6e6;

/* ЕЩЕ ЛУЧШЕ - использование различных оттенков для разных элементов */
--text-primary: #e6e6e6;
--text-secondary: #a3a3a3;
--text-hint: #7373a3;
```

### 3. Адаптация цветов акцента

Цвета акцента также должны быть адаптированы для темной темы:

```css
[data-theme="dark"] {
  --primary-color: #4dabf7; /* Более светлый синий для темной темы */
  --success-color: #69db7c; /* Более светлый зеленый */
  --warning-color: #ffd43b; /* Более светлый желтый */
  --error-color: #ff6b6b; /* Более светлый красный */
}
```

## Адаптация изображений и иконок

При использовании темных тем важно адаптировать изображения и иконки:

```css
[data-theme="dark"] {
  /* Инвертируем иконки при необходимости */
  .icon {
    filter: invert(1) hue-rotate(180deg);
  }
  
  /* Или используем отдельные версии изображений */
  .logo {
    content: url('/images/logo-dark.svg');
  }
  
  /* Для изображений с прозрачным фоном */
  .image-with-transparent-bg {
    background-color: var(--color-bg-secondary);
  }
}
```

## Анимации и переходы

При переключении тем полезно использовать плавные переходы:

```css
/* Плавные переходы между темами */
:root {
  --transition-duration: 0.3s;
}

html {
  transition: background-color var(--transition-duration) ease,
              color var(--transition-duration) ease;
}

/* Для отдельных элементов */
.card, .button, .input {
  transition: background-color var(--transition-duration) ease,
              border-color var(--transition-duration) ease,
              color var(--transition-duration) ease;
}
```

## Проверка доступности темных тем

При создании темных тем важно учитывать доступность:

1. **Проверьте контрастность** - используйте инструменты для проверки контраста
2. **Тестируйте с пользователями** - особенно с людьми с нарушениями зрения
3. **Учитывайте разные типы слепоты** - цветовая слепота может повлиять на восприятие
4. **Используйте дополнительные индикаторы** - не только цвет, но и текст или иконки

## Рекомендации по созданию темных тем

1. **Не используйте чистый черный фон** - #000000 может создавать ореолы, используйте темно-серые оттенки
2. **Снижайте яркость текста** - избегайте чистого белого текста, используйте светло-серые оттенки
3. **Адаптируйте все цвета** - не только текст и фон, но и акценты, границы и тени
4. **Предоставляйте опцию отключения** - не все пользователи любят темные темы
5. **Тестируйте на разных устройствах** - особенно на OLED-экранах

## Заключение

Темные темы - важная часть современного пользовательского опыта, особенно в условиях низкой освещенности. Правильная реализация темных тем улучшает доступность и снижает усталость глаз, но требует внимательного подхода к цветам и контрастности.
