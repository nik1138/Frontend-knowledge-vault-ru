# CSS Селекторы

Этот файл описывает различные типы CSS-селекторов.

CSS-селекторы — это шаблоны, которые определяют, к каким HTML-элементам применять CSS-правила. Они играют ключевую роль в каскадных стилях, позволяя точно указать, какие элементы страницы должны быть отформатированы определенным образом.

## Типы селекторов:

1. **Селекторы тегов**: Применяются ко всем элементам указанного типа.
2. **Селекторы классов**: Применяются к элементам с определённым классом.
3. **Селекторы ID**: Применяются к элементу с определённым ID.
4. **Селекторы атрибутов**: Применяются к элементам с определёнными атрибутами.
5. **Псевдоклассы и псевдоэлементы**: Применяются к элементам в определённых состояниях или частям элементов.
