# CSS-функции

CSS-функции — это специальные выражения, которые позволяют выполнять вычисления, манипуляции с цветами, работу с изображениями и другие операции прямо в CSS. Они играют ключевую роль в создании динамических и адаптивных стилей, позволяя разработчикам создавать более гибкие и поддерживаемые стили.

## Классификация CSS-функций

CSS-функции можно разделить на несколько категорий в зависимости от их назначения:

- **Математические функции** — выполняют вычисления (например, `calc()`, `min()`, `max()`)
- **Цветовые функции** — манипуляции с цветами (например, `rgb()`, `hsl()`, `color-mix()`)
- **Геометрические функции** — работа с размерами и позиционированием (например, `min()`, `max()`, `clamp()`)
- **Функции для работы с изображениями** — вставка и манипуляции с изображениями (например, `url()`, `image()`)
- **Функции трансформаций** — преобразования элементов (например, `rotate()`, `scale()`, `translate()`)
- **Функции фильтров** — применение визуальных эффектов (например, `blur()`, `brightness()`, `contrast()`)

## Основные CSS-функции

### calc()

Функция `calc()` позволяет выполнять математические вычисления в CSS, комбинируя различные единицы измерения.

```css
.element {
  width: calc(100% - 20px);
  height: calc(100vh - 80px);
  margin: calc(2rem + 10px);
}
```

Эта функция особенно полезна для создания адаптивных макетов, где нужно вычесть фиксированное значение из процентной ширины.

### min() и max()

Функции `min()` и `max()` позволяют выбирать минимальное или максимальное значение из списка.

```css
.responsive-text {
  font-size: min(4vw, 24px); /* Выберет меньшее из двух значений */
}

.container {
  width: max(300px, 50%); /* Выберет большее из двух значений */
}
```

### clamp()

Функция `clamp()` задает минимальное, предпочтительное и максимальное значение.

```css
.clamped-text {
  font-size: clamp(16px, 4vw, 32px); /* Минимум 16px, предпочтительно 4vw, максимум 32px */
}
```

### var()

Функция `var()` используется для работы с CSS-переменными [[CSS Variables]].

```css
:root {
  --main-color: #3498db;
  --padding: 1rem;
}

.element {
  color: var(--main-color);
  padding: var(--padding);
  border: 2px solid var(--main-color, #ccc); /* #ccc как fallback */
}
```

### attr()

Функция `attr()` позволяет получать значение атрибута HTML-элемента.

```css
/* Отображение значения атрибута data-label в псевдоэлементе */
.element::before {
  content: attr(data-label);
}

.width-based {
  width: attr(data-width px); /* px как тип единицы измерения */
}
```

### url()

Функция `url()` используется для вставки внешних ресурсов, таких как изображения.

```css
.background-image {
  background-image: url('/path/to/image.jpg');
}

.list-item {
  list-style-image: url('bullet.svg');
}
```

## Современные CSS-функции

### color-mix()

Функция `color-mix()` позволяет смешивать цвета с определенной пропорцией.

```css
.mixed-color {
  background-color: color-mix(in srgb, red 70%, blue 30%);
}

.semi-transparent {
  color: color-mix(in srgb, black 80%, transparent);
}
```

### light-dark()

Функция `light-dark()` позволяет задавать разные значения для светлой и темной темы.

```css
.element {
  background-color: light-dark(white, #333);
  color: light-dark(black, white);
}
```

## Вложенные функции и сложные выражения

CSS-функции можно вкладывать друг в друга для создания сложных вычислений:

```css
.complex-calc {
  width: calc(var(--base-size) * (1 + 0.1 * var(--multiplier)));
  padding: calc(min(5vw, 20px) - 2px);
  background-color: color-mix(in srgb, var(--primary-color) 70%, var(--secondary-color) 30%);
}
```

## Совместимость и браузерная поддержка

Поддержка различных CSS-функций различается в браузерах:

- `calc()` — поддерживается всеми современными браузерами
- `min()`, `max()`, `clamp()` — поддержка с 2019 года в современных браузерах
- `color-mix()` — новая функция, поддержка ограничена, требует префиксы или полифиллы
- `light-dark()` — новая функция, поддержка в Safari и некоторых других браузерах

Для проверки поддержки конкретных функций рекомендуется использовать [Can I Use](https://caniuse.com) или аналогичные ресурсы.

## Лучшие практики использования функций

1. **Используйте `calc()` для адаптивных размеров**, особенно при вычитании фиксированных значений из процентных
2. **Применяйте `clamp()` для типографики**, чтобы обеспечить читаемость на разных устройствах
3. **Комбинируйте функции с CSS-переменными** для более гибкой настройки
4. **Проверяйте браузерную поддержку** перед использованием новых функций
5. **Используйте fallback значения**, особенно при работе с `var()`

```css
/* Хорошая практика: комбинация переменных и calc() */
:root {
  --base-font-size: 1rem;
  --font-scale-ratio: 1.2;
}

.responsive-heading {
  font-size: calc(var(--base-font-size) * var(--font-scale-ratio));
}
```

## Заключение

CSS-функции являются мощным инструментом для создания динамических и адаптивных стилей. Они позволяют выполнять вычисления, манипуляции с цветами и другие операции непосредственно в CSS, что делает стили более гибкими и поддерживаемыми. Современные функции, такие как `color-mix()` и `light-dark()`, открывают новые возможности для создания адаптивных интерфейсов.
