# CSS Box Model

Этот файл описывает CSS-модель коробки (box model).

CSS Box Model — это фундаментальная концепция, определяющая, как элементы отображаются и измеряются в веб-документе. Каждый элемент на странице можно представить как прямоугольную коробку, которая состоит из контента, внутренних отступов (padding), границы (border) и внешних отступов (margin).

## Компоненты Box Model:

1. **Content**: Область, в которой отображается контент элемента (текст, изображения и т.д.)
2. **Padding**: Пространство между контентом и границей элемента
3. **Border**: Граница, окружающая padding и content
4. **Margin**: Пространство снаружи границы элемента
