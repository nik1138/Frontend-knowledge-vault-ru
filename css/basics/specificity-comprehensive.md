# CSS Специфичность

Этот файл описывает концепцию специфичности в CSS.

Специфичность в CSS — это механизм, который определяет, какие стили будут применены к элементу, когда существует несколько конфликтующих объявлений. Это ключевая концепция каскада CSS, которая помогает браузеру решить, какое CSS-правило должно быть применено к элементу, если несколько правил применяются к одному и тому же элементу.

## Как вычисляется специфичность:

CSS использует четырехмерную систему для вычисления специфичности:
1. Инлайны (Inline styles) — стили, примененные непосредственно к элементу
2. ID — количество ID-селекторов в селекторе
3. Классы, атрибуты и псевдоклассы — количество селекторов классов, атрибутов и псевдоклассов
4. Теги и псевдоэлементы — количество селекторов тегов и псевдоэлементов
