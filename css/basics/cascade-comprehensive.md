# CSS Каскад

Этот файл описывает концепцию каскада в CSS.

Каскад — это фундаментальный механизм CSS, определяющий, как браузер разрешает конфликты между различными CSS-правилами, когда они применяются к одному и тому же элементу. Понимание каскада критически важно для написания эффективного и предсказуемого CSS-кода.

## Основные принципы каскада:

1. **Источники CSS-стилей**: Стили могут происходить из различных источников (браузерные стили, пользовательские стили, авторские стили), каждый из которых имеет свой приоритет.
2. **Специфичность селекторов**: Более специфичные селекторы имеют больший приоритет.
3. **Порядок объявления**: При равной специфичности последнее объявление имеет приоритет.
