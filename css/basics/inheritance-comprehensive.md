# CSS Наследование

Этот файл описывает концепцию наследования в CSS.

Наследование — это механизм CSS, при котором дочерние элементы получают определённые CSS-свойства от своих родительских элементов. Это позволяет создавать согласованные стили по всему документу без необходимости явно задавать свойства для каждого элемента.

## Основные принципы наследования:

1. **Наследуемые свойства**: Некоторые свойства (например, `color`, `font-family`, `text-align`) автоматически наследуются дочерними элементами.
2. **Не наследуемые свойства**: Большинство свойств (например, `background`, `border`, `margin`, `padding`) не наследуются по умолчанию.
3. **Управление наследованием**: Ключевые слова `inherit`, `initial`, `unset` позволяют управлять наследованием.
