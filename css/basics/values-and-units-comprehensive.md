# CSS Значения и Единицы измерения

Этот файл описывает различные типы значений и единиц измерения, используемых в CSS.

CSS значения и единицы измерения определяют, как задавать размеры, расстояния, цвета и другие свойства элементов. Понимание различных типов значений и единиц измерения критически важно для эффективной работы с CSS.

## Основные категории:

1. **Числовые значения**: 
   - Абсолютные единицы (px, cm, mm, in, pt, pc)
   - Относительные единицы (em, rem, %, vw, vh, vmin, vmax)

2. **Цветовые значения**: 
   - Ключевые слова (red, blue, black и т.д.)
   - Шестнадцатеричные значения (#ff0000)
   - RGB/RGBA значения
   - HSL/HSLA значения

3. **Функциональные значения**:
   - calc()
   - var() для CSS-переменных
   - transform-функции
