# CSS единицы измерения

Этот файл описывает единицы измерения в CSS.

CSS предоставляет различные единицы измерения для определения размеров, расстояний и других числовых значений. Понимание различных единиц измерения критически важно для создания гибких и адаптивных макетов.

## Абсолютные единицы:

- px (пиксели): Фиксированный размер, не зависящий от других элементов
- cm, mm, in, pt, pc: Традиционные единицы измерения

## Относительные единицы:

- em: Относительно размера шрифта родительского элемента
- rem: Относительно размера шрифта корневого элемента
- %: Относительно размера родительского элемента
- vw, vh: Относительно размера вьюпорта (1% от ширины/высоты окна)
- vmin, vmax: Относительно меньшего/большего измерения вьюпорта
