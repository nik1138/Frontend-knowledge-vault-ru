# Кастомные свойства CSS (CSS Variables)

## Определение

Кастомные свойства CSS (также известные как CSS переменные) - это сущности, определяемые разработчиком, которые содержат определенные значения, используемые повторно в CSS-коде. Они позволяют управлять значениями в одном месте и использовать их в различных частях стилей.

## Синтаксис

Кастомные свойства определяются с префиксом `--`:

```css
--main-color: #3498db;
--main-padding: 1rem;
--font-size: 1.2em;
```

Для использования кастомных свойств используется функция `var()`:

```css
.header {
  color: var(--main-color);
  padding: var(--main-padding);
  font-size: var(--font-size);
}
```

## Область видимости

### Глобальные переменные
Определяются на уровне `:root` и доступны везде:

```css
:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --font-size-base: 16px;
}
```

### Локальные переменные
Могут быть определены в любом селекторе и применяются только к соответствующему элементу и его потомкам:

```css
.card {
  --card-padding: 1.5rem;
  --card-bg: #f8f9fa;
  
  padding: var(--card-padding);
  background-color: var(--card-bg);
}
```

## Практическое применение

### Цветовые схемы

```css
:root {
  /* Основные цвета */
  --color-primary: #3498db;
  --color-secondary: #2ecc71;
  --color-accent: #e74c3c;
  --color-text: #2c3e50;
  --color-background: #ffffff;
  
  /* Цвета состояний */
  --color-success: #27ae60;
  --color-warning: #f39c12;
  --color-error: #e74c3c;
}

/* Использование в компонентах */
.button {
  background-color: var(--color-primary);
  color: var(--color-background);
  border: 1px solid var(--color-primary);
}

.button:hover {
  background-color: var(--color-secondary);
}
```

### Размеры и отступы

```css
:root {
  /* Размеры шрифта */
  --font-size-xs: 0.75rem;
  --font-size-sm: 0.875rem;
  --font-size-base: 1rem;
  --font-size-lg: 1.125rem;
  --font-size-xl: 1.25rem;
  --font-size-2xl: 1.5rem;
  --font-size-3xl: 1.875rem;
  
  /* Отступы */
  --spacing-xs: 0.25rem;
  --spacing-sm: 0.5rem;
  --spacing-md: 1rem;
  --spacing-lg: 1.5rem;
  --spacing-xl: 2rem;
  --spacing-2xl: 3rem;
  
  /* Размеры элементов */
  --border-radius: 0.375rem;
  --border-width: 1px;
}

/* Использование в компонентах */
.text-small {
  font-size: var(--font-size-sm);
}

.card {
  padding: var(--spacing-lg);
  border-radius: var(--border-radius);
  border: var(--border-width) solid #e2e8f0;
}
```

### Адаптивные значения

```css
:root {
  --container-max-width: 1200px;
  --sidebar-width: 250px;
  --header-height: 60px;
}

/* Адаптивные переменные */
@media (max-width: 768px) {
  :root {
    --sidebar-width: 0px;
    --header-height: 50px;
  }
}

.layout {
  display: grid;
  grid-template-columns: var(--sidebar-width) 1fr;
  min-height: calc(100vh - var(--header-height));
}
```

## Совмещение с единицами измерения

Кастомные свойства могут содержать значения с единицами измерения:

```css
:root {
  --base-font-size: 16px;
  --line-height-ratio: 1.5;
  --container-width: 80vw;
  --card-height: 30vh;
  --transition-duration: 0.3s;
  --box-shadow-blur: 10px;
}

.typography {
  font-size: var(--base-font-size);
  line-height: var(--line-height-ratio);
}

.card {
  width: var(--container-width);
  height: var(--card-height);
  box-shadow: 0 0 var(--box-shadow-blur) rgba(0,0,0,0.1);
}

.transition {
  transition: all var(--transition-duration) ease;
}
```

## Использование с calc()

Кастомные свойства можно использовать внутри функции `calc()`:

```css
:root {
  --base-font-size: 16px;
  --font-scale-ratio: 1.2;
}

.responsive-text {
  font-size: calc(var(--base-font-size) * var(--font-scale-ratio));
}

.calculated-width {
  width: calc(100% - var(--sidebar-width));
}

.offset-position {
  left: calc(var(--spacing-lg) * 2);
}
```

## Темизация интерфейсов

### Светлая и темная темы

```css
:root {
  --color-bg: #ffffff;
  --color-text: #000000;
  --color-border: #e2e8f0;
}

[data-theme="dark"] {
  --color-bg: #1a202c;
  --color-text: #ffffff;
  --color-border: #4a5568;
}

.component {
  background-color: var(--color-bg);
  color: var(--color-text);
  border: 1px solid var(--color-border);
}
```

### Темы с помощью JavaScript

```css
:root {
  --primary-hue: 200;
  --saturation: 80%;
  --lightness: 50%;
}

.dynamic-color {
  background-color: hsl(
    var(--primary-hue), 
    var(--saturation), 
    var(--lightness)
  );
}
```

```javascript
// Изменение темы через JavaScript
document.documentElement.style.setProperty('--primary-hue', 250);
```

## Современные возможности (2025 год)

### CSS-хуки (CSS Hooks)
Новые возможности для интеграции с JavaScript:

```css
.element {
  width: var(--dynamic-width, 100px);
  height: var(--dynamic-height, 100px);
}
```

```javascript
// Динамическое изменение значений
const element = document.querySelector('.element');
element.style.setProperty('--dynamic-width', '200px');
```

### Поддержка в современных браузерах
К 2025 году кастомные свойства полностью поддерживаются во всех современных браузерах, что делает их стандартом для создания дизайн-систем.

## Рекомендации по использованию в 2025 году

- Широкое использование кастомных свойств для создания дизайн-систем
- Интеграция с современными фреймворками и библиотеками
- Создание темизируемых интерфейсов
- Использование в сочетании с CSS Grid и Flexbox
- В российской практике кастомные свойства становятся стандартом для крупных проектов

## Преимущества и недостатки

### Преимущества
- Централизованное управление значениями
- Легкость темизации интерфейсов
- Повышение поддерживаемости кода
- Возможность динамического изменения значений через JavaScript
- Совместимость с современными подходами к разработке

### Недостатки
- Отсутствие поддержки в старых браузерах (IE)
- Сложность отладки в некоторых случаях
- Не подходят для всех сценариев использования
- Требуют дисциплины в организации кода

## Лучшие практики

### Структурирование переменных
```css
:root {
  /* Цвета */
  --color-primary-50: #eff6ff;
  --color-primary-100: #dbeafe;
  --color-primary-500: #3b82f6;
  --color-primary-900: #1e3a8a;
  
  /* Типографика */
  --font-family-base: 'Inter', system-ui, sans-serif;
  --font-weight-normal: 400;
  --font-weight-bold: 700;
  
  /* Размеры */
  --size-unit: 8px;
  --size-1: calc(var(--size-unit) * 1); /* 8px */
  --size-2: calc(var(--size-unit) * 2); /* 16px */
  --size-4: calc(var(--size-unit) * 4); /* 32px */
}
```

### Использование fallback значений
```css
.element {
  color: var(--primary-color, #3498db);
  background: var(--bg-gradient, linear-gradient(to right, #fff, #eee));
}
```

## Заключение

Кастомные свойства CSS являются мощным инструментом для создания гибких, поддерживаемых и темизируемых интерфейсов. Их правильное использование позволяет централизовать управление стилями и создавать масштабируемые дизайн-системы. В условиях развития веб-стандартов и увеличения сложности интерфейсов, кастомные свойства становятся неотъемлемой частью современного CSS.
