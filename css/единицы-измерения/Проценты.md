# Проценты в CSS

## Определение

Проценты (%) в CSS - это относительная единица измерения, значение которой вычисляется относительно соответствующего свойства родительского элемента или содержащего блока. Проценты широко используются для создания гибких и адаптивных макетов.

## Как работают проценты

Проценты вычисляются на основе:
- Ширины родительского элемента (для свойств ширины, горизонтальных отступов)
- Высоты содержащего блока (для свойств высоты, вертикальных отступов)
- Размера шрифта родителя (для свойств, связанных с шрифтом)

## Применение процентов

### Для размеров и позиционирования

```css
.container {
  width: 100%;
}

.sidebar {
  width: 25%; /* 25% от ширины родителя */
  float: left;
}

.content {
  width: 75%; /* 75% от ширины родителя */
  float: right;
}
```

### Для внутренних отступов (padding)

```css
.aspect-ratio-box {
  width: 100%;
  padding-top: 56.25%; /* Соотношение сторон 16:9 */
  position: relative;
}

.aspect-ratio-content {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
```

### Для внешних отступов (margin)

```css
.centered-element {
  width: 80%;
  margin: 0 auto; /* Центрирование по горизонтали */
}

.offset-element {
  margin-left: 10%;
  width: 90%;
}
```

### Для свойств фона

```css
.background-element {
  background-position: 50% 50%; /* Центрирование фона */
  background-size: 100% auto; /* Адаптивный размер фона */
}
```

## Особенности вычисления процентов

### Относительно ширины родителя

Следующие свойства вычисляют проценты относительно ширины родительского элемента:
- `width`, `max-width`, `min-width`
- `height`, `max-height`, `min-height` (в некоторых случаях)
- `padding`, `margin`
- `left`, `right`, `text-indent`

### Относительно высоты родителя

Высота (`height`) в процентах вычисляется относительно высоты содержащего блока, но только если высота родительского элемента явно задана.

```css
.parent {
  height: 400px; /* Необходимо задать высоту родителя */
}

.child {
  height: 50%; /* Будет 200px (50% от 400px) */
}
```

### Для свойств шрифта

Проценты в свойствах шрифта вычисляются относительно родительского значения:
- `font-size: 120%` увеличивает размер на 20% относительно родительского
- `line-height: 150%` устанавливает высоту строки в 1.5 раза больше размера шрифта

## Практические применения

### Создание адаптивных сеток

```css
.grid-container {
  display: flex;
  flex-wrap: wrap;
}

.grid-item {
  width: 33.333%; /* Три колонки */
  padding: 1%;
  box-sizing: border-box;
}

/* Адаптация для мобильных устройств */
@media (max-width: 768px) {
  .grid-item {
    width: 50%; /* Две колонки на планшетах */
  }
}

@media (max-width: 480px) {
  .grid-item {
    width: 100%; /* Одна колонка на мобильных */
  }
}
```

### Создание резиновых макетов

```css
.responsive-container {
  width: 90%;
  max-width: 1200px;
  min-width: 320px;
  margin: 0 auto;
}

.responsive-text {
  font-size: 4vw; /* Адаптивный размер шрифта */
  line-height: 1.4;
}
```

### Создание адаптивных медиа элементов

```css
.responsive-image {
  max-width: 100%;
  height: auto;
}

.video-container {
  position: relative;
  width: 100%;
  padding-bottom: 56.25%; /* 16:9 Aspect Ratio */
}

.video-container iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
```

## Рекомендации по использованию в 2025 году

- Проценты остаются важной частью адаптивной верстки
- Особенно актуальны при создании мобильных интерфейсов
- В российской практике часто комбинируются с Flexbox и Grid
- Используются для создания адаптивных изображений и видео
- Важно учитывать особенности вычисления в разных контекстах

## Преимущества и недостатки

### Преимущества
- Гибкость и адаптивность
- Автоматическая подстройка под размеры родителя
- Хорошая совместимость с различными размерами экранов
- Эффективность при создании отзывчивых интерфейсов

### Недостатки
- Сложность точного контроля размеров
- Зависимость от размеров родительских элементов
- Потенциальные проблемы с вычислением высоты
- Требуют понимания контекста вычисления

## Современные альтернативы

Хотя проценты остаются важными, в 2025 году разработчики также активно используют:
- Flexbox для гибких раскладок
- CSS Grid для сложных сеток
- Viewport единицы для адаптации под размеры окна
- Кастомные свойства для создания переменных
