# CSS Display и Positioning

Этот файл описывает свойства display и positioning в CSS.

Свойства display и position играют ключевую роль в определении того, как элементы отображаются и позиционируются на веб-странице. Понимание этих свойств необходимо для создания сложных макетов.

## Свойство Display:

Свойство display определяет, как элемент должен отображаться:
- block: элемент отображается как блочный
- inline: элемент отображается как строчный
- inline-block: элемент отображается как строчный блок
- flex: элемент становится flex-контейнером
- grid: элемент становится grid-контейнером
- none: элемент не отображается

## Свойство Position:

Свойство position определяет, как элемент позиционируется:
- static: элемент позиционируется по умолчанию
- relative: элемент позиционируется относительно своего обычного положения
- absolute: элемент позиционируется относительно ближайшего позиционированного предка
- fixed: элемент позиционируется относительно окна просмотра
- sticky: элемент переключается между relative и fixed
