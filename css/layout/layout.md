# Верстка

Этот файл описывает веб-верстку.

Веб-верстка — это процесс создания HTML-разметки и применения к ней CSS-стилей для отображения веб-страниц. Это важный этап веб-разработки, на котором макет превращается в интерактивную веб-страницу.

## Основные аспекты верстки:

1. **Семантическая разметка**: Использование правильных HTML-тегов для структуры документа
2. **Стилизация**: Применение CSS для визуального оформления
3. **Адаптивность**: Создание макетов, которые корректно отображаются на различных устройствах
4. **Доступность**: Обеспечение возможности использования сайта людьми с ограниченными возможностями

## Техники верстки:

- Float-based layouts (устаревшие)
- Flexbox для одномерных макетов
- Grid для двумерных макетов
- Responsive design
