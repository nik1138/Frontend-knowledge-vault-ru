# CSS Grid

Этот файл описывает CSS Grid.

CSS Grid Layout — это двумерная система раскладки, позволяющая создавать сложные макеты с контролем по осям X и Y. Grid особенно полезен для создания сложных макетов страниц и компонентов.

## Основные свойства:

### Свойства grid-контейнера:
- display: grid
- grid-template-columns
- grid-template-rows
- grid-template-areas
- grid-gap

### Свойства grid-элементов:
- grid-column
- grid-row
- grid-area
- align-self
- justify-self
