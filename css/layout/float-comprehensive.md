# CSS Float

Этот файл описывает свойство float в CSS.

CSS float — это свойство, которое позволяет элементу смещаться влево или вправо в контейнере, позволяя тексту и инлайн-элементам обтекать его. Хотя float изначально предназначался для создания эффектов, подобных обтеканию изображений текстом в текстовых редакторах, он также использовался для создания макетов страниц до появления Flexbox и Grid.

## Основные значения:

- left: Элемент плавает влево
- right: Элемент плавает вправо
- none: Элемент не плавает (по умолчанию)
- inherit: Наследует значение от родителя

## Очистка float:

После использования float часто требуется очистка (clear), чтобы следующие элементы не обтекали плавающий элемент.
