# CSS Layout

Этот файл описывает системы раскладки в CSS.

CSS Layout охватывает различные методы организации элементов на веб-странице. С развитием CSS появилось множество мощных инструментов для создания сложных макетов.

## Основные системы раскладки:

1. **Normal Flow**: Стандартный поток документа
2. **Float**: Традиционный метод раскладки (устаревший для макетов)
3. **Positioning**: Точное позиционирование элементов
4. **Flexbox**: Одномерная система раскладки
5. **Grid**: Двумерная система раскладки
6. **Multi-column**: Раскладка в несколько колонок

## Современные подходы:

- Flexbox для раскладки в одном измерении
- Grid для раскладки в двух измерениях
- Комбинации различных методов для сложных макетов
