# CSS Flexbox

Этот файл описывает Flexbox в CSS.

CSS Flexible Box Layout (Flexbox) — это одномерная система раскладки, идеально подходящая для распределения пространства между элементами в строке или столбце. Flexbox особенно полезен для создания гибких макетов, которые адаптируются к различным размерам экрана.

## Основные свойства:

### Свойства flex-контейнера:
- display: flex
- flex-direction
- justify-content
- align-items
- flex-wrap

### Свойства flex-элементов:
- flex-grow
- flex-shrink
- flex-basis
- align-self
