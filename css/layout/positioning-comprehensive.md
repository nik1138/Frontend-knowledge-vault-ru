# CSS Positioning

Этот файл описывает позиционирование элементов в CSS.

CSS positioning позволяет точно контролировать расположение элементов на веб-странице. Свойство position определяет, как элемент позиционируется в документе.

## Типы позиционирования:

- static: Элемент позиционируется по умолчанию (в нормальном потоке документа)
- relative: Элемент позиционируется относительно своего обычного положения
- absolute: Элемент позиционируется относительно ближайшего позиционированного предка
- fixed: Элемент позиционируется относительно окна просмотра
- sticky: Элемент переключается между relative и fixed в зависимости от позиции прокрутки

## Связанные свойства:

- top, right, bottom, left: Определяют смещение позиционированного элемента
- z-index: Определяет порядок наложения элементов
