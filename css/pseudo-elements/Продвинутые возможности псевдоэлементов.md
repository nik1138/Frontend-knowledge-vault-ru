# Продвинутые возможности псевдоэлементов

## Введение

Псевдоэлементы CSS предоставляют мощные возможности для добавления декоративного и функционального контента к элементам без изменения HTML-разметки. В этом руководстве мы рассмотрим продвинутые возможности псевдоэлементов `::before` и `::after`, а также другие менее известные псевдоэлементы.

## Продвинутые техники с ::before и ::after

### Создание сложных геометрических фигур

```css
/* Круг */
.circle::before {
  content: '';
  position: absolute;
  width: 50px;
  height: 50px;
  border-radius: 50%;
  background-color: #007bff;
}

/* Треугольник */
.triangle::after {
  content: '';
  position: absolute;
  width: 0;
  height: 0;
  border-left: 10px solid transparent;
  border-right: 10px solid transparent;
  border-bottom: 20px solid #28a745;
}

/* Звезда */
.star {
  position: relative;
  display: inline-block;
  width: 100px;
  height: 100px;
}

.star::before,
.star::after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100px;
  height: 100px;
  background: #ffc107;
  clip-path: polygon(
    50% 0%, 
    61% 35%, 
    98% 35%, 
    68% 57%, 
    79% 91%, 
    50% 70%, 
    21% 91%, 
    32% 57%, 
    2% 35%, 
    39% 35%
  );
}

.star::after {
  transform: rotate(36deg);
}
```

### Создание сложных декоративных элементов

```css
.quote-box {
  position: relative;
  padding: 20px 20px 20px 60px;
  background: #f8f9fa;
  border-left: 4px solid #007bff;
  font-style: italic;
}

.quote-box::before {
  content: """;
  position: absolute;
  top: 10px;
  left: 10px;
  font-size: 60px;
  color: #007bff;
  line-height: 1;
  font-family: Georgia, serif;
}

.quote-box::after {
  content: "";
  position: absolute;
  bottom: 10px;
  right: 20px;
  width: 30px;
  height: 30px;
  background: 
    radial-gradient(circle at 0 0, transparent 14px, #007bff 15px, #007bff 17px, transparent 18px),
    radial-gradient(circle at 100% 0, transparent 14px, #007bff 15px, #007bff 17px, transparent 18px),
    radial-gradient(circle at 0 100%, transparent 14px, #007bff 15px, #007bff 17px, transparent 18px),
    radial-gradient(circle at 100% 100%, transparent 14px, #007bff 15px, #007bff 17px, transparent 18px);
}
```

## Множественные псевдоэлементы

### Использование нескольких ::after элементов

CSS позволяет использовать только один ::before и один ::after на элемент, но мы можем создавать дополнительные "псевдоэлементы" с помощью дополнительных DOM-элементов:

```css
.complex-shape {
  position: relative;
  width: 200px;
  height: 100px;
  background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
}

.complex-shape::before {
  content: '';
  position: absolute;
  top: -10px;
  left: -10px;
  right: -10px;
  bottom: -10px;
  background: inherit;
  border-radius: 10px;
  z-index: -1;
  filter: blur(10px);
  opacity: 0.5;
}

.complex-shape::after {
  content: '';
  position: absolute;
  top: 5px;
  left: 5px;
  right: 5px;
  bottom: 5px;
  border: 2px solid rgba(255,255,255,0.3);
  border-radius: 8px;
  pointer-events: none;
}
```

## Продвинутые анимации с псевдоэлементами

### Анимированные границы

```css
.animated-border {
  position: relative;
  padding: 20px;
  background: white;
}

.animated-border::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  border: 2px solid;
  border-image: linear-gradient(45deg, #ff6b6b, #4ecdc4, #45b7d1, #96ceb4) 1;
  animation: border-rotate 3s linear infinite;
  pointer-events: none;
}

@keyframes border-rotate {
  0% {
    transform: rotate(0deg);
  }
  100% {
    transform: rotate(360deg);
  }
}
```

### Анимированный фон с псевдоэлементами

```css
.animated-bg {
  position: relative;
  overflow: hidden;
}

.animated-bg::before {
  content: '';
  position: absolute;
  top: -50%;
  left: -50%;
  width: 200%;
  height: 200%;
  background: 
    radial-gradient(circle at center, rgba(255,255,255,0.1) 0%, transparent 70%),
    linear-gradient(45deg, #ff6b6b, #4ecdc4, #45b7d1);
  animation: rotate-bg 8s linear infinite;
  z-index: 0;
}

@keyframes rotate-bg {
  0% {
    transform: rotate(0deg);
  }
  100% {
    transform: rotate(360deg);
  }
}

.animated-bg > * {
  position: relative;
  z-index: 1;
}
```

## Продвинутые текстовые эффекты

### Текст с градиентной заливкой

```css
.gradient-text {
  position: relative;
  display: inline-block;
  font-size: 2em;
  font-weight: bold;
  color: transparent;
  background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
  -webkit-background-clip: text;
  background-clip: text;
}

.gradient-text::before {
  content: attr(data-text);
  position: absolute;
  top: 0;
  left: 0;
  color: #000;
  z-index: -1;
  filter: blur(5px);
  opacity: 0.5;
}
```

### Текст с 3D-эффектом

```css
.text-3d {
  position: relative;
  font-size: 3em;
  font-weight: bold;
  color: #007bff;
  text-shadow: 
    1px 1px 0 #ccc,
    2px 2px 0 #bbb,
    3px 3px 0 #aaa;
}

.text-3d::before {
  content: attr(data-text);
  position: absolute;
  top: 2px;
  left: 2px;
  color: #6c757d;
  z-index: -1;
}
```

## Псевдоэлементы для вспомогательных функций

### Индикаторы загрузки

```css
.loading-indicator {
  position: relative;
  display: inline-block;
  padding-left: 30px;
}

.loading-indicator::before {
  content: '';
  position: absolute;
  left: 0;
  top: 50%;
  transform: translateY(-50%);
  width: 20px;
  height: 20px;
  border: 3px solid #f3f3f3;
  border-top: 3px solid #007bff;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

@keyframes spin {
  0% { transform: translateY(-50%) rotate(0deg); }
  100% { transform: translateY(-50%) rotate(360deg); }
}
```

### Счетчики и бейджи

```css
.notification::after {
  content: attr(data-count);
  position: absolute;
  top: -8px;
  right: -8px;
  background: #dc3545;
  color: white;
  border-radius: 50%;
  width: 20px;
  height: 20px;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 0.75em;
  font-weight: bold;
}
```

## Псевдоэлементы для декоративных разделителей

### Стилизованные разделители

```css
.divider {
  position: relative;
  text-align: center;
  margin: 20px 0;
}

.divider::before {
  content: '';
  position: absolute;
  top: 50%;
  left: 0;
  right: 0;
  height: 1px;
  background: linear-gradient(to right, transparent, #ccc, transparent);
  z-index: 0;
}

.divider span {
  position: relative;
  background: white;
  padding: 0 15px;
  z-index: 1;
}

/* Декоративный разделитель с иконками */
.decorative-divider {
  position: relative;
  height: 20px;
  margin: 20px 0;
}

.decorative-divider::before,
.decorative-divider::after {
  content: '';
  position: absolute;
  top: 50%;
  width: 40%;
  height: 1px;
  background: #ddd;
}

.decorative-divider::before {
  left: 0;
}

.decorative-divider::after {
  right: 0;
}

.decorative-divider::before::after {
  content: '❦';
  position: absolute;
  right: -10px;
  top: -8px;
  color: #007bff;
  font-size: 1.2em;
}

.decorative-divider::after::after {
  content: '❦';
  position: absolute;
  left: -10px;
  top: -8px;
  color: #007bff;
  font-size: 1.2em;
  transform: rotate(180deg);
}
```

## Современные псевдоэлементы

### ::marker - стилизация маркеров списков

```css
.custom-list {
  list-style: none;
  padding-left: 0;
}

.custom-list li {
  position: relative;
  padding-left: 30px;
  margin-bottom: 10px;
}

.custom-list li::marker {
  color: #007bff;
  font-size: 1.2em;
  font-weight: bold;
}

/* Альтернативный подход с ::before */
.modern-list {
  list-style: none;
  padding-left: 0;
}

.modern-list li::before {
  content: '▶';
  position: absolute;
  left: 0;
  color: #007bff;
  transition: transform 0.2s ease;
}

.modern-list li:hover::before {
  transform: translateX(5px);
}
```

### ::selection - стилизация выделенного текста

```css
.highlighted-text::selection {
  background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
  color: white;
}

/* Стилизация ввода */
input::selection {
  background: rgba(0, 123, 255, 0.5);
  color: white;
}
```

## Практические примеры

### Интерактивная карточка с псевдоэлементами

```css
.interactive-card {
  position: relative;
  overflow: hidden;
  padding: 20px;
  background: white;
  border-radius: 8px;
  transition: transform 0.3s ease;
}

.interactive-card::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: linear-gradient(90deg, transparent, rgba(255,255,255,0.4), transparent);
  transition: left 0.5s;
  z-index: 1;
}

.interactive-card:hover::before {
  left: 100%;
}

.interactive-card::after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  border: 2px solid transparent;
  border-radius: 8px;
  transition: border-color 0.3s ease;
  z-index: 2;
}

.interactive-card:hover::after {
  border-color: #007bff;
}
```

### Прогресс-бар с псевдоэлементами

```css
.progress-container {
  position: relative;
  width: 100%;
  height: 20px;
  background: #e9ecef;
  border-radius: 10px;
  overflow: hidden;
}

.progress-bar {
  position: relative;
  height: 100%;
  background: linear-gradient(90deg, #007bff, #0069d9);
  border-radius: 10px;
  width: var(--progress, 0%);
  transition: width 0.3s ease;
}

.progress-bar::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: 
    linear-gradient(
      45deg,
      rgba(255, 255, 255, .2) 25%,
      transparent 25%,
      transparent 50%,
      rgba(255, 255, 255, .2) 50%,
      rgba(255, 255, 255, .2) 75%,
      transparent 75%,
      transparent
    );
  background-size: 30px 30px;
  animation: progress-animation 1s linear infinite;
  border-radius: 10px;
}

@keyframes progress-animation {
  0% {
    background-position: 0 0;
  }
  100% {
    background-position: 30px 30px;
  }
}
```

## Заключение

Продвинутые возможности псевдоэлементов открывают широкие возможности для создания сложных визуальных эффектов без дополнительных DOM-элементов. Понимание этих возможностей позволяет создавать более чистую HTML-разметку и более гибкие стили.
