# Container Queries

Этот файл описывает контейнерные запросы в CSS.

Контейнерные запросы — это новая возможность CSS, позволяющая применять стили в зависимости от размеров родительского контейнера, а не от размера окна просмотра. Это позволяет создавать более адаптивные и переиспользуемые компоненты.

## Основные особенности:

- Зависимость от размера контейнера, а не от вьюпорта
- Позволяет создавать более модульные и переиспользуемые компоненты
- Улучшает создание адаптивных интерфейсов

## Синтаксис:

```css
@container (min-width: 400px) {
  .card {
    display: flex;
  }
}
```

## Поддержка браузерами:

На момент написания этого файла поддержка контейнерных запросов в браузерах может быть ограничена или требовать префиксов.
