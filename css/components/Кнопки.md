# Кнопки

## Обзор

Кнопки являются одними из самых важных интерактивных элементов веб-интерфейса. Они инициируют действия, управляют навигацией и улучшают пользовательский опыт. В 2025 году требования к стилям и функциональности кнопок становятся всё более строгими, особенно с учётом доступности (accessibility) и адаптивности.

## Основные свойства CSS для стилизации кнопок

### 1. Визуальное оформление
- `background-color` — задаёт цвет фона кнопки.
- `color` — определяет цвет текста.
- `border` — стиль, ширина и цвет границы.
- `border-radius` — скругление углов.
- `box-shadow` — добавляет тени, особенно полезно для состояния `:hover` или `:focus`.

### 2. Размеры и отступы
- `padding` — внутренние отступы, влияющие на восприятие размера кнопки.
- `font-size` — размер шрифта.
- `min-height` и `min-width` — минимальные размеры, важны для доступности.

### 3. Поведение
- `cursor: pointer` — изменение курсора при наведении.
- `transition` — плавные анимации при изменении состояния.
- `outline: none` — удаление стандартной рамки фокуса (заменить на `box-shadow` для доступности!).

## Практические рекомендации

### 1. Стандартные состояния
Кнопка должна иметь следующие состояния:
- `:default` — обычное состояние.
- `:hover` — при наведении.
- `:active` — при нажатии.
- `:focus` — при фокусе (важно для доступности!).

Пример:
```css
.button {
  background-color: #007bff;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 4px;
  cursor: pointer;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: #0056b3;
}

.button:active {
  transform: translateY(1px);
}

.button:focus {
  outline: none;
  box-shadow: 0 0 0 3px rgba(0, 123, 255, 0.25);
}
```

### 2. Типы кнопок
- **Primary** — основные действия.
- **Secondary** — второстепенные действия.
- **Danger** — действия, требующие осторожности.
- **Ghost** — минимальный визуальный акцент.

### 3. Адаптивность
Кнопки должны корректно отображаться на всех устройствах. Используйте `rem` или `em` для размеров шрифта и `flexbox` или `grid` для выравнивания.

### 4. Доступность
- Убедитесь, что кнопки можно активировать с клавиатуры (Tab + Enter/Space).
- Используйте `aria-label` или `title`, если текст кнопки неясен.
- Контрастность текста и фона должна соответствовать WCAG 2.1 AA (коэффициент не менее 4.5:1).

## Примеры

### Стильная кнопка с анимацией
```css
.animated-button {
  background-color: #6f42c1;
  color: white;
  border: none;
  padding: 12px 24px;
  font-size: 16px;
  border-radius: 6px;
  cursor: pointer;
  transition: all 0.3s ease;
  position: relative;
  overflow: hidden;
}

.animated-button::before {
  content: '';
  position: absolute;
  top: 50%;
  left: 50%;
  width: 0;
  height: 0;
  background: rgba(255, 255, 255, 0.3);
  border-radius: 50%;
  transform: translate(-50%, -50%);
  transition: width 0.5s, height 0.5s;
}

.animated-button:hover::before {
  width: 300px;
  height: 300px;
}

.animated-button span {
  position: relative;
  z-index: 1;
}
```
