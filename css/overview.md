# CSS - Каскадные таблицы стилей

CSS (Cascading Style Sheets) - это язык описания слоев, используемый для описания внешнего вида документа, написанного на языке разметки, например HTML. CSS описывает, как элементы должны отображаться на экране, на бумаге, в речи или на других носителях.

## Назначение CSS

CSS отделяет содержимое документа (структуру, представленную HTML) от представления (визуального оформления). Это позволяет:

- Упростить управление стилями и внешним видом документа
- Поддерживать единообразие оформления на всех страницах сайта
- Сократить объем кода за счет централизованного управления стилями
- Улучшить доступность и адаптивность веб-сайтов

## Основные концепции

CSS строится на нескольких ключевых концепциях:

- [[selectors]] - определяют, к каким элементам применяются стили
- [[cascade]] - механизм определения приоритета стилей
- [[specificity]] - система вычисления приоритета селекторов
- [[inheritance]] - передача стилей от родительских элементов к дочерним

## История и развитие

CSS был разработан в 1996 году Консорциумом Всемирной паутины (W3C) для отделения стиля и оформления от содержимого HTML. С тех пор он прошел несколько этапов развития:

- CSS1 (1996) - основные возможности форматирования
- CSS2 (1998) - позиционирование, табличные стили, медиазапросы
- CSS2.1 (2011) - улучшенная стабильность и совместимость
- CSS3 (2011-настоящее время) - модульная архитектура, новые возможности

CSS3 не является единым стандартом, а представляет собой набор модулей, каждый из которых развивается независимо. Это позволяет более гибко внедрять новые функции и обеспечивает лучшую совместимость между браузерами.

## Современные возможности

Современный CSS предоставляет мощные инструменты для создания сложных веб-интерфейсов:

- Flexbox и Grid для сложных раскладок
- Пользовательские свойства CSS (переменные)
- Модули для анимаций и переходов
- Медиазапросы для адаптивного дизайна
- Поддержка переменных шрифтов и контейнерных запросов
