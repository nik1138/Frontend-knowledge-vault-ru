# Transform-origin

## Описание

Свойство `transform-origin` в CSS определяет точку, относительно которой применяются трансформации элемента. По умолчанию трансформации применяются относительно центра элемента (50% 50%), но с помощью `transform-origin` можно изменить эту точку, что особенно полезно при создании анимаций и сложных визуальных эффектов.

## Синтаксис

```css
transform-origin: x-axis y-axis z-axis;
```

### Значения

- **x-axis**: Определяет горизонтальную позицию точки трансформации
  - `left`, `center`, `right`
  - Проценты: `0%`, `50%`, `100%`
  - Длина: `10px`, `2em` и т.д.
  
- **y-axis**: Определяет вертикальную позицию точки трансформации
  - `top`, `center`, `bottom`
  - Проценты: `0%`, `50%`, `100%`
  - Длина: `10px`, `2em` и т.д.
  
- **z-axis**: Определяет положение по оси Z (для 3D-трансформаций)
  - Длина: `10px`, `2em` и т.д.

## Примеры использования

### Базовое использование

```css
.element {
  transform: rotate(45deg);
  transform-origin: left top;
}
```

В этом примере элемент будет вращаться вокруг своей левой верхней точки, а не вокруг центра.

### 3D-трансформации

```css
.cube-face {
  transform: rotateY(45deg);
  transform-origin: 0 50% 0;
}
```

Здесь элемент вращается вокруг оси Y с точкой вращения по левому краю.

### Анимация с изменением точки трансформации

```css
@keyframes swing {
  0% {
    transform: rotate(0deg);
    transform-origin: left top;
  }
  50% {
    transform: rotate(20deg);
    transform-origin: right top;
  }
  100% {
    transform: rotate(0deg);
    transform-origin: center center;
  }
}

.swing-element {
  animation: swing 3s infinite;
}
```

## Практические применения

### 1. Создание анимаций открытия/закрытия

```css
.door {
  transform-origin: left center;
  transition: transform 0.5s ease;
}

.door:hover {
  transform: rotateY(-90deg);
}
```

Используется для создания эффектов "дверей" или "страниц", которые открываются при наведении.

### 2. Анимация "качания" элементов

```css
.swing {
  transform-origin: top center;
  animation: swing 2s infinite alternate ease-in-out;
}

@keyframes swing {
  from { transform: rotate(-10deg); }
  to { transform: rotate(10deg); }
}
```

### 3. Создание эффектов "поворота карты"

```css
.card {
  transform-style: preserve-3d;
  transform-origin: center center 100px;
  transition: transform 0.8s;
}

.card.flipped {
  transform: rotateY(180deg);
}
```

## Важные особенности

> [!tip] Совет
> Используйте `transform-origin` в сочетании с `transform-box` для более точного контроля над областью трансформации.

> [!warning] Важно
> Значение `transform-origin` влияет только на элементы, к которым применено свойство `transform`.

> [!note] Примечание
> Свойство `transform-origin` не наследуется, поэтому каждый элемент должен иметь свою собственную точку трансформации.

## Совместимость с браузерами

Свойство `transform-origin` поддерживается во всех современных браузерах, включая:

- Chrome 36+
- Firefox 16+
- Safari 9+
- Edge 12+
- Opera 23+

## Практические рекомендации

1. **Используйте `transform-origin` для улучшения пользовательского опыта**
   - Эффекты вращения вокруг углов элемента создают более естественные анимации
   - Помогает создать ощущение физичности интерфейса

2. **Комбинируйте с `transform-box`**
   - Используйте `transform-box: fill-box` для трансформаций SVG
   - `transform-box: border-box` для элементов с бордерами

3. **Оптимизация производительности**
   - Избегайте частого изменения `transform-origin` в анимациях
   - Используйте GPU-ускорение через `will-change: transform`

## Примеры из российской практики

В российских веб-разработках часто используется `transform-origin` для создания:

- Эффектов "открывающихся" меню на сайтах банков и финансовых организаций
- Анимаций переключения вкладок в личных кабинетах
- Интерактивных элементов на сайтах электронной коммерции (например, эффекты "поворота" карточек товаров)

## Заключение

Свойство `transform-origin` является мощным инструментом для контроля над визуальными эффектами в CSS. Правильное его использование позволяет создавать более плавные и естественные анимации, улучшая общее восприятие интерфейса пользователя.
