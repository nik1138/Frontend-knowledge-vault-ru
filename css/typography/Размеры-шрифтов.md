# Размеры шрифтов

## Общее описание

Размеры шрифтов играют ключевую роль в веб-дизайне и доступности. Правильная типографика обеспечивает читаемость, иерархию и удобство восприятия информации. В 2025 году в России приоритет отдается адаптивной типографике, учитывающей различные устройства и потребности пользователей.

## Единицы измерения размера шрифта

### Абсолютные единицы

Абсолютные единицы задают фиксированный размер шрифта, не зависящий от контекста:

- `px` - пиксели
- `pt` - пункты (points)
- `pc` - пика (picas)
- `mm` - миллиметры
- `cm` - сантиметры
- `in` - дюймы

```css
.absolute-sizes {
  font-size: 16px;  /* Наиболее распространенный размер */
  font-size: 12pt;  /* Используется в печати */
}
```

> [!warning] Важно
> Использование абсолютных единиц не рекомендуется для веб-дизайна, так как они не масштабируются при изменении настроек браузера пользователя.

### Относительные единицы

Относительные единицы обеспечивают гибкость и доступность:

#### em

Единица `em` зависит от размера шрифта родительского элемента:

```css
.parent {
  font-size: 16px;
}

.child {
  font-size: 1.5em; /* 16px * 1.5 = 24px */
}
```

> [!tip] Совет
> Используйте `em` для элементов, размер которых должен масштабироваться относительно родительского контекста, например, для иконок или отступов.

#### rem

Единица `rem` (root em) зависит от размера шрифта корневого элемента (`html`):

```css
html {
  font-size: 16px; /* Базовый размер */
}

.title {
  font-size: 2rem; /* 2 * 16px = 32px */
}

.subtitle {
  font-size: 1.5rem; /* 1.5 * 16px = 24px */
}
```

> [!tip] Совет
> Используйте `rem` для основных размеров текста, чтобы обеспечить согласованность по всему сайту и поддержку пользовательских настроек масштабирования.

#### %

Проценты работают аналогично `em`, но выражены в процентах от родительского элемента:

```css
.parent {
  font-size: 16px;
}

.child {
  font-size: 125%; /* 16px * 1.25 = 20px */
}
```

#### vw, vh, vmin, vmax

Адаптивные единицы, зависящие от размеров вьюпорта:

```css
.responsive-title {
  font-size: 4vw; /* 4% от ширины вьюпорта */
}

.responsive-text {
  font-size: 2.5vh; /* 2.5% от высоты вьюпорта */
}
```

> [!warning] Важно
> Используйте вьюпортные единицы с осторожностью, так как они могут привести к слишком большим или маленьким размерам на некоторых устройствах.

## Ключевые слова размера шрифта

CSS предоставляет ключевые слова для размеров шрифта:

```css
.font-sizes {
  font-size: xx-small;  /* Очень маленький */
  font-size: x-small;   /* Особо маленький */
  font-size: small;     /* Маленький */
  font-size: medium;    /* Средний (по умолчанию) */
  font-size: large;     /* Большой */
  font-size: x-large;   /* Особо большой */
  font-size: xx-large;  /* Очень большой */
  font-size: xxx-large; /* Особо-особо большой */
}
```

## Иерархия типографики

### Базовая шкала размеров

Рекомендуемая шкала размеров шрифтов в проектах 2025 года:

```css
:root {
  --font-size-xs: 0.75rem;      /* 12px */
  --font-size-sm: 0.875rem;     /* 14px */
  --font-size-base: 1rem;       /* 16px */
  --font-size-md: 1.125rem;     /* 18px */
  --font-size-lg: 1.25rem;      /* 20px */
  --font-size-xl: 1.5rem;       /* 24px */
  --font-size-2xl: 1.875rem;    /* 30px */
  --font-size-3xl: 2.25rem;     /* 36px */
  --font-size-4xl: 3rem;        /* 48px */
  --font-size-5xl: 3.75rem;     /* 60px */
}
```

### Пример иерархии заголовков

```css
h1 {
  font-size: 2.5rem;    /* ~40px */
}

h2 {
  font-size: 2rem;      /* ~32px */
}

h3 {
  font-size: 1.75rem;   /* ~28px */
}

h4 {
  font-size: 1.5rem;    /* ~24px */
}

h5 {
  font-size: 1.25rem;   /* ~20px */
}

h6 {
  font-size: 1.125rem;  /* ~18px */
}
```

## Адаптивная типографика

### Fluid Typography

Техника "жидкой" типографики позволяет плавно изменять размер шрифта в зависимости от ширины экрана:

```css
.fluid-title {
  font-size: clamp(1.5rem, 4vw, 3rem);
  /* Минимум 24px, идеал 4% от ширины вьюпорта, максимум 48px */
}
```

### Media Queries для типографики

```css
body {
  font-size: 1rem;      /* 16px на мобильных */
}

@media (min-width: 768px) {
  body {
    font-size: 1.125rem; /* 18px на планшетах */
  }
}

@media (min-width: 1024px) {
  body {
    font-size: 1.25rem;  /* 20px на десктопах */
  }
}
```

## Доступность и размеры шрифтов

### Поддержка пользовательских настроек

Использование относительных единиц позволяет уважать пользовательские настройки:

```css
/* Плохо */
.content {
  font-size: 16px;
}

/* Хорошо */
.content {
  font-size: 1rem;
}
```

### Минимальные рекомендуемые размеры

- Основной текст: не менее 16px (1rem)
- Подписи к изображениям: не менее 14px (0.875rem)
- Вспомогательный текст: не менее 12px (0.75rem)

> [!tip] Совет
> В России согласно ГОСТ Р 50577-2019 и рекомендациям по доступности, минимальный размер текста для электронных документов должен быть не менее 14 пунктов, что эквивалентно примерно 18-19px на экране.

## Современные подходы

### CSS Custom Properties для типографики

Использование CSS переменных для управления размерами:

```css
:root {
  --font-scale-ratio: 1.2; /* Соотношение масштаба */
  --font-size-3: calc(var(--font-size-2) * var(--font-scale-ratio));
  --font-size-2: calc(var(--font-size-1) * var(--font-scale-ratio));
  --font-size-1: 1rem;
}

h1 { font-size: var(--font-size-3); }
h2 { font-size: var(--font-size-2); }
```

### CSS Math Functions

Использование математических функций для вычисления размеров:

```css
.responsive-text {
  font-size: calc(1rem + 0.5vw);
}
```

### Container Queries

Для более точного контроля типографики в компонентах:

```css
.card {
  container-type: inline-size;
}

@container (min-width: 300px) {
  .card-title {
    font-size: 1.2rem;
  }
}

@container (min-width: 500px) {
  .card-title {
    font-size: 1.5rem;
  }
}
```

## Российские особенности

### Читаемость кириллицы

Кириллические символы часто требуют немного большего размера для комфортного чтения:

```css
/* Увеличенный размер для русскоязычных сайтов */
body {
  font-size: 1.125rem; /* Вместо стандартных 1rem */
}
```

### Популярные размеры в российском веб-дизайне

- Основной текст: 16-18px
- Подзаголовки: 20-24px
- Заголовки: 28-36px
- Заголовки верхнего уровня: 36-48px

### Адаптация под российские устройства

Учитывая разнообразие устройств, используемых в России, рекомендуется тестировать типографику на различных экранах, особенно на устройствах с низкой плотностью пикселей.

## Лучшие практики

1. Используйте `rem` для основных размеров шрифтов
2. Применяйте систему модульной шкалы для согласованности
3. Обеспечивайте адаптивность через clamp() и media queries
4. Проверяйте читаемость на различных устройствах
5. Учитывайте потребности пользователей с нарушениями зрения
6. Тестируйте типографику с кириллическими текстами
