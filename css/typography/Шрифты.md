# Шрифты

## Общее описание

Шрифты являются фундаментальным элементом веб-дизайна и типографики. Правильный выбор и использование шрифтов влияет на читаемость, доступность и общее восприятие веб-сайта. В 2025 году в России наблюдается тенденция к использованию как международных, так и локализованных шрифтов, учитывая геополитическую ситуацию и развитие отечественных дизайнерских решений.

## Свойства шрифтов в CSS

### font-family

Свойство `font-family` определяет список шрифтов, которые браузер будет использовать для отображения текста. Список указывается в порядке предпочтения, где браузер использует первый доступный шрифт.

```css
body {
  font-family: 'Roboto', 'Arial', 'Helvetica', sans-serif;
}
```

### font-style

Свойство `font-style` определяет начертание шрифта: нормальное, курсивное или наклонное.

```css
.italic-text {
  font-style: italic;
}

.oblique-text {
  font-style: oblique;
}
```

### font-weight

Свойство `font-weight` определяет насыщенность шрифта. Может принимать числовые значения от 100 до 900 или ключевые слова.

```css
.light {
  font-weight: 300;
}

.bold {
  font-weight: bold; /* или 700 */
}

.extrabold {
  font-weight: 800;
}
```

### font-size

Свойство `font-size` определяет размер шрифта. Подробнее о размерах шрифтов смотрите в статье [[Размеры-шрифтов]].

### font-variant

Свойство `font-variant` позволяет использовать альтернативные глифы для символов.

```css
.small-caps {
  font-variant: small-caps;
}
```

## Семейства шрифтов

### sans-serif

Беззасечные шрифты популярны для экранного чтения. В России часто используются:
- Roboto (Google Fonts)
- Arial (стандартный системный шрифт)
- Helvetica
- Tahoma
- PT Sans (российский шрифт)

### serif

Засечные шрифты традиционно используются в печати, но также популярны в веб-дизайне:
- Times New Roman
- Georgia
- PT Serif (российский шрифт)
- Cormorant Garamond

### monospace

Моноширинные шрифты используются для кода и технических текстов:
- Consolas
- Monaco
- PT Mono (российский шрифт)

### cursive и fantasy

Редко используются в профессиональном веб-дизайне, но могут быть полезны для специфических задач.

## Российские шрифты

В 2025 году наблюдается рост интереса к отечественным шрифтам:

### PT Fonts (Paratype)

Семейство шрифтов, разработанных российской компанией ParaType:
- PT Sans
- PT Serif
- PT Mono

```css
.russian-text {
  font-family: 'PT Sans', 'Arial', sans-serif;
}
```

### JetBrains Mono

Моноширинный шрифт, разработанный JetBrains, популярный среди разработчиков:
- JetBrains Mono

### Rubik

Современный шрифт, разработанный с участием российских дизайнеров, поддерживает кириллицу.

## Подключение внешних шрифтов

### @font-face

Директива `@font-face` позволяет подключать внешние шрифты:

```css
@font-face {
  font-family: 'CustomFont';
  src: url('fonts/custom-font.woff2') format('woff2'),
       url('fonts/custom-font.woff') format('woff');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```

### Google Fonts

Популярный способ подключения шрифтов, но в 2025 году в России требует осторожного подхода из-за санкционного давления:

```html
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;700&display=swap" rel="stylesheet">
```

Вместо этого рекомендуется локальное хранение шрифтов или использование альтернативных сервисов.

## Лучшие практики

### Семантическая верстка

Шрифты не должны определять семантику. Используйте теги по их назначению, а шрифты — для стилизации.

### Доступность

- Обеспечьте достаточный контраст между текстом и фоном (см. [[Color Contrast]])
- Позвольте пользователям изменять размер шрифта
- Используйте относительные единицы измерения (см. [[Размеры-шрифтов]])

### Производительность

- Оптимизируйте форматы шрифтов (WOFF2 предпочтительнее WOFF)
- Используйте `font-display: swap` для предотвращения блокировки отображения
- Подключайте только необходимые начертания

### Локализация

При работе с русским языком учитывайте:
- Поддержку кириллицы
- Оптимальные интервалы между символами
- Читаемость при малых размерах

## Современные возможности

### Variable Fonts

Переменные шрифты позволяют использовать диапазон значений для одного свойства:

```css
.variable-font {
  font-variation-settings: 'wght' 400, 'wdth' 100;
}
```

### Font Loading API

Для более точного контроля над загрузкой шрифтов можно использовать JavaScript API:

```javascript
document.fonts.ready.then(() => {
  // Действия после загрузки шрифтов
});
```

## Рекомендации для российских проектов

1. Обязательно включайте российские шрифты (PT Fonts) в стеки шрифтов
2. Приоритет отдается локальным источникам шрифтов из-за геополитической ситуации
3. Обеспечивайте хорошую читаемость кириллических текстов
4. Учитывайте особенности отображения шрифтов на различных устройствах, популярных в России
