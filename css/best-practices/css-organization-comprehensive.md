# CSS Organization

Этот файл описывает организацию CSS-кода.

CSS Organization — это практика структурирования CSS-файлов и кода для обеспечения его читаемости, поддерживаемости и масштабируемости. Правильная организация позволяет легко находить, изменять и расширять стили.

## Принципы организации CSS:

1. **Модульность**: Разделение кода на логические модули
2. **Последовательность**: Следование единообразному стилю написания кода
3. **Документирование**: Комментирование сложных участков кода
4. **Архитектура файлов**: Логическая структура файлов и папок

## Структуры файлов:

- Разделение по компонентам
- Разделение по слоям (атомы, молекулы, организмы)
- Разделение по функциональности
