# Полный справочник CSS-селекторов

## Введение

CSS-селекторы — это шаблоны, используемые для выбора элементов HTML, к которым применяются стили. Понимание селекторов критически важно для эффективного стилизования веб-страниц.

## Основные селекторы

### Селектор элемента (тега)

Выбирает все элементы указанного типа.

```css
p {
  color: blue;
}
```

[[html/elements/paragraph|Параграф]]

### Селектор класса

Выбирает элементы с определённым атрибутом `class`.

```css
.highlight {
  background-color: yellow;
}
```

[[css/classes|Классы в CSS]]

### Селектор ID

Выбирает элемент с определённым атрибутом `id`.

```css
#main-header {
  font-size: 2em;
}
```

[[html/attributes/id|ID Атрибут]]

## Комбинаторы

### Потомок (пробел)

Выбирает элементы, являющиеся потомками другого элемента.

```css
div p {
  margin: 10px;
}
```

### Дочерний (>) 

Выбирает прямых потомков.

```css
ul > li {
  list-style: none;
}
```

### Сосед по следующему элементу (+)

Выбирает элемент, следующий непосредственно за другим.

```css
h2 + p {
  margin-top: 0;
}
```

### Общий сосед (~)

Выбирает элементы, следующие после другого элемента на том же уровне.

```css
h2 ~ p {
  color: gray;
}
```

## Псевдоклассы

### Состояния

Определяют состояние элемента.

```css
a:hover {
  color: red;
}

input:focus {
  border: 2px solid blue;
}

button:active {
  transform: scale(0.98);
}
```

[[css/pseudo-classes|Псевдоклассы]]

### Структурные

Выбирают элементы на основе их позиции в дереве документа.

```css
/* Первый и последний элемент */
li:first-child { color: green; }
li:last-child { color: red; }

/* Нечетные/четные строки */
tr:nth-child(odd) { background: #f2f2f2; }
tr:nth-child(even) { background: #ffffff; }

/* Третий элемент */
p:nth-child(3) { font-weight: bold; }
```

### Относящиеся к вводу

Применяются к формам и их элементам.

```css
input:required { border-color: red; }
input:valid { border-color: green; }
input:invalid { border-color: red; }
input:checked + label { font-weight: bold; }
```

[[html/forms/input|Формы HTML]]

### Языковые

Выбирают элементы на основе языка.

```css
p:lang(en) { quotes: '"' '"'; }
p:lang(ru) { quotes: '«' '»'; }
```

## Псевдоэлементы

Создают виртуальные элементы для стилизации частей содержимого.

```css
/* Первый символ абзаца */
p::first-letter {
  font-size: 2em;
  color: navy;
}

/* Первая строка абзаца */
p::first-line {
  font-weight: bold;
}

/* Псевдоэлемент перед содержимым */
.quote::before {
  content: '"';
}

/* Псевдоэлемент после содержимого */
.quote::after {
  content: '"';
}

/* Выделение текста */
::selection {
  background: #ffcc00;
}
```

[[css/pseudo-elements|Псевдоэлементы]]

## Селекторы атрибутов

Выбирают элементы на основе наличия или значений атрибутов.

```css
/* Элемент с атрибутом title */
[title] {
  border-bottom: 1px dotted;
}

/* Атрибут со значением */
[type="submit"] {
  background: red;
}

/* Атрибут содержит значение */
[href*="example"] {
  color: purple;
}

/* Атрибут начинается с */
[href^="https"] {
  color: green;
}

/* Атрибут заканчивается на */
[href$=".pdf"] {
  color: red;
}
```

## Расчет специфичности

Специфичность определяет, какой CSS-стиль будет применен, когда несколько правил конфликтуют.

### Порядок важности:

1. Встроенные стили (`style="..."`) - 1000
2. ID селекторы - 100
3. Классы, атрибуты, псевдоклассы - 10
4. Элементы, псевдоэлементы - 1

```css
/* Специфичность: 1 (1 элемент) */
p { color: blue; }

/* Специфичность: 11 (1 класс + 1 элемент) */
p.highlight { color: red; }

/* Специфичность: 100 (1 ID) */
#main p { color: green; }

/* Специфичность: 110 (1 ID + 1 класс) */
#main .highlight { color: yellow; }
```

## Производительность

- Избегайте чрезмерно сложных селекторов
- Используйте ID и классы вместо селекторов элементов
- Ограничьте использование универсального селектора `*`
- Не используйте лишние каскады

```css
/* Медленный селектор */
div ul li span a { color: blue; }

/* Быстрый селектор */
.nav-link { color: blue; }
```

## Лучшие практики

- Используйте осмысленные имена классов
- Следуйте методологии БЭМ
- Организуйте селекторы по модульности
- Документируйте сложные селекторы
- Избегайте `!important` если возможно

[[css/bem|Методология БЭМ]]

## Примеры сложных селекторов

```css
/* Стилизация всех ссылок в заголовках, кроме активных */
h1 a:not(:active),
h2 a:not(:active),
h3 a:not(:active) {
  text-decoration: none;
}

/* Стилизация каждого третьего элемента списка */
li:nth-child(3n) {
  background-color: #f0f0f0;
}

/* Стилизация элементов формы с определённым атрибутом */
input[type="text"][required]:not(:placeholder-shown) {
  border: 2px solid green;
}
```

## Заключение

CSS-селекторы — мощный инструмент для точечной стилизации элементов. Понимание их работы, специфичности и производительности позволяет создавать более эффективные и поддерживаемые стили.
