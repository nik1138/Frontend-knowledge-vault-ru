# Паттерны для проверки грамматических структур в тексте

В этом разделе рассматриваются регулярные выражения, которые могут помочь в анализе и проверке грамматических структур в тексте. Хотя полный грамматический анализ требует сложных NLP-библиотек, простые проверки можно реализовать с помощью регулярных выражений.

## Простые грамматические проверки

### Поиск предложений

```javascript
// Паттерн для поиска предложений (предложения заканчиваются точкой, восклицательным или вопросительным знаком)
const sentencePattern = /[A-Z][^.!?]*[.!?]/g;

const text = "Привет! Как дела? Я изучаю регулярные выражения.";
const sentences = text.match(sentencePattern);
console.log(sentences); // ["Привет!", " Как дела?", " Я изучаю регулярные выражения."]
```

### Поиск вопросительных предложений

```javascript
// Паттерн для выделения вопросительных предложений
const questionPattern = /[A-Z][^?!]*\?/g;

const text = "Привет! Как дела? Я изучаю регулярные выражения.";
const questions = text.match(questionPattern);
console.log(questions); // [" Как дела?"]
```

### Поиск восклицательных предложений

```javascript
// Паттерн для выделения восклицательных предложений
const exclamationPattern = /[A-Z][^.!]*!/g;

const text = "Привет! Как дела? Я изучаю регулярные выражения!";
const exclamations = text.match(exclamationPattern);
console.log(exclamations); // ["Привет!"]
```

## Проверка структуры предложений

### Поиск простых предложений (с одним подлежащим и сказуемым)

```javascript
// Простая проверка на наличие слова, которое может быть глаголом (оканчивается на -ет, -ит, -ат, -ют)
const simpleSentencePattern = /\b\w+\s+(?:\w+)*\s+\w*(?:ет|ит|ат|ют|ут|ят)\b/gi;

const text = "Кот спит на диване. Девочка читает книгу.";
const simpleSentences = text.match(simpleSentencePattern);
console.log(simpleSentences); // ["Кот спит", "девочка читает"]
```

### Поиск слов с определенными суффиксами (например, прилагательные)

```javascript
// Паттерн для поиска прилагательных (в русском языке часто заканчиваются на -ый, -ий, -ой, -ая, -ое и т.д.)
const adjectivePattern = /\b\w+(?:ый|ий|ой|ая|ое|ые|их|ым|ыми|его|ому|ему|ей|ую|ое|ыми|ом|ем|ен|на|но|ны|не|ни|ко|ка|ко|ке|ки|ка|ки|ке)\b/gi;

const text = "Красивая девушка читает интересную книгу.";
const adjectives = text.match(adjectivePattern);
console.log(adjectives); // ["Красивая", "интересную"]
```

## Проверка орфографии и пунктуации

### Поиск слов с повторяющимися буквами (потенциальные опечатки)

```javascript
// Паттерн для поиска слов с тремя и более повторяющимися буквами подряд
const repeatedLettersPattern = /\b\w*([а-яёa-z])\1{2,}\w*\b/gi;

const text = "Он ооочень устал. Это был прееелестный вечер.";
const repeatedWords = text.match(repeatedLettersPattern);
console.log(repeatedWords); // ["ооочень", "прееелестный"]
```

### Проверка пробелов перед знаками препинания

```javascript
// Паттерн для поиска пробелов перед знаками препинания (неправильное форматирование)
const wrongSpacingPattern = /\s[,.:;!?)\]]/g;

const text = "Он сказал , что устал .";
const wrongSpacing = text.match(wrongSpacingPattern);
console.log(wrongSpacing); // [" ,", " ."]
```

### Проверка отсутствия пробелов после знаков препинания

```javascript
// Паттерн для поиска отсутствующих пробелов после точки, запятой и других знаков
const missingSpacePattern = /[,.:;!?)\]](?!\s)/g;

const text = "Он сказал,что устал.";
const missingSpaces = text.match(missingSpacePattern);
console.log(missingSpaces); // [","]
```

## Анализ сложных грамматических конструкций

### Поиск сложных предложений (с союзами)

```javascript
// Паттерн для поиска сложных предложений с союзами (и, или, но, а, что, чтобы и т.д.)
const complexSentencePattern = /\b\w+[^.!?]*(?:и|или|но|а|что|чтобы|как|когда|если|пока)[^.!?]*[.!?]/gi;

const text = "Я пошёл в магазин, и купил хлеб. Он учится, потому что хочет сдать экзамен.";
const complexSentences = text.match(complexSentencePattern);
console.log(complexSentences); // ["Я пошёл в магазин, и купил хлеб.", " Он учится, потому что хочет сдать экзамен."]
```

### Поиск конструкций "не" с глаголами

```javascript
// Паттерн для поиска отрицательных форм глаголов с "не"
const negativeVerbPattern = /\bне\s+\w*(?:ет|ит|ат|ют|ут|ят|ла|ло|ли|ть|ешь|ете|ем|им|ите|им|ет|ют|ут|ят)\b/gi;

const text = "Он не читает книгу. Она не пойдёт сегодня.";
const negativeVerbs = text.match(negativeVerbPattern);
console.log(negativeVerbs); // ["не читает", "не пойдёт"]
```

## Применение в frontend разработке

### Создание компонента проверки грамматики текста

```typescript
class GrammarChecker {
  private patterns: Map<string, RegExp> = new Map();

  constructor() {
    this.patterns.set('sentences', /[A-ZА-ЯЁ][^.!?]*[.!?]/g);
    this.patterns.set('questions', /[A-ZА-ЯЁ][^?!]*\?/g);
    this.patterns.set('exclamations', /[A-ZА-ЯЁ][^.!]*!/g);
    this.patterns.set('repeatedLetters', /\b\w*([а-яёa-z])\1{2,}\w*\b/gi);
    this.patterns.set('adjectives', /\b\w+(?:ый|ий|ой|ая|ое|ые|их|ым|ыми|его|ому|ему|ей|ую|ое|ыми|ом|ем|ен|на|но|ны|не|ни|ко|ка|ко|ке|ки|ка|ки|ке)\b/gi);
  }

  checkGrammar(text: string): Map<string, string[]> {
    const results = new Map<string, string[]>();
    
    for (const [name, pattern] of this.patterns) {
      const matches = text.match(pattern) || [];
      results.set(name, matches);
    }
    
    return results;
  }

  highlightIssues(text: string): string {
    let highlightedText = text;
    
    // Подсветка потенциальных опечаток
    const repeatedMatches = text.match(this.patterns.get('repeatedLetters')!);
    if (repeatedMatches) {
      for (const match of repeatedMatches) {
        highlightedText = highlightedText.replace(
          new RegExp(match.replace(/[.*+?^${}()|[\]\\]/g, '\\$&'), 'g'),
          `<mark class="grammar-issue">${match}</mark>`
        );
      }
    }
    
    return highlightedText;
  }
}

// Использование в компоненте
const checker = new GrammarChecker();
const text = "Привет! Как дела? Я ооочень устал.";
const results = checker.checkGrammar(text);
console.log(results);

// Для подсветки в интерфейсе
const highlighted = checker.highlightIssues(text);
document.getElementById('textDisplay')!.innerHTML = highlighted;
```

## Ограничения и важные замечания

> [!warning] Важное замечание
> Регулярные выражения имеют ограничения при анализе грамматических структур. Они подходят для простых проверок, но не могут заменить полноценный грамматический анализатор. Для сложных задач рекомендуется использовать специализированные NLP-библиотеки.

> [!tip] Совет
> Используйте регулярные выражения для предварительной проверки текста, а затем применяйте более сложные алгоритмы для точного анализа грамматических ошибок.

## Заключение

Регулярные выражения могут быть полезны для базовой проверки грамматических структур в тексте. Хотя они не могут полностью заменить специализированные инструменты для анализа языка, они могут помочь в простых задачах проверки текста на клиентской стороне.
