# Основы регулярных выражений

Регулярные выражения (regexp) - это мощный инструмент для поиска, сопоставления и манипуляции строками на основе шаблонов. В веб-разработке регулярные выражения особенно полезны для валидации форм, обработки пользовательского ввода и парсинга данных.

## Что такое регулярные выражения?

Регулярное выражение - это последовательность символов, которая задает шаблон поиска. В JavaScript регулярные выражения могут быть созданы двумя способами:

```javascript
// Литерал регулярного выражения
const regex1 = /pattern/flags;

// Конструктор RegExp
const regex2 = new RegExp('pattern', 'flags');
```

## Простые примеры

### Поиск подстроки

```javascript
const text = "Привет, мир!";
const regex = /мир/;

console.log(regex.test(text)); // true
console.log(text.match(regex)); // ['мир']
```

### Замена подстроки

```javascript
const text = "Привет, мир!";
const result = text.replace(/мир/, "JavaScript");

console.log(result); // "Привет, JavaScript!"
```

## Применение в веб-разработке

Регулярные выражения часто используются для:

- Валидации email: `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
- Проверки формата телефонного номера
- Валидации паролей по сложности
- Обработки URL-адресов
- Фильтрации пользовательского ввода

## Основные символы

- `.` - любой символ, кроме новой строки
- `^` - начало строки
- `$` - конец строки
- `*` - ноль или более повторений
- `+` - одно или более повторений
- `?` - ноль или одно повторение
- `|` - логическое ИЛИ
- `()` - группировка
- `[]` - набор символов

## Практическое применение

```javascript
// Проверка валидности email
function validateEmail(email) {
  const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return emailRegex.test(email);
}

// Проверка формата телефонного номера
function validatePhone(phone) {
  const phoneRegex = /^\+?[\d\s\-\(\)]+$/;
  return phoneRegex.test(phone);
}

// Валидация пароля (минимум 8 символов, одна заглавная, одна цифра)
function validatePassword(password) {
  const passwordRegex = /^(?=.*[A-Z])(?=.*\d).{8,}$/;
  return passwordRegex.test(password);
}
```
