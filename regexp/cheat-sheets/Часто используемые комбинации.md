# Часто используемые комбинации

Эта шпаргалка содержит популярные сочетания метасимволов регулярных выражений, которые часто применяются в фронтенд-разработке.

## Основные комбинации

### Любой символ (кроме новой строки)
```javascript
const pattern = /./;
const text = "Привет";
console.log(pattern.test(text)); // true
```

### Один или более
```javascript
const pattern = /\d+/; // одна или более цифр
const text = "Цена: 12345 рублей";
const match = text.match(pattern);
console.log(match[0]); // "12345"
```

### Ноль или более
```javascript
const pattern = /\d*/; // ноль или более цифр
const text = "Цена: рублей";
const match = text.match(pattern);
console.log(match[0]); // "" (ноль цифр)
```

### Ноль или один
```javascript
const pattern = /colou?r/; // 'u' может быть, а может и не быть
console.log(pattern.test("color")); // true
console.log(pattern.test("colour")); // true
```

### Диапазон символов
```javascript
const pattern = /[a-z]/; // строчная буква
const text = "Привет World";
const match = text.match(pattern);
console.log(match[0]); // "W"
```

### Исключающий диапазон
```javascript
const pattern = /[^0-9]/; // не цифра
const text = "123a456";
const match = text.match(pattern);
console.log(match[0]); // "a"
```

## Часто используемые комбинации

### Только цифры
```javascript
const digitsOnly = /^\d+$/;
console.log(digitsOnly.test("12345")); // true
console.log(digitsOnly.test("123a45")); // false
```

### Только буквы
```javascript
const lettersOnly = /^[a-zA-Zа-яА-ЯёЁ]+$/;
console.log(lettersOnly.test("Тест")); // true
console.log(lettersOnly.test("Тест123")); // false
```

### Буквы и цифры
```javascript
const lettersAndDigits = /^[a-zA-Zа-яА-ЯёЁ0-9]+$/;
console.log(lettersAndDigits.test("Тест123")); // true
console.log(lettersAndDigits.test("Тест123!")); // false
```

### Буквы, цифры и подчеркивание
```javascript
const alphanumericUnderscore = /^\w+$/;
console.log(alphanumericUnderscore.test("test_123")); // true
console.log(alphanumericUnderscore.test("test-123")); // false (дефис не входит в \w)
```

### Пробельные символы
```javascript
const whitespace = /\s+/;
const text = "Текст   с    пробелами";
const parts = text.split(whitespace);
console.log(parts); // ["Текст", "с", "пробелами"]
```

### Не-пробельные символы
```javascript
const nonWhitespace = /\S+/;
const text = "  Текст  ";
const match = text.match(nonWhitespace);
console.log(match[0]); // "Текст"
```

### Количество повторений
```javascript
// Ровно 3 цифры
const exactlyThreeDigits = /^\d{3}$/;
console.log(exactlyThreeDigits.test("123")); // true
console.log(exactlyThreeDigits.test("12")); // false

// От 3 до 5 цифр
const threeToFiveDigits = /^\d{3,5}$/;
console.log(threeToFiveDigits.test("123")); // true
console.log(threeToFiveDigits.test("12345")); // true
console.log(threeToFiveDigits.test("12")); // false
console.log(threeToFiveDigits.test("123456")); // false

// Не менее 3 цифр
const atLeastThreeDigits = /^\d{3,}$/;
console.log(atLeastThreeDigits.test("123")); // true
console.log(atLeastThreeDigits.test("1234567")); // true
console.log(atLeastThreeDigits.test("12")); // false
```

### Альтернация (или/или)
```javascript
const pattern = /red|blue|green/;
console.log(pattern.test("Цвет: red")); // true
console.log(pattern.test("Цвет: blue")); // true
console.log(pattern.test("Цвет: yellow")); // false
```

### Группировка
```javascript
// Найти "color" или "colour"
const pattern = /colou?r/;
console.log(pattern.test("color")); // true
console.log(pattern.test("colour")); // true

// Группировка с квантификатором
const repeatedWord = /(\w+)\s+\1/;
console.log(repeatedWord.test("word word")); // true
console.log(repeatedWord.test("word different")); // false
```

### Позиционные комбинации
```javascript
// Начало строки
const startsWithNumber = /^\d/;
console.log(startsWithNumber.test("123abc")); // true
console.log(startsWithNumber.test("abc123")); // false

// Конец строки
const endsWithNumber = /\d$/;
console.log(endsWithNumber.test("abc123")); // true
console.log(endsWithNumber.test("123abc")); // false

// Вся строка - только цифры
const onlyDigits = /^\d+$/;
console.log(onlyDigits.test("12345")); // true
console.log(onlyDigits.test("123a45")); // false
```

## Практические комбинации для фронтенда

### Извлечение параметров из URL
```javascript
const url = "https://example.com/page?param1=value1&param2=value2";
const paramPattern = /[?&]([^=]+)=([^&]*)/g;
let match;
const params = {};

while ((match = paramPattern.exec(url)) !== null) {
  params[decodeURIComponent(match[1])] = decodeURIComponent(match[2]);
}

console.log(params); // {param1: "value1", param2: "value2"}
```

### Проверка формата времени (HH:MM)
```javascript
const timePattern = /^([01]?[0-9]|2[0-3]):[0-5][0-9]$/;
console.log(timePattern.test("12:30")); // true
console.log(timePattern.test("25:00")); // false
```

### Проверка формата IPv4
```javascript
const ipPattern = /^(\d{1,3}\.){3}\d{1,3}$/;
console.log(ipPattern.test("192.168.1.1")); // true
console.log(ipPattern.test("999.999.999.999")); // true (но значения вне диапазона 0-255)
```

### Проверка формата цвета в HEX
```javascript
const hexColorPattern = /^#([A-Fa-f0-9]{6}|[A-Fa-f0-9]{3})$/;
console.log(hexColorPattern.test("#ff0000")); // true
console.log(hexColorPattern.test("#f00")); // true
console.log(hexColorPattern.test("ff0000")); // false
```
